## 1. 产品概述

自由职业者工时记录与账单生成应用，帮助用户高效管理多项目并行时的工时记录、日报生成和简易账单发送，解决手动记录耗时、汇总困难的痛点。

## 2. 核心功能

### 2.1 功能模块

1. **项目列表面板**：左侧展示所有项目，支持切换当前项目
2. **工时记录列表**：按日期分组展示当前项目的所有工时记录
3. **计时器功能**：悬浮计时按钮，支持开始/停止计时，记录任务详情
4. **日报生成**：一键生成最近7天的Markdown格式工作日报
5. **账单发送**：工时累计超过10小时后可模拟发送账单

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 项目列表面板 | 左侧300px宽面板，卡片式展示项目，点击切换 |
| 首页 | 工时记录区 | 顶部显示项目名和本周工时，下方按日期分组列表 |
| 首页 | 计时器按钮 | 右下角悬浮圆形按钮，点击开始/停止计时 |
| 首页 | 记录弹窗 | 停止计时后弹出，填写任务描述、时间微调、标签选择 |
| 首页 | 日报预览区 | 生成日报后在下方展开，支持复制和发送账单 |

## 3. 核心流程

用户进入应用 → 查看左侧项目列表 → 选择当前项目 → 查看工时记录 → 点击计时器开始工作 → 结束工作点击停止 → 填写任务详情并保存 → 点击生成日报 → 查看/复制日报 → 工时达标后发送账单

## 4. 用户界面设计

### 4.1 设计风格

- **主题色**：#6C63FF（紫色主色调）
- **辅助色**：#10B981（绿色-生成日报）、#3B82F6（蓝色-复制）、#FF4757（红色-计时中）
- **背景色**：#F1F5F9（主背景）、#F8F9FA（侧边栏）、#FFFFFF（卡片）
- **文字色**：#1E293B（主文字）、#64748B（次要文字）
- **分隔线**：#E2E8F0
- **圆角**：卡片8px、按钮6px、弹窗12px
- **字体**：无衬线字体，标题20px加粗，正文14px
- **按钮效果**：悬停背景加深10%，过渡0.2s ease-out

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 项目列表 | 卡片式布局，左侧彩色边框，悬停效果 |
| 首页 | 工时记录 | 日期分组，分隔线，标签徽章 |
| 首页 | 计时器 | 圆形悬浮按钮，脉动动画，颜色状态切换 |
| 首页 | 记录弹窗 | 居中模态框，表单输入，标签选择按钮 |
| 首页 | 日报预览 | 展开动画，等宽字体，操作按钮 |

### 4.3 响应式

- **桌面端**：左侧固定300px项目面板，右侧主内容区
- **移动端（<768px）**：左侧面板折叠为顶部下拉导航菜单
- **触控优化**：按钮最小尺寸44px，适合触控操作

### 4.4 动效设计

- 计时器脉动动画：transform缩放1.05→1，1s循环
- 日报预览展开：max-height 0→800px，0.3s ease过渡
- 按钮悬停：背景色加深，0.2s ease-out
- 列表项添加：淡入效果

# PodNotes 个人播客与音频书签笔记管理应用 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
PodNotes 是一款面向播客爱好者的音频学习笔记工具，帮助用户在收听播客过程中随时标记重要时刻、添加笔记，并通过标签和节目分类进行高效管理，同时提供学习统计功能激励持续学习。

### 1.2 目标用户
- 播客重度听众
- 学习者和知识工作者
- 需要记录音频内容要点的用户

### 1.3 核心价值
- **即时标记**：收听时一键打点，不错过任何重要内容
- **笔记关联**：笔记与时间戳自动绑定，快速回听上下文
- **高效检索**：按节目、标签多维度过滤浏览
- **学习可视化**：统计图表直观展示学习进度

|------|------|--------|
| 节目列表展示 | 首页以卡片形式展示已导入的播客节目 | P0 |

## 3. 非功能需求

### 3.1 性能要求
- 音频打点响应延迟 < 100ms
- 50条笔记渲染时间 < 200ms
- 图表动画帧率 ≥ 30fps

### 3.2 技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **图表库**: Recharts
- **样式**: 原生CSS + CSS变量

### 3.3 模块架构
- **AudioPlayer模块**: 音频加载、播放控制、打点标记
- **NoteManager模块**: 笔记CRUD、分类过滤、数据存储
- **StatisticsChart模块**: 图表渲染、数据可视化

## 4. 数据模型

### 4.1 Podcast（播客节目）
```typescript
interface Podcast {
  id: string;
  title: string;
  audioUrl: string;
  coverColor: string;
  duration: number; // 秒
  createdAt: number;
}
```

### 4.2 Note（笔记）
```typescript
interface Note {
  id: string;
  podcastId: string;
  timestamp: number; // 秒
  content: string;
  tags: string[];
  createdAt: number;
  updatedAt: number;
}
```

### 4.3 ListeningSession（收听记录）
```typescript
interface ListeningSession {
  id: string;
  podcastId: string;
  date: string; // YYYY-MM-DD
  duration: number; // 秒
}
```

## 5. 界面设计规范

### 5.1 颜色系统
| 用途 | 颜色值 |
|------|--------|
| 导航栏背景 | #1E1B4B |
| 导航选中项 | #312E81 |
| 内容区背景 | #F3F4F6 |
| 卡片背景 | #F8F9FA |
| 主色调1 | #6366F1 |
| 主色调2 | #8B5CF6 |
| 强调色（标记点） | #F59E0B |
| 边框色 | #E0E3E7 / #E5E7EB |

### 5.2 动画规范
- 过渡时长：0.2s
- 缓动函数：ease
- 应用范围：所有按钮、卡片悬浮、页面切换

### 5.3 响应式断点
- 移动端：< 768px（底部Tab导航）
- 桌面端：≥ 768px（左侧导航）

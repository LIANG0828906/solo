## 1. 产品概述

交互式钢琴谱练习与演奏评价应用，用户可通过键盘或鼠标在浏览器中演奏旋律，系统自动与标准乐谱对比并给出准确率评分。

- 主要目的：帮助钢琴学习者通过互动方式练习曲目，获得实时反馈
- 解决问题：传统钢琴学习缺乏即时、量化的演奏准确度反馈
- 目标用户：钢琴初学者、音乐爱好者

## 2. 核心功能

### 2.1 功能模块

1. **主界面**：乐谱选择、演奏控制、钢琴键盘、评价结果展示
2. **钢琴键盘组件**：88键可交互钢琴，支持鼠标和键盘输入
3. **音乐引擎模块**：音符生成、乐谱解析、播放控制
4. **录音评价模块**：用户演奏录制、与标准乐谱对比、准确率计算
5. **数据可视化模块**：节奏偏差散点图、练习准确率折线图

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主界面 | 乐谱选择区 | 下拉菜单选择内置三段乐谱（小星星、欢乐颂、卡农片段） |
| 主界面 | 演奏控制区 | 开始演奏按钮、停止按钮、练习模式段落选择 |
| 主界面 | 乐谱进度指示 | 红色竖线指示器随时间移动，显示当前演奏位置 |
| 主界面 | 钢琴键盘区 | 88键交互钢琴，按键高亮、键盘映射提示 |
| 主界面 | 评价结果面板 | 准确率百分比、彩色进度条、统计数据 |
| 主界面 | 节奏偏差图表 | 散点图显示每个音符的时间偏差 |
| 主界面 | 练习记录区 | 段落练习次数、准确率变化折线图 |

## 3. 核心流程

用户选择乐谱 → 点击开始演奏 → 系统播放标准乐谱音频并录制用户输入 → 演奏结束自动评价 → 展示准确率和节奏偏差分析 → 用户可选择段落进行循环练习

```mermaid
    D --> E["展示准确率和节奏偏差"]
    E --> F["选择练习段落"]
    F --> G["循环练习并记录每次成绩"]
    G --> H["展示准确率变化曲线"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景#1a1a2e，面板#16213e，文字#e0e0e0
- **钢琴键配色**：白键象牙白#f8f4e8，黑键深灰#2c2c2c；按下时白键浅蓝#a0c4ff，黑键深蓝#4a6fa5
- **按钮风格**：毛玻璃效果（backdrop-filter: blur(4px) + box-shadow），圆角设计
- **交互反馈**：按钮hover缩放1.05倍，点击波纹效果
- **键盘动画**：按键0.1秒亮度渐隐回弹动画

### 4.2 页面设计概览

| 模块 | UI元素 |
|-----|--------|
| 乐谱选择区 | 深色下拉选择器，毛玻璃效果 |
| 控制按钮区 | 毛玻璃按钮组：开始/停止演奏、练习模式切换 |
| 乐谱进度条 | 横向时间轴，红色竖线指示器，当前音符高亮 |
| 钢琴键盘 | 居中布局，键盘映射浮动标签，按键状态动画 |
| 评价结果面板 | 右侧浮动卡片，彩色进度条，统计数字 |
| 节奏偏差散点图 | X轴音符序号，Y轴偏差毫秒，绿/红色点 |
| 练习记录区 | 折线图，段落选择器 |

### 4.3 响应式设计

- 桌面端（≥768px）：88键完整钢琴，评价面板右侧浮动，图表完整展示
- 移动端（<768px）：61键钢琴（裁剪两侧），散点图折叠为抽屉，面板改为底部堆叠
- 触摸优化：钢琴键触摸区域增大，防止误触

### 4.4 性能要求

- 钢琴键按下反馈延迟 ≤ 50ms
- 演奏模式实时处理无卡顿
- 录音时间戳精度达到毫秒级
- 使用 Web Audio API 生成方波音色

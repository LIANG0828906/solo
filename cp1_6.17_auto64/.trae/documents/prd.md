# JSON Schema 浏览与差异对比应用 PRD

## 1. 产品概述

面向前端工程师的在线 JSON Schema 浏览与版本差异对比工具，帮助开发者快速检索 Schema 定义、直观对比字段变化，避免手动比对文档导致的遗漏。

- 核心价值：提升 Schema 评审效率，降低版本迭代中的字段遗漏风险
- 目标用户：前端工程师、后端工程师、API 设计人员

## 2. 核心功能

### 2.1 功能模块

1. **Schema 浏览模块**：树形结构展示 JSON Schema，支持展开/折叠、多文件加载与切换
2. **Diff 对比模块**：并排展示两个版本 Schema，自动标注字段增删改，支持仅显示差异模式

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主应用 | Schema 浏览面板 | 树形渲染 Schema、文件上传、文件切换、字段选中 |
| 主应用 | Diff 对比面板 | 并排双树视图、差异高亮、差异统计、仅显示差异切换、详情弹窗 |

## 3. 核心流程

### 3.1 主要用户流程

用户上传多个 JSON Schema 文件 → 在左侧浏览面板选中一个文件作为旧版本 → 选中另一个文件作为新版本 → 右侧自动计算并展示差异 → 可切换"仅显示差异"模式过滤无变化节点 → 点击差异节点查看详情。

### 3.2 数据流向

```mermaid
flowchart LR
    A["文件上传"] --> B["schemaLoader 解析"]
    B --> C["Zustand Store (schemas 列表)"]
    C --> D["左侧 SchemaTree 渲染"]

    E --> F["diff 工具计算差异"]
    F --> G["右侧 SchemaDiff 并排渲染"]
    G --> H["差异统计与高亮展示"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色主题，适合开发工具场景
- **主背景**：#1e1e2e
- **侧栏背景**：#181825
- **卡片背景**：#2a2a40
- **强调色**：
  - object 类型：蓝色 #3b82f6
  - array 类型：紫色 #8b5cf6
  - string 类型：绿色 #10b981
  - number 类型：橙色 #f59e0b
  - boolean 类型：红色 #ef4444
- **字体**：14px 行高 32px，等宽编程字体
- **动效**：展开折叠 0.3s ease，三角旋转 0.2s ease

### 4.2 页面设计概述

| 页面区域 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 左侧浏览面板 | SchemaTree | 文件选择器、文件列表、递归树节点、类型标签、展开折叠三角、选中指示条 |
| 右侧 Diff 面板 | SchemaDiff | 差异统计卡片、仅显示差异开关、左右并排双树、差异连线、高亮背景、详情弹窗 |

### 4.3 响应式设计

- 桌面端（≥768px）：左右分栏布局，左侧 320px 固定宽度，右侧自适应
- 移动端（<768px）：上下布局，浏览面板在上，Diff 面板在下，支持拖拽调节高度
- 拖拽调节：使用 resize handle 实现面板大小调整

### 4.4 视觉细节

- 树节点连线：SVG 绘制，颜色 #cbd5e1，线宽 1.5px
- 节点悬浮：背景变亮 #3a3a55
- 选中状态：左侧 4px 蓝色指示条 #3b82f6
- 新增字段：绿色背景 #d1fae5
- 删除字段：红色删除线 + 背景 #fecaca
- 修改字段：黄色背景 #fef9c3
- 差异统计卡片：圆角 8px，背景 #2a2a40，内边距 12px

## 5. 性能要求

- 2000+ 字段 Schema 渲染 ≤ 500ms
- 500+ 字段 Schema Diff 计算 ≤ 200ms
- 展开折叠动画流畅不卡顿

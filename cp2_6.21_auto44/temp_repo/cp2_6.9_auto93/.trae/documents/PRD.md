## 1. 产品概述

宋代香铺虚拟调香与线上售卖全栈应用，用户以汴京调香师身份体验香料选择、合香调配、试香品鉴及库存订单管理的完整流程。

- 核心价值：传承中华香文化，提供沉浸式宋代调香体验
- 目标用户：香文化爱好者、传统文化体验者
- 市场定位：文化体验类Web应用，融合教育与娱乐

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 调香师 | 无需注册，直接使用 | 香料选择、合香调配、试香品鉴、库存管理、订单处理 |

### 2.2 功能模块

1. **香铺主场景**：宋代风格香铺可视化，香料架、乳钵、试香台
2. **调香面板**：香料拖拽、配比调节、合香生成、研磨动画
3. **试香品鉴**：烟雾动画、三维度评分、合香命名入库
4. **库存与订单**：香品库存管理、订单列表、发货操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 香铺主场景 | 香料架 | 三层木质货架，18只瓷瓶，悬停旋转放大，点击显示香料信息，支持拖拽 |
| 香铺主场景 | 乳钵 | 铜制乳钵，接收拖拽香料，显示混合粉末，支持研磨动画 |
| 香铺主场景 | 试香台 | 三足铜香炉，接收合香球，点燃产生烟雾动画 |
| 调香面板 | 配方调节 | 滑块调节香料比例（0-10份），总份数上限20 |
| 调香面板 | 研磨功能 | 点击研磨，5秒螺旋粒子动画，生成合香球 |
| 试香品鉴 | 评分系统 | 醇厚度、持久度、扩散度三维度百分比评分 |
| 库存管理 | 库存列表 | 显示香品名称、配方、评分、库存量 |
| 订单管理 | 订单处理 | 订单列表、发货操作、库存校验与扣减 |

## 3. 核心流程

用户进入香铺 → 点击/拖拽香料至乳钵 → 调节配方比例 → 研磨生成合香球 → 拖拽至香炉试香 → 查看品鉴评分 → 命名存入库存 → 处理客户订单 → 扣减库存

```mermaid
flowchart LR
    A["进入香铺"] --> B["选择香料"]
    B --> C["拖拽至乳钵"]
    C --> D["调节配比"]
    D --> E["研磨合香"]
    E --> F["生成合香球"]
    F --> G["拖拽至香炉"]
    G --> H["点燃试香"]
    H --> I["品鉴评分"]
    I --> J["命名入库"]
    J --> K["处理订单"]
    K --> L["扣减库存"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：木色#8b6f47、青瓷色#5f9ea0、暗红色#8b0000
- **辅助色**：青砖#7a8a7a、青铜#b87333、红木#8b4513
- **按钮风格**：圆角木质按钮，悬停上浮，点击涟漪效果
- **字体**：标题使用宋体/楷体，正文使用雅致宋体，营造古典氛围
- **布局风格**：宋代庭院式对称布局，留白适度，虚实结合
- **背景**：宣纸纹理背景，古朴典雅

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 香铺主场景 | 香料架 | 三层木质结构，18只渐变瓷瓶，悬停Y轴旋转15度放大1.1倍 |
| 香铺主场景 | 乳钵 | 青铜色圆钵，径向渐变内侧，接收拖拽高亮 |
| 香铺主场景 | 试香台 | 三足暗红圆鼎，顶盖镂空，烟雾粒子效果 |
| 调香面板 | 配方区 | 滑块+数字输入，实时计算总分，进度条显示 |
| 调香面板 | 研磨按钮 | 木质按钮，涟漪效果，螺旋粒子动画 |
| 试香品鉴 | 评分面板 | 三维度环形进度条，百分比显示 |
| 库存订单 | 标签页 | 库存/订单切换，抽屉式订单面板（Pad端） |

### 4.3 响应式设计

- **桌面端（>=1024px）**：三栏布局，香料架左、操作区中、库存订单右
- **Pad端（768-1023px）**：两列香料架，订单面板折叠为抽屉式
- **触控优化**：增大点击区域，优化拖拽手势

### 4.4 动画与交互

- 瓷瓶悬停：Y轴旋转15度，放大1.1倍，0.3秒过渡
- 合香球滑落：framer-motion spring弹性缓动
- 烟雾粒子：CSS动画，20+粒子，大小透明度随机，螺旋上升
- 按钮点击：涟漪扩散效果
- 研磨动画：requestAnimationFrame螺旋粒子搅拌
- 页面加载： staggered reveal 渐入效果

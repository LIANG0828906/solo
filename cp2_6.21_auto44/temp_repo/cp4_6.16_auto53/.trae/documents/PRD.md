## 1. 产品概述

BeatGrid是一款面向复古游戏爱好者社群的浏览器端像素风格节奏游戏。玩家需要跟随音乐节拍按压对应方向的音符，获得连击评分，体验怀旧街机风格的音乐游戏乐趣。

- 核心目标：提供一款轻量级、可直接在浏览器运行的复古像素风节奏游戏
- 目标用户：复古游戏爱好者、音游玩家、休闲游戏用户
- 市场价值：填补浏览器端像素风音游空白，满足怀旧玩家需求

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 直接进入游戏，体验完整功能 |

### 2.2 功能模块

1. **游戏主界面**：4条竖轨音符下落、判定线、分数显示、生命值显示、连击显示
2. **击中判定系统**：Perfect/Good/Miss三级判定、分数计算、生命值管理
3. **粒子特效系统**：击中爆炸粒子、心碎动画、连击特效
4. **音频系统**：Web Audio API生成节拍音效、击中音效
5. **结果界面**：最终分数、最高连击、评级显示、重玩按钮
6. **暂停系统**：Esc暂停、空格继续

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 游戏主界面 | 轨道渲染 | 4条彩色竖轨，扫描线纹理间隔，音符从顶部下落 |
| 游戏主界面 | 判定系统 | 半透明判定线区域，检测按键时机与音符位置偏差 |
| 游戏主界面 | HUD显示 | 左上角心形生命值、分数、右上角连击数特效 |
| 游戏主界面 | 粒子特效 | Perfect击中时彩色粒子爆炸，Good时消失，Miss时判定线变红 |
| 结果界面 | 评级展示 | 逐帧放大像素字体显示S/A/B/C/D评级，黑白闪烁格子背景 |
| 结果界面 | 重玩功能 | 点击按钮重置游戏状态，重新开始 |

## 3. 核心流程

玩家进入游戏后，音符根据预定义节奏序列从顶部下落。玩家需要在音符到达判定线时按下对应方向键（上/下/左/右）。系统根据按键时机判定Perfect/Good/Miss，累计分数和连击数。Miss时扣除生命值，生命值归零则游戏结束，进入结果界面显示最终评级。游戏过程中可按Esc暂停，空格继续。

```mermaid
flowchart TD
    A["开始游戏"] --> B["音符生成与下落"]
    B --> C["按键判定"]
    C --> D{"判定结果"}
    D -->|Perfect| E["+300分，连击+1，播放完美音效"]
    D -->|Good| F["+100分，连击+1，播放良好音效"]
    D -->|Miss| G["0分，连击清零，生命值-1"]
    G --> H{"生命值 > 0"}
    E --> I["继续游戏"]
    F --> I
    H -->|是| I
    H -->|否| J["游戏结束"]
    I --> K{"是否暂停"}
    K -->|是(Esc)| L["暂停遮罩，等待空格"]
    L --> I
    K -->|否| B
    J --> M["结果界面：分数、连击、评级"]
    M --> N{"点击重玩"}
    N -->|是| A
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景 #1a1a2e，霓虹像素风格
- **轨道颜色**：上轨#ff6b6b（红）、下轨#51cf66（绿）、左轨#339af0（蓝）、右轨#fcc419（黄）
- **像素字体**：使用Press Start 2P或类似8位像素风格字体
- **按钮样式**：像素化边框，悬停时有霓虹发光效果
- **整体风格**：复古街机像素风，配合扫描线、CRT效果

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 游戏主界面 | 轨道区域 | 4条彩色竖轨，1px边框，扫描线纹理，音符16x16像素方块带残影 |
| 游戏主界面 | 判定线 | 半透明白色闪烁虚线，Miss时变红闪烁 |
| 游戏主界面 | HUD | 左上角5个心形像素图标（心碎动画）、分数、右上角连击闪烁特效 |
| 游戏主界面 | 粒子特效 | Perfect时8个彩色像素方块向外扩散消失 |
| 暂停界面 | 暂停遮罩 | 半透明黑色遮罩，中央像素字体"按下空格继续" |
| 结果界面 | 评级展示 | 逐帧放大像素字体S/A/B/C/D，黑白闪烁格子背景 |
| 结果界面 | 数据展示 | 总分数、最高连击数、重玩按钮 |

### 4.3 响应性

- 桌面优先设计，移动端自动适配屏幕宽度
- 建议在桌面端使用键盘操作获得最佳体验
- 触控设备可考虑虚拟方向键（本次版本不实现）

### 4.4 动画与特效

- 音符下落：匀速运动，尾部拖出2个同色残影（透明度递减）
- 击中特效：Perfect时彩色粒子爆炸，Good时淡出消失
- 连击特效：连击>10时右上角像素文字逐帧变色闪烁
- 心碎动画：心形图标裂开然后消失
- 判定线闪烁：正常时白色虚线闪烁，Miss时红色闪烁
- 评级动画：逐帧放大的像素字体
- 背景效果：扫描线纹理、微弱CRT弯曲效果

## 1. 产品概述

开源项目Issue智能管理工具，旨在帮助维护者高效处理大量Issue，解决重复Issue识别困难、关键问题易被淹没、手动标签效率低三大核心痛点。目标用户为开源项目维护者及社区管理者，通过文本相似度算法和N-gram标签匹配实现Issue智能筛选与分类。

## 2. 核心功能

### 2.1 功能模块

1. **Issue列表页**：批量Issue浏览与筛选、批量操作、骨架屏加载
2. **Issue详情页**：完整Issue信息展示、相似Issue检测与标记、智能标签建议、评论与操作

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| Issue列表页 | 筛选栏 | 关键词搜索（聚焦边框#6366F1）、状态筛选下拉、标签筛选下拉 |
| Issue列表页 | Issue卡片列表 | 卡片展示标题、摘要、标签徽章、状态、相对时间；左侧彩色边框按标签着色；悬停阴影加深；复选框多选 |
| Issue列表页 | 批量操作栏 | 选中2+后顶部固定栏，支持批量标记待处理/批量添加标签/批量关闭，绿色Toast提示 |
| Issue列表页 | 骨架屏 | 首次加载6个灰色圆角卡片渐变动画1.5秒循环 |
| Issue详情页 | 详情展示 | 完整描述、评论列表、操作按钮 |
| Issue详情页 | 相似Issue面板 | 自动Jaccard相似度计算，超过0.6阈值显示重复面板，红色/橙色相似度标识，"标记为重复"按钮 |
| Issue详情页 | 智能标签建议 | N-gram特征匹配预定义标签，气泡卡片展示建议标签及确信度百分比条，采纳/拒绝操作 |

## 3. 核心流程

**主流程**：用户打开应用 → 骨架屏加载 → 展示Issue列表 → 通过筛选栏过滤 → 点击卡片进入详情 → 查看相似Issue/获取标签建议 → 返回列表进行批量操作

**相似检测流程**：打开Issue详情 → 自动调用Jaccard相似度算法与所有Issue比对 → 超过阈值则显示重复面板 → 用户可标记为重复

**智能标签流程**：点击"智能建议"按钮 → N-gram特征计算 → 弹出建议气泡 → 用户采纳或拒绝 → 采纳后标签添加到Issue

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：柔和灰蓝背景(#F8FAFC)，靛蓝按钮主色(#6366F1)
- **文字**：主要深灰(#1E293B)，辅助中灰(#64748B)
- **按钮样式**：圆角8px，悬停#4F46E5，点击#4338CA，禁用#A5B4FC
- **卡片样式**：圆角12px，阴影0 2px 8px rgba(0,0,0,0.06)，左侧4px彩色边框
- **标签颜色**：Bug红色#EF4444、增强蓝色#3B82F6、文档紫色#8B5CF6、优化绿色#10B981、其他灰色#64748B
- **布局**：最大宽度1200px居中，卡片间距12px，内边距16px
- **字体**：系统字体栈，标题16px bold，描述14px，标签12px
- **动画**：统一0.2s ease过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| Issue列表页 | 筛选栏 | 背景#F8FAFC，圆角12px，内边距16px，搜索框+状态下拉+标签下拉 |
| Issue列表页 | Issue卡片 | 白色背景，圆角12px，左彩色边框4px，阴影0 2px 8px，悬停阴影加深0.4-16过渡0.2s |
| Issue列表页 | 批量操作栏 | 固定top0，白色背景，阴影0 2px 10px，z-index50 |
| Issue列表页 | 骨架屏 | 6个灰色圆角卡片，渐变动画1.5s循环 |
| Issue详情页 | 重复面板 | 背景#FEF2F2，圆角8px，边框#FECACA，内边距16px |
| Issue详情页 | 标签建议气泡 | 白色背景，圆角12px，阴影0 4px 20px rgba(0,0,0,0.15) |

### 4.3 响应式适配

- 桌面优先设计，最大宽度1200px居中
- <768px：列表单列全宽，搜索和筛选栏垂直堆叠
- >1200px：保持居中布局
- 所有过渡动画0.2s ease

### 4.4 性能要求

- 100张卡片渲染保持60FPS
- 300条Issue内相似度计算<200ms
- 标签匹配建议响应<100ms
- localStorage数据持久化

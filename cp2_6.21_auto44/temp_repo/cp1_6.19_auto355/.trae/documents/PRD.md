## 1. 产品概述

微型在线问卷调查与可视化分析平台，支持用户创建、发布问卷并收集回复，自动生成数据图表分析报告。面向需要快速收集意见、进行市场调研或满意度调查的个人和小型团队。

- 核心价值：零配置、即用即走的问卷工具，提供专业级数据可视化分析
- 目标用户：市场调研人员、产品经理、教育工作者、活动组织者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 问卷创建者 | 无需注册，本地会话 | 创建/编辑/发布问卷，查看数据分析看板 |
| 问卷填写者 | 无需注册，访问URL | 浏览并填写已发布问卷，提交回复 |

### 2.2 功能模块
1. **问卷编辑器**：题目管理（增删改）、拖拽排序、问卷发布
2. **问卷填写页**：题目展示、输入控件、进度条、提交与感谢页
3. **数据看板**：题目统计卡片、时间序列趋势图、回复详情表格

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 问卷列表 | 展示已有问卷列表，创建新问卷入口 |
| 问卷编辑页 | 题目编辑区 | 添加/编辑/删除题目，支持单选/多选/评分三种类型 |
| 问卷编辑页 | 拖拽排序 | 按住手柄拖拽调整题目顺序，带半透明跟随和弹性动画 |
| 问卷编辑页 | 发布按钮 | 生成唯一URL，锁定问卷不可编辑 |
| 问卷填写页 | 题目展示 | 渲染问卷标题、说明和所有题目 |
| 问卷填写页 | 输入控件 | radio单选、checkbox多选、星级评分点击 |
| 问卷填写页 | 进度条 | 底部显示已答/总题目比例 |
| 问卷填写页 | 感谢页面 | 提交后显示确认信息 |
| 数据看板 | 概览区 | 问卷标题、总回复数、平均完成时间 |
| 数据看板 | 题目统计卡片 | 单选饼图、多选柱状图、评分星级分布 |
| 数据看板 | 时间序列图 | 每日回复数量折线图，支持拖拽选择时间段放大 |
| 数据看板 | 回复详情表 | 每行显示一名用户全部答案，支持按选项筛选 |

## 3. 核心流程

```mermaid
flowchart LR
    A["创建者进入首页"] --> B["创建新问卷"]
    B --> C["添加/编辑题目"]
    C --> D["拖拽调整顺序"]
    D --> E["发布问卷生成URL"]
    E --> F["分享URL给填写者"]
    F --> G["填写者访问问卷"]
    G --> H["填写题目"]
    H --> I["提交回复"]
    I --> J["显示感谢页"]
    J --> K["数据实时同步"]
    K --> L["创建者查看数据看板"]
    L --> M["图表分析与筛选"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：#1976D2（蓝色），辅助色：#FFB300（琥珀色）
- 背景色：#F5F5F5，卡片背景：白色
- 圆角：8px，阴影：box-shadow: 0 2px 8px rgba(0,0,0,0.1)
- 按钮：主色填充，hover时加深10%
- 字体：系统默认无衬线字体，标题字号18-24px，正文14px

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 问卷编辑页 | 题目卡片 | 拖拽手柄、题目类型标签、编辑/删除按钮、展开/折叠 |
| 问卷编辑页 | 拖拽效果 | 半透明题目卡片跟随鼠标、放置位置高亮指示、0.2s弹性动画 |
| 数据看板 | 图表加载 | 从下方淡入上滑，stagger 0.1s 交错动画 |
| 数据看板 | Tooltip | 毛玻璃效果 backdrop-filter: blur(6px) |

### 4.3 响应式
- 桌面端优先设计，移动端 < 768px 自适应
- 看板图表由两列布局切换为单列
- 回复详情表格变为横向滚动
- 触摸设备优化拖拽和点击区域

## 5. 性能指标
- 问卷编辑器第10个题目时，拖拽排序响应 < 50ms
- 1000条回复、10道题目时，看板首次加载 < 3秒
- 图表切换筛选响应 < 200ms

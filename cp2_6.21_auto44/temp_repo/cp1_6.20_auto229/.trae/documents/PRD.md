## 1. 产品概述

Brew Journal 是一款面向咖啡爱好者的手冲咖啡记录与风味图谱应用，帮助用户系统化记录冲煮参数、量化风味感受，并通过社区分享发现更多冲煮方案。

- 核心目标：解决咖啡爱好者冲煮变量记录零散、风味感受难以量化对比的痛点
- 目标用户：家庭咖啡爱好者、精品咖啡从业者、咖啡冲煮研究者
- 产品价值：建立个性化的咖啡风味数据库，通过数据驱动优化冲煮技艺

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 模拟注册（本地存储） | 管理咖啡豆档案、记录冲煮参数、生成风味图谱、浏览社区、抄作业 |

### 2.2 功能模块

1. **个人主页**：咖啡豆档案卡片展示、冲煮记录时间线、快捷入口
2. **冲煮记录页**：参数录入表单、流速图实时预览、风味雷达图打分
3. **社区页**：横向滚动卡片流、点赞排序、抄作业功能
4. **风味时间线**：同一豆子的多次冲煮记录时间轴展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 个人主页 | 咖啡豆档案区 | 毛玻璃卡片展示，悬停上浮并显示产地经纬度 |
| 个人主页 | 风味探索时间线 | 右侧时间轴，圆点颜色随综合评分渐变，点击展开详情 |
| 冲煮记录页 | 参数录入表单 | 水温滑块、研磨度旋钮、粉水比选择、注水时间输入 |
| 冲煮记录页 | 流速图 | Canvas绘制滴滤V60流速曲线，随参数实时变化 |
| 冲煮记录页 | 风味雷达图 | 六维可拖拽雷达图，弹性回拉动画 |
| 社区页 | 横向滚动卡片 | 每屏3张，含豆子名、参数、雷达图缩略图、抄作业按钮 |

## 3. 核心流程

用户添加咖啡豆档案 → 进入冲煮记录页面 → 调整冲煮参数（实时预览流速图） → 完成冲煮后进行风味打分 → 保存记录 → 时间线自动更新 → 社区分享/浏览他人记录 → 抄作业一键填入参数

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，主背景 #1a1a2e，卡片背景 #16213e，强调色 #e94560
- **文字颜色**：主色 #eee，辅助色 #aaa
- **卡片风格**：圆角 12px，毛玻璃效果，悬停上浮 4px
- **按钮风格**：圆角 8px，强调色填充，悬停缩放 1.02 倍
- **字体**：现代无衬线字体，清晰的层次结构
- **动效**：统一 0.3s ease 过渡，悬停 0.2s ease，弹性动画 cubic-bezier

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 个人主页 | 咖啡豆档案 | 毛玻璃卡片网格、产地经纬度悬停显示、添加按钮 |
| 个人主页 | 风味时间线 | 垂直时间轴、评分渐变色圆点、滑入式详情卡片 |
| 冲煮记录页 | 参数表单 | 滑块控件、旋钮控件、数字输入、Canvas 流速图 |
| 冲煮记录页 | 雷达图 | Canvas 六维图、拖拽控制点、弹性动画 |
| 社区页 | 卡片流 | 横向滚动、缩略雷达图、抄作业按钮、点赞数 |

### 4.3 响应式设计

- 桌面端（≥768px）：左右两栏布局，左栏 65% 个人记录区，右栏 35% 档案和社区概览
- 移动端（<768px）：上下堆叠布局，顶部导航收缩为汉堡菜单
- 触控优化：增大点击区域，滑动手势支持

### 4.4 性能要求

- 雷达图和流速图帧率 ≥ 50fps
- 页面首次加载时间 ≤ 2.5 秒（预置 20 条模拟数据）
- 动画流畅无卡顿

## 1. 产品概述

代码量子态是一个交互式代码可视化Web应用，将用户输入的代码片段转换为量子态概率云图，通过视觉化方式展示代码结构与复杂度。目标用户为开发者、编程学习者和代码审计人员，帮助他们直观理解代码结构、识别复杂区域和潜在问题。

- 核心功能：代码解析、量子态模拟、交互式概率云可视化
- 产品价值：将抽象的代码结构转化为直观的可视化图形，提升代码理解效率

## 2. 核心功能

### 2.1 功能模块

1. **代码编辑器模块**：支持JavaScript/TypeScript代码输入，语法高亮，实时同步
2. **代码解析模块**：将代码解析为AST，提取变量、函数、控制流等结构
3. **量子态模拟模块**：根据代码复杂度生成量子态概率分布数据
4. **概率云可视化模块**：使用D3.js绘制交互式力导向图，支持拖拽、缩放、节点交互

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 标题栏 | 显示应用名称"代码量子态"，深色背景 |
| 主页面 | 代码编辑器 | 左侧40%区域，支持代码输入、语法高亮、500字符限制 |
| 主页面 | 概率云图 | 右侧60%区域，D3力导向图展示量子态概率分布 |
| 主页面 | 分隔条 | 可拖拽调整左右区域宽度比例 |
| 主页面 | 重置按钮 | 右下角固定按钮，恢复云图初始视图 |

## 3. 核心流程

用户在左侧代码编辑器输入JavaScript/TypeScript代码 → 代码实时解析为AST结构 → 量子态模拟模块根据代码复杂度计算概率分布 → 概率云图组件使用D3力导向布局渲染节点和连接 → 用户可与节点交互（点击查看详情、拖拽平移、滚轮缩放）

```mermaid
flowchart LR
    A["用户输入代码"] --> B["实时解析为AST"]
    B --> C["量子态模拟生成概率数据"]
    C --> D["D3力导向图渲染"]
    D --> E["节点交互（点击/拖拽/缩放）"]
    E --> F["显示节点详情弹窗"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰#252526背景，蓝色#00BFFF到紫色#8A2BE2渐变作为数据颜色
- **文字颜色**：白色#FFFFFF
- **字体**：代码使用Fira Code等宽字体，界面使用系统无衬线字体
- **布局风格**：左右分栏布局，顶部标题栏，整体简洁科技感
- **动画效果**：节点脉冲动画、边框过渡、平滑拖拽

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|---------|---------|-------------|
| 主页面 | 标题栏 | 高度50px，背景#1E1E1E，白色16px加粗字体居中 |
| 主页面 | 代码编辑器 | 背景#1E1E1E，1px半透明白色边框，聚焦时蓝色边框过渡0.2s |
| 主页面 | 语法高亮 | 关键字蓝色#569CD6，字符串橙色#CE9178，注释绿色#6A9955 |
| 主页面 | 概率云图 | SVG力导向图，节点10-40px半径，蓝紫渐变，半透明连接线 |
| 主页面 | 节点交互 | 点击放大1.5倍，显示详情弹窗，关联节点脉冲动画 |
| 主页面 | 分隔条 | 4px宽#555555，悬停#888888，col-resize光标 |
| 主页面 | 重置按钮 | 30px圆形，背景#333333，悬停#555555，白色R字母 |

### 4.3 响应性

- 桌面端优先设计，左右分栏布局
- 最小宽度：云图区域400px，编辑器区域自适应
- 分隔条拖拽支持调整比例
- 云图支持缩放范围0.5-3倍

### 4.4 性能要求

- 编辑器输入响应延迟低于50ms
- 节点数≤30时，拖拽缩放帧率≥30fps

## 1. 产品概述

家庭植物医生诊断日志是一款面向家庭园艺爱好者的植物健康管理工具，帮助用户通过拍照记录和症状描述来诊断植物病害，并生成个性化的植物健康档案，追踪每盆植物的健康变化。

- 目标用户：养花种菜的家庭用户、园艺爱好者
- 核心价值：降低植物养护门槛，让普通用户也能科学地识别和应对植物病害

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 添加植物、记录症状、查看诊断、管理健康档案 |

### 2.2 功能模块

1. **植物列表页**：植物健康卡片展示、搜索筛选、滚动加载
2. **植物详情页**：照片轮播、基本信息卡片、诊断时间线
3. **症状记录模块**：症状选择表单、护理情况输入、诊断匹配
4. **诊断结果面板**：病因概率排序、严重程度渐变条、护理建议

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 植物列表页 | 搜索栏 | 按植物名称搜索，按摆放位置筛选（阳台/客厅/厨房等） |
| 植物列表页 | 植物健康卡片 | 渐变色背景、植物图标、缩略图、名称、上次诊断日期徽章，悬停浮现操作按钮 |
| 植物列表页 | 添加植物弹窗 | 上传1-3张照片（裁剪/旋转/进度条）、填写名称/购买日期/摆放位置 |
| 植物详情页 | 照片轮播 | 箭头切换、圆点指示器、支持更新照片 |
| 植物详情页 | 基本信息卡片 | 植物名称、摆放位置、已养护天数 |
| 植物详情页 | 诊断时间线 | 时间倒序、颜色圆点（绿/橙/红）、点击展开诊断报告、滚动加载更多 |
| 症状记录页 | 症状选择表单 | 症状类型选择（叶片发黄/枯萎/虫害/霉斑/生长缓慢/烂根），每个配emoji图标 |
| 症状记录页 | 护理情况输入 | 出现日期、浇水/施肥/光照（开关+滑块组合） |
| 诊断结果面板 | 病因匹配列表 | Top3病因、概率百分比、典型症状描述、护理措施、严重程度渐变色条 |

## 3. 核心流程

1. 用户添加植物：上传照片 → 填写信息 → 生成健康卡片
2. 记录症状：选择症状类型 → 填写护理情况 → 提交诊断
3. 查看诊断：查看匹配病因 → 确认归档 → 时间线更新
4. 追踪历史：浏览时间线 → 展开详细报告 → 查看变化趋势

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：橄榄绿（#6B8E23）搭配奶白色（#FFFDF5）和深棕色（#5C4033）
- 按钮风格：圆角设计，悬停有波纹反馈动画
- 字体：Nunito（标题/数字）+ Noto Sans SC（中文正文）
- 布局风格：左右分栏布局，左侧植物列表，右侧详情档案
- 图标/emoji风格：园艺主题线条图标 + 症状类型emoji

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 植物列表页 | 搜索栏 | 圆角输入框，浅绿背景，搜索图标 |
| 植物列表页 | 植物健康卡片 | 圆角卡片，渐变色背景（绿→浅绿），左上角诊断日期徽章，悬停浮现3个操作按钮（平滑过渡） |
| 植物列表页 | 添加植物弹窗 | 居中弹窗，照片上传区（虚线边框），进度条加载动画，表单输入 |
| 植物详情页 | 照片轮播 | 大尺寸轮播图，左右箭头，底部圆点指示器 |
| 植物详情页 | 基本信息卡片 | 圆角白色卡片，信息标签式排列，养护天数高亮 |
| 植物详情页 | 诊断时间线 | 垂直时间线，颜色圆点，点击展开（从两侧展开动画），滚动加载 |
| 症状记录页 | 症状选择 | 网格布局，emoji图标+文字标签，选中高亮 |
| 症状记录页 | 护理输入 | 开关控件+滑块组合，日期选择器 |
| 诊断结果面板 | 病因列表 | 概率百分比，从警告黄到危险红的渐变色条，护理措施列表 |

### 4.3 响应式设计

- 桌面端：左右分栏布局，时间线单列垂直排列
- 平板端：时间线改为双列布局，两侧分布节点
- 手机端：卡片全宽，时间线单列，弹窗全屏适配

### 4.4 动画与交互

- 所有按钮和链接：点击波纹反馈动画
- 卡片悬停：底部操作按钮平滑浮现
- 时间线节点：点击展开诊断报告（从两侧展开动画）
- 弹窗：平滑弹出过渡
- 照片上传：进度条加载动画，缩略图渐进式加载
- 页面加载：整体帧率维持60FPS以上

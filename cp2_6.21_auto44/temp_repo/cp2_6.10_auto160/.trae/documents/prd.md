## 1. 产品概述

金陵书坊·芸窗雅集是一款沉浸式明代江南书坊互动体验应用，让用户化身游逛金陵书坊的文人，在仿古书架间浏览、翻阅线装古籍，交互式品味古籍版式、插图和批注。

- 核心价值：通过数字化还原明代江南书坊文化，提供沉浸式古籍阅读和批注体验
- 目标用户：传统文化爱好者、古籍研究者、文学艺术爱好者
- 产品定位：寓教于乐的文化体验应用

## 2. 核心 Features

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览书架、阅读古籍、添加和编辑个人批注 |

### 2.2 Feature 模块

1. **主界面**：仿明式书房场景，左侧书架区 + 右侧阅读区
2. **书架模块**：按经史子集四部分类的古籍展示，支持虚拟滚动
3. **阅读器模块**：线装书双页布局，古籍版式 + 现代译文/批注对照
4. **批注系统**：文本高亮 + 朱笔批点，本地存储持久化
5. **动画交互**：翻页动画、悬停效果、页面过渡、惯性滚动

### 2.3 页面详情

| 页面名称 | 模块名称 | Feature 描述 |
|----------|----------|-------------|
| 主页面 | 书架导航区 | 经史子集四部分类标签，滚动浏览古籍书脊 |
| 主页面 | 书籍卡片 | 悬停放大显示书名作者卡片，点击进入阅读 |
| 主页面 | 阅读面板 | 线装书双页展开，左侧古籍原文，右侧现代译文 |
| 主页面 | 批注编辑 | 选中文本高亮，弹出批注框，支持朱笔批点 |
| 主页面 | 翻页控制 | 上下页按钮，书页卷曲翻转动画 |

## 3. 核心流程

用户打开应用 → 浏览经史子集分类书架 → 鼠标悬停查看书籍详情 → 点击书籍进入阅读模式 → 翻阅书页浏览古籍内容 → 选中文本添加批注 → 批注自动保存到本地存储 → 返回书架继续浏览其他书籍

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：古纸色 #f5e6c8（背景）、墨色 #2a1a0a（文字）
- **辅助色**：深褐色 #5d4037（书架木纹）、朱红色 #c41e3a（批点印章）
- **背景**：仿宣纸纹理，细腻的纤维质感
- **按钮**：雕版印刷墨线边框效果，按压时的凹陷反馈
- **字体**：标题使用宋体风格字体，正文使用楷体风格，体现古籍韵味
- **布局**：桌面端左右两栏，移动端自适应上下结构
- **动效**：书页卷曲翻转动画、墨点晕染过渡、微光闪烁反馈

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主页面 | 书架区域 | 深褐色木纹书架，经史子集分类标签，书脊书名 |
| 主页面 | 悬停卡片 | 古纸色卡片，墨线边框，书名作者信息 |
| 主页面 | 阅读面板 | 双页线装书，栏线、鱼尾、版心版式元素 |
| 主页面 | 批注系统 | 朱红色高亮，弹出式批注框，墨笔书写效果 |
| 主页面 | 动效系统 | 翻页卷曲动画，淡入缩放过渡，惯性滚动 |

### 4.3 响应式设计

- 桌面端（≥1024px）：左右两栏布局，书架占40%宽度，阅读区占60%
- 平板端（768px-1024px）：书架占35%，阅读区占65%
- 移动端（<768px）：上下布局，书架在上可折叠，阅读区在下全屏显示
- 触摸优化：增大点击区域，支持滑动翻页和双指缩放

### 4.4 视觉细节

- **宣纸纹理**：使用SVG或CSS渐变模拟宣纸的纤维质感和细微斑点
- **木纹效果**：多层线性渐变模拟书架的深褐色木纹
- **墨线边框**：1.5px实线边框，模拟雕版印刷的墨线效果
- **版心元素**：每页古籍包含上鱼尾、下鱼尾、栏线、版心文字
- **朱批效果**：高亮文本使用半透明朱红色，批注框带印章风格边角

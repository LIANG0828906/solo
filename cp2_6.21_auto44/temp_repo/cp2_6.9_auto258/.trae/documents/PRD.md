# 云锦织机3D交互模拟器 PRD

## 1. 产品概述

### 1.1 项目背景
云锦是中国传统的高档丝织品，被誉为"寸锦寸金"，但其复杂的织造工艺缺乏直观的交互演示手段。本项目旨在通过3D交互技术，让用户能够在浏览器中模拟古代织造署的云锦织机，实时预览织锦图案，解决传统工艺传承中视觉化不足的问题。

### 1.2 产品定位
一款面向文化传承、教育展示、工艺爱好者的Web端3D交互模拟器，让用户能够以第一人称视角体验云锦织造过程，直观理解云锦图案生成原理。

### 1.3 目标用户
- 云锦工艺爱好者
- 服装设计与纺织专业师生
- 博物馆与文化机构访客
- 对中国传统工艺感兴趣的大众

## 3. 界面设计需求

### 3.1 视觉风格
- **整体风格**：明代家具的沉稳与云锦的华贵对比
- **主色调**：#3e2723（深褐色）
- **辅助色**：#ffe0b2（暖米色）
- **背景色**：#1a0f0a（深棕色）

### 3.2 交互元素
| 需求编号 | 需求描述 | 优先级 |
|---------|---------|--------|
| UI-01 | 织机周围散布半透明黄铜色光晕（强度0.3，半径0.5单位） | P0 |
| UI-02 | 按钮和交互元素有微光效提示（周期2s，透明度0.3-0.6交替） | P0 |
| UI-03 | 拖拽操作有磁性吸附反馈（吸附距离0.3单位） | P0 |
| UI-04 | 吸附时触发粒子爆发（15个粒子，丝线色，持续0.5s） | P0 |
| UI-05 | 响应式适配1280x720到1920x1080分辨率 | P0 |
| UI-06 | 使用Google Fonts的Ma Shan Zheng字体 | P0 |

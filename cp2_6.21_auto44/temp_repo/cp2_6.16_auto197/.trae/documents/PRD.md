## 1. 产品概述

SpriteForge 是一款面向独立游戏开发者的像素动画帧编辑器，帮助用户快速制作战斗场景动画原型。用户可上传精灵表图片，切割出独立帧，在时间轴上排列组合成循环动画，并实时预览播放效果。

- **核心目标**：降低像素动画制作门槛，提供直观的帧级编辑体验
- **目标用户**：独立游戏开发者、像素艺术爱好者
- **市场价值**：轻量级、浏览器端即用，无需安装复杂软件

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 全部功能使用、导出动画 |

### 2.2 功能模块

1. **精灵表导入模块**：上传精灵表图片，框选帧边界，自动网格切割
2. **帧管理模块**：帧列表展示、重命名、删除、多选批量操作
3. **时间轴编辑模块**：拖拽排序、持续时间设置、右键菜单操作
4. **动画预览模块**：实时播放、速度调节、循环控制、帧信息显示
5. **导出模块**：合并为竖排精灵表 PNG，下载导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑器 | 导入面板 | 文件上传、精灵表预览、选框切割、确认生成帧 |
| 主编辑器 | 帧列表面板 | 帧缩略图网格、帧编号、重命名、删除、多选操作 |
| 主编辑器 | 预览画布 | 256x256 画布、帧居中显示、红色虚线边框、播放控制 |
| 主编辑器 | 时间轴 | 水平帧卡片、拖拽排序、持续时间输入、右键菜单 |
| 主编辑器 | 导出功能 | 进度条、旋转加载图标、自动下载 PNG |

## 3. 核心流程

用户上传精灵表图片 → 拖拽选框定义单帧尺寸 → 确认切割生成帧列表 → 拖拽帧到时间轴排序 → 设置每帧持续时间 → 调节播放速度预览 → 导出竖排精灵表

```mermaid

  A["上传精灵表"] --> B["框选帧边界"]
  B --> C["确认切割生成帧"]
  C --> D["帧列表管理"]
  D --> E["时间轴排序"]
  E --> F["设置持续时间"]
  F --> G["预览动画播放"]
  G --> H["导出精灵表"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景 #1E1E2E，面板 #2D2D44，边框 #3B3B5C
- **强调色**：蓝色 #3B82F6（选框/选中）、绿色 #22C55E（播放）、红色 #EF4444（删除/边框）
- **文字色**：浅灰色 #D1D5DB
- **字体**：JetBrains Mono 等宽字体
- **按钮风格**：圆角 4px，悬停高亮，点击反馈
- **布局风格**：三栏式布局，面板式结构，标题栏渐变
- **图标风格**：简约线性图标，使用 lucide-react

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主编辑器 | 左栏面板 | Tab 切换（Import/Frames）、帧缩略图网格、操作按钮 |
| 主编辑器 | 中间预览区 | 画布居中、播放按钮、帧信息显示、速度滑块 |
| 主编辑器 | 右栏时间轴 | 水平滚动帧卡片、拖拽指示线、右键菜单 |
| 主编辑器 | 全局 | 深色主题、面板边框、渐变标题栏、悬停动画 |

### 4.3 响应式

- **桌面端**（>900px）：三栏布局，左栏 280px，右栏 300px，中间自适应
- **移动端**（<900px）：左栏折叠为顶部 Tabs，右栏折叠为底部面板，中间占满全宽
- **触摸优化**：增大点击区域，支持触摸拖拽

### 4.4 动效设计

- 帧缩略图悬停：放大 1.1 倍，0.2s 过渡
- 选中帧：蓝色发光边框
- 拖拽卡片：透明度 0.3，轻微旋转 2 度
- 删除动画：缩小并淡出
- 导出进度：白色进度条从左向右填充，旋转加载图标

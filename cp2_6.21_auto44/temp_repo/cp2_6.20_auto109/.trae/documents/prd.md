## 1. 产品概述

AI战棋编辑器是一款在线战棋单位AI行为树编辑与战斗模拟应用，用户可创建并编辑不同职业战棋单位的AI行为树，并在6x8网格地图上模拟单位自动执行寻路、攻击、防守等战术动作。

- 核心价值：为战棋游戏开发者和AI爱好者提供可视化行为树编辑工具，支持实时战斗模拟与回放
- 目标用户：游戏开发者、AI策略研究者、战棋游戏爱好者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 使用所有编辑器和模拟器功能 |

### 2.2 功能模块
1. **行为树编辑器**：节点拖拽、连线、属性编辑、树状图实时预览
2. **网格地图组件**：6x8网格渲染、地形编辑、单位显示、移动范围高亮
3. **战斗模拟器**：开始/暂停/单步/重置控制、动画过渡、战斗日志
4. **日志回放系统**：时间线展示、伤害计算详情、JSON导出

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 行为树编辑器 | 节点工具面板、画布、属性侧面板，支持拖拽创建节点、连线、编辑属性 |
| 主页面 | 网格地图 | 6x8网格地形、单位图标、移动范围高亮、地形绘制工具 |
| 主页面 | 模拟控制面板 | 开始/暂停/单步/重置按钮、单位行为树分配 |
| 主页面 | 日志面板 | 战斗日志列表、自动滚动、淡入动画、JSON导出 |

## 3. 核心流程

用户从左侧工具面板拖拽节点到画布构建行为树 → 在属性面板编辑节点参数 → 为红蓝双方单位分配行为树 → 点击开始模拟 → 观看单位按行为树自动行动 → 查看战斗日志并导出JSON

```mermaid

    A["创建行为树节点"] --> B["连线构建树结构"]
    B --> C["编辑节点属性"]
    C --> D["分配行为树到单位"]
    D --> E["开始战斗模拟"]
    E --> F["单位自动执行行动"]
    F --> G["生成战斗日志"]
    G --> H["导出JSON回放"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，背景#1a1a2e，卡片背景#16213e，品牌色#0f3460
- **辅助色**：战士红色、弓箭手绿色、法师蓝色、刺客紫色
- **文字颜色**：浅灰色#e0e0e0
- **按钮风格**：圆角矩形，hover变深色品牌色
- **字体**：现代无衬线字体，清晰易读
- **布局风格**：左右分栏布局，左侧工具面板400px，右侧地图和模拟器
- **动效**：节点发光效果、单位脉动光环、移动平滑插值、攻击闪烁效果

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 节点工具面板 | 节点卡片（宽180px高40px，圆角8px），hover边框变品牌色 |
| 主页面 | 行为树画布 | 深色背景#0f0f1a，贝塞尔曲线连线，选中节点发光效果 |
| 主页面 | 网格地图 | 6x8格子，绿白交替，单位圆形图标，移动范围半透明覆盖 |
| 主页面 | 模拟控制栏 | 开始/暂停/单步/重置按钮，圆角矩形 |
| 主页面 | 日志面板 | 时间戳+动作描述，自动滚动，淡入动画 |

### 4.3 响应式
- 桌面端（≥1024px）：左右分栏布局，左侧面板400px固定宽度
- 移动端（<1024px）：左侧面板折叠为抽屉式侧边栏，从左侧滑出
- 网格地图自动缩放以适应可用区域

### 4.4 性能指标
- 节点拖拽保持60fps
- 动画过渡300ms内完成
- 日志超过50条启用虚拟滚动
- 单位移动插值流畅无卡顿

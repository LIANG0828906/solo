## 1. 产品概述

播客互动展示页面生成器，帮助独立播客创作者快速生成包含嘉宾信息、时间轴、高光时刻和互动问答的节目展示页面，解决手动编辑HTML繁琐、缺乏统一模板、听众无法标记感兴趣片段并留言的问题。

### 目标用户
- 独立播客创作者：需要快速生成美观的节目展示页
- 播客听众：在展示页上收听节目、查看时间轴、留言互动

### 产品价值
- 统一的节目展示模板，降低创作者技术门槛
- 丰富的互动功能（时间轴跳转、留言标记时间点）提升听众参与感
- 数据统计看板帮助创作者了解听众行为

---

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 播客创作者 | 无需登录（后台管理页） | 创建节目、查看统计数据 |
| 听众 | 无需登录 | 收听节目、查看时间轴、发表留言、标记时间点 |

### 2.2 功能模块

1. **后台管理页**：节目信息表单输入、提交生成节目、跳转展示页
2. **节目展示页**：封面与音频播放器、可交互时间轴、留言互动区
3. **统计看板**：访问次数、留言数、最多留言时间段柱状图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 后台管理页 | 节目表单 | 输入播客名称、期数、封面图URL、音频URL、嘉宾信息、节目简介，底部边框动效 |
| 后台管理页 | 提交按钮 | 调用API生成唯一节目ID，跳转到展示页 |
| 节目展示页 | 封面播放器区 | 圆形封面图、自定义音频播放器、波形可视化Canvas |
| 节目展示页 | 时间轴 | 可展开的时间轴条目，点击跳转播放位置，留言标记橙色圆点 |
| 节目展示页 | 留言区 | 昵称+留言提交，留言淡入动画，无限滚动加载，标记时间点按钮 |
| 统计看板 | 数据概览 | 总访问次数、总留言数统计卡片 |
| 统计看板 | 柱状图 | 最多留言时间段柱状图，弹性动画，Chart.js渲染 |

## 3. 核心流程

### 创作者流程
创作者在后台填写节目信息 → 提交生成节目 → 获取唯一ID → 跳转至节目展示页 → 分享链接给听众

### 听众流程
听众访问节目展示页 → 播放音频 → 查看/展开时间轴 → 点击时间戳跳转 → 发表留言 → 标记留言关联时间点 → 时间轴显示橙色标记点

```mermaid
    D --> E["听众访问页面"]
    E --> F["播放音频"]
    F --> G["查看时间轴"]
    F --> H["发表留言"]
    H --> I["标记时间点"]
    I --> J["时间轴显示标记"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：深色科技风，沉浸式播客体验
- **主色调**：紫色 #6C63FF、青色 #4ECDC4
- **辅助色**：粉色 #FF6584、橙色 #FF9F43（时间点标记）
- **背景色**：深蓝紫 #1A1A2E（页面背景）、#16213E（卡片背景）
- **文字色**：#E8E8E8（主文字）
- **卡片风格**：10px圆角、轻微内阴影、卡片间距20px
- **动效风格**：流畅过渡、300ms cubic-bezier 缓动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 后台管理页 | 表单区 | 输入框淡蓝底部边框动效、卡片式布局、紫色主按钮 |
| 节目展示页 | 播放器区 | 圆形封面图（浅灰阴影）、播放按钮旋转动画、进度条可拖动、音量滑块、倍速切换、Canvas波形（蓝紫渐变） |
| 节目展示页 | 时间轴 | 时间戳链接（点击跳转+滚动动画）、条目展开收起（高度平滑过渡）、橙色标记点 |
| 节目展示页 | 留言区 | 圆角输入框（焦点变主题色）、留言淡入动画、无限滚动加载、旋转紫色加载环 |
| 统计看板 | 数据卡片 | 三个统计卡片、数据数字动画、柱状图弹性动画 |

### 4.3 响应式设计
- **设计策略**：桌面端优先，移动端自适应
- **断点**：768px
- **移动端适配**：
  - 单列布局
  - 播放器固定在底部
  - 封面图缩小为圆形头像显示在左上角
  - 波形可视化简化显示
  - 触摸交互优化

### 4.4 动效设计
- 页面路由切换：左右滑入滑出，300ms cubic-bezier
- 播放按钮：播放时旋转动画
- 输入框焦点：底部边框淡蓝色扩展，200ms过渡
- 时间轴展开：高度从0到auto平滑过渡
- 留言新增：淡入动画
- 柱状图数据更新：弹性动画
- 加载状态：旋转紫色圆环

## 5. 性能要求

- 场景切换和音频播放保持60FPS流畅度
- 时间轴展开和留言新增无卡顿
- 首次加载页面资源（除音频文件外）总大小 ≤ 500KB

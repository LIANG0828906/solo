## 1. 产品概述

本项目是一款面向家庭咖啡烘焙师的在线烘焙模拟与风味预测应用，旨在解决烘焙师难以科学记录烘焙曲线、复盘风味差异以及分享配方的问题。目标用户为家庭咖啡烘焙爱好者和小型精品咖啡馆，通过数据化的烘焙记录与交互式风味品鉴，提升烘焙的科学与可复现性。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（模拟） | 创建批次、品鉴记录、浏览社区、克隆配方、发表评论 |

### 2.2 功能模块

1. **新建批次页面**：实时烘焙曲线可视化、阶段数据面板、标记注释
2. **批次详情页面**：完整曲线展示、品鉴历史、评论列表
3. **风味品鉴页面**：交互式风味轮盘、二级风味选择、品鉴笔记
4. **社区页面**：公开批次卡片列表、筛选与分页、配方克隆
5. **社区详情页面**：完整信息展示、评论发表

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 新建批次 (/batch/new) | 实时曲线图 | Chart.js绘制温度-时间曲线，X轴0-15分钟，Y轴100-250℃，曲线颜色橙色渐变到深红，每秒更新一个模拟数据点 |
| 新建批次 (/batch/new) | 阶段数据面板 | 显示当前豆温、环境温、升温速率(RoR) |
| 新建批次 (/batch/new) | 标记注释 | 用户可点击"记录标记"按钮在当前时间点打注释标签（如"一爆开始"），以浅黄色圆点显示在曲线上 |
| 批次详情 (/batch/:id) | 曲线展示 | 显示完整烘焙曲线图及所有标记点 |
| 批次详情 (/batch/:id) | 品鉴历史 | 展示与该批次关联的所有品鉴笔记 |
| 批次详情 (/batch/:id) | 评论列表 | 显示用户评论，支持新增评论 |
| 风味品鉴 (/taste) | 风味轮盘 | 中央可交互圆形轮盘，外层12个扇形区域代表风味大类，悬停外扩1.2倍并显示二级风味列表 |
| 风味品鉴 (/taste) | 二级选择面板 | 点击风味大类后右侧弹出二级选择面板，点击二级风味加入品鉴笔记 |
| 风味品鉴 (/taste) | 品鉴笔记 | 以标签形式显示已选风味，每个标签带×按钮可删除，提交后关联烘焙批次 |
| 社区 (/community) | 筛选栏 | 按豆种（埃塞俄比亚、哥伦比亚、巴西、肯尼亚）或烘焙度（浅、中浅、中、中深、深）筛选 |
| 社区 (/community) | 批次卡片 | 每张卡片展示豆种名称、烘焙日期、总用时、风味标签（最多5个）、评分（1-5星） |
| 社区 (/community) | 配方克隆 | 点击"克隆此配方"按钮复制批次参数到新批次 |
| 社区 (/community) | 分页浏览 | 每页9张卡片，支持翻页 |
| 社区详情 (/community/:id) | 完整信息 | 完整曲线图、品鉴笔记和评论列表 |
| 社区详情 (/community/:id) | 评论发表 | 文本框最多200字符，带字数计数 |

## 3. 核心流程

### 3.1 烘焙批次创建流程
用户进入"新建批次"页面 → 点击开始烘焙 → 系统模拟三阶段温度曲线（Phase1: 22→150℃, Phase2: 150→205℃, Phase3: 205→190℃） → 用户在关键节点打标记注释 → 烘焙完成自动保存记录 → 跳转至批次详情页

### 3.2 风味品鉴流程
用户点击"开始品鉴" → 进入品鉴页面 → 与风味轮盘交互选择风味大类 → 在二级面板选择具体风味 → 风味标签添加到笔记区 → 可删除或继续添加 → 提交品鉴关联到烘焙批次

### 3.3 社区克隆流程
用户进入社区页面 → 使用筛选条件查找批次 → 浏览卡片信息 → 点击进入详情或直接克隆 → 克隆后参数复制到新批次 → 用户可修改后重烘

## 4. 界面设计

### 4.1 设计风格

- **主色调**：温暖咖啡色调，米色到浅棕渐变背景（#FFF8F0至#F0E6D0）
- **导航栏**：深棕色（#4A2C2A）白色文字，置顶固定
- **曲线图区域**：纯白背景（#FFFFFF），16px圆角，2px浅棕（#C4A882）边框
- **按钮样式**：圆角矩形（8px），棕色字，浅橙色背景（#F4A460），悬浮加深为#D2691E并上移2px
- **风味轮盘**：中央放射渐变背景，外圈发光描边
- **表单输入框**：焦点状态2px深棕边框+淡棕阴影（0 0 8px rgba(74,44,42,0.3)）
- **字体**：选用具有手写质感的衬线字体作为标题，搭配清晰的无衬线字体作为正文
- **布局**：内容区最大宽度1200px，居中，两端留白，响应式

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 新建批次 | 实时曲线图 | 白色背景卡片，橙色→深红渐变曲线，浅黄色标记圆点，16px圆角，2px浅棕边框 |
| 新建批次 | 数据面板 | 三列数据卡片（豆温/环境温/RoR），深棕文字，米色背景 |
| 新建批次 | 标记按钮 | 浅橙色圆角按钮，悬浮加深上移 |
| 风味品鉴 | 风味轮盘 | 12色渐变扇形（#FFD700→#8B4513），悬停外扩1.2倍，放射渐变背景，外圈发光 |
| 风味品鉴 | 二级面板 | 右侧滑出面板，浅色背景，标签式二级选项 |
| 风味品鉴 | 笔记区 | 标签样式，×删除按钮，浅棕背景 |
| 社区 | 筛选栏 | 下拉选择框，深棕边框，聚焦阴影 |
| 社区 | 卡片网格 | 3×3网格，卡片含豆种/日期/时长/风味标签/星级评分 |
| 社区 | 分页 | 页码按钮，当前页深棕色高亮 |
| 社区详情 | 曲线图 | 同新建批次曲线风格 |
| 社区详情 | 评论区 | 文本框+字数计数+提交按钮 |

### 4.3 响应式

- 桌面优先设计，内容区最大宽度1200px居中
- 社区卡片：大屏3列，中屏2列，小屏1列
- 风味轮盘：根据屏幕尺寸自适应缩放
- 移动端导航栏折叠为汉堡菜单

## 1. 产品概述

面向中小型敏捷团队的冲刺规划与燃尽图追踪工具，帮助 Scrum Master 创建冲刺、分配任务并实时监控进度。目标用户为采用 Scrum 流程的开发团队，核心价值在于轻量化、可视化的冲刺管理与工时追踪。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| Scrum Master | 默认角色 | 创建/编辑冲刺、分配任务、查看燃尽图 |
| 团队成员 | 被分配至冲刺 | 查看任务、更新任务状态 |

### 2.2 功能模块

1. **冲刺仪表盘页**：冲刺概览信息、燃尽图展示、冲刺内任务列表、冲刺创建入口
2. **待办事项面板**：任务 CRUD 操作、拖拽分配、多条件筛选（优先级/负责人/状态）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 冲刺仪表盘页 | 冲刺概览区 | 展示当前冲刺名称、起止日期、团队成员、进度百分比 |
| 冲刺仪表盘页 | 燃尽图区 | 基于 Chart.js 绘制，红色折线显示实际剩余工时，灰色虚线显示理想线，悬停显示数值和日期 |
| 冲刺仪表盘页 | 冲刺任务列表 | 展示冲刺内已分配任务卡片，支持拖入/移出 |
| 冲刺仪表盘页 | 冲刺创建按钮 | 渐变色按钮，弹出创建冲刺表单（名称、起止日期、团队成员） |
| 待办事项面板 | 任务卡片列表 | 支持拖拽分配、优先级标记、点击查看详情 |
| 待办事项面板 | 筛选栏 | 按优先级（高/中/低）、负责人、状态筛选，切换时淡入淡出过渡 |
| 任务详情模态窗 | 详情/编辑表单 | 展示和编辑任务标题、描述、负责人、优先级、预估工时，保存后同步全局状态 |

## 3. 核心流程

用户打开应用后，左侧为待办事项面板，右侧为冲刺仪表盘。用户可通过冲刺创建按钮新建冲刺，从左侧待办面板拖拽任务卡片到右侧冲刺区域完成分配。燃尽图根据冲刺内任务的工时数据实时更新。点击任务卡片弹出详情模态窗可查看和编辑任务信息。

## 4. 用户界面设计

### 4.1 设计风格

- 主色：#1a1a2e（深色背景），辅助色：#16213e（面板背景），强调色：#e94560（按钮、高亮）
- 按钮样式：圆角渐变按钮（创建冲刺按钮从 #e94560 到 #0f3460），点击时凹陷反馈
- 字体：JetBrains Mono（数据/数字）+ Outfit（UI文本），大小层级清晰
- 布局：左侧固定 300px 待办面板（半透明磨砂背景），右侧冲刺仪表盘
- 图标：lucide-react 图标库
- 拖拽交互：半透明跟随状态，目标区域高亮，松开后弹性动画，缓动 ease-out 300ms

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 冲刺仪表盘页 | 燃尽图区 | 全宽显示，占仪表盘上方 60% 高度，红色折线+灰色虚线，悬停 tooltip |
| 冲刺仪表盘页 | 冲刺任务列表 | 圆角卡片带阴影，悬停阴影加深上移 4px |
| 冲刺仪表盘页 | 创建冲刺按钮 | 渐变色 #e94560→#0f3460，凹陷反馈 |
| 待办事项面板 | 面板整体 | 固定 300px 宽，半透明磨砂背景 backdrop-blur |
| 待办事项面板 | 筛选栏 | 下拉选择器，切换时卡片列表淡入淡出 ≤200ms |
| 待办事项面板 | 任务卡片 | 可拖拽，半透明跟随，优先级色标 |
| 任务详情模态窗 | 编辑表单 | 输入框获焦放大动画，保存按钮按压反馈 |
| 全局 | 响应式 | 屏幕<768px 时左侧面板折叠为可滑出侧栏 |

### 4.3 响应式设计

- 桌面优先（≥768px）：左侧固定 300px 面板 + 右侧仪表盘
- 移动端（<768px）：左侧面板折叠为可滑出侧栏，通过汉堡按钮触发滑入/滑出
- 拖拽在移动端降级为点击选择分配方式

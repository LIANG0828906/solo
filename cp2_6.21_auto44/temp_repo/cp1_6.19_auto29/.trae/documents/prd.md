## 1. 产品概述

Pet Paradise（宠物天堂）是一款像素风格的虚拟宠物养成与社交广场 Web 应用。用户可领养像素宠物（小猫、小狗或小龙），通过喂食、玩耍和清洁来维持宠物状态，同时在公共广场浏览和点赞其他用户的宠物动态。

- 目标用户：喜爱休闲养成类游戏、对像素风格有偏好的年轻用户群体
- 核心价值：结合养成互动与社交分享，打造温暖治愈的虚拟宠物社区体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 注册时选择宠物类型 | 养成自己的宠物、浏览广场、点赞他人宠物 |

### 2.2 功能模块

1. **宠物养成页面**：宠物展示、状态面板、交互按钮、状态衰减与恢复机制
2. **社交广场页面**：宠物卡片列表、分页、点赞功能、宠物详情查看

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 宠物养成页面 | 宠物展示区 | 显示16x16像素风格宠物形象，根据状态播放动画（跳跃/转圈/躺倒呼吸/消失告别） |
| 宠物养成页面 | 状态面板 | 四个渐变状态条（饱食度、快乐度、清洁度、精力值），低于30%时红色闪烁（0.5s频率），综合健康评分星级显示 |
| 宠物养成页面 | 交互按钮 | 喂食/玩耍/清洁三个按钮，点击触发宠物动画（0.3s过渡）并提升对应状态值 |
| 社交广场页面 | 卡片列表 | 每页12张卡片，显示宠物名字、类型图标、主人昵称、健康评分星级、点赞数；悬停放大1.1倍+阴影加深0.3s |
| 社交广场页面 | 分页导航 | 支持翻页浏览所有活跃宠物 |
| 社交广场页面 | 宠物详情弹窗 | 点击卡片进入宠物主页查看详情，点赞按钮心形填充动画（0.2s放大缩小），点赞数实时更新 |

## 3. 核心流程

### 3.1 注册领养流程
用户打开应用 → 选择宠物类型（小猫/小狗/小龙）→ 为宠物命名 → 进入宠物养成页面开始互动

### 3.2 养成互动流程
用户查看状态面板 → 发现某项状态偏低 → 点击对应交互按钮（喂食/玩耍/清洁）→ 宠物播放动画 → 状态值提升 → UI在100ms内更新

### 3.3 状态衰减流程
每5分钟自然衰减 → 非线性衰减曲线（初始1小时慢，后加速）→ 某项状态降为0 → 宠物显示躺倒+呼吸动画 → 用户交互恢复 → 若30分钟无交互 → 宠物消失+告别动画

### 3.4 社交广场流程
用户切换到广场页面 → 浏览宠物卡片列表 → 悬停查看卡片放大效果 → 点击卡片查看详情 → 点赞 → 点赞数实时更新

## 4. 用户界面设计

### 4.1 设计风格

- 主色：#FF8C42（温暖橙色）
- 辅色：#F4D03F（明亮金黄）
- 背景色：#FFF8E7（浅米色）
- 按钮风格：圆角12px + 柔和阴影，暖色调配色
- 字体：像素风标题字体 + 圆润正文字体
- 布局：桌面端左右两栏（60%/40%），移动端上下堆叠
- 动画：统一使用 ease-out 缓动函数

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 宠物养成页面 | 宠物展示区 | 像素风格宠物形象居中展示，带动画状态切换，背景带柔和渐变 |
| 宠物养成页面 | 状态面板 | 四个水平渐变状态条（红→绿），低值闪烁，星级评分，交互按钮行 |
| 社交广场页面 | 卡片网格 | 3列4行网格布局（桌面端），每张卡片圆角12px+阴影，悬停放大1.1倍+0.3s阴影加深 |
| 社交广场页面 | 宠物详情弹窗 | 模态弹窗，宠物大图+完整状态+点赞按钮（心形填充动画0.2s）|

### 4.3 响应式设计

- 桌面优先设计
- 宠物养成页面：桌面端左右两栏（60%+40%），768px以下自动堆叠为上下结构
- 社交广场：桌面端3列，平板2列，手机1列
- 触控优化：按钮最小44px点击区域，卡片间距适配手指操作

### 4.4 动画规格

| 动画名称 | 触发条件 | 效果 | 时长 | 缓动 |
|----------|----------|------|------|------|
| 跳跃动画 | 喂食/玩耍/清洁按钮点击 | 宠物上移后回落 | 0.3s | ease-out |
| 状态条闪烁 | 状态值<30% | 红色透明度循环 | 0.5s周期 | - |
| 躺倒呼吸 | 任一状态=0 | 缩放0.95↔1.0 | 2s周期 | - |
| 告别动画 | 状态0持续30分钟 | 宠物淡出+伤感提示 | 1.5s | ease-out |
| 卡片悬停 | 鼠标悬停卡片 | 放大1.1倍+阴影加深 | 0.3s | ease-out |
| 点赞动画 | 点击点赞按钮 | 心形放大缩小 | 0.2s | ease-out |

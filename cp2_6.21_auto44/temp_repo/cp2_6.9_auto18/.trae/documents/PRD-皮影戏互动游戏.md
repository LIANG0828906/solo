## 1. 产品概述

皮影戏互动游戏是一款在浏览器中模拟古代皮影戏表演与皮影制作全流程的互动应用，解决传统皮影戏中影人雕刻、上色、操纵杆绑定以及幕前演出难以在屏幕上直观体验和反复排练的问题。

- 主要用途：让用户在数字环境中体验皮影制作与表演的完整流程，传承非遗文化
- 目标用户：文化爱好者、教育工作者、学生群体以及对传统艺术感兴趣的普通用户
- 市场价值：将传统皮影戏数字化，降低学习门槛，提供沉浸式互动体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地存储 | 制作皮影、表演剧目、保存作品、导出分享 |

### 2.2 功能模块

1. **工坊界面**：皮革选择、轮廓绘制、裁剪分离、部件上色、关节装配
2. **戏台界面**：幕布表演、灯光模拟、影操纵、倒计时评分、剧情模式
3. **收藏与分享**：本地归档、JSON导出、GIF导出、回放功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工坊界面 | 皮革选择 | 三种皮革（牛皮、驴皮、羊皮）选择，带纹理效果 |
| 工坊界面 | 轮廓绘制 | 鼠标绘制直线和曲线，笔触宽3px，颜色#333 |
| 工坊界面 | 裁剪分离 | 沿轮廓裁剪，废料淡出动画，6个部件落入材料盘 |
| 工坊界面 | 部件上色 | 6种颜料拖拽上色，带透明度效果 |
| 工坊界面 | 关节装配 | 竹签绑定关节，物理铰链模拟，阻尼系数0.3 |
| 戏台界面 | 幕布渲染 | 800x600半透明幕布，drop-shadow投影效果 |
| 戏台界面 | 灯光模拟 | 油灯径向渐变光源，随鼠标轻微偏移，火焰闪烁 |
| 戏台界面 | 影人操纵 | 拖拽关节控制动作，响应延迟<50ms |
| 戏台界面 | 倒计时评分 | 60秒演出计时，基于流畅度和多样性评分 |
| 戏台界面 | 剧情模式 | 三个预设剧本，台词滚动提示，关键动作触发道具特效 |
| 戏台界面 | 录像回放 | 保存30秒帧序列，15fps回放，支持慢放快进 |
| 收藏分享 | JSON导出 | 影人外观和关节参数导出 |
| 收藏分享 | GIF导出 | Web Worker处理，10fps，256色，480px宽 |
| 收藏分享 | 本地归档 | 封面预览，重新加载编辑或播放 |

## 3. 核心流程

```mermaid
flowchart LR
    A["选择皮革"] --> B["绘制轮廓"]
    B --> C["裁剪分离"]
    C --> D["部件上色"]
    D --> E["关节装配"]
    E --> F["进入戏台"]
    F --> G["选择剧本"]
    G --> H["幕前表演"]
    H --> I["倒计时结束"]
    I --> J["评分展示"]
    J --> K["保存/导出"]
    K --> L["本地收藏"]
```

用户在工坊完成皮影制作后进入戏台，可选择剧情模式或自由表演，表演结束后获得评分并可保存分享。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暗红#8b0000、土黄#d4a76a、米白#f5f0e0
- **字体**：楷体，体现中国传统文化特色
- **按钮样式**：圆角6px，0.5px内边框#b88a4a，悬停阴影加深至8px
- **布局风格**：工坊左右分栏（60%/40%），中间8px分隔条可拖拽；戏台黑底场景
- **视觉元素**：竹编纹理、皮革颗粒、油灯火焰、幕布发光

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工坊界面 | 皮革操作区 | 皮革纹理背景、绘制画布、裁剪按钮 |
| 工坊界面 | 颜料架 | 6格颜料（红#c0392b、绿#27ae60、蓝#2980b9、黄#f1c40f、金#d4a017、黑#2c3e50） |
| 工坊界面 | 材料盘 | 竹编圆形，直径120px，浅黄色#e8d5a0 |
| 工坊界面 | 五金盒 | 铜铆钉和竹签图标 |
| 戏台界面 | 幕布区域 | 白色半透明#ffffff 60%透明度，发光效果text-shadow: 0 0 20px #fff |
| 戏台界面 | 油灯 | 径向渐变#ffd700到#ff8c00，半径200px，CSS动画闪烁 |
| 戏台界面 | 台词面板 | 宋体14px，颜色#555，当前行高亮#c0392b |
| 戏台界面 | 控制栏 | 暂停/播放、回放速度、倒计时显示 |
| 收藏界面 | 作品列表 | 卡片式布局，封面为轮廓预览 |

### 4.3 响应式设计

- **桌面端（>768px）**：工坊左右分栏布局，标准尺寸交互元素
- **移动端（≤768px）**：自动切换上下布局，交互元素缩小至70%
- **触摸优化**：拖拽目标区域放大，确保移动端可操作

### 4.4 交互动效

- **触觉反馈**：点击元素transform: scale(0.95)，0.1s动画
- **拖拽效果**：被拖拽对象opacity 0.6，跟随鼠标移动
- **目标闪烁**：合法目标区域金色边框闪烁0.3秒
- **废料淡出**：半透明淡出动画
- **火焰闪烁**：opacity 0.8到1循环，周期1.2s
- **道具飞入**：0.5秒ease-out动画

## 5. 性能要求

- **影人操纵响应延迟**：低于50ms
- **幕前表演帧率**：稳定30fps以上，使用requestAnimationFrame优化
- **GIF导出**：主线程阻塞不超过200ms，Web Worker处理帧压缩

## 1. 产品概述
PodChapter 是一款面向独立播客创作者和音频内容团队的浏览器端互动式播客章节编辑与发布工具，解决现有剪辑工具无法让听众在时间轴上点击章节标记直接跳转、也无法在章节间嵌入投票、链接或图文弹窗等互动元素的问题，从而提升播客收听体验的双向互动率和用户参与度。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 播客创作者 | 无需注册 | 上传音频、添加章节与互动元素、发布分享链接 |
| 听众 | 无需注册 | 通过分享链接收听、点击章节跳转、参与互动 |

### 2.2 功能模块
1. **编辑器页面**: 音频上传与波形渲染、播放控制、章节标记添加与编辑、互动元素配置、侧边栏章节管理、发布与预览
2. **预览/播放页面**: 播客标题与封面、播放器控件、章节列表与互动体验、评论区占位

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 音频上传区 | 上传MP3/WAV音频文件（最大50MB），上传后即时解码渲染波形 |
| 编辑器页面 | 播放控制栏 | 播放/暂停按钮、进度条（已播放/未播放双色）、时间显示 |
| 编辑器页面 | 波形时间轴 | Canvas渲染音频波形（深灰背景#2D3748，波形色#4FD1C5），刻度标尺每5秒tick，章节标记浮动气泡 |
| 编辑器页面 | 章节编辑面板 | 弹出面板输入章节标题（2-20字）和描述（≤150字），互动元素下拉选择 |
| 编辑器页面 | 互动元素配置 | 投票（1-4选项、计票百分比进度条）、外部链接（URL输入、新标签页打开）、图文弹窗（图片+文字、暂停自动弹出） |
| 编辑器页面 | 侧边栏 | 章节列表显示编号、标题、互动类型图标，点击跳转时间点，编辑/删除操作 |
| 编辑器页面 | 发布按钮 | 生成JSON元数据、创建唯一分享链接、复制链接或预览 |
| 预览页面 | 播放器区域 | 播客标题、封面占位、播放器控件 |
| 预览页面 | 章节与互动区 | 章节列表、互动元素体验（投票、链接、弹窗） |
| 预览页面 | 评论区占位 | 评论输入框与评论列表占位 |

## 3. 核心流程

用户打开应用 → 上传音频文件 → 系统解码并渲染波形 → 用户播放音频并在时间轴上定位 → 点击"添加章节"在当前位置创建章节标记 → 填写章节标题和描述 → 选择互动元素类型并配置（投票/链接/弹窗） → 在侧边栏管理所有章节 → 点击"发布"生成分享链接 → 复制链接或预览最终效果

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暗色主题（背景#1A202C，文字#E2E8F0），强调色青绿#4FD1C5
- 按钮：添加章节按钮#38A169圆角6px，播放按钮圆形44px青绿渐变，发布按钮渐变蓝圆角8px
- 字体：Inter或无衬线字体，刻度10px，正文14px
- 布局：左右分栏（主编辑区+侧边栏280px），响应式（<768px侧边栏折叠为底部横条60px）
- 图标：使用lucide-react图标库（投票条形图标、链接图标、窗口图标等）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 编辑器页面 | 播放控制栏 | 圆形播放/暂停按钮44px青绿渐变，进度条70%宽度双色，白色圆形播放头14px，悬停放大48px 0.15s过渡 |
| 编辑器页面 | 波形时间轴 | Canvas 80px高度，背景#2D3748，波形#4FD1C5渐变线#81E6D9→#319795，刻度标尺10px #718096 |
| 编辑器页面 | 章节标记 | 金色菱形12px #ECC94B带序号，悬停放大16px显示标题，浮动气泡在波形上方 |
| 编辑器页面 | 侧边栏 | 280px宽#1A202C背景，暗色滚动条#2D3748/滑动块#4A5568，列表项含编号/标题/互动图标 |
| 编辑器页面 | 互动元素 | 输入框圆角4px边框#CBD5E0聚焦#3182CE，投票百分比进度条，弹窗rgba(0,0,0,0.7)圆角12px |
| 编辑器页面 | 发布按钮 | 右上角渐变蓝#4299E1→#3182CE圆角8px白色文字，点击0.5s旋转加载动画 |
| 预览页面 | 整体布局 | 顶部标题+封面，中间播放器，下方章节列表+评论区，最小宽度320px响应式 |

### 4.3 响应式设计
- 桌面优先（≥768px）：左右分栏布局，侧边栏280px固定宽度
- 移动端（<768px）：侧边栏折叠为底部横条60px可上滑展开，播放器控制栏自适应缩小
- 所有交互元素悬停上浮translateY(-2px)阴影加深0.1s过渡，点击缩放scale:0.97 0.05s

### 4.4 性能要求
- 播放音频时波形渲染帧率≥30fps
- 章节标记拖动或编辑时界面响应延迟≤100ms

## 1. 产品概述
动态地牢陷阱生成与效果预览应用，面向Roguelike地牢探险游戏开发者，提供3D可视化陷阱预览、参数调试和实时状态监控功能，帮助快速设计和调试陷阱机制。

## 2. 核心功能

### 2.1 功能模块
1. **主场景页**：3D地牢房间渲染、陷阱模型展示与动画、触发范围可视化
2. **控制面板**：陷阱类型切换、参数调节滑块
3. **调试面板**：实时碰撞箱、伤害范围、状态监控
4. **重置功能**：一键恢复默认参数

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景 | 3D地牢房间 | 石纹墙壁、方格石板地面、陷阱模型与周期动画 |
| 主场景 | 陷阱动画 | 尖刺弹出/缩回动画、落石坠落、毒气扩散、刀片旋转 |
| 主场景 | 触发范围指示 | 半透明红色圆圈显示触发半径范围 |
| 控制面板 | 陷阱类型选择 | 下拉选择4种陷阱类型，切换时0.5秒淡入淡出过渡 |
| 控制面板 | 参数调节 | 触发半径0.5-3.0、伤害值1-10、持续时间1-5秒滑块 |
| 调试面板 | 实时数据 | 碰撞箱位置、伤害范围、状态（待命/触发/冷却），每帧更新 |
| 重置按钮 | 一键重置 | 圆形带刷新图标，点击恢复默认参数和动画 |

## 3. 核心流程
用户进入应用 → 查看3D地牢房间默认尖刺陷阱 → 通过右侧面板切换陷阱类型 → 调节触发半径/伤害/持续时间参数 → 观察3D场景实时变化 → 通过左下角调试面板监控数据 → 点击重置按钮恢复默认

## 4. 用户界面设计

### 4.1 设计风格
- 主背景色：#1a1a2e
- 控制面板：#16213e 半透明卡片
- 强调色：#e94560（红粉色）
- 滑块轨道：#0f3460，滑块圆点：#e94560
- 按钮悬停：#e94560到#0f3460渐变
- 按钮点击：缩放0.95按压动效
- 字体：Cinzel（标题）+ Rajdhani（UI文字）
- 毛玻璃效果：调试面板半透明背景

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主场景 | 3D渲染区域 | 占据左侧约75%宽度，Three.js Canvas全高 |
| 主场景 | 触发范围红圈 | 半透明红色圆环在地面上 |
| 控制面板 | 右侧面板 | #16213e半透明卡片，垂直排列下拉框和滑块 |
| 控制面板 | 陷阱类型下拉 | 深色背景下拉框，4个选项 |
| 控制面板 | 参数滑块 | 自定义样式，轨道#0f3460，圆点#e94560，右侧数值标记 |
| 调试面板 | 左下角数据 | 毛玻璃背景，显示碰撞箱/伤害范围/状态 |
| 重置按钮 | 右上角 | 圆形按钮，刷新图标，悬停渐变 |

### 4.3 响应式适配
- 桌面优先设计，目标分辨率1920x1080和1366x768
- 3D场景和控制面板比例自适应
- 界面元素自动居中缩放

### 4.4 3D场景指引
- 环境：阴暗地牢氛围，微弱暖色点光源
- 灯光：环境光低强度 + 点光源模拟火把
- 相机：透视相机，俯视45度角，可鼠标旋转
- 焦点元素：房间中央陷阱模型
- 动画：尖刺弹出/缩回、落石坠落、毒气粒子扩散、刀片旋转
- 性能目标：稳定60FPS，参数调整响应<200ms

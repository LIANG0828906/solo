## 1. 产品概述

双镜头取景器模拟与胶片摄影实验室是一款沉浸式复古相机网页应用，为摄影爱好者和复古美学爱好者提供模拟双镜头反光照相机（TLR）腰平取景器的拍摄体验。用户可以通过两个并列的圆形取景窗口观察实时画面，应用多层胶片模拟滤镜，并在胶片条上回放拍摄的照片。

- 目标用户：复古摄影爱好者、胶片相机玩家、视觉艺术创作者
- 核心价值：在数字世界中重现胶片摄影的仪式感和独特美学

## 2. 核心功能

### 2.1 功能模块

1. **双镜头取景器**：两个圆形取景窗口，左侧为实时取景带对焦辅助线，右侧为构图辅助网格线
2. **胶片滤镜处理引擎**：颗粒效果、色偏效果、晕影效果三层滤镜可调节
3. **快门拍摄系统**：金属质感快门按钮，带闪光动画反馈
4. **胶片条滚动回放**：最近6张照片的横向胶片条，带滚动动画和胶片齿孔效果
5. **手动对焦模式**：虚拟对焦环，景深计算和合焦提示
6. **滤镜参数控制面板**：颗粒强度、色偏切换、晕影范围调节

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|

| 主页面 | 快门按钮 | 圆形金属质感按钮，按下动画 + 全屏闪光效果 |

## 3. 核心流程

### 3.1 拍摄流程

用户打开页面 → 摄像头（或模拟画面）实时显示在双取景窗中 → 用户调节缩放和对焦 → 调整滤镜参数 → 按下快门 → 全屏闪光 → 滤镜处理引擎处理图像 → 图像存入状态 → 胶片条向左滚动展示新照片

```mermaid
flowchart LR
    A["进入页面"] --> B["初始化摄像头/模拟画面"]
    B --> C["双取景窗实时显示"]
    C --> D["用户调节缩放/对焦/滤镜"]
    D --> E["按下快门按钮"]
    E --> F["触发全屏闪光动画"]
    F --> G["捕获当前帧画面"]
    G --> H["FilterProcessor依次应用三层滤镜"]
    H --> I["Base64图像存入Zustand Store"]
    I --> J["胶片条向左滚动动画"]
    J --> K["新照片显示在胶片条最左侧"]
```

### 3.2 照片预览流程

用户点击胶片条缩略图 → 模态框淡入显示原图 → 点击关闭或背景 → 模态框淡出

## 4. 用户界面设计

### 4.1 设计风格

- **设计主题**：深色复古相机美学，模拟经典TLR相机的质感
- **主色调**：深棕色 #3E2F23 到 黑色 #1A1A1A 径向渐变背景
- **强调色**：琥珀色 #D4A574（高亮、滑块、按钮）
- **辅助色**：冷青色 #7EB5D1（色偏选项）、银灰色（快门按钮金属质感）
- **取景器边框**：黑色圆环
- **棱镜装饰**：等腰梯形，颜色 #B8A88A，模拟相机顶部棱镜

**按钮风格**：
- 圆角设计（8px圆角）
- 半透明黑色背景（#00000099）
- 悬停时0.2s背景色过渡到不透明
- 点击时0.95倍缩放反馈

**字体**：
- 使用具有复古感的衬线字体或等宽字体，营造摄影器材仪表盘氛围
- 标题使用粗体，参数数值使用等宽字体

**布局风格**：
- 居中对称布局，取景器区域占顶部60%
- 下方控制区域和胶片条
- 两个取景窗之间有棱镜造型装饰

### 4.2 页面设计概览

| 页面/模块 | 核心UI元素 | 风格要点 |
|----------|-----------|---------|
| 双镜头取景器 | 黑色圆环边框、十字对焦线、网格线、棱镜装饰 | 对称居中，真实相机取景器质感 |
| 快门按钮 | 银灰色金属质感、按下弹起动画 | 圆形60px直径，3D立体效果 |
| 滤镜控制面板 | 圆角12px半透明背景、琥珀色滑块、切换按钮 | 复古仪表盘风格 |
| 胶片条 | 深灰背景 #2A2A2A、胶片齿孔、方形缩略图 | 真实胶片卷动效果 |
| 对焦指示器 | 蓝色半圆环刻度、绿色/红色合焦光圈 | 精密仪器刻度风格 |

### 4.3 响应式设计

- **桌面端（≥700px）**：双取景窗左右并列，直径240px，间距20px
- **移动端（<700px）**：双取景窗上下排列，直径缩小至160px
- 胶片条高度自适应：桌面100px，移动60px
- 触摸优化：按钮尺寸适合点击，滑块支持触摸拖动

### 4.4 动画与交互

- **快门闪光**：0.1秒全屏白色闪光覆盖
- **快门按钮**：按下时弹起0.3秒动画
- **胶片滚动**：新照片从左侧插入，0.5秒平移过渡（framer-motion）
- **悬停效果**：按钮和滑块0.2秒背景色过渡
- **点击反馈**：0.95倍缩放
- **取景器刷新**：requestAnimationFrame驱动，≥24fps

## 5. 性能约束

- 取景窗口刷新率：≥24fps，使用requestAnimationFrame驱动
- 快门到胶片条显示：≤150ms，使用Canvas ImageData直接操作
- 滤镜处理：单次ImageData读取，三层滤镜顺序应用，避免多次Canvas转换

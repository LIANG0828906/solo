## 1. 产品概述

Skill Forge 是一款面向独立游戏开发者的战斗模拟沙箱应用，帮助开发者快速测试和平衡角色技能数值。解决开发初期缺乏真实对局数据、手动调整数值效率低下且难以全面验证各种技能组合强度的问题。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 独立游戏开发者 | 无需注册，直接使用 | 配置角色、运行模拟、查看统计、保存历史 |

### 2.2 功能模块

1. **角色配置模块**：创建两个自定义角色，从预设技能库选择最多4个技能，设置基础属性
2. **战斗模拟模块**：回合制对战逻辑，速度决定行动顺序，技能冷却独立计算
3. **战斗日志模块**：逐帧输出每回合详细行为记录，支持自动滚动和暂停
4. **统计面板模块**：展示双方战斗数据对比，柱状图和雷达图呈现
5. **历史记录模块**：保存最多10套配置方案和战斗记录到本地存储

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 角色配置区 | 左右双角色配置面板，技能选择、属性滑块调节 |
| 主页面 | 战斗日志区 | 实时战斗记录输出，自动滚动/暂停切换 |
| 主页面 | 统计面板区 | 柱状图和雷达图展示战斗数据对比 |
| 主页面 | 历史记录区 | 卡片式展示保存的配置，支持载入和删除 |

## 3. 核心流程

用户配置两个角色的属性和技能 → 点击开始模拟 → 战斗引擎自动运行回合制对战 → 战斗日志实时输出 → 战斗结束后展示统计数据 → 用户可保存配置到历史记录 → 点击历史卡片可重新载入配置

```mermaid
    B --> C["战斗引擎运行回合制对战"]
    C --> D["战斗日志实时输出"]
    D --> E["战斗结束展示统计数据"]
    E --> F["保存配置到历史记录"]
    F --> G["点击历史卡片载入配置"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主背景：深灰蓝(#1a2332)渐变
- 面板风格：磨砂半透明（rgba(30,40,55,0.85)，圆角16px，边框1px rgba(255,255,255,0.08)）
- 技能图标：圆形色块，焰红#ff5252、冰蓝#448aff、治愈绿#69f0ae、电紫#e040fb、盾橙#ffab40
- 选中状态：放大1.1倍且外圈发光
- 柱状图：蓝色(#2196f3)到红色(#f44336)渐变
- 雷达图：背景半透明浅灰(#e0e0e0)，数据线宽3px带圆点标记
- 历史卡片：尺寸260x120px，圆角12px，毛玻璃背景，3px左侧色条（绿色已验证/黄色待测试）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 角色配置区 | 滑块带实时数值显示和0.15s平滑过渡动画，技能圆形图标 |
| 主页面 | 战斗日志区 | 可滚动日志区域，自动/手动切换按钮 |
| 主页面 | 统计面板区 | recharts柱状图和雷达图，悬停显示精确数值 |
| 主页面 | 历史记录区 | 卡片式布局，命名、删除、载入功能 |

### 4.3 响应性

桌面端优先设计，响应式适配不同屏幕尺寸。

### 4.4 动效设计

- 滑块数值平滑过渡动画（0.15s）
- 技能选中放大发光效果
- 战斗弹道和碰撞溅射粒子效果（粒子大小4-8px，存在0.3s后消散）
- 历史卡片悬停效果

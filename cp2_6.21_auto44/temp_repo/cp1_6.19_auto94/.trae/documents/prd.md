## 1. 产品概述

微型城市天际线生长模拟与交互探索应用，让用户通过滑块和点击控制不同区域（商业区、住宅区、工业区）的开发强度，实时观察城市从空地逐渐长成高楼林立的3D景观。

- 目标用户：城市模拟爱好者、交互3D体验探索者
- 核心价值：通过直觉式交互让用户体验城市生长的动态过程，融合元胞自动机算法与实时3D渲染

## 2. 核心功能

### 2.1 功能模块

1. **3D城市场景页面**：50x50网格城市地面、3D建筑群、粒子效果、鼠标交互
2. **控制面板**：区域开发强度滑块、生长速度滑块、重置按钮、折叠功能

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D城市场景 | 城市地面 | 50x50方格网格（边长2单位），中心浅灰#D3D3D3向边缘深灰#696969渐变 |
| 3D城市场景 | 区域覆盖层 | 商业区#4A90D9/住宅区#50B86C/工业区#D94A4A，透明度0.3，悬停0.6+标签 |
| 3D城市场景 | 建筑群 | InstancedMesh渲染，最多2500栋，2秒生长动画+0.3s弹性缓动 |
| 3D城市场景 | 建筑颜色渐变 | 商业区#A8D0E6→#4A90D9，住宅区#A8E6A8→#50B86C，工业区#E6A8A8→#D94A4A |
| 3D城市场景 | 建筑信息卡 | 点击弹出悬浮卡（rgba(255,255,255,0.95)/圆角12px/微阴影），显示类型/高度/楼层数/区域，0.3s飞出动画 |
| 3D城市场景 | 粒子系统 | 白色光点（0.1-0.3单位），上升0.5单位/秒，生命周期1.5秒，总数≤200 |
| 3D城市场景 | 视角控制 | 鼠标拖拽视角，滚轮缩放 |
| 控制面板 | 开发强度滑块 | 商业/住宅/工业各0-100，对应区域颜色为激活色 |
| 控制面板 | 生长速度滑块 | 0.5-5倍速 |
| 控制面板 | 重置按钮 | 红色#D94A4A，1秒均匀收缩回地面，区域覆盖层淡出 |
| 控制面板 | 折叠按钮 | 右上角箭头图标，收起为40px窄条，0.3s弹性滑入 |

## 3. 核心流程

用户打开应用 → 看到平坦50x50网格 → 通过滑块调整区域开发强度 → 城市建筑在2秒内逐渐生长 → 点击建筑查看信息卡 → 调整生长速度控制动画节奏 → 点击重置按钮还原空地

## 4. 界面设计

### 4.1 设计风格

- 深色主题，背景深蓝灰#0D1117
- 控制面板：半透明深灰rgba(22,27,34,0.9)，10px圆角，20px上下内边距
- 滑块轨道：深灰#21262D，手柄：亮蓝#58A6FF（悬停外发光）
- 区域激活色：商业#4A90D9、住宅#50B86C、工业#D94A4A
- 重置按钮：红色#D94A4A，点击缩小0.95倍
- 字体：系统无衬线字体，层次通过大小和颜色区分

### 4.2 界面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 3D城市场景 | 地面网格 | 50x50方格，中心浅灰到边缘深灰渐变 |
| 3D城市场景 | 建筑群 | 基于InstancedMesh的长方体群，颜色随开发强度渐变 |
| 3D城市场景 | 信息卡 | 半透明白色背景，圆角12px，微阴影，指向箭头 |
| 控制面板 | 滑块组 | 三个区域滑块+生长速度滑块，轨道#21262D，手柄#58A6FF |
| 控制面板 | 重置按钮 | 红色#D94A4A，圆角，点击缩放动画 |
| 控制面板 | 折叠按钮 | 箭头图标，切换面板展开/收起 |

### 4.3 响应式

- 桌面优先设计，控制面板固定右侧300px
- 3D场景自适应填满剩余空间
- 面板可折叠为40px窄条

### 4.4 3D场景指引

- 环境光+方向光组合，营造城市夜景氛围
- 相机：透视相机，初始45度俯视，OrbitControls拖拽+缩放
- 建筑：BoxGeometry长方体，InstancedMesh批量渲染
- 粒子：PointsMaterial + BufferGeometry，GPU加速
- 性能目标：30FPS以上，draw call最小化

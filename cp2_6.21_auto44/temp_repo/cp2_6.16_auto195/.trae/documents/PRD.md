# CrystalGarden 水晶花园 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 项目背景
CrystalGarden 是一个基于 Three.js 的交互式 3D 虚拟水晶花园体验。用户可以在浏览器中探索一片由参数化生成的水晶组成的奇幻世界，通过鼠标交互与水晶产生互动。

### 1.2 目标用户
- 3D 可视化爱好者
- 创意交互体验探索者
- WebGL 技术学习者

### 1.3 核心价值
提供沉浸式、可交互的虚拟水晶花园观赏体验，通过精美的视觉效果和流畅的动画带来视觉愉悦。

## 2. 功能需求

### 2.1 场景环境
- **背景**: 深紫色渐变（#1A0F2E 到 #2D1B4E）的夜晚天空，带有缓慢变化的极光飘动效果
- **地形**: 网格抖动生成的轻微起伏地形，地表铺满半透明荧光点（淡蓝色 #B5D8FF，大小 0.5-1.5 像素，缓慢呼吸闪烁）

### 2.2 水晶系统
- **参数化生成**: 每株水晶由 6-12 个面组成棱柱，高度 0.8-2.5 单位随机
- **颜色**: 从预设调色板随机选取（紫水晶 #9B59B6、蓝宝石 #3498DB、翡翠 #2ECC71、琥珀 #F39C12）
- **透明度**: 0.6-0.9 随机
- **棱边描边**: 发光描边，颜色略亮，宽度 0.02 单位
- **生长位置**: 水晶生长在地形凸起处，总数不超过 300 株

### 2.3 用户交互
- **视角控制**: 鼠标拖拽旋转视角，滚轮拉近拉远
- **悬停效果 (Hover)**: 水晶微微发光并放大 1.05 倍，周围出现扩散浅色光环（半径增大 0.3 单位，1 秒后消失）
- **点击绽放**: 晶体沿中心向外分裂成 6-8 个小碎片，旋转飞出；从中心喷出 30-50 个彩色粒子（大小 0.1-0.3 单位，生命周期 2 秒）；0.5 秒后重构为新的随机形状（颜色微变）

### 2.4 UI 界面
- **悬浮工具栏**: 右上角半透明磨砂玻璃效果
  - 背景: #FFFFFF 透明度 0.1，边框 #FFFFFF 透明度 0.2，圆角 12px
  - **"重新生长全部"按钮**: 所有水晶 1 秒内缩回地面，重新长出新的随机组合，伴随地面微弱震动
  - **"地形起伏"滑块**: 范围 0-2，控制地形波动幅度，拖动时实时重新生成，过渡平滑

## 3. 非功能需求

### 3.1 性能要求
- 稳定 60 FPS
- 水晶数量限制在 300 株以内
- 动画流畅无卡顿

### 3.2 技术栈
- TypeScript
- Three.js 0.160
- Vite 构建工具
- uuid 库

### 3.3 兼容性
- 现代浏览器支持（Chrome、Firefox、Safari、Edge）
- WebGL 2.0 支持

## 4. 视觉设计

### 4.1 整体风格
- 梦幻、神秘的夜晚水晶花园
- 深紫色调为主，配合多彩水晶的发光效果
- 极光背景营造奇幻氛围

### 4.2 动效设计
- 平滑过渡动画
- 粒子效果轻盈飘逸
- 水晶绽放与重构动画自然流畅
- 地形变化过渡平滑无跳变

## 5. 数据流向

```
main.ts ──实例化──> Garden.ts ──管理──> Crystal.ts (多个实例)
   │                  │
   └──事件绑定──> UI.ts ──回调──> Garden 的重置/地形更新方法
                        │
                        └──> 鼠标交互 ──> raycaster 检测 ──> Crystal.animateBurst()
```

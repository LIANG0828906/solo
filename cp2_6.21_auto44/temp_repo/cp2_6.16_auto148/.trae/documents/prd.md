## 1. 产品概述

ArtCollab 是一个面向小型艺术团体的展览与艺术作品协作管理平台，支持策展人创建和管理展览、艺术家提交和审核作品、观众收藏评价作品，并通过热度仪表盘追踪展览影响力。项目采用纯前端架构（React + TypeScript + Vite + Zustand + IndexedDB），无后端依赖，数据全部持久化在浏览器本地。

- 目标用户：策展人、艺术家、艺术爱好者/观众
- 核心价值：降低艺术展览组织的协作门槛，提供一站式的展览发布、作品征集、审核和互动体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 策展人 | 默认角色，可切换 | 创建展览、审核作品、查看热度仪表盘 |
| 艺术家 | 默认角色，可切换 | 提交作品至展览、查看审核结果 |
| 观众 | 默认角色，可切换 | 浏览展览与作品、收藏作品、发表评论和评分 |

### 2.2 功能模块

1. **首页**：展览网格展示、搜索过滤、热度排序、创建展览入口
2. **展览详情页**：作品画廊、作品提交入口、热度仪表盘
3. **我的收藏页**：收藏作品列表、取消收藏

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 顶部导航栏 | 毛玻璃效果导航，包含Logo、展览链接、我的收藏链接，移动端汉堡菜单 |
| 首页 | 搜索与排序 | 按展览名称搜索，按热度排序（切换时卡片淡入动画0.4s） |
| 首页 | 展览卡片网格 | 展示所有展览卡片（封面缩略图、标题、日期、热度），已结束展览半透明+标签 |
| 首页 | 创建展览浮动按钮 | 右下角圆形渐变紫色按钮，弹出创建表单 |
| 首页 | 创建展览表单 | 半透明遮罩表单，填写名称、简介、封面URL、起止日期 |
| 展览详情页 | 作品画廊 | 展示审核通过的作品，支持收藏心形图标动画 |
| 展览详情页 | 提交作品按钮 | 右上角毛玻璃按钮，点击打开侧边滑入表单面板（0.3s缓出） |
| 展览详情页 | 作品提交表单 | 填写作品名称、描述、图片URL、联系方式，含字段校验 |

| 展览详情页 | 热度仪表盘 | 横向柱状图（每日新增收藏/评论趋势，CSS绘制渐变柱子），总访问量和总收藏量及对比箭头 |
| 我的收藏页 | 收藏作品列表 | 展示已收藏作品卡片，支持取消收藏 |

## 3. 核心流程

### 3.1 展览创建流程

策展人在首页点击右下角浮动按钮 → 弹出创建展览表单（半透明遮罩） → 填写展览信息 → 提交后展览卡片出现在首页网格 → 按开始日期升序排列

### 3.2 作品提交与审核流程

艺术家/观众进入展览详情页 → 点击右上角提交作品按钮 → 侧边滑入表单面板 → 填写作品信息并提交 → 作品进入待审核列表 → 策展人审核（通过/退回） → 通过则作品公开在画廊，退回则删除并显示退回原因

### 3.3 收藏与评价流程

观众浏览作品 → 点击心形图标收藏（缩放动画0.8→1.0） → 作品卡片显示收藏数（数字动画增长） → 可发表评论和星级评分

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深紫 #4A154B（标题、边框）、渐变紫 #7B2D8E → #A23BCC（按钮、链接）
- **背景色**：#F8F6FA（主背景）
- **文本色**：#2C3E50（深灰主文本）
- **卡片**：12px圆角、2px实线边框、hover向上浮起6px+阴影加深
- **字体**：标题使用 Playfair Display（优雅艺术感），正文使用 DM Sans（现代清晰）
- **图标**：lucide-react 图标库
- **布局**：全屏响应式，桌面端三栏网格，平板（≤1024px）两栏，手机（≤640px）单栏+汉堡菜单

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 毛玻璃效果（backdrop-blur 8px），Logo + 导航链接，移动端汉堡菜单（全屏遮罩淡入0.3s） |
| 首页 | 展览网格 | 三栏响应式网格，卡片hover浮起+阴影动画，已结束卡片半透明+灰色标签12px |
| 首页 | 创建按钮 | 圆形悬浮右下角，渐变紫#7B2D8E→#A23BCC |
| 首页 | 创建表单 | 半透明遮罩+居中表单，输入框聚焦紫色发光 |
| 展览详情页 | 作品画廊 | 网格展示作品卡片，图片懒加载+占位渐变，心形收藏图标（红色填充/灰色描边） |
| 展览详情页 | 提交面板 | 从右侧滑入0.3s缓出，毛玻璃按钮背景模糊6px |
| 展览详情页 | 星级评分 | 1-5星星形按钮，逐个高亮填充金色 |
| 展览详情页 | 热度仪表盘 | CSS横向柱状图，渐变#7B2D8E→#A23BCC，悬停显示数字；大字统计+绿色/红色对比箭头 |
| 全局 | Toast通知 | 右上角半透明深紫(#4A154B, 0.9)，右侧滑入3s渐隐 |
| 全局 | 路由切换 | 旧页面向左滑出，新页面向右滑入，0.4s ease-in-out |
| 全局 | 图片加载 | 占位符渐变（灰色→浅紫），加载完成fade-in 0.5s |

### 4.3 响应式设计

- 桌面端（>1024px）：三栏网格布局
- 平板端（≤1024px）：两栏网格布局
- 手机端（≤640px）：单栏布局，导航栏变为汉堡菜单，全屏遮罩式菜单淡入0.3s
- 卡片和表单在各断点下自适应宽度

### 4.4 动画与交互

- 卡片hover：向上浮起6px + box-shadow从`0 2px 8px 淡紫`过渡到`0 8px 24px 深紫`
- 收藏心形：缩放0.8→1.0弹性动画，收藏数数字动画增长
- 星级评分：逐个高亮填充金色
- 排序切换：卡片淡入动画（opacity 0.4s）
- 路由切换：左滑出+右滑入（0.4s ease-in-out）
- Toast：右侧滑入3s后渐隐
- 输入框聚焦：紫色边框+发光动画（box-shadow 0 0 8px #A23BCC）
- 图片加载：占位渐变→fade-in 0.5s

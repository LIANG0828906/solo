## 1. 产品概述

个人知识卡片管理器是一款帮助用户将零散知识点写成卡片、通过标签和双向链接自由组织、形成个人知识图谱的Web应用。目标用户为知识工作者、学生和终身学习者，核心价值在于让碎片化知识得以结构化沉淀与关联发现。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册（本地单用户） | 创建、编辑、删除卡片；管理标签；浏览知识图谱 |

### 2.2 功能模块

1. **主界面**：左侧标签栏 + 右侧卡片列表区域，顶部操作栏（新建卡片）
2. **卡片编辑器**：Markdown编辑区 + 实时预览区 + 标签输入 + 保存按钮
3. **卡片详情页**：卡片内容展示 + 底部双向链接列表

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主界面 | 标签侧边栏 | 展示所有标签（彩色圆点+名称+数量），点击标签筛选卡片，带淡入动画刷新 |
| 主界面 | 卡片列表 | 三栏响应式网格布局展示卡片预览，悬停上浮+阴影加深，点击进入详情 |
| 主界面 | 操作栏 | 新建卡片按钮，搜索框（可选） |
| 卡片编辑器 | Markdown编辑区 | 左侧输入Markdown文本，支持常用Markdown语法 |
| 卡片编辑器 | 实时预览区 | 右侧实时渲染Markdown预览 |
| 卡片编辑器 | 标签输入 | 支持添加最多5个标签，圆角矩形pill样式，不同标签颜色不同 |
| 卡片编辑器 | 保存按钮 | 保存卡片至localStorage，延迟<50ms |
| 卡片详情页 | 卡片内容 | 渲染后的Markdown内容展示 |
| 卡片详情页 | 双向链接列表 | 底部展示与当前卡片标题有相互引用关系的其他卡片链接，点击跳转0.3s fade过渡 |

## 3. 核心流程

1. **创建卡片流程**：用户点击"新建卡片" → 进入卡片编辑器 → 输入Markdown内容（实时预览） → 添加标签 → 保存到localStorage
2. **浏览卡片流程**：用户在主界面查看卡片列表 → 点击标签筛选 → 点击卡片进入详情页
3. **知识关联流程**：用户在详情页底部看到双向链接 → 点击跳转到关联卡片（0.3s fade过渡）

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖米色（#F5F0E8）背景 + 深橄榄绿（#3D4F3C）文字与强调色
- **辅助色**：柔和棕色（#8B7355）、淡金色（#C9B896）
- **按钮样式**：圆角按钮，深橄榄绿底色配白色文字，悬停时轻微亮度变化
- **字体**：正文使用系统无衬线字体栈，标题使用 Georgia 或类似衬线字体增添书卷气
- **布局风格**：左侧固定标签栏（240px宽） + 右侧自适应内容区，卡片三栏响应式网格
- **卡片样式**：白色卡片底、圆角12px、16px间距，悬停上浮2px + box-shadow加深
- **标签样式**：圆角矩形pill形状（border-radius: 999px），不同标签配不同颜色圆点

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主界面 | 标签侧边栏 | 固定左侧240px，暖米色略深背景，彩色圆点+标签名+数量，点击高亮 |
| 主界面 | 卡片列表 | 三栏CSS Grid（gap: 16px），手机端单栏，每张卡片：标题、内容摘要、标签pills |
| 主界面 | 操作栏 | 顶部固定，深橄榄绿背景，白色"新建卡片"按钮 |
| 卡片编辑器 | 编辑/预览区 | 左右分栏，左侧深色输入区，右侧白底预览区 |
| 卡片编辑器 | 标签区 | pill形状标签，X删除按钮，添加标签输入框 |
| 卡片详情页 | 内容区 | 居中宽阅读区，渲染后Markdown |
| 卡片详情页 | 双向链接 | 底部卡片列表，关联卡片标题可点击，0.3s fade过渡 |

### 4.3 响应式设计

- 桌面端（≥1024px）：三栏卡片网格 + 左侧标签栏
- 平板端（768px-1023px）：两栏卡片网格 + 可折叠标签栏
- 手机端（<768px）：单栏卡片列表 + 底部标签栏/汉堡菜单

### 4.4 动画效果

- 标签筛选：卡片列表淡入动画（opacity 0→1, 0.2s ease）
- 卡片悬停：transform translateY(-2px) + box-shadow加深（0.2s ease）
- 页面切换（详情页进出）：opacity 0→1（0.3s ease）
- 新建卡片弹窗：scale 0.95→1 + opacity 0→1（0.2s ease）

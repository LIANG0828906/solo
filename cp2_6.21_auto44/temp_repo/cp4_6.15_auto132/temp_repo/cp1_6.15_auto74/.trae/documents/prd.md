## 1. 产品概述

街头风味美食导航工具——让用户在交互式地图上发现周边隐藏的街头小吃摊和特色小店，获取摊点评分与排队时长预估，帮助用户快速找到心仪的街头美味。

- 核心目标：解决街头小吃"难发现、不知道好不好吃、不知道要等多久"三大痛点
- 目标用户：城市探索者、美食爱好者、夜市逛吃人群

## 2. 核心功能

### 2.1 功能模块

1. **地图主界面**：满屏交互式地图，展示小吃摊图标，支持拖拽缩放
2. **信息卡片**：点击图标弹出摊点详情卡片
3. **筛选面板**：左侧抽屉式筛选面板，支持类别/价格/排队时长过滤
4. **定位功能**：浏览器定位，定位到用户当前位置

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 地图主页 | 全屏地图 | Leaflet地图 + OSM瓦片，深灰到黑色渐变底图色调 |
| 地图主页 | 摊点标记 | 数十个emoji图标标记，按类别着色，缩放时平滑缩放 |
| 地图主页 | 信息卡片 | 弹出式卡片，包含摊点名称/评分星级/排队时长/食物缩略图 |
| 地图主页 | 筛选面板 | 左侧抽屉，毛玻璃背景，类别多选/价格滑块/排队时长分组开关 |
| 地图主页 | 定位按钮 | 右上角蓝色渐变圆形按钮，定位后显示脉冲光晕 |
| 地图主页 | 提示条 | 定位失败时从顶部滑入的红色提示条 |

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：暖色调街头夜市风格，热烈而温暖
- **主色调**：暖黄(#FFB800)、橙色(#FF6B35)、深红(#E63946)
- **辅助色**：深灰(#1A1A2E)到黑色(#0F0F0F)渐变（地图背景）
- **卡片风格**：圆角12px，微弱投影，悬停上浮-2px
- **字体**：正文用系统字体，标题用特色字体
- **布局**：全屏地图为主，叠加UI覆盖层
- **图标**：食物类别用彩色emoji（🔥烤串、🧋奶茶、🥞煎饼、🍬糖葫芦等）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 地图主页 | 全屏地图 | 深色瓦片底图，全屏铺满，平滑拖拽 |
| 地图主页 | 摊点标记 | 彩色emoji图标，缩放时0.3s缓动缩放(80%-120%) |
| 地图主页 | 信息卡片 | 底部弹出0.4s弹性动画，圆角12px，暖色渐变边框 |
| 地图主页 | 评分星级 | 1-5星半星显示，悬停填充，黄色星形 |
| 地图主页 | 排队时长 | 绿/黄/红彩色小条，对应短/中/长等待 |
| 地图主页 | 筛选面板 | 左侧25%宽滑入，毛玻璃10px模糊，半透明白色背景 |
| 地图主页 | 类别标签 | 选中为圆形色块，未选中为浅灰圆角矩形 |
| 地图主页 | 价格滑块 | 双头滑块5-50元，绿到橙渐变 |
| 地图主页 | 排队时长开关 | 三个可折叠分组，0.2s折叠动画 |
| 地图主页 | 定位按钮 | 右上角蓝色渐变圆形，白色箭头图标 |
| 地图主页 | 脉冲光晕 | 蓝色0.5s呼吸动画，向内收缩再向外扩散 |
| 地图主页 | 提示条 | 红色背景白色文字，0.3s顶部滑入，3s后滑出 |

### 4.3 响应式设计

- 桌面端：筛选面板占25%宽度，卡片居中弹出
- 移动端：卡片撑满宽度，筛选面板全屏滑入
- 地图触控手势支持：双指缩放、单指拖拽

### 4.4 动画规范

- 标记缩放：0.3s缓动，低缩放80%，高缩放120%
- 卡片弹出：0.4s弹性动画
- 筛选面板：滑入/滑出动画
- 标记更新：0.3s淡入淡出，新增从中心放大出现，移除向中心缩小消失
- 悬停效果：translateY -2px，0.2s过渡
- 折叠动画：0.2s
- 提示条：0.3s滑入，3s后滑出
- 脉冲光晕：0.5s呼吸循环

## 5. 性能要求

- 地图拖拽和标记动画保持50FPS以上
- 使用CSS transform和opacity进行动画（GPU加速）
- 减少DOM操作，批量更新标记

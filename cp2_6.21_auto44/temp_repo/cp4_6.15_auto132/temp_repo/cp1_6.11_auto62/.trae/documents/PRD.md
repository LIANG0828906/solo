## 1. 产品概述

手绘草图转3D几何体Web应用，帮助设计师在概念设计阶段快速将2D手绘灵感转化为可交互的3D模型，大幅提升创意迭代效率。

- 核心价值：将2D手绘草图实时识别并转化为可旋转、可编辑的3D几何体
- 目标用户：工业设计师、产品设计师、3D艺术家、创意工作者
- 市场价值：降低3D建模门槛，加速从概念到可视化的创意流程

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 设计师用户 | 无需注册，直接使用 | 绘制草图、生成3D模型、调节参数、导出模型 |

### 2.2 功能模块

1. **2D手绘与形状识别模块**：手绘画布、轨迹采样、形状识别算法、识别结果展示
2. **3D生成与交互模块**：Three.js 3D场景、几何体生成、材质渲染、相机控制、动画效果
3. **参数控制面板**：尺寸调节、旋转控制、颜色选择、实时预览
4. **对比画廊模块**：几何体克隆保存、缩略图展示、参数快照、快速切换
5. **导出与重置模块**：GLTF格式导出、场景一键清空

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主应用页面 | 2D绘图画布 | 鼠标/触摸绘制，亮蓝色3px线条，白色背景带网格线，实时轨迹采样 |
| 主应用页面 | 形状识别器 | RDP降采样算法，识别圆形/矩形/三角形/多边形，显示置信度，绿色虚线拟合轮廓 |
| 主应用页面 | 3D渲染场景 | Three.js场景，深蓝紫渐变背景+雾效，环境光+方向光，OrbitControls交互 |
| 主应用页面 | 参数控制面板 | 尺寸滑块(0.5-3.0)、XYZ旋转滑块(0-360°)、6色选择器、毛玻璃效果 |
| 主应用页面 | 对比画廊 | 最多8个克隆体卡片(120x120px)，缩略图+参数标签，点击高亮切换 |
| 主应用页面 | 工具栏 | 生成3D按钮、导出GLTF按钮、清空场景按钮 |

## 3. 核心流程

用户打开应用后，在2D画布上绘制草图 → 系统实时采样轨迹点 → 用户松开鼠标后自动识别形状类别 → 显示识别结果和拟合轮廓 → 点击"生成3D"按钮创建对应几何体 → 可调节参数实时更新 → 可克隆保存至对比画廊 → 可导出GLTF或清空场景

```mermaid
    D --> E["RDP降采样+形状匹配"]

    F --> G["点击生成3D按钮"]
    G --> H["Three.js生成几何体"]
    H --> I["参数面板实时调节"]
    I --> J["克隆保存至画廊"]
    J --> K["导出GLTF/清空场景"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景#1a1a2e，面板#16213e，边框#0f3460
- **强调色**：亮蓝色#00BFFF（绘制线条）、绿色虚线（拟合轮廓）、6种主题色（#FF6B6B, #4ECDC4, #45B7D1, #F9CA24, #A29BFE, #FFEAA7）
- **文字颜色**：#e0e0e0
- **按钮样式**：圆角8px，悬停变浅（过渡0.2s），点击缩放0.95（0.1s）
- **字体**：现代无衬线字体，清晰易读
- **布局风格**：四栏布局（参数面板-2D画布-3D场景-对比画廊），桌面端横向排列，移动端抽屉+堆叠
- **视觉效果**：毛玻璃面板（backdrop-filter: blur(10px)）、内阴影、渐变背景、雾效、光晕边框

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主应用 | 参数面板 | 固定宽280px，圆角8px，内阴影，毛玻璃，滑块控件，颜色按钮，操作按钮 |
| 主应用 | 2D画布 | 45%视口宽，白色背景，20px网格线（0.5px浅灰），最小400x300px |
| 主应用 | 3D场景 | 剩余宽度，深蓝紫渐变(#0a0a2e→#1a1a3e)，雾效，与画布等高 |
| 主应用 | 对比画廊 | 固定宽200px，卡片120x120px，缩略图+标签，选中发光边框 |
| 主应用 | 分隔线 | 1px #333深灰色分隔各面板 |
| 主应用 | 加载动画 | 居中60px直径旋转加载圈 |

### 4.3 响应式设计

- 桌面端（>1024px）：四栏横向布局，参数面板(280px) → 2D画布(45%) → 3D场景(剩余) → 画廊(200px)
- 移动端（≤1024px）：左侧面板和右侧画廊折叠为抽屉，汉堡图标展开/收起，画布和3D场景上下堆叠
- 触摸优化：支持触摸笔绘制，手势缩放旋转3D场景

### 4.4 3D场景指导

- **环境**：深蓝到紫色渐变背景(#0a0a2e到#1a1a3e)，添加指数雾效营造深度感
- **光照**：柔和环境光(AmbientLight 0.5强度) + 右上角方向光(DirectionalLight 0.8强度，带阴影)
- **相机**：PerspectiveCamera，fov 60，初始距离5，OrbitControls支持360°旋转，缩放范围0.5-5倍
- **构图**：几何体居中放置，地面网格辅助线，留有充足负空间
- **交互**：鼠标悬停显示白色光晕边框(0.5px)，点击弹出工具栏（删除/克隆/重置）
- **动画**：生成时从原点展开动画(1.5s ease-out)，参数调节实时更新无延迟
- **性能**：帧率保持30fps以上（6个几何体场景），响应延迟≤50ms

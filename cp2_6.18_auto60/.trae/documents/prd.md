## 1. 产品概述

为小型手工艺人打造的微型商城应用，解决无技术能力搭建电商平台、无法轻松管理商品和订单的痛点。目标用户为陶瓷、编织、木雕等手工制品的制作者和购买者，提供商品展示、在线购买、收藏、简易后台管理等一站式服务。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 浏览者 | 无需注册 | 浏览商品、搜索筛选、查看详情 |
| 购买者 | 无需注册 | 浏览、搜索、加购、结算、收藏 |
| 制作者 | 无需注册 | 后台添加/编辑/删除商品、查看订单 |

### 2.2 功能模块

1. **首页画廊**：瀑布流卡片展示所有在售商品，支持搜索与筛选
2. **商品详情页**：轮播大图、制作者信息、规格参数、加购与收藏
3. **购物车抽屉**：侧滑抽屉展示购物车商品、数量调整、结算
4. **收藏夹页**：网格视图展示已收藏商品
5. **简易后台**：商品管理（增删改）和订单记录查看

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页画廊 | 搜索栏 | 固定页头搜索栏，支持按商品名和制作者名实时搜索（300ms防抖） |

| 简易后台 | 订单记录 | 表格形式展示商品名、买家、数量、总价、下单时间 |

## 3. 核心流程

**浏览购买流程**：用户进入首页 → 浏览/搜索商品 → 点击卡片查看详情 → 加入购物车/收藏 → 打开购物车抽屉 → 调整数量 → 结算 → toast提示成功

**制作者管理流程**：进入后台 → 添加新商品（填写表单、上传图片） → 查看/编辑/删除已有商品 → 查看订单记录

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖白背景（#FFF7ED），陶土橙辅助色（#D97706），森林绿按钮色（#059669）
- **按钮风格**：统一圆角8px，过渡0.2s ease
- **字体**：系统默认无衬线，商品名称使用Noto Serif SC
- **布局风格**：卡片式布局，页面最大宽度1200px居中
- **图标风格**：Lucide线性图标
- **卡片风格**：白色背景，圆角16px，阴影0 4px 12px rgba(0,0,0,0.08)
- **动画**：卡片入场淡入上移0.3s，悬停上浮4px阴影加深，收藏心形放大回弹0.2s

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页画廊 | 固定导航栏 | 暖白背景，搜索框居中，购物车图标右侧，汉堡菜单768px以下 |
| 首页画廊 | 瀑布流卡片 | 4列布局，卡片宽280px，4:3封面图渐变占位符，价格橙色加粗 |
| 商品详情 | 轮播区 | 大图+左右箭头+圆点指示器，渐隐渐显切换 |
| 商品详情 | 信息区 | 制作者头像40px圆形，规格参数表格，加购绿色按钮 |
| 购物车抽屉 | 侧滑面板 | 宽320px，白色背景，从右滑入0.3s ease-out |
| 收藏夹 | 网格卡片 | 同首页卡片样式，带取消收藏按钮 |
| 简易后台 | 表单 | 输入框+下拉+图片上传，提交按钮绿色 |
| 简易后台 | 商品列表 | 表格展示，操作列编辑/删除按钮 |
| 简易后台 | 订单表格 | 表头：商品名、买家、数量、总价、下单时间 |

### 4.3 响应式设计

- **4列**：默认（>960px）
- **3列**：≤960px
- **2列**：≤720px
- **1列**：≤480px
- **导航栏**：768px以下折叠为汉堡菜单，从左侧滑入
- **触摸优化**：按钮最小点击区域44px，数量加减器增大触摸区域

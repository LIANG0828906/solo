## 1. 产品概述

迷你书架是一款面向社区二手书店的书籍收藏与整理应用，让用户能在虚拟书架中按喜好排列书籍，并通过拖拽交换来整理淘书清单。采用深色主题营造沉浸式阅读氛围，以流畅的拖拽交互和标签筛选系统提升书籍管理效率。

## 2. 核心功能

### 2.1 功能模块

1. **书架主页**：三列网格书架展示、搜索栏、标签筛选栏、悬浮添加按钮
2. **添加书籍弹窗**：书名/作者输入、封面颜色选择、标签选择、确认添加

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书架主页 | 搜索栏 | 圆角搜索框，实时过滤书名或作者匹配的书籍 |
| 书架主页 | 标签筛选栏 | 显示所有标签胶囊按钮，选中标签后高亮对应卡片，其余半透明 |
| 书架主页 | 书架网格 | 三列网格展示书籍卡片，支持拖拽排列，响应式适配 |
| 书架主页 | 悬浮添加按钮 | 右下角圆形按钮，点击弹出添加表单 |
| 书架主页 | 书籍卡片 | 显示封面颜色、书名、作者，支持拖拽手柄和删除按钮 |
| 添加书籍弹窗 | 表单蒙层 | 半透明黑色背景 + 模糊效果 |
| 添加书籍弹窗 | 表单内容 | 书名、作者输入框，封面颜色选择器（6个预设色块），添加按钮 |

## 3. 核心流程

用户打开应用后看到书架主页，可通过搜索栏或标签筛选快速定位书籍。长按卡片可拖拽到目标位置重新排列，松开后自动插入并平滑过渡。点击右下角悬浮按钮弹出添加表单，填写信息后添加到书架。卡片右上角X按钮可删除书籍。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝黑 #0F172A（背景）、#1E293B（书架区域）、#1E1E2E（搜索栏）
- 强调色：#4F46E5（主按钮）、#6366F1（选中标签）
- 危险色：#EF4444（删除悬停）
- 圆角风格：卡片12px、搜索栏/标签20px、弹窗16px
- 字体：书名白色16px居中，作者灰色12px居中底部
- 布局：网格卡片布局，固定顶栏，可滚动书架区域
- 阴影：卡片 0 4px 6px rgba(0,0,0,0.3)

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 书架主页 | 搜索栏 | 圆角20px，背景#1E1E2E，白色文字，占位符"搜索书名或作者" |
| 书架主页 | 标签筛选栏 | 胶囊按钮，圆角20px，默认#334155，选中#6366F1 |
| 书架主页 | 书架网格 | 三列网格，卡片200×280px，圆角12px，封面颜色背景 |
| 书架主页 | 书籍卡片 | 白色书名16px居中，灰色作者12px底部，右上X删除按钮 |
| 书架主页 | 悬浮按钮 | 直径48px，背景#4F46E5，白色加号图标 |
| 书架主页 | 拖拽占位 | 虚线边框占位框 |
| 添加书籍弹窗 | 蒙层 | 半透明黑，backdrop-filter: blur(4px) |
| 添加书籍弹窗 | 表单 | 宽400px，背景#1E293B，圆角16px，6个预设色块 |

### 4.3 响应式适配

- 桌面端（≥1024px）：三列网格
- 平板端（768px-1023px）：两列网格
- 手机端（<768px）：单列网格

### 4.4 交互动画

- 卡片悬停：上移3px + 阴影加深，transition平滑
- 拖拽交换：transition 0.3s ease
- 删除卡片：缩小消失动画
- 标签筛选：选中标签对应卡片增加2px彩色描边，其余卡片opacity 0.4
- 所有过渡动画平滑

## 1. 产品概述

音乐可视化播放器是一款基于 Web Audio API 的在线音频可视化应用，解决用户无法直观感受音频频谱和波形动态变化的问题。通过实时 FFT 频谱分析，将音频数据转化为频谱柱状图、波形曲线、圆形环绕和粒子瀑布流四种沉浸式可视化效果，并支持录制分享。

- 目标用户：音乐爱好者、音频创作者、视觉艺术爱好者
- 核心价值：将听觉体验转化为视觉盛宴，让用户"看见"音乐

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：音频可视化主区域、播放控制、参数调节、录制功能

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 可视化主区域 | Canvas 渲染四种可视化模式（频谱柱状图、波形曲线图、圆形环绕频谱图、粒子频谱瀑布流），接收实时频谱数据，60fps 动态渲染 |
| 主页面 | 音频引擎 | 加载本地 MP3/WAV/OGG 文件，Web Audio API 解析频谱（FFT），提供播放/暂停/seek/音量控制，发射频谱数据和播放状态事件 |
| 主页面 | 控制面板 | 文件选择器、播放/暂停、进度条（含缓冲区域显示）、音量滑块、模式切换按钮、自定义参数滑块（柱子数量/粒子大小/背景色/敏感度）、录制按钮 |
| 主页面 | 录制功能 | 捕获 Canvas 流+音频流为 WebM 视频，30 秒上限，红点闪烁录制指示，自动下载 |

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：深灰 `#1a1a2e`，辅以紫色 `#6c63ff` 主色调、金色 `#ffd166` 强调色
- 暗色科技感风格，频谱图和波形图带微弱发光阴影效果
- 按钮/滑块悬停缩放 `transform: scale(1.05)` + 背景色渐变
- 播放进度条和音量滑块自定义样式，滑块圆形按钮带紫色渐变光晕
- 圆角统一 8px，系统无衬线字体

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 可视化主区域 | 深色背景 Canvas、频谱柱状图（低频暖色到高频冷色渐变）、波形曲线（带发光阴影）、圆形环绕、粒子瀑布流 |
| 主页面 | 控制面板 | 左右两栏布局（左70%可视化+右30%控制面板320px固定宽度）、文件上传区域、播放/暂停按钮、进度条（浅色缓冲区域）、音量滑块、模式切换按钮组、参数滑块组、录制按钮（红点闪烁）、面板可收起/展开 |
| 主页面 | 响应式布局 | 桌面端左右两栏，移动端（<768px）上下布局、控制面板固定底部、高度可拖拽 |

### 4.3 响应式设计

- 桌面优先设计，左右两栏布局
- 移动端（宽度 < 768px）自动切换为上下布局
- 控制面板在移动端固定在底部，高度可拖拽调整
- Canvas 可视化区域自适应容器宽高

### 4.4 动画与交互

- 四种可视化模式切换时平滑过渡（淡入淡出或位移滑动）
- 录制按钮红点闪烁动画
- 按钮悬停缩放和背景渐变
- 控制面板收起/展开滑动动画
- 进度条拖拽实时响应（<50ms）

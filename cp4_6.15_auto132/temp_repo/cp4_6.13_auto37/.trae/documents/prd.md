## 1. 产品概述

中世纪炼金术主题策略游戏，玩家通过混合魔法材料合成药剂，并在回合制自动战斗中检验药剂效果。核心循环为"采集材料 → 合成药剂 → 战斗验证 → 解锁配方"，目标用户为喜欢策略组合与收集探索的休闲玩家。

## 2. 核心功能

### 2.1 功能模块
1. **合成台页面**：材料选择、合成操作、产物展示、配方图鉴
2. **战场页面**：角色对战、生命值条、战斗日志、结算弹窗
3. **库存管理**：材料分组列表、采集按钮、冷却倒计时

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 合成台 | 材料槽位 | 两个圆形水晶容器，点击从库存选择材料，显示名称/图标/稀有度星级 |
| 合成台 | 合成按钮 | 选定两种材料后点击合成，调用calculateResult计算产物 |
| 合成台 | 产物展示 | 显示产物卡片（名称/效果/稀有度/图标），未知配方显示"废弃残渣" |
| 合成台 | 配方图鉴 | 按稀有度排序的折叠卡片列表，已解锁烫金边框，未解锁暗灰+锁图标 |
| 战场 | 角色区域 | 玩家和AI敌人立绘、生命值条（渐变红绿+百分比）、药剂图标 |
| 战场 | 战斗控制 | 开始战斗按钮，调用simulateBattle模拟三回合 |
| 战场 | 战斗日志 | 逐行弹出文字日志，每行间隔800ms |
| 战场 | 结算弹窗 | 战斗结束弹出胜负结果 |
| 全局 | 库存面板 | 材料按植物/矿物/魔法精华分组折叠，数量角标 |
| 全局 | 采集按钮 | 30秒冷却，旋转炼金瓶动画，随机获得1-2种新材料 |

## 3. 核心流程

玩家进入游戏后，从库存中选择材料放入合成台槽位，点击合成按钮尝试合成药剂。合成成功获得新药剂并解锁配方，失败获得残渣和少量经验。随后装备药剂进入战场，进行三回合自动战斗验证效果。战斗结束后可继续采集新材料循环游戏。

```mermaid
flowchart TD
    "采集材料" --> "选择材料放入槽位"
    "选择材料放入槽位" --> "点击合成"

    "计算结果" --> "成功？"
    "成功？" -->|"是"| "获得药剂 + 解锁配方"
    "成功？" -->|"否"| "获得残渣 + 少量经验"
    "获得药剂 + 解锁配方" --> "装备药剂"
    "获得残渣 + 少量经验" --> "采集材料"
    "装备药剂" --> "开始战斗"
    "开始战斗" --> "三回合模拟"
    "三回合模拟" --> "战斗结算"
    "战斗结算" --> "采集材料"
```

## 4. 界面设计

### 4.1 设计风格
- 主色调：暗金(#C9A84C)、深紫(#4A1A6B)、琥珀色(#D4A017)
- 背景：深褐色羊皮纸纹理（CSS渐变+纹理叠加）
- 合成台：木质边框(box-shadow+圆角)+铆钉装饰
- 按钮：悬停上浮动画(translateY(-4px)+阴影加强)，点击缩小恢复
- 卡片：悬停平滑放大，烫金/暗灰边框区分配方状态
- 字体：MedievalSharp(标题) + Cinzel(正文)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 合成台 | 材料槽位 | 圆形水晶容器，流光动画(CSS关键帧光晕旋转) |
| 合成台 | 产物展示 | 卡片式展示，稀有度星级，效果描述 |
| 合成台 | 配方图鉴 | 折叠卡片，烫金边框/暗灰锁图标，悬停放大+"请自行探索" |
| 战场 | 角色区域 | 暗红丝绒背景，生命值渐变条，减血闪烁 |
| 战场 | 战斗日志 | 逐行弹出文字，800ms间隔 |
| 全局 | 库存面板 | 分组折叠列表，数量角标 |
| 全局 | 采集按钮 | 旋转炼金瓶动画，冷却倒计时 |

### 4.3 响应式设计
- 桌面端：左右分栏布局（合成台2/3，战场1/3）
- 移动端（<768px）：上下布局（合成台在上，战场在下），卡片等比缩小

### 4.4 动画与交互
- 合成成功：产物卡片弹出动画
- 战斗日志：逐行淡入弹出
- 采集：旋转炼金瓶渐变色动画
- 生命值条：减血闪烁+平滑过渡
- 所有交互元素：悬停上浮、点击缩放

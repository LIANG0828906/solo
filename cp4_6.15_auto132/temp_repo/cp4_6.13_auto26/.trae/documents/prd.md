## 1. 产品概述

互动式学习卡牌游戏在线平台，帮助教育工作者将传统课堂知识点转化为趣味卡牌对战游戏，解决学生参与度低、记忆枯燥的问题。教师创建卡牌游戏并实时监控，学生通过游戏码加入进行知识竞答，系统自动生成战绩报告。

- 目标用户：中小学教师及学生（课堂场景）
- 核心价值：将被动学习变为主动竞技，通过游戏化机制提升课堂互动和知识掌握度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 教师 | 账号密码登录 | 创建/管理卡牌游戏、控制游戏流程、查看实时数据统计、导出报告 |
| 学生 | 输入游戏码加入 | 加入游戏房间、选择虚拟形象、打出手牌、查看个人战绩 |

### 2.2 功能模块

1. **教师工作台**：游戏创建与管理、卡牌编辑、出牌规则设定、游戏控制台
2. **游戏房间**：学生加入与准备、实时对局、积分排行、通知横幅
3. **战绩报告**：个人/全班成绩统计、卡牌对错标识、排行榜、PDF导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 教师登录表单 | 账号密码输入，登录后跳转教师工作台 |
| 教师工作台 | 游戏列表 | 显示已创建的游戏，支持新建/编辑/删除 |

| 教师工作台 | 规则设置 | 设置问答限时、每回合抽牌上限、胜利条件 |
| 教师工作台 | 游戏控制台 | 实时查看学生作答结果和正确率，控制游戏开始/结束 |
| 加入游戏页 | 游戏码输入 | 学生输入游戏码或扫描二维码加入房间 |

## 3. 核心流程

### 3.1 教师创建游戏流程
教师登录 → 点击新建游戏 → 设定主题 → 添加卡牌（知识卡/道具卡，≥10张）→ 设置出牌规则和胜利条件 → 生成游戏码 → 等待学生加入

### 3.2 学生游戏流程
输入游戏码 → 选择动物头像 → 进入房间等待 → 游戏开始 → 查看手牌 → 点击翻转卡牌 → 打出卡牌 → 系统判定得分 → 回合结束抽新牌 → 达成胜利条件或游戏结束 → 查看战绩报告

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：渐变蓝紫（#667eea → #764ba2），白色和浅灰色辅助底色
- **按钮样式**：圆角矩形（border-radius: 12px），悬停缩放（transform: scale(1.05)），平滑过渡（transition: all 0.3s ease）
- **字体**：标题使用 Playfair Display，正文使用 Noto Sans SC
- **布局风格**：桌面端左右分栏（70%/30%），移动端上下堆叠，响应式设计
- **特效风格**：玻璃拟态（backdrop-filter: blur(10px)，半透明背景），3D卡牌翻转，淡入淡出列表过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 渐变背景居中卡片，玻璃拟态登录框，圆角输入框，渐变按钮 |
| 教师工作台 | 游戏列表 | 左侧游戏卡片网格，右侧快捷操作面板，顶部导航栏 |
| 教师工作台 | 卡牌编辑器 | 卡牌列表+编辑面板双栏，Markdown预览区，类型标签切换 |
| 教师工作台 | 规则设置 | 表单布局，数字输入+滑块，实时预览规则摘要 |
| 教师工作台 | 游戏控制台 | 实时数据面板，学生作答状态表格，正确率柱状图 |
| 加入游戏页 | 游戏码输入 | 居中大卡片，游戏码输入框+加入按钮，二维码扫描区 |

| 游戏房间 | 通知横幅 | 顶部浮层，浅绿背景深绿文字，3秒淡出 |
| 战绩报告 | 个人报告 | 统计卡片网格，卡牌红绿角标列表，环形进度图 |
| 战绩报告 | 排行榜 | 前三名金银铜飘带动画，排名列表 |
| 战绩报告 | PDF导出 | 渐变导出按钮，悬停缩放效果 |

### 4.3 响应式设计

- **桌面端（≥768px）**：左右分栏布局，卡牌区域占70%，积分榜占30%
- **移动端（<768px）**：上下堆叠布局，卡牌自适应缩小为两列网格
- **触摸优化**：卡牌点击区域增大，按钮间距适当加宽

### 4.4 动画规范

- 卡牌翻转：3D翻面动画，使用 requestAnimationFrame 驱动，保持60fps
- 列表过渡：淡入淡出效果（opacity + transform）
- 通知横幅：滑入/滑出动画，3秒自动移除
- 头像动画：12种动物各自独立CSS动画（兔子耳朵抖动、猫眼眨动等）
- 排行榜前3名：金银铜飘带飘动动画
- 按钮交互：悬停缩放1.05倍，0.3s ease过渡

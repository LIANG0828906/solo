## 1. 产品概述

社区二手书交换平台，旨在为书友提供一个便捷的二手书籍流转空间。用户通过昵称快速登录后，可浏览、搜索和交换二手书籍，促进知识共享与社区互动。

- 目标用户：喜爱阅读、希望以书换书的社区成员
- 核心价值：降低阅读成本、促进书籍循环利用、建立书友社交连接

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 输入昵称登录（无需注册） | 浏览书籍、发起交换、查看交换记录 |

### 2.2 功能模块

1. **登录页**：昵称输入登录，模拟身份认证
2. **书籍列表页**：展示全部书籍，支持按书名/作者/分类筛选，搜索防抖300ms
3. **书籍详情页**：完整书籍信息、主人信息、交换条件、联系主人按钮
4. **交换弹窗**：填写留言与期望换得的书，确认后生成请求并更新状态
5. **交换记录页**：展示发起/收到的交换请求，状态流转，未读标记

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 登录表单 | 输入昵称提交登录，成功后跳转首页 |
| 书籍列表页 | 搜索筛选栏 | 按书名/作者搜索，按分类下拉筛选，300ms防抖 |
| 书籍列表页 | 书籍卡片网格 | 展示封面缩略图、书名、作者、城市、交换条件，悬停上浮动画 |
| 书籍详情页 | 书籍信息区 | 高清封面、详细简介、交换条件 |
| 书籍详情页 | 主人信息区 | 昵称、信用等级、已完成交换数 |
| 书籍详情页 | 操作按钮 | 联系主人/交换按钮，触发交换弹窗 |
| 交换弹窗 | 双方信息 | 当前用户与书籍主人昵称展示 |
| 交换弹窗 | 表单 | 留言输入、期望换得的书（可选） |
| 交换弹窗 | 操作 | 确认/取消按钮，按压波纹反馈 |
| 交换记录页 | 记录列表 | 按时间倒序，封面/书名/对方昵称/状态，未读红点 |
| 交换记录页 | 状态操作 | 待确认→已接受/已拒绝，已接受→已完成 |

## 3. 核心流程

用户打开应用 → 未登录自动跳转登录页 → 输入昵称登录 → 进入书籍列表页 → 浏览/筛选书籍 → 点击卡片进入详情 → 点击交换/联系主人 → 弹出交换弹窗 → 填写留言确认 → 书籍状态变更为"交换中" → 交换记录页查看请求 → 对方确认/拒绝/完成

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暖木色（#8B6914）搭配深绿色（#2D5016），营造书本与自然联想
- 辅助色：米白色背景（#FBF7F0）、深棕文字（#3E2723）、浅绿点缀（#A5D6A7）
- 按钮风格：圆角8px，暖木色填充，按压波纹反馈
- 字体：标题使用衬线体风格，正文使用无衬线体
- 布局：固定顶部导航栏，卡片式网格布局，响应式适配
- 卡片：圆角12px阴影卡片，悬停上浮+阴影加深动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 居中卡片，暖木色标题，输入框柔和边框，登录按钮 |
| 书籍列表页 | 导航栏 | 固定顶部，深绿背景，用户昵称+登出按钮，移动端汉堡菜单 |
| 书籍列表页 | 筛选区 | 搜索框+分类下拉，柔和边框淡入效果，移动端折叠 |
| 书籍列表页 | 卡片网格 | 圆角阴影卡片，封面缩略图，书名/作者/城市/条件标签 |
| 书籍详情页 | 信息区 | 左侧高清封面，右侧详情，主人信息卡片 |
| 交换弹窗 | 弹窗体 | 居中遮罩层，双方昵称，留言输入框，确认/取消按钮 |
| 交换记录页 | 记录列表 | 时间倒序卡片，未读红点标记，状态标签可操作 |

### 4.3 响应式

- 桌面端（≥768px）：多列卡片网格，筛选区横向排列
- 移动端（<768px）：单列卡片，筛选区折叠为汉堡菜单，导航栏简化

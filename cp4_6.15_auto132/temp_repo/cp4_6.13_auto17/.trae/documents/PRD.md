## 1. 产品概述

冰箱食材管家是一款面向家庭用户的本地食材管理与菜谱推荐应用，帮助用户减少食物浪费、简化日常烹饪决策。用户可以管理冰箱食材库存、根据库存智能推荐菜谱、规划本周三餐，并自动生成购物清单。

- 核心价值：减少食物浪费，解决"今天吃什么"的决策难题
- 目标用户：家庭主厨、独居青年、注重健康饮食的上班族
- 产品定位：轻量级、纯本地、离线可用的食材管理工具

## 2. 核心功能

### 2.1 功能模块

1. **食材管理面板**：食材的增删改查、过期提醒、搜索筛选
2. **菜谱推荐区域**：智能匹配算法、分类展示（完全匹配/缺少1-2样）、菜谱详情
3. **本周计划面板**：7天日历视图、拖拽排餐、食材消耗统计
4. **购物清单面板**：自动生成清单、分类展示、勾选入库

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主应用 | 三栏布局 | 左侧食材管理（可折叠）、中间菜谱推荐、右侧计划/清单切换 |
| 食材管理 | 食材列表 | 按保质期升序排列、3天内过期红色高亮、搜索与分类筛选 |
| 食材管理 | 食材表单 | 模态框新增/编辑、表单验证、淡入动画 |
| 菜谱推荐 | 菜谱卡片 | 毛玻璃效果、悬停上移、点击展开详情、匹配状态标识 |
| 本周计划 | 日历网格 | 7天×3餐格子、拖拽放置、弹性动画、食材消耗提示 |
| 购物清单 | 清单列表 | 按超市分区分类、勾选已购买、自动入库 |

## 3. 核心流程

### 3.1 主要用户流程

1. 用户打开应用 → 加载本地IndexedDB数据 → 展示食材库存和推荐菜谱
2. 用户添加/编辑食材 → 表单验证 → 保存到IndexedDB → 实时更新菜谱推荐
3. 用户点击"智能推荐" → 基于库存匹配菜谱 → 分类展示结果
4. 用户拖拽菜谱到日历 → 更新本周计划 → 重新计算食材消耗与购物清单
5. 用户勾选购物清单项 → 自动增加对应食材库存 → 更新所有视图

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#4CAF50（清新绿）
- **辅助色**：#81C784（浅绿）
- **背景色**：#F1F8E9（淡绿米白）
- **警示色**：#F44336（过期提醒红色）
- **卡片风格**：圆角16px、毛玻璃效果（backdrop-filter: blur(8px)）
- **字体**：系统字体栈，标题加粗，正文清晰易读
- **图标风格**：线性图标（lucide-react），与主题色一致

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主布局 | 三栏结构 | 左侧可折叠面板、中间内容区、右侧切换面板、顶部导航栏 |
| 食材列表 | 食材卡片 | 名称+类别标签、数量单位、保质期徽章、存放位置图标、过期红色高亮 |
| 菜谱卡片 | 推荐卡片 | 菜名、食材清单（打勾/缺失）、烹饪时间、难度等级、悬停上移动效 |
| 日历视图 | 餐次格子 | 日期标题、三餐格子、拖拽放置区、半透明幽灵元素、弹性缩放动画 |
| 购物清单 | 分类列表 | 超市分区标题、复选框、数量显示、已购买删除线效果 |

### 4.3 响应式设计

- **桌面端（>768px）**：三栏布局，左右面板固定宽度，中间自适应
- **平板端（≤768px）**：两栏布局，左侧面板折叠为图标栏，右侧与中间合并切换
- **触控优化**：增大点击区域，支持触摸拖拽

### 4.4 动效设计

- 模态框淡入：0.3s ease-out
- 卡片悬停：上移5px + 阴影加深，0.2s过渡
- 拖拽放置：弹性缩放动画（scale 0.9 → 1.05 → 1）
- 列表更新：平滑高度过渡
- 过期提醒：脉冲呼吸动画

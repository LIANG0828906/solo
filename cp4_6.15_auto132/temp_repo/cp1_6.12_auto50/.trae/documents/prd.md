## 1. 产品概述
多轨音频协作编排应用，为小型乐队和音乐爱好者提供轻量级在线协作平台，支持创建项目、添加多类音轨、录制音频片段、拖拽排序、实时回放和合成导出WAV文件。

- 核心目标：让多人实时协作编排简单歌曲片段，降低音乐创作门槛
- 目标用户：小型乐队、独立音乐人、音乐爱好者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 协作用户 | 直接访问 | 创建项目、录制音频、拖拽编排、回放、导出 |

### 2.2 功能模块
1. **项目页面**：项目创建、项目列表、项目名称输入
2. **编排工作台页面**：音轨列表、时间轴视图、录音控制、播放控制、导出功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 项目页面 | 项目创建 | 输入项目名称，自动生成包含鼓/贝斯/吉他/人声四轨的空项目 |
| 编排工作台 | 音轨面板（左侧） | 显示音轨列表，每轨含录音/静音/独享/删除按钮，支持拖拽重排序 |
| 编排工作台 | 时间轴视图（右侧） | 顶部时间刻度，片段按时间排列，支持拖拽对齐1/16拍网格 |
| 编排工作台 | 录音模块 | 点击录音按钮通过麦克风录制，实时波形显示，停止后片段追加到音轨末尾 |
| 编排工作台 | 播放控制 | 播放/暂停按钮，播放头同步移动，所有音轨同步播放 |
| 编排工作台 | 导出模块 | 合成所有音轨为WAV，显示进度条和预计剩余时间，完成后自动下载 |
| 编排工作台 | 实时协作 | WebSocket广播片段拖动、录音事件给同项目其他用户 |

## 3. 核心流程

### 3.1 项目创建流程
用户输入项目名称 → 系统创建项目并生成四条默认音轨 → 跳转至编排工作台

### 3.2 录音编排流程
选择音轨 → 点击录音按钮 → 麦克风采集音频 → 实时波形显示 → 停止录音 → 片段追加到音轨末尾 → 拖拽调整位置 → 对齐1/16拍网格

### 3.3 回放导出流程
点击播放 → 所有音轨同步播放 → 播放头匀速移动 → 暂停/恢复 → 点击导出 → 离线混音 → 进度条显示 → WAV下载

## 4. 用户界面设计

### 4.1 设计风格
- 主题：深色科技感扁平风
- 主色调：背景 #1a1a2e，卡片 #16213e
- 辅色：鼓红色(#e74c3c)、贝斯蓝色(#3498db)、吉他绿色(#2ecc71)、人声紫色(#9b59b6)
- 文字颜色：浅灰 #e0e0e0
- 按钮渐变：#0f3460 到 #533483
- 圆角：8px 全局统一
- 间距：音轨间隔 12px
- 阴影：0 4px 15px rgba(0,0,0,0.3)
- 图标：lucide-react 统一图标系列
- 动画时长：0.3s / 0.5s 统一

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 项目页面 | 项目创建表单 | 居中卡片，输入框+创建按钮，渐变背景 |
| 编排工作台 | 音轨面板 | 左侧固定宽度，音轨卡片列表，每卡含图标/颜色/操作按钮 |
| 编排工作台 | 时间轴视图 | 右侧自适应，顶部时间刻度线，片段彩色波形块，播放头竖线 |
| 编排工作台 | 录音按钮 | 圆形按钮，录音时红色脉冲动画，波纹扩散效果 |
| 编排工作台 | 片段块 | 半透明彩色波形，宽度对应时长，拖动弹性缓动 |
| 编排工作台 | 播放控制栏 | 底部固定栏，播放/暂停/停止/导出按钮 |
| 编排工作台 | 导出进度 | 模态对话框，进度条0%-100%渐变填充，预计剩余时间 |

### 4.3 响应式适配
- 桌面优先设计（≥768px）：左右两栏布局
- 移动端（<768px）：左侧面板折叠为可弹出侧边栏，时间轴自适应宽度，按钮图标化+触控区域≥44px

### 4.4 动画细节
- 录音按钮：红色脉冲动画 + 波纹扩散
- 播放头：线性插值匀速移动
- 片段拖拽：弹性缓动(ease-out)，释放时自动对齐
- 重叠警告：红色边框动画持续0.3秒
- 音轨拖拽排序：阴影加深，松开后平滑移动
- 播放高亮：播放头经过片段时高亮显示

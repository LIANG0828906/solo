## 1. 产品概述
西园雅集·笔墨丹青是一款互动式水墨画鉴赏工具，以宋代文人雅集为主题，让用户以文人视角在园林中漫步，通过点击不同场景区域触发琴、棋、书、画等雅集活动的动画与文字解说。

- **核心价值**：将传统文化与现代交互技术结合，为用户提供沉浸式的宋代文人生活体验
- **目标用户**：传统文化爱好者、艺术鉴赏者、学生群体
- **市场定位**：文化教育类互动应用，可用于博物馆导览、传统文化普及

## 2. 核心 Features

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览园林地图、体验雅集活动、查看文字解说 |

### 2.2 功能模块
1. **园林地图页面**：水墨风格园林鸟瞰图，四个可交互区域（竹林、曲水、石桌、亭台）
2. **活动详情页面**：活动动画展示、古文解说、逐字字幕、毛笔书写效果
3. **全局交互系统**：水墨涟漪动画、卷轴展开动画、背景色渐变切换
4. **水墨粒子系统**：模拟墨色在宣纸上晕染的粒子动画效果

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 园林地图 | 鸟瞰图展示 | SVG绘制水墨风格园林，包含竹林、曲水、石桌、亭台区域 |
| 园林地图 | 区域交互 | 点击区域触发水墨涟漪动画，跳转至对应活动 |
| 活动详情 | 主题背景 | 根据活动类型渐变切换主题色（抚琴暖棕、对弈墨黑、挥毫纸白、赏画朱红） |
| 活动详情 | 古文解说 | 随机选取约100字的古文描述展示 |
| 活动详情 | 逐字动画 | 文字逐字显示，速度可调，毛笔跟随模拟书写效果 |
| 活动详情 | 卷轴动效 | 卡片以卷轴展开方式入场，退场时反方向卷起 |
| 全局 | 水墨粒子 | 背景粒子动画模拟墨色晕染，粒子数≤200 |

## 3. 核心流程
用户进入应用 → 查看水墨园林鸟瞰图 → 点击感兴趣的区域（如竹林抚琴）→ 页面平滑过渡，背景色渐变 → 卷轴展开显示活动详情 → 逐字显示古文解说，毛笔跟随移动 → 可返回地图探索其他区域

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：淡青灰 #c8d5d5（背景）、细金线 #d4a843（边框/按钮）
- **活动主题色**：暖棕 #b8956a（抚琴）、墨黑 #333333（对弈）、纸白 #f5f0e0（挥毫）、朱红 #bf4342（赏画）
- **字体**：Google Fonts - Noto Serif SC（书法体）
- **视觉风格**：宋代山水画风格，绢本质感背景（CSS纹理叠加）
- **按钮风格**：细金线勾勒，圆角设计，悬停时轻微发光
- **整体氛围**：清雅、古朴、文人气息

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 园林地图 | 鸟瞰图 | SVG绘制的水墨园林，可点击区域带有半透明高亮效果，悬停时显示活动名称 |
| 园林地图 | 可交互区域 | 点击时产生水墨涟漪扩散动画 |
| 活动详情 | 背景 | 绢本质感纹理，活动主题色渐变填充 |
| 活动详情 | 卷轴卡片 | 由中心向两侧展开的卷轴动效，细金边框 |
| 活动详情 | 古文解说 | 约100字古文，书法字体，逐字显示 |
| 活动详情 | 毛笔动画 | 毛笔图标跟随文字显示进度移动，模拟书写 |
| 活动详情 | 速度控制 | 字幕速度调节滑块，返回按钮 |

### 4.3 响应式设计
- **桌面端（≥768px）**：横向布局，左侧园林地图，右侧活动卡片并列展示
- **移动端（<768px）**：纵向堆叠布局，地图缩小至屏幕宽度，活动卡片全屏展示
- **触控优化**：可点击区域最小48×48px，适配手指触控

### 4.4 动画与交互细节
- **水墨涟漪**：点击区域时，从点击位置向外扩散3-4圈半透明墨色圆环
- **卷轴展开**：入场时卡片高度从0向上下同时展开，退出时反向收起
- **毛笔跟随**：毛笔图标位于当前显示文字的右侧，带有轻微的上下浮动效果
- **背景渐变**：切换活动时，背景色在1.5秒内平滑过渡到新主题色

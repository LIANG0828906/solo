## 1. 产品概述

创意工坊（CreativeForge）是一款面向个人创作者和小团队的轻量级创意项目管理工具，解决灵感零散不易收集、创作计划缺乏结构、进度全凭脑记容易遗漏三大痛点。

- 核心目标：将"灵感闪现→项目规划→执行追踪→复盘沉淀"的完整创作流程数字化、可视化
- 目标用户：独立写作者、设计师、音乐人、视频创作者、小型创意团队

## 2. 核心功能

### 2.1 用户角色

无需角色区分，单用户本地使用模式。

### 2.2 功能模块

1. **灵感看板页面**：浮动灵感输入框、灵感卡片网格、分类/标签过滤、灵感转项目入口
2. **项目详情页面**：项目基本信息、任务拆分与管理、甘特图视图（可拖拽）、依赖关系编辑
3. **复盘报告页面**：项目复盘统计、耗时分析、进度图表、Markdown导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 灵感看板 | 浮动输入框 | 页面右下角常驻FAB按钮，点击展开浮动输入面板（标题+描述），选择分类和标签后快速保存，自动打时间戳 |
| 灵感看板 | 灵感卡片网格 | 卡片展示标题、描述、分类图标、标签、创建时间，按创建时间降序排列，新卡片底部浮入动画 |
| 灵感看板 | 过滤器 | 顶部分类标签栏（文字/图像/音乐/视频/其他）+ 标签搜索框，联合过滤 |
| 灵感看板 | 灵感转项目 | 卡片操作菜单中"转化为项目"，自动填充项目标题和描述 |
| 项目详情 | 项目信息卡 | 显示/编辑项目标题、目标描述、起止日期、优先级 |
| 项目详情 | 任务列表 | 项目下的分阶段任务列表，支持增删改，每项包含名称、预计工时、依赖关系 |
| 项目详情 | 甘特图 | 时间轴横向展示，任务条以柔和渐变色区分阶段，依赖关系用贝塞尔曲线+箭头连接，任务条可拖拽调整开始日期和持续天数 |
| 项目详情 | 进度时间线 | 每个任务下的进度日志列表，按时间排列，每条左侧颜色标记（绿=完成/蓝=推进/红=阻塞），支持文字+可选图片 |
| 复盘报告 | 统计概览 | 总耗时、各阶段耗时占比饼图、平均每日推进速度 |
| 复盘报告 | 导出功能 | 导出为Markdown格式文本，或导出全部数据为JSON压缩zip |

## 3. 核心流程

用户打开应用后，首先在灵感看板快速记录灵感。当灵感成熟时，将其转化为创作项目。在项目详情页中，用户拆分任务、设置依赖、通过甘特图规划时间线。执行过程中在任务下添加进度日志。项目完成后，系统自动生成复盘报告供回顾和导出。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，背景 #1a1a2e，卡片 #16213e，强调色 #e94560
- **辅助色**：分类图标色彩（文字=#8be9fd、图像=#ffb86c、音乐=#bd93f9、视频=#50fa7b、其他=#f1fa8c）
- **毛玻璃效果**：卡片使用 `backdrop-filter: blur(12px)` + 半透明背景 + `box-shadow` 悬浮阴影
- **按钮风格**：圆角胶囊按钮，点击时弹性缩放 `scale(0.95)→scale(1.05)` 过渡 0.2s
- **字体**：标题使用 Outfit（粗体有力），正文使用 Noto Sans SC（中文友好）
- **布局风格**：左侧导航栏 + 右侧内容区，灵感卡片网格布局

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 灵感看板 | 浮动输入框 | FAB按钮（强调色圆形），展开后为毛玻璃面板，包含标题输入、描述文本区、分类选择器、标签输入、提交按钮 |
| 灵感看板 | 灵感卡片网格 | 三列网格，每张卡片毛玻璃背景，左上分类色条，标题加粗，描述省略两行，底部标签药丸+时间戳，悬浮时轻微上浮+阴影加深 |
| 灵感看板 | 过滤器 | 顶部水平分类标签栏（带图标），下方标签搜索输入框 |
| 项目详情 | 甘特图 | 横向时间轴（日/周切换），任务行高度40px，任务条渐变色圆角矩形，依赖线为贝塞尔曲线+箭头，拖拽手柄在任务条两端 |
| 项目详情 | 进度时间线 | 左侧竖线连接各日志节点，节点为颜色圆点（绿/蓝/红），右侧显示时间+内容+可选缩略图 |
| 复盘报告 | 统计面板 | 顶部三个统计卡片（总耗时/日均速度/完成率），中部饼图，底部阶段明细表 |

### 4.3 响应式设计

- 桌面端（≥1024px）：侧边导航常驻，内容区最大宽度 1200px 居中
- iPad横屏（768-1023px）：侧边导航折叠为图标模式，任务面板折叠为可展开抽屉
- 窄屏适配：甘特图区域横向可滚动，卡片网格从三列变为两列

### 4.4 动效规范

- 灵感卡片新增：从底部浮入 + 淡入，duration 0.4s ease-out
- 按钮/卡片点击：`scale(0.95)→scale(1.05)→scale(1)`，总过渡 0.2s
- 页面切换：淡入淡出，duration 0.3s
- 甘特图拖拽：实时跟随鼠标，无延迟感

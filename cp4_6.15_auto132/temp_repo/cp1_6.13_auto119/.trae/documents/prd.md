# QuantumClash 量子对战游戏 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
QuantumClash 是一款基于量子态叠加和测量原理的回合制策略对战游戏，将量子力学概念转化为直观的游戏玩法，让玩家在策略对战中体验量子世界的奇妙特性。

### 1.2 核心玩法
- 两位玩家轮流在 5x5 网格上操作
- 每回合可放置一个量子粒子（红/黄/蓝三色代表不同能级）
- 粒子默认处于叠加态，同时占据相邻两个格子
- 玩家选择"测量"时粒子坍缩到其中一个格子并产生连锁反应
- 同色粒子被冲击波击中会湮灭（造成伤害）
- 不同色粒子被击中会纠缠（改变颜色）

### 1.3 目标用户
- 喜欢策略对战游戏的玩家
- 对量子物理概念感兴趣的用户
- 追求新颖游戏机制的玩家

## 2. 功能需求

### 2.1 量子场模块
- 5x5 网格管理
- 粒子叠加态数据结构
- 粒子放置逻辑
- 粒子坍缩测量逻辑
- 冲击波传播与连锁反应
- 同色湮灭（伤害计算）
- 不同色纠缠（颜色变化）

### 2.2 渲染模块
- 深蓝到紫色径向渐变背景
- 网格细线边框（灰色半透明）
- 叠加态粒子（半透明双色块）
- 坍缩闪烁动画（两色交替闪烁3次，0.5秒）
- 冲击波特效（圆形扩散，半径增大至30px）
- 碎裂动画（4片碎片飞散，0.4秒）
- 颜色渐变动画（0.3秒过渡）
- 粒子浮动效果
- 生命值能量条
- 倒计时条
- 回合切换提示文字
- 伤害/纠缠结果文字

### 2.3 交互控制模块
- 放置阶段：点击空格子选择位置
- 选中格子金色描边高亮
- 底部颜色选择面板（三个圆形按钮）
- 按钮悬停放大效果
- 测量阶段：点击叠加态粒子触发坍缩
- 点击压扁弹跳动画（0.2秒）
- 伤害/纠缠结果文字显示（1.5秒渐隐）

### 2.4 回合管理模块
- 双方玩家轮换操作
- 15秒倒计时条
- 超时自动随机操作
- 回合切换提示动画
- 玩家生命值显示（左右能量条）
- 生命值分段消失动画（每段0.1秒）

## 3. 视觉设计

### 3.1 整体风格
- 量子科幻风格
- 深色调为主，辅以霓虹色彩
- 神秘而富有科技感

### 3.2 配色方案
- 背景：深蓝到紫色径向渐变
- 粒子颜色：红色、黄色、蓝色
- 高亮：金色
- UI：半透明暗色面板

### 3.3 动画风格
- 流畅的过渡动画
- 量子感的闪烁与波动效果
- 富有冲击力的粒子特效

## 4. 性能要求
- 测量后场状态更新：≤ 50ms
- 粒子动画帧率：≥ 30fps
- 响应延迟：≤ 100ms

## 5. 技术约束
- 使用 TypeScript 开发
- Canvas API 进行渲染
- Vite 作为构建工具
- 代码模块化拆分

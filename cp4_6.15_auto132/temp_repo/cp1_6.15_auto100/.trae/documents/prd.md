## 1. 产品概述

为小型独立咖啡馆打造的会员点单与积分管理系统，让常客可以在线浏览菜单、下单并累积积分，店主可在后台管理菜单、查看订单和调整积分规则。解决预算有限无法使用大型SaaS的痛点，提供一个轻量、温馨的数字化运营工具。

- 目标用户：独立咖啡馆常客（会员端）、店主（后台管理端）
- 核心价值：零成本部署、操作直观、营造咖啡馆温馨氛围的线上体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册/登录方式 | 核心权限 |
|------|-------------|----------|
| 会员 | 输入昵称和手机号注册，自动生成会员卡号；手机号登录 | 浏览菜单、下单、查看积分和消费记录 |
| 店主 | 固定密码（admin）登录后台 | 管理菜单（增/编/下架）、查看所有订单、手动调整会员积分 |

### 2.2 功能模块

1. **首页（点单页）**：菜单分类展示（饮品/甜品/轻食）、商品卡片（Emoji+名称+价格）、数量选择器、购物车角标
2. **购物车面板**：侧滑面板、商品列表与数量编辑、总价计算、确认下单
3. **会员中心**：注册/登录、积分余额、消费记录
4. **后台管理**：商品管理表单、订单列表（筛选）、积分调整

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页（点单页） | 导航栏 | 固定顶部，深咖啡色，左侧☕图标+店名，右侧积分余额+购物车图标+角标 |
| 首页（点单页） | 菜单分类 | 饮品/甜品/轻食三个分类标签切换 |
| 首页（点单页） | 商品卡片 | 220px×280px，含Emoji图片、名称、价格，悬停上浮2px+阴影加深（0.2s），点击弹出数量选择器（+/-弹性动画） |
| 首页（点单页） | 积分提示 | 下单后顶部弹出1.25倍放大再回缩动画的积分入账提示 |
| 购物车面板 | 侧滑面板 | 从右侧滑入（0.3s缓出），宽320px，浅米色背景，商品列表带分割线和删除按钮（红色缩小消失0.3s） |
| 购物车面板 | 下单确认 | 展示总价，确认下单后状态变为"制作中"，积分=消费金额×10%向下取整 |
| 会员中心 | 注册/登录 | 昵称+手机号注册，自动生成会员卡号；手机号登录后显示积分和消费记录 |
| 后台管理 | 商品管理 | 新增/编辑/下架商品，表单实时校验 |
| 后台管理 | 订单列表 | 按时间倒序，可筛选全部/制作中/已完成 |
| 后台管理 | 积分调整 | 手动加/减会员积分，需填写原因，操作按钮涟漪效果（0.4s半径扩散） |

## 3. 核心流程

**会员点单流程**：会员登录 → 浏览菜单分类 → 选择商品加入购物车 → 打开购物车确认下单 → 订单状态"制作中" → 积分自动入账 → 顶部弹出积分提示

**店主后台流程**：店主密码登录 → 管理商品（增/编/下架） → 查看订单列表并筛选 → 手动调整会员积分

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖木色和咖啡色（背景 #F5E6D0，主按钮 #6F4E37，文字 #3E2723）
- **辅助色**：深咖啡色导航栏、浅米色购物车面板
- **按钮风格**：圆角按钮，主按钮咖啡色，悬停时加深
- **字体**：正文使用系统无衬线字体，标题加粗
- **布局风格**：卡片式布局，固定顶部导航栏
- **图标风格**：使用 Emoji 替代商品图片（☕️🍰🥐等），lucide-react 图标库用于UI图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 首页 | 导航栏 | 深咖啡色背景，100px高，左侧☕+店名，右侧积分+购物车图标+角标 |
| 首页 | 商品卡片 | 220px×280px，4px圆角，浅阴影，悬停上浮2px+阴影加深0.2s |
| 首页 | 数量选择器 | +/-按钮带弹性动画的弹出层 |
| 首页 | 积分提示 | 顶部1.25倍放大再回缩动画toast |
| 购物车 | 侧滑面板 | 右侧滑入0.3s缓出，320px宽，浅米色，商品列表带分割线 |
| 购物车 | 删除按钮 | 点击变红并缩小至消失0.3s |
| 后台管理 | 深色主题 | 背景#263238，卡片#37474F，表格行交替色 |
| 后台管理 | 涟漪效果 | 操作按钮0.4s半径扩散涟漪 |

### 4.3 响应式设计

- **桌面端优先**：完整布局，商品卡片网格排列
- **移动端（<768px）**：导航栏折叠为汉堡菜单，商品卡片双列流式布局，购物车侧滑面板覆盖全屏
- **触摸优化**：按钮最小44px点击区域，滑动操作支持

### 4.4 动画与交互

- 商品卡片悬停：上浮2px + 阴影加深，0.2秒过渡
- 购物车数量更新：0.2秒缩放动画
- 购物车面板：右侧滑入0.3秒缓出
- 数量选择器+/-按钮：弹性动画
- 积分入账提示：1.25倍放大再回缩动画
- 删除按钮：变红+缩小至消失0.3秒
- 后台涟漪效果：0.4秒半径扩散

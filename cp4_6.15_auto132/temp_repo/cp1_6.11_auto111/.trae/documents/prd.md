## 1. 产品概述

2D角色技能连招编辑器与回放演示器，面向游戏开发者，提供可视化编辑连招序列与实时Canvas动画回放能力，快速验证技能衔接流畅性与碰撞伤害判定。

## 2. 核心功能

### 2.1 功能模块

1. **连招编辑器**：左侧区域，管理技能序列列表，支持拖拽重排、单击编辑技能属性
2. **回放演示器**：右侧区域，Canvas动画回放角色动作与技能特效，含播放控制栏与DPS指示器
3. **连招逻辑引擎**：计算伤害、帧数、连击判定

### 2.2 页面详情

| 页面区域 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 左侧编辑器 | 技能序列列表 | 展示技能卡片列表，支持拖拽排序，最多8个技能 |
| 左侧编辑器 | 添加技能按钮 | 底部按钮，点击添加默认技能卡片 |
| 左侧编辑器 | 属性编辑面板 | 点击卡片弹出，编辑名称/伤害/帧数属性 |
| 右侧回放区 | Canvas画布 | 绘制角色精灵、技能特效、伤害数字 |
| 右侧回放区 | 控制栏 | 播放/暂停/重置按钮，当前帧/总帧数显示 |
| 右侧回放区 | DPS指示器 | 半透明面板，实时显示伤害/秒 |
| 右侧回放区 | 连招标签 | 连击3次/5次时弹出特效文字 |

## 3. 核心流程

用户通过左侧编辑器添加/排列/编辑技能卡片 → 点击播放 → 右侧Canvas按序执行技能动画（前摇→判定→后摇）→ 判定帧期间显示特效和伤害 → 连击判定触发连招标签 → 全部技能执行完毕结束

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色赛博朋克
- **主色**：深灰 #2A2A3E（编辑器）、深蓝黑 #1A1A2E（回放区）
- **强调色**：橙色 #FF8C42（角色/分割线）、紫色 #6A5ACD（分割线渐变终点）
- **分割线**：窄光谱渐变 #FF8C42 → #6A5ACD
- **卡片风格**：圆角矩形，渐变背景 #3B3B52 → #4A4A6A，悬停上浮+阴影
- **按钮风格**：圆角8px，播放绿/暂停橙/重置红，按压动画scale 0.9
- **字体**：系统默认sans-serif，标题白色#FFF，DPS面板白色#FFF 16px，连击文字24-32px金色描边#FFD700

### 4.2 页面设计详情

| 页面区域 | 模块名称 | UI元素 |
|---------|---------|--------|
| 左侧编辑器 | 技能卡片 | 圆角矩形220x60px，渐变背景，白色文字，悬停上浮3px+阴影，点击缩放0.97 |
| 左侧编辑器 | 属性编辑面板 | 输入框背景#2A2A3E，边框#5A5A7A，圆角6px |
| 左侧编辑器 | 添加技能按钮 | 底部居中 |
| 右侧回放区 | Canvas | 100%宽90%高，深蓝黑背景 |
| 右侧回放区 | 角色精灵 | 简笔小人50x80px，颜色#FF8C42，底部中央 |
| 右侧回放区 | 技能特效 | 白色弧线/橙色粒子/蓝色光圈 |
| 右侧回放区 | 控制栏 | 播放(绿)/暂停(橙)/重置(红)按钮+帧数文本 |
| 右侧回放区 | DPS面板 | 半透明#222244圆角10px，淡入动画0.2s |
| 右侧回放区 | 连招标签 | 3连击24px/5连击32px，白色金色描边，弹入动画 |

### 4.3 响应式

桌面优先设计，左右分栏布局（45%+55%），暂不需要移动端适配。

### 4.4 动画规范

- 所有交互过渡0.2s平滑transition
- Canvas动画60FPS，每帧绘制≤12ms，requestAnimationFrame驱动
- 拖拽响应延迟≤50ms
- 伤害面板淡入0.2s
- 连招标签弹入效果（gsap动画）

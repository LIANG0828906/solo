## 1. 产品概述

在线壁纸设计工具，基于CSS艺术风格生成与主题共创功能，用户可自由组合预设模板、调整配色渐变、叠加几何图案，并将作品保存为高清PNG下载。面向设计师、开发者及创意爱好者，提供零门槛的壁纸创作体验。

## 2. 核心功能

### 2.1 功能模块

1. **主编辑页面**：模板选择面板、画布编辑区、导出控制栏三区联动

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑页面 | 模板选择面板 | 网格卡片预览6+种预设壁纸模板，选中时卡片放大并伴随波纹扩散动画 |
| 主编辑页面 | 画布编辑区 | Canvas 2D实时渲染壁纸，支持渐变背景、几何图案叠加、微纹理等效果 |
| 主编辑页面 | 配色控制面板 | 主色/辅色选择器（Hex和HSL双模式），渐变角度调整（0-360°），实时反映画布 |
| 主编辑页面 | 图案叠加控制 | 最多3层几何图案叠加，每层可调透明度、缩放、旋转角度；图案类型：圆点阵、六边形蜂窝、波浪线、三角网格 |
| 主编辑页面 | 导出控制栏 | 2x(1920x1080)和4x(3840x2160)分辨率导出，Worker并行处理，进度指示 |
| 主编辑页面 | 状态栏 | 显示当前缩放比例和颜色数值 |

## 3. 核心流程

用户打开工具 → 浏览预设模板列表 → 选择模板应用至画布 → 调整配色（主色/辅色/渐变角度）→ 叠加几何图案（选择类型/调整参数）→ 实时预览效果 → 选择导出分辨率 → 点击导出 → Worker处理 → 下载PNG

## 4. 界面设计

### 4.1 设计风格

- **风格**：极简毛玻璃（Glassmorphism）
- **主色调**：随当前选中配色动态适配（如主色为蓝色时面板边框泛蓝光）
- **辅色调**：半透明白/灰，微光边框
- **字体**：Google Fonts - Outfit（显示字体）+ DM Sans（UI字体）
- **布局**：三栏布局，左侧模板面板，中间画布区，右侧配色/图案控制面板
- **按钮风格**：圆角微光，悬停时轻微上浮阴影和颜色渐变反馈
- **面板风格**：半透明模糊背景 + 微光边框

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主编辑页面 | 模板选择面板 | 网格卡片、波纹扩散选中动画、卡片放大效果 |
| 主编辑页面 | 画布编辑区 | Canvas画布居中展示、实时渲染 |
| 主编辑页面 | 配色控制面板 | 弹出式颜色选择器、Hex/HSL模式切换、拖拽反馈、数值实时显示 |
| 主编辑页面 | 图案控制面板 | 图案类型选择、透明度滑块、缩放滑块、旋转角度滑块 |
| 主编辑页面 | 顶部操作栏 | 固定定位、毛玻璃背景、工具标题、重置按钮 |
| 主编辑页面 | 底部状态条 | 当前缩放比例、颜色数值显示 |
| 主编辑页面 | 导出控制栏 | 2x/4x分辨率按钮、导出进度指示 |

### 4.3 响应式

- 桌面端（>768px）：三栏横向布局
- 移动端（≤768px）：自动折叠为纵向流式布局，面板可折叠展开

### 4.4 动效

- 模板切换：200ms补间动画过渡，60fps
- 颜色调整：200ms补间动画过渡
- 图案增减：200ms补间动画过渡
- 选中模板：卡片放大 + 波纹扩散
- 按钮悬停：上浮阴影 + 颜色渐变
- 颜色选择器：弹出式动画，拖拽反馈

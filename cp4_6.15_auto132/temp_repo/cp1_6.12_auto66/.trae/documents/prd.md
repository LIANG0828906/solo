## 1. 产品概述

情绪脉动（MoodPulse）是一款帮助用户记录日常情绪波动并自动发现情绪模式的移动优先 Web 应用。它解决的核心问题是：人们难以从零散的心情笔记中归纳出情绪诱因和规律。
- 目标用户：希望提升自我觉察、理解情绪触发因素的普通用户
- 核心价值：将碎片化的情绪记录转化为可操作的情绪洞察，帮助用户主动改善情绪健康

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 记录情绪、查看日历与趋势、获取模式洞察 |

### 2.2 功能模块
1. **情绪记录页**：快速情绪录入表单、6种基础情绪选择、备注输入、活动标签选择
2. **日志列表页**：月视图日历、情绪趋势折线图、日期展开记录列表
3. **洞察页**：30天情绪模式分析、情绪频率统计、推荐活动建议

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 情绪记录页 | 情绪选择器 | 展示6种基础情绪（快乐/悲伤/愤怒/焦虑/平静/疲惫），每种配有emoji和柔和渐变背景色，点击选中 |
| 情绪记录页 | 备注输入框 | 最多100字的简短备注输入 |
| 情绪记录页 | 活动标签选择 | 预定义活动标签（工作、运动、社交、阅读等），支持多选 |
| 情绪记录页 | 记录卡片 | 圆角卡片表单，从底部滑入带0.3秒弹性动画，提交后缩小飞入日历对应日期格子 |
| 日志列表页 | 月视图日历 | 每天情绪以彩色圆点表示，星期标题固定顶部，点击展开当天记录列表 |
| 日志列表页 | 记录列表 | 按时间倒序排列，带渐入动画，展示情绪类型、备注、活动标签、时间 |
| 日志列表页 | 趋势折线图 | 本周情绪趋势，平滑曲线插值，可拖拽节点，悬停弹出情绪摘要 |
| 日志列表页 | 日期范围滑块 | 折线图下方分布区间滑块，可调整日期范围 |
| 洞察页 | 洞察卡片 | 圆角阴影，加载时旋转90度回正动画，包含三个数据块 |
| 洞察页 | 情绪对统计 | 最常出现的情绪-活动组合（如"焦虑-工作"） |
| 洞察页 | 波动时段 | 情绪波动最剧烈的时段分析 |
| 洞察页 | 活动推荐 | 基于数据推荐可改善情绪的活动 |

## 3. 核心流程

用户打开应用后，默认进入日志列表页查看当月情绪日历。点击右下角的记录按钮，弹出情绪记录卡片从底部滑入。用户选择情绪类型、添加备注和活动标签后提交，卡片动画飞入日历对应日期。用户可点击日历中某天展开详细记录，也可切换到洞察页查看30天情绪模式分析。

## 4. 界面设计

### 4.1 设计风格
- 主背景色：暖白色 #F5F0EB
- 情绪配色体系：快乐 #FFD166（柠檬黄）、悲伤 #7B68EE（薰衣草紫）、愤怒 #EF476F（珊瑚红）、焦虑 #457B9D（灰蓝色）、平静 #8CB369（鼠尾草绿）、疲惫 #B8B8B8（米灰色）
- 卡片样式：毛玻璃效果（背景模糊12px、半透明白色）、圆角12px、悬停上浮translateY(-4px)并增加阴影
- 字体：标题使用细圆体字重，正文使用无衬线字体
- 按钮风格：圆角胶囊按钮，带柔和渐变色
- 图标风格：emoji作为情绪标识图标
- 布局风格：卡片式布局，日历网格布局

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 情绪记录页 | 情绪选择器 | 6个圆形情绪按钮，每个带emoji和渐变背景色，选中时放大并发光 |
| 情绪记录页 | 备注输入 | 圆角输入框，半透明白色背景，100字计数器 |
| 情绪记录页 | 活动标签 | 胶囊形状标签按钮，选中时填充对应情绪色 |
| 情绪记录页 | 记录卡片 | 毛玻璃卡片，底部滑入弹性动画0.3s，提交后缩小飞入动画 |
| 日志列表页 | 月视图日历 | 7列网格，星期标题固定顶部，日期格内彩色圆点，悬停微放大 |
| 日志列表页 | 记录列表 | 从日期格下方展开，列表项渐入动画，每项含情绪圆点+备注+标签+时间 |
| 日志列表页 | 趋势折线图 | Recharts平滑曲线，可拖拽节点，悬停弹出摘要tooltip |
| 日志列表页 | 范围滑块 | 双端滑块控件，调整折线图日期范围 |
| 洞察页 | 洞察卡片 | 毛玻璃效果圆角阴影卡片，加载时旋转90度回正动画 |
| 洞察页 | 情绪对 | 横向条形图展示情绪-活动组合频率 |
| 洞察页 | 时段分布 | 24小时环形图展示情绪时段分布 |
| 洞察页 | 推荐卡片 | 带图标的活动推荐列表 |

### 4.3 响应式设计
- 桌面端：多列布局，日历7列网格，折线图宽幅展示
- 移动端：单列布局，日历单元格缩小为圆形icon，情绪卡片在底部以抽屉形式展示
- 触摸优化：拖拽节点增大触摸区域，滑动切换月份

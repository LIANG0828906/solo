## 1. 产品概述

本产品为小型独立书店打造的一体化管理应用，解决书店老板用 Excel 管理图书库存效率低下、客户购书时缺乏个性化推荐体验的痛点。

- 目标用户：小型独立书店老板（管理员）和购书客户
- 核心价值：提供轻量级库存管理 + 线上订单处理 + 个性化图书推荐服务

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 书店管理员 | 后台入口访问 | 查看仪表盘数据、管理库存、处理订单 |
| 客户用户 | 无需登录 | 浏览图书、获取推荐、加入购物车、提交订单 |

### 2.2 功能模块

1. **管理员仪表盘**：当日订单统计卡片、已发货数、低库存预警、库存管理表格
2. **客户首页**：动态轮播精选图书、"猜你喜欢"推荐区域、购物车浮窗
3. **图书详情页**：大图展示、库存状态提示、同分类推荐
4. **购物车模态**：数量调整、总价计算、结算下单、成功动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 管理员仪表盘 | 统计卡片 | 显示当日订单数、已发货数、低库存预警数，带数字计数动画 |
| 管理员仪表盘 | 库存表格 | 展示所有图书（书名、作者、ISBN、库存、价格），支持模糊搜索，行点击展开详情 |
| 管理员仪表盘 | 库存修改 | 展开面板显示封面、分类、库存调整按钮，修改后高亮渐隐 |
| 客户首页 | 轮播组件 | 精选图书轮播，右侧滑入+缩放动画，圆点指示器悬停放大变色 |
| 客户首页 | 猜你喜欢 | 基于浏览历史/随机畅销书推荐，网格卡片布局，悬浮上移+按钮滑入 |
| 客户首页 | 购物车图标 | 右上角浮动图标，加入商品时跳动动画并显示数量 |
| 图书详情页 | 书籍信息 | 大图、作者、ISBN、分类、库存状态（缺货/仅剩X本/有货）、简介 |
| 图书详情页 | 库存紧张提示 | 仅剩1-3本时显示橙色脉冲闪烁提示条 |
| 图书详情页 | 同分类推荐 | 水平滚动列表，两侧半圆形渐变遮罩，悬停透明化 |
| 购物车模态 | 购物车列表 | 书名、单价、数量调节器（带按压动画）、小计 |
| 购物车模态 | 结算流程 | 确认对话框 → 模拟提交 → 成功动画（打勾+光晕粒子） |

## 3. 核心流程

客户浏览首页 → 查看轮播/猜你喜欢推荐 → 点击图书进入详情 → 加入购物车（图标跳动）→ 点击购物车图标查看购物车 → 调整数量 → 结算下单 → 确认订单 → 提交成功动画

管理员访问后台 → 查看仪表盘统计卡片 → 搜索/浏览库存表格 → 点击行展开详情 → 修改库存量 → 确认修改 → 行高亮渐隐

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白（#FAF6F0）、浅棕（#D4A574）、深褐（#5D4037）
- **强调色**：亮橙色（#FF8C42）用于按钮和高亮
- **按钮风格**：圆角矩形，轻微阴影，悬停阴影加深
- **字体**：圆润无衬线体（Nunito / system-ui rounded）
- **布局风格**：卡片式布局，顶部导航栏，内容区域留白充足
- **图标风格**：Lucide React 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 管理员仪表盘 | 统计卡片 | 暖色调背景、数字计数动画、图标左侧点缀 |
| 管理员仪表盘 | 库存表格 | 斑马纹行、悬停高亮、展开平滑高度过渡 |
| 客户首页 | 轮播组件 | 大卡片、右侧滑入+1.02倍缩放、圆点悬停放大变主题色 |
| 客户首页 | 猜你喜欢 | 4列网格、卡片悬浮上移4px、按钮从底部滑入 |
| 图书详情页 | 信息区 | 左图右文平板端上下布局、库存脉冲闪烁条 |
| 图书详情页 | 同分类推荐 | 水平滚动、两侧半圆形渐变遮罩悬停透明 |
| 购物车模态 | 结算区 | 半透明模糊背景、白色圆角卡片、打勾光晕粒子动画 |

### 4.3 响应式设计

- **桌面端（≥1024px）**：图书网格4列，详情页左右分栏
- **平板端（768-1023px）**：图书网格2列，详情页图片占满宽度文字在下
- **手机端（<768px）**：图书网格单列，详情页堆叠布局，精简导航

### 4.4 动效设计

- 所有状态变化使用 CSS transition（200-300ms ease）
- 数字计数动画从 0 递增到实际值
- 卡片悬停阴影加深 + 轻微上移
- 按钮按压时 scale(0.96) 反馈
- 库存修改后行高亮 → 1.5s 渐隐
- 结算成功：勾号从中心放大 + 光晕粒子扩散

## 1. 产品概述

城市生成沙盒是一个基于Web的交互式3D可视化应用，让参观者能够像玩沙盒游戏一样自由探索程序化生成的三维城市街区。应用面向数字孪生城市概念展示，提供沉浸式的城市探索体验。

- 主要目的：通过交互式3D可视化展示数字孪生城市概念，让用户直观理解城市规划与生成过程
- 目标用户：展览参观者、城市规划学习者、科技爱好者
- 市场价值：创新的数字孪生展示方式，可用于科技馆、城市规划馆、教育机构等场景

## 2. 核心功能

### 2.1 用户角色
本应用为单用户体验，无需注册登录。

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 自由探索3D城市、调整生成参数、观察交通流、查看建筑信息 |

### 2.2 功能模块
1. **3D城市场景**：程序化生成的城市街区、建筑、道路、植被、动态交通流
2. **城市生成引擎**：基于参数的网格化城市布局生成，支持实时重建
3. **交互控制面板**：侧边参数调节面板，支持街区规模、建筑高度、密度、树木覆盖率等参数调整
4. **视角导航系统**：轨道控制器、相机飞行、建筑选中与高亮
5. **信息展示系统**：建筑信息卡、UI动画效果

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景页面 | 3D渲染模块 | 全屏3D城市场景，渐变深空背景，星星粒子系统 |
| 主场景页面 | 城市生成模块 | 4x4街区默认布局，建筑高度5-30单位，颜色随高度渐变 |
| 主场景页面 | 交通流模块 | 随机颜色小球模拟汽车，沿道路网格行驶，路口随机转向 |
| 主场景页面 | 交互控制模块 | 侧边控制面板，滑块调节参数，开关控制交通流显示 |
| 主场景页面 | 视角导航模块 | 鼠标拖拽旋转、滚轮缩放、点击建筑飞行视角 |
| 主场景页面 | 信息展示模块 | 建筑顶部信息卡，显示高度和编号 |

## 3. 核心流程

```mermaid
flowchart LR
    A["用户进入页面"] --> B["初始化Three.js场景"]
    B --> C["加载默认城市参数"]
    C --> D["生成城市数据"]
    D --> E["渲染3D城市模型"]
    E --> F["启动渲染循环"]
    F --> G["用户交互"]
    G --> H["调整参数/点击建筑"]
    H --> I["更新Zustand状态"]
    I --> J["触发城市重建/相机飞行"]
    J --> F
```

用户进入页面后，系统自动初始化3D场景并生成默认城市。用户可以通过鼠标拖拽旋转视角、滚轮缩放、点击建筑查看详情，或通过侧边面板调整城市参数，参数变化会实时触发城市平滑重建。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深空紫蓝渐变背景（#0A0A23 → #1B1B3A），建筑暖冷渐变（#D4A373 → #4A6FA5）
- **强调色**：紫色（#7C6FFF）用于滑块、开关等交互元素，金色（#FFD700）用于建筑高亮
- **按钮样式**：Toggle开关（44×24px），圆角设计，选中时底色变紫色
- **字体**：现代无衬线字体，清晰易读
- **布局风格**：沉浸式全屏3D场景，右侧悬浮半透明控制面板
- **视觉效果**：毛玻璃效果、圆角12px、淡入动画、平滑过渡

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主场景页面 | 3D场景 | 全屏Canvas、渐变背景、星星粒子、城市模型、动态汽车 |
| 主场景页面 | 侧边面板 | 半透明背景rgba(30,30,50,0.85)、圆角12px、宽280px、距右20px距顶80px |
| 主场景页面 | 控制面板 | 滑块（轨道高6px圆角3px，滑块直径16px）、标签、数值显示、Toggle开关 |
| 主场景页面 | 信息卡 | 建筑顶部弹出、宽160px、背景rgba(0,0,0,0.7)、圆角8px、白色12px文字、0.3秒淡入 |
| 主场景页面 | 移动端菜单 | 汉堡按钮、从右向左滑入面板 |

### 4.3 响应式
- 桌面端（≥768px）：侧边面板固定显示在右侧
- 移动端（<768px）：面板折叠为汉堡菜单，点击后从右向左滑入
- 触摸优化：支持触摸拖拽旋转、双指缩放

### 4.4 3D场景指导
- **环境**：深空渐变背景，500个闪烁星星粒子，营造科技感氛围
- **光照**：平行光模拟太阳光，支持阴影投射，环境光提供基础照明
- **相机设置**：PerspectiveCamera，初始位置合理，OrbitControls控制，缩放范围5-200单位
- **构图**：城市居中，道路网格清晰，建筑层次分明
- **交互与动画**：建筑重建3秒平滑过渡，相机飞行0.8秒easeInOut曲线，汽车转弯平滑减速
- **后期处理**：建筑选中时外发光效果（#FFD700，强度0.5）
- **性能**：4x4规模下维持≥55 FPS，重建卡顿≤100ms

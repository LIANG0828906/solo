## 1. 产品概述

六博棋3D交互可视化应用是一款基于Web的古代棋盘游戏模拟器，旨在直观展示和反复模拟复杂的对弈逻辑。本应用使用Three.js构建三维陶制棋盘场景，实现棋子移动、吃子、河道事件触发以及胜负判定的全流程可视化，解决传统棋盘游戏复杂逻辑难以直观展示的问题。

- **目标用户**：游戏爱好者、历史文化研究者、对古代游戏规则学习者

- **产品价值**：将古老的六博棋以现代3D技术重现，让用户能够直观理解和交互体验古代棋类游戏的复杂规则和策略。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 完整游戏、调整骰子权重、回放棋谱 |

### 2.2 功能模块

1. **3D棋盘场景**：陶制棋盘、六条主路径、交叉河道、遮阳棚
2. **棋子系统**：红黑双方各6枚八角棱柱棋子、篆体文字
3. **骰子系统**：三枚六面木质骰子、投掷动画、点数计算
4. **游戏逻辑**：移动、吃子、河道事件、卡位逻辑
5. **胜负判定**：胜利条件检测、棋谱记录
6. **回放系统**：竹简卷轴棋谱展示、慢速回放
7. **控制面板**：骰子权重调整、概率饼图、统计直方图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|----------|--------------|
| 主游戏页 | 3D棋盘场景 | Three.js渲染陶土褐色棋盘，六条主路径、暗蓝色交叉河道、四角木柱遮阳棚，右键旋转、滚轮缩放、左键点击高亮可移动位置 |
| 主游戏页 | 棋子系统 | 红方(#cc3333)与黑方(#2a2a2a)各6枚八角棱柱棋子，底部刻篆体文字 |
| 主游戏页 | 骰子系统 | 点击木钵掷出三枚骰子，弹跳碰撞动画，自动求和计算步数 |
| 主游戏页 | 游戏逻辑 | 0.3秒缓动移动动画、吃子缩小消失+金色粒子爆散、河道随机事件（30%冲回起点、20%额外移动、50%无影响，水花动画，卡位逻辑 |
| 主游戏页 | 胜负判定 | 检测胜利条件（吃光对方棋子或抵达对方底线星位 |
| 主游戏页 | 回放系统 | 游戏结束弹出竹简卷轴棋谱展示（至少5条记录），"重播本局"按钮1/3速回放 |
| 主游戏页 | 控制面板 | 青竹筒样式，三个滑块调整骰面权重（1-5倍），概率饼图，模拟1000次统计直方图 |

## 3. 核心流程

```mermaid
flowchart TD
    A["开始游戏"] --> B["初始化棋盘与棋子"]
    B --> C["点击木钵掷骰子"]
    C --> D["骰子动画与点数计算"]
    D --> E["根据点数显示可移动位置"]
    E --> F["玩家点击选择移动位置"]
    F --> G{"是否进入河道?}
    G -->|是| H["触发河道随机事件"]
    H --> I["显示水花动画"]
    I --> J{"是否吃子?}
    G -->|否| J
    J -->|是| K["吃子动画（缩小+粒子爆散]
    J -->|否| L["移动棋子到目标位置"]
    K --> L
    L --> M{"是否胜利条件满足?}
    M -->|是| N["游戏结束"]
    M -->|否| O["切换玩家"]
    O --> C
    N --> P["弹出竹简棋谱"]
    P --> Q["点击重播本局回放"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖木色与土褐色调和
  - 棋盘土褐色 #b08858
  - 河道暗蓝色 #3a5a8a
  - 木柱 #6b4e3a
  - 锦帛 #d4a76a
  - 红方棋子 #cc3333
  - 黑方棋子 #2a2a2a
  - 金色星位 #ffd700
  - 骰子木色 #d2b48c
  - 竹简 #c2a06e 至 #8b6b3b

- **按钮风格**：悬停发白光(box-shadow: 0 0 8px rgba(255,255,255,0.7)，点击下压0.5px

- **字体**：全局字体按视口宽度等比调整（最小12px）

- **布局风格**：棋盘居中，右侧控制面板，下方锦纹饰带

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|----------|----------|
| 主游戏页 | 3D棋盘场景 | 居中Three.js场景，陶土质感，暖色调，缓动动画，粒子效果 |
| 主游戏页 | 棋子系统 | 八角棱柱，篆体文字，移动ease-out动画0.3秒 |
| 主游戏页 | 骰子系统 | 木质立方体，黑色圆点，弹跳碰撞动画 |
| 主游戏页 | 控制面板 | 青竹筒#6b8e23，滑块，概率饼图（红#ff3333至蓝#3333ff渐变），统计直方图 |
| 主游戏页 | 竹简卷轴 | 渐变背景，棋谱记录，回放按钮 |

### 4.3 响应性

- 桌面端优先，棋盘自适应窗口缩放
- 窗口缩小时棋盘自适应缩放
- 全局字体字号按视口宽度等比调整（最小12px）
- 移动端触控优化

### 4.4 3D场景指引

- **环境与氛围**：古风暖色环境光，环境贴图营造古代桌台
- **光照设置**：主光源+环境光+补光，突出陶土和木质质感
- **相机设置**：透视相机，初始俯视45度角，支持右键旋转、滚轮缩放
- **构图与焦点元素**：棋盘居中，焦点在棋盘中央
- **交互与动画**：棋子移动ease-out 0.3秒，吃子缩小0.2秒缩至0，粒子爆散，骰子弹跳碰撞动画，河道水花动画
- **后处理效果**：抗锯齿，柔和阴影
- **性能预算**：帧率不低于30FPS，粒子上限30个，每帧最多5个新粒子

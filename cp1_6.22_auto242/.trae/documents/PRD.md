## 1. 产品概述

调香师工作台是一款面向独立调香师的数字化管理工具，帮助用户摆脱手工笔记和复杂Excel表格，高效管理香料库存、记录实验配方、自动核算成本并生成专业报告。

- 核心价值：解决上百种香原料库存追踪、配方迭代与成本核算信息分散的痛点
- 目标用户：独立调香师、香水工作室、香水爱好者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 独立调香师 | 本地应用直接使用 | 完整的原料、配方、试香记录管理权限 |

### 2.2 功能模块

1. **原料库**：香料原料卡片列表、库存状态色标、香调家族与库存筛选
2. **配方管理**：配方创建编辑、百分比校验、拖拽排序、成本实时计算
3. **试香记录**：试香笔记卡片、星级评分、留香时长记录、气味演变描述
4. **报告中心**：成本分析饼图、PDF报告导出、配方版本历史时间轴

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 原料库 | 原料卡片列表 | 卡片式展示，左侧库存色标条，悬浮上升动画 |
| 原料库 | 筛选面板 | 按香调家族（花香/木质/辛香等）和库存量筛选 |
| 配方列表 | 配方卡片 | 顶部香调色条，展示配方名称与主香调 |
| 配方详情 | 配比编辑器 | 搜索添加原料、百分比输入、拖拽排序、实时成本 |
| 配方详情 | 版本历史 | 时间轴展示保存记录，支持预览与回滚 |
| 试香记录 | 试香卡片堆叠 | 星级评分动画、点击展开完整笔记 |
| 报告中心 | 成本分析 | 饼状成本分布图、每10ml成本估算 |
| 报告中心 | PDF导出 | 一键导出配方与成本报告为PDF |

## 3. 核心流程

用户从原料库开始，添加和管理香料原料库存；随后创建配方，从原料库中选取香料并调配比例；系统自动计算成本；可保存配方生成版本快照；进行试香后记录试香笔记与评分；最终生成并导出成本分析报告。

```mermaid

    A["原料入库"] --> B["创建配方"]
    B --> C["调配比例"]
    C --> D["成本核算"]
    D --> E["保存版本"]
    E --> F["试香记录"]
    F --> G["生成报告"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白 #FDFBF7（主背景）、深褐 #3C2415（正文）
- **辅助色**：淡金 #C9A96E（按钮/强调）、灰绿 #7C9A73（库存充足）
- **状态色**：橙色 #D4A373（低库存）、红色 #C47A7A（缺货）
- **卡片风格**：圆角、左侧色条、悬浮上升6px加阴影
- **按钮风格**：金色背景、米白文字、圆角6px、hover加深、点击缩放
- **字体**：优雅衬线体搭配现代无衬线体，营造复古温暖氛围
- **布局**：顶部导航栏 + 左侧可收起面板 + 右侧主内容区

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 原料库 | 原料卡片 | 库存色条、名称品牌、库存量、成本、悬浮上升动画 |
| 配方编辑器 | 配比行 | 拖拽手柄、原料名、百分比输入、成本显示、删除按钮 |
| 试香记录 | 评分星标 | 0.3秒缩放动画、1-5星交互 |
| 版本历史 | 时间轴 | 垂直时间线、版本节点、保存时间、备注 |
| 报告页面 | 成本饼图 | 各原料成本占比可视化 |

### 4.3 响应式

- 桌面端（≥1024px）：两栏布局，左侧导航面板320px宽
- 平板端（<1024px）：单栏堆叠，面板可滑出
- 手机端（<768px）：汉堡菜单导航，卡片100%宽度

### 4.4 动效与交互

- 卡片悬浮：上升6px + 阴影，过渡0.2秒 ease-out
- 面板展开/收起：0.3秒宽度过渡动画
- 按钮点击：0.1秒 scale 0.97 反馈
- 评分星标：0.3秒缩放动画
- 输入框聚焦：金色光环效果

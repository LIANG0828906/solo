## 1. 产品概述
视频关键帧提取与标记管理工具，面向纪录片剪辑师等视频工作者，解决批量视频素材快速浏览和标记关键片段的效率问题。支持自动按时间间隔提取关键帧、添加标签/评分/备注、搜索过滤及数据导出，替代手动逐个打开视频截图的低效流程。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传视频、提取关键帧、标记管理、搜索导出 |

### 2.2 功能模块
1. **主工作区**：视频上传、关键帧网格展示、搜索过滤、导出操作
2. **标记面板**：弹窗式标记编辑（标签/评分/备注）
3. **项目侧边栏**：历史项目管理与切换

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作区 | 视频上传区 | 选择视频按钮，支持MP4/WebM格式上传 |
| 主工作区 | 进度条 | 视频解析时显示提取进度 |
| 主工作区 | 关键帧网格 | 响应式网格展示所有关键帧缩略图，卡片含帧序号和时间戳 |
| 主工作区 | 搜索栏 | 按标签名或备注关键词实时过滤（300ms防抖） |
| 主工作区 | 评分筛选 | 右侧下拉按1-5星或全部筛选 |
| 主工作区 | 导出按钮 | 导出当前过滤结果为JSON或CSV |
| 标记面板 | 标签输入 | 按回车添加标签，最多5个，颜色从预设10色随机选取 |
| 标记面板 | 星级评分 | 5星点击切换 |
| 标记面板 | 备注文本框 | 200字以内文字备注 |
| 项目侧边栏 | 项目列表 | 显示视频文件名和标记数，点击恢复帧列表和标记状态 |

## 3. 核心流程

用户上传视频文件 → 系统自动解析提取关键帧（显示进度）→ 网格展示缩略图 → 用户点击缩略图打开标记面板 → 添加标签/评分/备注并保存 → 缩略图显示标记状态图标 → 用户搜索/过滤标记 → 导出标记数据为JSON/CSV

```mermaid
    "G" --> "H[缩略图显示标记状态图标]"
    "C" --> "I[搜索/过滤标记]"
    "I" --> "J[导出JSON/CSV]"
    "C" --> "K[侧边栏项目管理]"
    "K" --> "C"
```

## 4. 界面设计

### 4.1 设计风格
- 主背景 #0F172A，卡片背景 #1E293B，文字 #E2E8F0，强调色 #3B82F6
- 圆角卡片（8px），模态弹窗（16px圆角）
- 暗色主题，工业/专业工具感
- 字体：系统字体栈，清晰易读
- 按钮风格：圆角8px，强调色填充

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主工作区 | 侧边栏 | 宽260px，背景#1E293B，项目列表，选中项背景#334155+3px左蓝色边框 |
| 主工作区 | 搜索/过滤栏 | 顶部搜索输入框+右侧评分下拉筛选+导出按钮 |
| 主工作区 | 关键帧网格 | 中间70%宽度，每行4-5列，卡片180px宽，8px圆角，悬停0.2s缩放1.02+投影 |
| 标记面板 | 弹窗 | 居中模态，遮罩#00000080，弹窗背景#1E293B，16px圆角，0.3s淡入动画 |
| 主工作区 | 上传区 | 选择视频按钮+进度条 |

### 4.3 响应式
- 桌面优先设计
- 768px以下：单列网格，弹窗全屏
- 卡片和弹窗适配移动端

### 4.4 性能约束
- 关键帧提取使用requestAnimationFrame或Web Worker异步处理，主线程不阻塞超过200ms
- 帧网格滚动帧率保持50FPS以上

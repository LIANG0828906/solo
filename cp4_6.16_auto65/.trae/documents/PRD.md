# FitScribe 健身训练日志应用 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
FitScribe 是一款面向健身爱好者的训练日志与动作库管理应用，帮助用户系统化记录训练过程、管理个人动作库、创建训练模板，并通过数据可视化分析训练效果。

### 1.2 目标用户
- 健身爱好者（初级到中级）
- 需要系统化记录训练的人群
- 希望追踪训练进度和数据的用户

### 1.3 核心价值
- **高效记录**：快速记录每次训练的组数、次数和重量
- **动作管理**：建立个人动作库，支持分类筛选和搜索
- **模板复用**：保存常用训练组合，一键加载减少重复输入
- **数据洞察**：周度统计图表直观展示训练量和部位分布
- **数据安全**：本地存储 + 导出/导入功能，数据永不丢失

## 2. 功能需求

### 2.1 动作库管理
- **添加动作**：名称、部位（胸/背/腿/肩/臂/核心）、动作类型（自由重量/器械/自重）、文字说明
- **动作展示**：卡片网格布局，显示名称和部位标签
- **筛选功能**：按部位筛选动作
- **搜索功能**：按名称搜索，支持淡入动画
- **删除动作**：支持删除自定义动作

### 2.2 训练日志记录
- **选择动作**：从动作库中选择训练动作
- **记录数据**：组数、每组重复次数、负重重量（公斤）
- **日期时间**：设置训练日期和时间
- **日志列表**：按日期倒序排列，显示动作名称、组数、总重量
- **删除记录**：支持滑动删除动画效果
- **总重量计算**：自动计算组数 × 次数 × 重量

### 2.3 训练模板功能
- **创建模板**：保存一组动作和默认组次配置
- **加载模板**：快速填充训练表单
- **模板管理**：查看、删除已保存的模板

### 2.4 周度统计图表
- **折线图**：最近7天每日总训练组数
- **柱状图**：各部位总训练次数
- **交互效果**：平滑动画、悬停提示
- **独立页面**：专门的图表展示页面

### 2.5 数据导出/导入
- **导出功能**：全部训练记录导出为JSON文件下载
- **导入功能**：从JSON文件恢复数据

## 3. 界面设计

### 3.1 配色方案
- 背景色：#1a1a2e（深蓝紫色）
- 卡片背景：#16213e（深宝蓝）
- 主按钮：#0f3460（深海蓝）
- 高亮文字：#e94560（珊瑚红）
- 辅助色：白色文本、半透明毛玻璃效果

### 3.2 布局设计
- **桌面端**：左右两栏布局
  - 左侧：动作库管理面板
  - 右侧：训练日志表单和列表
- **移动端**：单列布局（< 768px）
  - 上部：动作库
  - 下部：训练日志

### 3.3 动效设计
- 页面加载：卡片依次出现（stagger动画）
- 搜索结果：淡入动画
- 卡片悬停：轻微上浮 + 阴影加深（transition 0.2s）
- 删除记录：向左滑动消失
- 图表渲染：平滑动画

### 3.4 视觉元素
- 毛玻璃效果卡片
- 渐变分割线
- 部位标签（不同颜色区分）
- 圆角设计 + 阴影层次

## 4. 性能要求
- 动作搜索和筛选响应时间 ≤ 100ms
- 图表渲染延迟 ≤ 200ms
- IndexedDB读写（200条记录）≤ 50ms
- 页面首次加载 ≤ 2s

## 5. 技术约束
- 框架：React 18 + TypeScript
- 构建工具：Vite
- 状态管理：Zustand
- 数据存储：IndexedDB（idb-keyval）
- 图表库：Recharts
- 路由：React Router v6
- ID生成：uuid

## 1. 产品概述
微缩景观策展平台是一个面向微缩模型爱好者、策展人和买家的在线社区与交易平台。模型师可发布作品并申请参展，策展人可组织线上主题展览并选品排序，普通用户可浏览、点赞、评论和购买参展作品，所有行为均有动态热度统计。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 模型师 | 注册登录 | 创建/管理微缩作品卡片，申请参展，查看个人工作室 |
| 策展人 | 注册登录 | 创建/管理主题展览，选品，拖拽排序，查看热度 |
| 普通用户 | 无需登录 | 浏览公开展览和作品，点赞/评论，查看热度 |

### 2.2 功能模块
1. **展览首页**：策展人列表、展览瀑布流卡片、封面缩略图、标题、参展作品计数
2. **展览详情页**：时间轴滚动布局、作品入场动画、拖拽排序（策展人模式）、点赞/评论、热度条
3. **个人工作室页**：瀑布流作品卡片、悬停放大预览、申请参展按钮、策展人选择弹窗
4. **作品发布页**：创建作品表单（名称、系列、比例尺、材质、图片URL、创作故事、是否可出售）
5. **轻量级作品册**：全屏图片转场、故事文本、点赞/评论功能、左右滑动切换多图

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 展览首页 | 策展人列表 | 展示所有策展人头像和名称 |
| 展览首页 | 展览瀑布流 | 网格布局展示展览卡片（封面缩略图+标题+作品计数），点击跳转详情 |
| 展览详情 | 时间轴布局 | 以时间轴滚动展示所有入选作品，每个作品带编号和热度值 |
| 展览详情 | 拖拽排序 | 策展人模式下可拖拽调整作品顺序，提交后立即保存 |
| 展览详情 | 热度条 | 实时显示点赞数和评论数，可视化热度 |
| 个人工作室 | 作品瀑布流 | 瀑布流展示作品卡片，悬停放大预览 |
| 个人工作室 | 申请参展 | 点击申请参展按钮弹出策展人列表供选择 |
| 作品发布 | 创建表单 | 输入作品名称、系列、比例尺、材质、多张图片URL、创作故事、是否可出售 |
| 轻量级作品册 | 图片转场 | 全屏图片轮播，framer-motion转场动画 |
| 轻量级作品册 | 互动区 | 底部点赞/评论功能，实时更新热度 |

## 3. 核心流程

模型师创建作品 → 申请参展选择策展人 → 策展人审核并组织展览 → 拖拽排序确定展序 → 普通用户浏览展览 → 点赞/评论实时反馈热度

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深墨绿(#1a2f23)搭配暖铜色(#c47d4e)点缀，营造微缩模型工坊的精工质感
- **辅助色**：象牙白(#f5f0e8)背景、深炭灰(#2d2d2d)文字、柔金色(#d4a855)高亮
- **按钮风格**：圆角微浮雕，hover时铜色描边发光
- **字体**：标题使用 Playfair Display（衬线体，体现手工艺感），正文使用 Noto Sans SC
- **布局风格**：卡片式布局，瀑布流排列，时间轴纵向滚动
- **动效**：卡片悬停微放大、framer-motion入场交错动画、图片转场滑动

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 展览首页 | 导航栏 | 深墨绿顶栏，Logo + 导航链接，右侧登录按钮 |
| 展览首页 | 策展人列表 | 水平滚动头像行，铜色边框高亮 |
| 展览首页 | 展览瀑布流 | 三列网格，卡片封面+标题+计数，悬停放大 |
| 展览详情 | 时间轴 | 左侧时间线，右侧作品卡片交错排列，编号+热度 |
| 展览详情 | 拖拽排序 | 策展人模式下卡片可拖拽，虚线占位提示 |
| 展览详情 | 热度条 | 卡片底部渐变色条，宽度=热度百分比 |
| 个人工作室 | 作品瀑布流 | Masonry布局，悬停放大1.05倍+阴影 |
| 个人工作室 | 申请参展弹窗 | 模态框，策展人列表+确认按钮 |
| 作品发布 | 表单 | 分组输入，图片URL可动态添加多行 |
| 轻量级作品册 | 全屏轮播 | 暗色背景，图片居中，左右箭头/滑动切换 |
| 轻量级作品册 | 互动区 | 底部半透明栏，心形点赞+评论输入框 |

### 4.3 响应式
- 桌面优先设计，1280px基准
- 平板（768px-1024px）：瀑布流双列
- 移动端（<768px）：单列堆叠，触摸优化拖拽

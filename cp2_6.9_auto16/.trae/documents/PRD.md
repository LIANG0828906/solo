# 古代冰鉴储冰管理互动游戏 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
一款以周代冰政文化为背景的交互式管理游戏，玩家扮演"凌人"（掌冰官员），管理冰窖储冰、调配冰鉴、为王室宴会提供冰镇服务。通过沉浸式的交互体验，展现古代藏冰文化与青铜礼器的魅力。

### 1.2 目标用户
- 对中国古代文化、青铜器感兴趣的用户
- 喜欢模拟经营、资源管理类游戏的玩家
- 教育场景下的文化展示与互动体验

### 1.3 核心价值
- **文化传播**：直观展示周代"凌人"官职的职责与冰鉴的使用方式
- **互动体验**：拖拽、放置等直观操作，带来沉浸式管理体验
- **美学呈现**：青铜、竹木、朱砂的古典色调与纹理，还原古风意境

## 2. 功能需求

### 2.1 凌阴（冰窖）场景

| 功能 | 描述 | 交互细节 |
|------|------|----------|
| 冰砖堆展示 | 6×5共30块冰砖堆叠，半透明浅蓝色#c8e6c9正方体，边缘磨砂感 | 表面水珠每2秒闪烁一次 |
| 冰砖拆卸 | 点击冰砖可拆下并拖拽 | 拖拽时冰砖跟随鼠标，缩放至80%大小 |
| 背包系统 | 右下角背包栏，最多容纳10块冰砖 | 落入背包触发水滴滴落声效 |
| 悬停反馈 | 冰砖悬停时放大1.1倍，增加微光效果 | box-shadow: 0 0 15px rgba(200, 230, 201, 0.8) |
| 点击反馈 | 点击时有0.2s按压缩放动画 | |

### 2.2 宴会厅场景

| 功能 | 描述 | 交互细节 |
|------|------|----------|
| 朱漆食案 | 中央摆放，桌面#8b4513 | |
| 青铜冰鉴组 | 大鉴（直径0.6，整坛酒）、中鉴（直径0.4，瓜果）、小鉴（直径0.2，单壶饮品） | |
| 冰鉴槽位 | 鉴体口沿下方0.5单位处有一圈槽位 | 拖拽冰砖到槽位自动排列 |
| 温度系统 | 每块冰砖降温1度，从室温25度开始下降 | 温度数值显示在冰鉴上方 |
| 雾气粒子效果 | 液体表面升腾雾气，粒子直径2-4px，白色半透明，上升速度0.5单位/秒 | 温度越低雾气越浓 |
| 冰块融化 | 每放置5分钟缩小5%，小于原始大小20%时消失，触发水花声 | |

### 2.3 竹简信息面板

| 功能 | 描述 |
|------|------|
| 批次记录 | 冰块来源批次、入库时间 |
| 状态监控 | 当前冰块大小（实时更新） |
| 使用统计 | 本次宴会已使用冰砖数量 |
| 损耗统计 | 总冰块损耗率（融化消失的冰块占比） |

### 2.4 场景切换
- 顶部Tab栏切换"凌阴"和"宴会厅"场景
- 切换时带有淡入淡出过渡动画

### 2.5 响应式布局
- 桌面端：左侧70%主操作区，右侧30%信息面板，中间竹节纹样分隔线
- 移动端（<900px）：信息面板收为底部抽屉，点击汉堡菜单弹出

## 3. 视觉设计

### 3.1 色调规范
| 用途 | 色值 | 说明 |
|------|------|------|
| 青铜锈色 | `#5d7a5a` | 冰鉴主体、边框、文字 |
| 竹简色 | `#d4c5a9` | 信息面板背景 |
| 朱砂红 | `#c0392b` | 强调色、按钮、食案边缘 |
| 冰砖色 | `#c8e6c9` | 半透明浅蓝色 |
| 地窖墙 | `#4a4a4a` | 深灰色石墙 |
| 地面 | `#6b7b6b` | 青砖色 |
| 背景 | `#2c2c2c` | 整体页面背景 |

### 3.2 纹理与装饰
- 背景：麻布纹理（CSS repeating-linear-gradient模拟经纬线）
- 分隔线：竹节纹样竖线（宽4px，重复图案）
- 字体：思源宋体（Google Fonts引入）

### 3.3 动画规范
| 动画 | 时长 | 缓动 |
|------|------|------|
| 悬停缩放 | 0.2s | ease-out |
| 点击按压 | 0.2s | ease-in-out |
| 场景切换 | 0.3s | ease-in-out |
| 水珠闪烁 | 2s | infinite |
| 粒子升腾 | 持续 | 线性 |

## 4. 性能要求

| 指标 | 要求 |
|------|------|
| 拖拽响应 | < 50ms |

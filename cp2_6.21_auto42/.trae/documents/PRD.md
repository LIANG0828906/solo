## 1. 产品概述

虚拟植物种植与社交分享平台，让用户在线种植虚拟植物并与社区分享。
- 解决用户无法随时种植真实植物的痛点，提供轻松愉悦的虚拟种植体验
- 结合社交元素，打造植物爱好者的分享交流社区

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 账号密码注册 | 种植养护植物、发布分享、点赞评论 |

### 2.2 功能模块

1. **登录注册页面**：用户登录、注册、毛玻璃卡片UI
2. **首页花园**：3x3网格展示最多9个花盆、植物生长SVG动画
3. **单株养护页面**：Three.js 3D植物模型、浇水/施肥/光照操作、粒子特效、状态仪表盘
4. **社区广场**：瀑布流布局、品种筛选、点赞排序、图片分享、评论互动

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 登录注册 | 毛玻璃卡片 | 居中圆形毛玻璃登录/注册表单，渐变绿色按钮 |
| 首页花园 | 花盆网格 | 3x3网格布局，SVG生长动画展示，点击进入养护页 |
| 单株养护 | 3D植物展示 | Three.js渲染绿色球体叶片+棕色圆柱茎，随风摆动 |
| 单株养护 | 养护操作 | 浇水(蓝粒子)、施肥(棕粒子)、光照(黄光晕)、拍照分享 |
| 单株养护 | 状态仪表盘 | 水分/肥料/光照三色进度条实时显示 |
| 社区广场 | 瀑布流展示 | 无限滚动瀑布流，图片懒加载 |
| 社区广场 | 筛选排序 | 品种彩色标签筛选、点赞数下拉排序 |
| 社区广场 | 互动功能 | 心形点赞按钮、50字评论框、显示最后3条评论 |

## 3. 核心流程

用户注册登录 → 首页查看花园 → 点击空花盆选择植物种子 → 进入单株养护页面 → 每日浇水/施肥/光照 → 植物生长从幼苗到开花 → 拍照分享到社区 → 浏览其他用户分享并点赞评论

## 4. 用户界面设计

### 4.1 设计风格
- **主题色**：清新自然暖绿色系，主背景淡绿到米白径向渐变(#e8f5e9 → #fffde7)
- **按钮风格**：渐变绿色(#66bb6a → #43a047)，圆角，悬浮上移3px加深阴影
- **字体**：系统无衬线字体，标题18-24px，正文14-16px
- **布局**：卡片式布局，圆角设计，柔和阴影
- **图标**：Lucide React图标库，自然绿色系

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 登录注册 | 毛玻璃卡片 | 宽360px，rgba(255,255,255,0.15)背景，8px模糊，1px #4caf5050边框，圆角20px |
| 首页花园 | 花盆卡片 | 宽200px高280px白色圆角卡片，2px #c8e6c9边框，4px圆角 |
| 单株养护 | 3D场景 | 60vh视口高度，绿色球体+棕色圆柱，动态摆动旋转 |
| 单株养护 | 操作按钮 | 4个圆形按钮分布周围，粒子特效反馈 |
| 单株养护 | 状态仪表盘 | 100%宽60px高，#f5f5f5灰色圆角背景，蓝/棕/黄进度条 |
| 社区广场 | 图片卡片 | 宽300px白色圆角卡片，8px圆角，2px #e0e0e0边框 |

### 4.3 响应式设计
- 桌面端优先设计，移动端自适应
- 花盆网格在平板变为2列，手机变为1列
- 瀑布流卡片宽度根据屏幕自动调整
- 触摸操作优化，按钮尺寸适合手指点击

### 4.4 3D场景指引
- **环境**：柔和绿色渐变背景，模拟自然光照
- **光照**：环境光+方向光组合，模拟自然日光
- **相机**：正交或透视相机，固定视角+轻微自动旋转
- **构成**：多组绿色球体(叶片)围绕棕色圆柱(茎)分层排列
- **动画**：正弦函数模拟风吹摆动，持续缓慢自动旋转
- **交互**：养护操作触发对应粒子特效

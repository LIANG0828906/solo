## 2. 技术选型说明

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **状态管理**：zustand（轻量级状态管理）
- **截图工具**：html-to-image（生成历史缩略图）
- **渲染方式**：SVG（矢量图形，保证高精度与交互支持）
- **样式方案**：CSS Modules + CSS Variables

## 4. 数据模型

### 4.1 配色方案

```typescript
interface ColorScheme {
  id: string;
  name: string;
  colors: string[]; // 5个颜色
}
```

### 4.2 纹样元素

```typescript
interface PatternElement {
  id: string;
  type: 'peony' | 'plum' | 'butterfly';
  x: number;
  y: number;
  scale: number;
  rotation: number;
  colors: string[];
}
```

### 4.3 历史记录

```typescript
interface HistoryRecord {
  id: string;
  timestamp: number;
  thumbnail: string;
  scheme: ColorScheme;
  elements: PatternElement[];
  fanShape: 'circle' | 'begonia';
}
```

### 4.4 应用状态

```typescript
interface AppState {
  currentScheme: ColorScheme | null;
  fanShape: 'circle' | 'begonia';
  customPalette: string[]; // 12个槽位
  history: HistoryRecord[];
  selectedElementId: string | null;
}
```

## 5. 核心模块说明

### 5.1 FanCanvas 团扇画布

- 使用 SVG 渲染扇面和纹样
- 支持纹样选中与颜色修改
- 支持拖拽移动纹样
- 支持形状切换动画
- 配色过渡动画

### 5.2 PalettePanel 调色板

- 6套预设古风配色展示
- 24色古风色系色板
- 12槽自定义调色盘
- 拖拽颜色填充
- 点击应用配色

### 5.3 HistoryPanel 历史记录

- 自动生成缩略图快照
- 最多保存15条记录
- 点击回退恢复
- 长按删除

## 1. 产品概述

设备借还管理系统是一款面向小团队或班级的轻量级共享设备管理应用，解决设备分散在不同成员手中、借用记录靠口头或表格沟通导致设备丢失、超期不还及预约冲突的问题。
- 目标用户：需要共享设备的小型团队、班级、实验室等组织
- 核心价值：通过系统化的借用/归还流程与实时状态追踪，消除设备管理混乱，提升设备利用率

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设 | 添加/删除设备、确认借用、确认归还、查看所有记录 |
| 普通用户 | 简单登录 | 浏览设备、提交借用申请、查看个人借阅历史 |

### 2.2 功能模块
1. **统计概览面板（首页）**：设备总数、当前借出数、超期未还数三大统计卡片
2. **设备管理页面**：设备列表卡片展示、搜索筛选、设备详情面板、管理员添加设备
3. **借用与归还流程**：借用申请表单、预约确认倒计时、管理员确认/归还
4. **预约冲突检测**：借用时自动检测7天内冲突、冲突提示横幅、日历小视图标注
5. **个人借阅历史**：按时间倒序展示、超期标记

## 3. 核心流程

### 3.1 借用流程
用户选择设备 → 填写借用表单（归还日期1-14天） → 系统检测冲突 → 无冲突则提交 → 设备状态变为"已预约"（10分钟倒计时） → 管理员确认 → 状态变为"借出" → 用户看到剩余天数（每日减少带缩放动画） → 管理员点击"归还" → 状态恢复"可借"

### 3.2 预约冲突检测流程
用户尝试借用 → 系统查询未来7天预约/借出记录 → 存在冲突则弹出红色横幅 + 日历标注 → 用户确认是否继续

## 4. 用户界面设计

### 4.1 设计风格
- **主背景**：#F8F9FA（极浅灰）
- **卡片背景**：白色，圆角12px，阴影0 2px 8px rgba(0,0,0,0.08)
- **标题色**：#2D3436（深灰）
- **正文色**：#636E72（中灰）
- **按钮主色调**：蓝紫渐变（from #6C5CE7 to #A29BFE），悬停渐变角度偏移45度+0.3s过渡
- **字体**：系统字体栈，标题加粗，正文常规
- **布局**：弹性布局，居中分布，卡片间距24px
- **动画**：列表逐行淡入（每行间隔0.05s，透明度0→1），剩余天数缩放动画0.3s，卡片悬停3px抬起

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 统计概览面板 | 统计卡片 | 三等宽卡片并排，弹性布局居中，间距24px，数字2.0rem加粗，图标32px/0.6透明 |
| 设备管理页面 | 设备卡片 | 白色圆角12px卡片，左上角8px状态圆点，悬停抬起3px+阴影加深 |
| 设备管理页面 | 详情面板 | 模态弹窗，完整设备信息+持有者+归还时间 |
| 借用申请页面 | 借用表单 | 日期选择器（1-14天限制），提交按钮蓝紫渐变 |
| 借用申请页面 | 倒计时条 | 黄→红渐变进度条，10分钟倒计时 |
| 借用申请页面 | 冲突提示 | 红色横幅+日历淡红色块（8px高/2px圆角） |
| 个人借阅历史 | 历史列表 | 表格式排列，超期行#FFE4E1背景+感叹号图标 |

### 4.3 响应式设计
- 桌面优先设计，移动端（<768px）自适应
- 统计卡片三列→纵向堆叠
- 侧边栏→顶部标签式导航
- 借阅列表卡片宽度自适应

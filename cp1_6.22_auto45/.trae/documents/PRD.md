## 1. 产品概述

在线花卉识别与养护日历应用，帮助花卉爱好者通过照片自动识别花卉品种，并提供个性化养护提醒，科学养护花卉。

- 主要用途：花卉品种识别、养护事件管理、种植位置可视化
- 目标用户：花卉爱好者、园艺初学者、家庭种植者
- 产品价值：降低花卉养护门槛，提供科学养护建议

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地存储数据 | 上传花卉照片、识别花卉品种、管理养护事件、查看养护日历、浏览种植地图 |

### 2.2 功能模块

1. **主页面：花卉识别上传、花卉卡片列表、养护日历视图、拖拽排序
2. **详情页面：花卉详情展示、养护事件管理、事件添加表单

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主页面 | 花卉上传模块 | 支持jpeg/png格式图片上传，模拟识别，加载动画，随机返回5种内置花卉 |
| 主页面 | 花卉卡片网格 | 卡片滑入动画，展示识别名称、置信度进度条、上传时间，低置信度警告边框 |
| 主页面 | 养护日历视图 | 按月视图展示，事件圆点标记，点击弹出事件详情 |
| 详情页面 | 花卉详情展示 | 放大图片展示、识别结果展示、养护日历 |
| 详情页面 | 养护事件管理 | 添加养护事件（浇水/施肥/修剪）、日期选择、备注输入 |
| 详情页面 | 种植地图 | Leaflet.js地图展示，标记花卉种植位置 |

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：草木绿 #7EC850
- 背景色：米白 #F5F0E8
- 按钮风格：渐变色按钮（#7EC850 到 #5BA34E），圆角12px，脉冲呼吸动画
- 字体：使用Google Fonts 优美字体组合
- 卡片风格：柔和圆角12px，浅阴影，悬停上浮4px加深阴影
- 图标风格：自然清新的线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|----------|----------|
| 主页面 | 上传区域 | 渐变色上传按钮、脉冲动画、文件选择器 |
| 主页面 | 花卉卡片网格 | CSS Grid自适应布局、卡片滑入动画、置信度渐变进度条 |
| 主页面 | 日历视图 | 月份滑动切换动画、事件彩色圆点标记、模态框淡入淡出 |
| 详情页面 | 花卉详情 | 大图展示、识别结果标签、养护建议区域 |
| 详情页面 | 事件表单 | 下拉选择器、日期选择器、备注输入框、加载状态按钮 |

### 4.3 响应式设计

- 桌面端：多列自适应网格布局
- 平板端：两列布局
- 移动端：单列布局
- 触摸优化：大点击区域，流畅滑动

### 4.4 动画与交互

- 卡片滑入动画：从底部滑入，300ms ease-out
- 模态框动画：中心放大淡入，缩小淡出
- 日历切换：左右滑动过渡，300ms ease-out
- 按钮悬停：脉冲呼吸动画
- 卡片悬停：上浮4px，阴影加深
- 事件添加成功：Web Audio API 清脆提示音

## 1. 产品概述

个人迷你骑行日志与路线规划应用——让骑行爱好者记录每次骑行的路线、距离、用时和海拔变化，并支持从已有路线片段拼接出新的骑行路线，自动生成可视化海拔剖面图和速度分析。

- 目标用户：骑行运动爱好者，需要轻量级日志记录和路线规划工具
- 核心价值：一站式骑行记录 + 路线拼接 + 数据可视化，替代零散的手动记录方式

## 2. 核心功能

### 2.1 功能模块

1. **骑行记录列表页**：展示所有骑行记录卡片，支持日期筛选、距离排序、月度统计
2. **路线拼接页**：从已有骑行记录中选取路段，拼接生成新路线及可视化图表

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 骑行记录列表 | 统计卡片 | 显示本月总距离🚀、平均速度⚡、累计爬升🏔、骑行次数📅，数字下方显示环比变化百分比（上升绿色↓下降红色↑，0.2s过渡） |
| 骑行记录列表 | 日期范围筛选 | 双滑块选择器，圆形滑块带#4CAF50填充色，滑动时背景渐变动画0.2s |
| 骑行记录列表 | 距离排序 | 升序/降序切换，列表项交错渐入动画，每项间隔0.05s |
| 骑行记录列表 | 骑行卡片列表 | 卡片背景根据骑行时长自动渐变（<1h浅蓝、1-2h暖黄、>2h暖橙），左上骑行图标，展示总距离和用时，点击展开详情 |
| 骑行卡片 | 创建表单 | 输入日期、骑行名称、总距离(km)、用时(hh:mm:ss)、平均心率(bpm)、海拔爬升(m)，简易地图标注≥3个关键地点（涟漪动画0.3s） |
| 骑行卡片 | 展开详情 | 海拔和速度数据以动画柱子填充（0.5s ease-out过渡） |
| 路线拼接 | 路段选取 | 从已保存记录中选取最多5段路段，按时间顺序拼接 |
| 路线拼接 | 节点标记 | 连接点显示灰色圆点，悬停放大为绿色并显示名称和经纬度（0.2s过渡） |
| 路线拼接 | 海拔剖面图 | 全程海拔波动，绿→红渐变色带表示海拔高低，X轴距离分段Y轴海拔，悬停显示海拔值和坡度% |
| 路线拼接 | 速度分析图 | 蓝色折线图表示每公里平均速度，最高点金色圆点标注 |

## 3. 核心流程

用户创建骑行记录 → 记录出现在列表页 → 可按日期/距离筛选排序 → 查看月度统计 → 选择多条记录进行路线拼接 → 查看合并海拔剖面图和速度分析图

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：运动绿(#4CAF50) + 深灰(#263238)作为基底
- 辅助色：骑行卡片根据时长动态渐变（浅蓝/暖黄/暖橙）
- 字体：Outfit（标题）+ DM Sans（正文），大号数字展示统计数据
- 布局：桌面端卡片网格布局，左侧导航切换视图
- 图标：react-icons + emoji图标（🚀⚡🏔📅）用于统计区
- 按钮：圆角、填充色、hover缩放效果

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 列表页 | 统计卡片 | 4列网格，大号数字+图标+环比小字，卡片微阴影 |
| 列表页 | 筛选区 | 双滑块选择器，绿色填充轨道 |
| 列表页 | 骑行卡片 | 圆角卡片，动态渐变背景，左上骑行图标，展开时柱状图动画 |
| 拼接页 | 路段选择器 | 复选框列表，选中项高亮，连接节点灰色/绿色过渡 |
| 拼接页 | 图表区 | 海拔剖面图（渐变色带）+ 速度折线图（金色标注最高点） |

### 4.3 响应式设计

- 桌面优先设计，最小宽度1024px
- 卡片网格在窄屏下从2列调整为1列
- 图表区域自适应宽度

### 4.4 动效规范

- 地图标注涟漪动画：0.3s
- 卡片展开柱状图填充：0.5s ease-out
- 筛选滑块背景渐变：0.2s
- 列表交错渐入：每项0.05s间隔
- 节点悬停放大变色：0.2s
- 环比数据颜色变化：0.2s

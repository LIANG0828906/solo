## 1. 产品概述
文创众筹管理平台，帮助小型文创社团在线创建和管理文创产品众筹项目，支持赞助人认筹并实时追踪进度。
- 解决众筹信息分散、认筹进度不透明、回报管理混乱的问题
- 目标用户：文创社团管理员、文创产品爱好者（赞助人）
- 核心价值：提供透明、高效、美观的众筹全流程管理体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 社团管理员 | 系统预置 | 创建众筹项目、管理项目状态、审核发布 |
| 赞助人 | 无需注册 | 浏览项目、认筹支持、查看进度和记录 |

### 2.2 功能模块
1. **项目列表页**：瀑布流展示已发布项目，按筹款进度排序
2. **项目详情页**：进度环展示、倒计时、回报档位选择、认筹表单、认筹记录
3. **后台管理**：创建新项目、项目列表管理、审核发布

## 3. 核心流程

### 管理员创建项目流程
```mermaid
flowchart LR
    A["填写项目信息"] --> B["添加回报档位<br/>(至少3个)"]
    B --> C["提交创建"]
    C --> D["项目进入<br/>'审核中'状态"]
    D --> E["管理员审核通过"]
    E --> F["自动发布到前台"]
```

### 赞助人认筹流程
```mermaid
flowchart LR
    A["浏览项目列表"] --> B["点击查看详情"]
    B --> C["选择回报档位"]
    C --> D["填写认筹信息"]
    D --> E["提交认筹"]
    E --> F["生成唯一认筹编号"]
    F --> G["进度环实时更新"]
    G --> H["显示认筹记录"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：温暖橙黄色系 (#FF9500, #FFB74D, #FFCC80)
- **背景色**：白色卡片 (#FFFFFF)，浅灰背景 (#FFF8F1)
- **强调色**：绿色渐变进度 (#4CAF50 → #81C784)
- **按钮风格**：圆角8px，悬停加深背景色并放大1.05倍，点击按压效果scale(0.95)
- **字体**：标题使用温暖圆润字体，正文使用易读字体，倒计时使用等宽字体 (monospace)
- **布局风格**：卡片式布局，圆角12px，柔和阴影，悬停上浮动画
- **图标风格**：线性简约图标，与橙黄主题配色一致

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 项目列表页 | 导航栏 | 渐变背景 (左#FF9500→右#FFB74D)，品牌文字，圆角按钮 |
| 项目列表页 | 项目卡片 | 圆角12px，阴影0 4px 12px rgba(255,149,0,0.1)，悬停translateY(-4px) |
| 项目列表页 | 进度环 | Canvas绘制，绿色渐变填充，0.5秒过渡动画 |
| 项目详情页 | 倒计时 | 等宽字体，每秒白→浅橙闪烁渐变 |
| 项目详情页 | 档位卡片 | 白色背景，边框hover变橙色，选中状态高亮 |
| 项目详情页 | 弹窗 | 半透明遮罩，圆角16px，滑入动画 |
| 认筹记录 | 记录项 | 时间轴样式，昵称标签，寄语缩略 |

### 4.3 响应式
- **桌面端**：每行3列瀑布流
- **平板端**：每行2列
- **手机端**：每行1列
- 触摸优化：按钮最小高度44px，点击区域放大

### 4.4 动画效果
- 页面加载：卡片 staggered 淡入 (animation-delay)
- 进度环：0.5秒 ease-out 从旧值过渡到新值
- 倒计时：每秒颜色渐变闪烁 (white → #FFCC80 → white)
- 按钮：悬停 transform scale(1.05)，点击 transform scale(0.95) 100ms
- 卡片：悬停 translateY(-4px)，box-shadow 加深

# 在线食谱创作与分享社区应用 - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
本产品是一个面向美食爱好者的在线食谱创作与分享社区，用户可以在平台上搜索、创建、编辑食谱，与其他用户互动交流。

### 1.2 目标用户
- 家庭主妇/煮夫：寻找日常菜谱灵感
- 美食爱好者：分享自创菜品
- 初学者：学习基础烹饪技巧

### 1.3 产品价值
- 提供便捷的食谱搜索与浏览体验
- 降低食谱创作与分享门槛
- 构建活跃的美食爱好者社区

---

## 2. 功能需求

### 2.1 食谱搜索与浏览

| 功能ID | 功能名称 | 功能描述 | 优先级 |
|--------|----------|----------|--------|
| FR-001 | 食谱列表展示 | 以卡片形式展示所有食谱，包含封面图、标题、作者、评分等信息 | P0 |
| FR-002 | 关键词搜索 | 支持按食谱标题、食材、标签进行模糊搜索 | P0 |
| FR-003 | 食谱详情 | 展示食谱完整信息，包括食材清单、制作步骤、烹饪时间等 | P0 |

### 2.2 食谱创作与编辑

| 功能ID | 功能名称 | 功能描述 | 优先级 |
|--------|----------|----------|--------|
| FR-101 | 创建食谱 | 用户可创建新食谱，填写标题、封面图、食材、步骤、烹饪时长等信息 | P0 |
| FR-102 | 编辑食谱 | 用户可编辑自己创建的食谱内容 | P0 |
| FR-103 | 删除食谱 | 用户可删除自己创建的食谱 | P1 |
| FR-104 | 食材管理 | 支持添加/删除/编辑食谱中的食材条目（食材名、用量） | P0 |
| FR-105 | 步骤管理 | 支持添加/删除/编辑烹饪步骤，支持步骤排序 | P0 |

### 2.3 社区互动

| 功能ID | 功能名称 | 功能描述 | 优先级 |
|--------|----------|----------|--------|
| FR-201 | 收藏食谱 | 用户可收藏感兴趣的食谱，在个人收藏夹中查看 | P0 |
| FR-202 | 取消收藏 | 用户可取消已收藏的食谱 | P0 |
| FR-203 | 星级评分 | 用户可对食谱进行 1-5 星评分 | P0 |
| FR-204 | 查看评分 | 展示食谱的平均星级和评分人数 | P0 |
| FR-205 | 发表评论 | 用户可在食谱下发表评论 | P0 |
| FR-206 | 查看评论 | 展示食谱下的所有评论，包含评论者、时间、内容 | P0 |

### 2.4 用户系统（本地模拟）

| 功能ID | 功能名称 | 功能描述 | 优先级 |
|--------|----------|----------|--------|
| FR-301 | 本地用户标识 | 使用本地存储维护当前用户身份，用于区分创建者和评论者 | P0 |
| FR-302 | 我的食谱 | 展示当前用户创建的所有食谱 | P1 |
| FR-303 | 我的收藏 | 展示当前用户收藏的所有食谱 | P1 |

## 3. 非功能需求

### 3.1 性能需求
- 页面首屏加载时间 < 2s
- 搜索响应时间 < 500ms
- 本地存储读写无明显延迟

### 3.2 兼容性需求
- 支持主流浏览器：Chrome、Firefox、Safari、Edge 最新两个版本
- 响应式设计，适配桌面端和平板端

### 3.3 可用性需求
- 界面操作直观，核心功能 3 步内可达
- 表单提供清晰的输入提示和错误反馈

### 3.4 数据存储
- 使用浏览器 localStorage 进行数据持久化
- 数据模型包含：食谱、评论、收藏、评分、用户

---

## 4. 视觉设计规范

### 4.1 色彩系统

| 色彩类型 | 色值 | 用途 |
|----------|------|------|
| 主色（背景色） | #F5E6D3（米白） | 页面背景、卡片背景 |
| 辅色 | #D4A574（暖棕） | 导航栏、按钮边框、次要文字 |
| 强调色 | #E88D3E（橙色） | 主按钮、链接、评分星星、高亮元素 |

### 4.2 设计风格
- 整体风格：温暖、简约、有食欲感
- 卡片采用圆角设计，柔和阴影
- 字体：无衬线字体，清晰易读

---

## 5. 数据模型（概念层）

### 5.1 食谱（Recipe）
- id: 唯一标识
- title: 食谱标题
- coverImage: 封面图 URL
- description: 简介
- ingredients: 食材列表 [{ name, quantity }]
- steps: 步骤列表 [{ order, content, image? }]
- cookTime: 烹饪时长（分钟）
- authorId: 作者 ID
- authorName: 作者名称
- tags: 标签列表
- createdAt: 创建时间
- updatedAt: 更新时间

### 5.2 评论（Comment）
- id: 唯一标识
- recipeId: 关联食谱 ID
- userId: 评论用户 ID
- userName: 评论用户名称
- content: 评论内容
- createdAt: 创建时间

### 5.3 收藏（Favorite）
- id: 唯一标识
- recipeId: 关联食谱 ID
- userId: 用户 ID
- createdAt: 收藏时间

### 5.4 评分（Rating）
- id: 唯一标识
- recipeId: 关联食谱 ID
- userId: 用户 ID
- score: 评分（1-5）
- createdAt: 评分时间

---

## 6. 页面结构

### 6.1 页面清单
1. 首页 - 食谱列表与搜索
2. 食谱详情页 - 查看食谱完整信息与互动
3. 创建/编辑食谱页 - 表单填写与预览
4. 我的收藏页 - 收藏食谱列表

### 6.2 导航结构
- 顶部导航栏：Logo、搜索框、创建食谱按钮、我的收藏入口

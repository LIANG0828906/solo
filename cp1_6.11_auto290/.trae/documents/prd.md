## 1. 产品概述

古代书院体能考核在线系统，用于管理弟子骑射、投壶、蹴鞠三项传统体能考核的成绩登记、排行榜生成与月度训练计划制定。面向书院教师与学生两类用户，实现考核全流程数字化管理。

- 解决传统纸质记录效率低、排名计算繁琐、训练计划传达不便等问题
- 目标用户：古代书院教师（管理端）与弟子（学生端）

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 学生 | 姓名+学号登录 | 提交考核成绩、查看排行榜、查看训练计划 |
| 教师 | 工号+密码登录 | 增删改查成绩、生成排行榜、制定训练计划 |

### 2.2 功能模块

1. **学生登录/首页**：当月训练计划列表、排名变化趋势折线图
2. **骑射考核页**：Canvas靶面绘制、10环制成绩输入
3. **投壶考核页**：Canvas青铜壶绘制、投中计分输入
4. **蹴鞠考核页**：Canvas球门绘制、力度条与角度选择器
5. **排行榜页**：月度各考核项前十名展示
6. **训练计划页**：教师编辑发布月度训练任务

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 角色切换 | 学生/教师登录入口，输入对应凭证 |
| 学生首页 | 训练计划列表 | 当月计划按日期排序，过期项灰色显示 |
| 学生首页 | 排名趋势 | 近三个月排名折线图 |
| 骑射考核 | 靶面展示 | Canvas绘制10环靶（红心#FF4500、黄#FFD700、蓝#1E90FF、灰#D3D3D3） |
| 骑射考核 | 成绩输入 | 选择环数（1-10），提交至后端 |
| 投壶考核 | 壶口展示 | Canvas绘制青铜壶+蓝色云纹，壶口闪烁提示 |
| 投壶考核 | 成绩输入 | 记录10支箭中投中数（0-10） |
| 蹴鞠考核 | 球门展示 | Canvas绘制白色门框+球网 |
| 蹴鞠考核 | 成绩输入 | 力度条+角度选择器，5球逐个评分0-10 |
| 排行榜 | 综合排名 | 前十名纵向列表，前三金银铜勋章 |
| 训练计划 | 计划编辑 | 教师创建训练日期、项目、内容、目标分数 |
| 教师管理 | 成绩管理 | 按日/周/月筛选，手动删除或修改成绩 |

## 3. 核心流程

**学生考核流程**：学生登录 → 选择考核项目 → 查看考核图示 → 输入成绩 → 提交保存 → 查看排行榜

**教师管理流程**：教师登录 → 查看考核记录（按时间筛选）→ 修改/删除成绩 → 查看排行榜 → 制定训练计划

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白背景 #F5F0E1，深褐按钮边框 #4A2C1A
- **辅助色**：淡黄输入框 #FFF8DC，朱砂提示 #8B0000
- **字体**：标题使用隶书体（font-family: 'LiSu', 'STLiti', serif），正文使用宋体
- **按钮风格**：深褐色圆角按钮，悬停时微亮
- **布局风格**：仿古书卷风格，卡片式布局，带卷轴边框装饰
- **动画**：页面切换0.3秒淡入淡出，成绩提交0.5秒卷轴展开动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 登录表单 | 卷轴式卡片，隶书标题，深褐按钮 |
| 学生首页 | 训练计划 | 卡片列表，过期项灰色，日期排序 |
| 学生首页 | 排名趋势 | 简易折线图，近三月数据 |
| 骑射页 | 左侧靶面 | Canvas 10环靶，红心/黄/蓝/灰四色环 |
| 骑射页 | 右侧表单 | 淡黄输入框，深褐提交按钮 |
| 投壶页 | 中央壶 | Canvas青铜壶，蓝色云纹，闪烁壶口 |
| 投壶页 | 下方表单 | 投中计数，10箭逐次记录 |
| 蹴鞠页 | 中央球门 | Canvas白色门框+球网 |
| 蹴鞠页 | 下方控件 | 力度滑条+角度选择器，5球逐个评分 |
| 排行榜 | 纵向列表 | 前三名金#D4AF37/银#C0C0C0/铜#CD7F32勋章 |
| 训练计划 | 编辑表单 | 日期、项目、内容、目标分数输入 |

### 4.3 响应式适配

- 桌面优先设计，移动端（<768px）自适应
- 移动端考核页面：改为上下布局（上为成绩输入，下为考核图示）
- 移动端排行榜：单列显示

### 4.4 Canvas绘制规范

- 骑射靶面：10环同心圆，中心#FF4500（10-9环）、#FFD700（8-7环）、#1E90FF（6-5环）、#D3D3D3（4-1环）
- 投壶壶身：青铜色壶体，蓝色云纹装饰，壶口闪烁动画
- 蹴鞠球门：白色门框，灰色球网网格线
- 动画帧率不低于30fps

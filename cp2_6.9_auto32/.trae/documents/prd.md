## 1. 产品概述

唐代金银器坊全栈Web应用，模拟唐代少府监金银作坊的管理流程，让用户以掌作匠师身份管理银器制作全链条。

- 核心目的：通过沉浸式交互体验，还原唐代金银器制作工艺，实现从订单、制作到入库的全流程可视化管理
- 目标用户：对唐代工艺文化感兴趣的用户、游戏化管理系统的使用者
- 产品价值：将传统工艺与现代Web技术结合，提供兼具教育性和娱乐性的交互体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 掌作匠师 | 直接进入 | 订单管理、工匠指派、工艺操作、品级评定、库存监控、报表查看 |

### 2.2 功能模块

1. **订单管理面板**：订单列表展示、紧急程度标注、订单详情加载
2. **作坊工作台**：工匠槽位、银器工位、拖拽操作、工艺动画、实时状态更新
3. **原料库存监控**：库存量展示、消耗统计、低库存预警
4. **品级判定系统**：工序完成检测、综合评分、品级评定、考语展示
5. **统计报表面板**：入库统计图表、原料消耗分析、时间筛选

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作台 | 订单列表 | 左侧展示所有银器订单，红黄绿标注紧急程度，点击加载详情 |
| 主工作台 | 工匠工位 | 中央4个工匠槽位，可拖拽原料和工具触发操作 |
| 主工作台 | 银器渲染区 | 3D半透明SVG展示银器坯件，实时更新重量和形状 |
| 主工作台 | 原料/工具区 | 左侧原料区（银块、锡、铜），右侧工具区（锤子等） |
| 主工作台 | 顶部状态栏 | 三种原料库存量进度条，当日消耗数字 |
| 主工作台 | 预警横幅 | 原料低于阈值时底部红色闪烁预警 |
| 报表面板 | 入库簿 | 柱状图展示7日成品入库（按品级分组），饼图展示原料消耗占比 |
| 品级弹窗 | 结果展示 | 完成5道工序后弹窗显示品级、考语和彩带特效 |

## 3. 核心流程

```mermaid
flowchart LR
    A["进入作坊"] --> B["查看订单列表"]
    B --> C["点击订单加载坯件"]
    C --> D["拖拽原料/工具到工匠"]
    D --> E["触发工艺动画"]
    E --> F["更新银器状态"]
    F --> G["完成5道工序"]
    G --> H["品级评定弹窗"]
    H --> I["成品入库"]
    I --> J["生成统计报表"]
```

用户流程简述：用户进入应用后，左侧展示待处理订单列表，点击某一订单后中央工作台加载对应银器坯件和原料清单。用户通过拖拽原料（银块、锡、铜）到工匠身上触发熔炼，拖拽工具（锤子）触发锤揲。完成熔炼→锤揲→錾刻→鎏金→抛光五道工序后，系统根据完成时间和失误次数评定品级，弹出考语和特效，成品自动入库。顶部状态栏实时监控原料库存，低于阈值时触发预警。右侧入库簿可查看统计报表。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：唐风朱红 `#c0392b`、浅金 `#d4af37`
- **背景**：仿古宣纸纹理（CSS重复渐变+图案叠加）
- **按钮风格**：圆角6px，微凸阴影，朱红底色配金色描边
- **字体**：思源宋体（正文），配合古风装饰性标题
- **布局风格**：三栏式布局，卡片式组件，深木色工作台模拟红木案几
- **动画风格**：framer-motion实现0.2-0.4秒缓动，拖拽放大1.1倍半透明，放置失败弹性回弹

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主工作台 | 订单列表 | 宽240px，卡片微凸阴影圆角6px，红黄绿紧急度标签，隐藏滚动条 |
| 主工作台 | 中央工作台 | flex:1，深木色`#6b4423`背景，素色锦缎纹理，4个圆形工匠头像带星标技能等级 |
| 主工作台 | 银器SVG | 半透明银色`#c0c0c0`，重量数字渐变，形状随工序变化 |
| 主工作台 | 原料/工具区 | 彩色圆点标识原料（银灰、锡白、铜橙），工具图标可拖拽 |
| 主工作台 | 顶部状态栏 | 条状进度条展示库存，低于阈值红色1Hz闪烁 |
| 报表面板 | 入库簿 | 宽300px，可折叠，柱状图（上上金、上中银、中灰、下深灰），饼图原料占比 |
| 品级弹窗 | 考语展示 | 居中弹窗，彩带动画，古风考语文案 |

### 4.3 响应式设计

- **桌面端（>768px）**：三栏式布局（左订单/中工作台/右报表）
- **平板端（≤768px）**：上中下布局（顶部状态栏+订单、中部工作台、底部报表）
- **触摸优化**：拖拽区域扩大，点击目标最小44x44px

### 4.4 交互特效

- **拖拽反馈**：元素放大1.1倍，透明度0.8，阴影加深
- **工艺动画**：熔炼（银块液态流动0.5s）、锤揲（锤子上下敲击0.3s循环）
- **完成标识**：每道工序完成后工匠槽位绿色对勾
- **品级特效**：彩带飘落动画，金光闪烁效果
- **预警动画**：进度条1Hz红色闪烁，底部横幅滑入

## 1. 产品概述

社区咖啡馆在线点单与取餐管理应用，为顾客提供便捷的手机/电脑端浏览菜单、下单并预约取餐服务，同时为咖啡师提供后台订单实时查看与库存管理功能。

- 核心目标：提升咖啡馆运营效率，减少顾客排队等待时间，优化点单体验
- 目标用户：社区咖啡馆顾客（C端）、咖啡师/店家（B端）
- 市场价值：本地化、轻量化的咖啡馆数字化解决方案

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 顾客 | 无需登录 | 浏览菜单、添加购物车、下单、预约取餐、查询订单状态 |
| 咖啡师 | 后台管理 | 实时接收订单、管理饮品库存、更新订单状态 |

### 2.2 功能模块

1. **菜单浏览页**：分类标签切换、商品卡片展示、商品详情弹窗
2. **购物车面板**：商品增删改、数量调整、价格计算
3. **取餐预约**：时间段选择、订单提交、订单确认
4. **订单状态追踪**：进度条展示、状态自动刷新、四阶段进度

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 菜单浏览页 | 分类标签栏 | 冰饮/热饮/轻食三个分类，横向滑动切换动画（0.3秒缓动） |
| 菜单浏览页 | 商品卡片网格 | 2列网格布局，卡片含缩略图、名称、价格，暖色调阴影 |
| 菜单浏览页 | 商品详情浮层 | 点击卡片弹出，从卡片位置放大展开，半透明黑色遮罩 |
| 购物车栏 | 悬浮购物车 | 底部悬浮浅灰色条，加入商品时上浮并显示数量徽标动画 |
| 购物车面板 | 购物车详情 | 从底部向上滑动展开（0.4秒弹性缓动），显示商品列表、数量加减、小计 |
| 取餐时间选择 | 时间选择器 | 显示当前30分钟内时段，每15分钟一个选项，横向可滚动标签 |
| 订单确认 | 提交订单 | 加载状态动画，成功弹出确认对话框（渐入0.3秒），显示订单号和取餐时间 |
| 订单追踪 | 进度条 | 四阶段（已提交→制作中→取餐中→已完成），每10秒自动刷新，平滑过渡动画 |

## 3. 核心流程

顾客进入应用 → 浏览菜单（按分类筛选）→ 点击商品查看详情 → 添加到购物车 → 展开购物车确认商品 → 选择取餐时间 → 提交订单 → 查看订单状态实时更新 → 取餐完成

    E --> F["调整数量"]

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#8B4513（咖啡棕）
- **辅助色**：#F5DEB3（小麦色）、#FFF8DC（奶油色）背景
- **强调色**：#D2691E（肉桂色）按钮，#CD853F（秘鲁色）高亮
- **卡片圆角**：12px
- **卡片阴影**：rgba(139, 90, 43, 0.2) 暖色调阴影
- **字体**：标题使用有衬线字体体现咖啡馆质感，正文使用清晰易读的无衬线字体
- **整体风格**：温暖、舒适、有质感的社区咖啡馆氛围

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 菜单浏览页 | 分类标签 | 咖啡棕底色标签，选中态小麦色文字，横向滑动动画 |
| 菜单浏览页 | 商品卡片 | 圆角12px，柔和阴影，缩略图占卡片宽80%，价格带¥符号 |
| 购物车栏 | 悬浮条 | 浅灰色背景，底部悬浮，徽标弹入动画 |
| 购物车面板 | 展开面板 | 半屏高度，底部滑入，弹性缓动 |
| 订单进度条 | 四阶段进度 | 咖啡色高亮当前阶段，绿色已完成阶段，箭头渐变指示 |

### 4.3 响应式设计

- 桌面端：最大宽度1200px居中，2列商品网格，购物车悬浮右下角
- 移动端：单列布局，购物车栏固定底部不可滚动
- 图片自适应：卡片内图片按比例缩放
- 触控优化：按钮最小尺寸44px，适合手指点击

### 4.4 微动效设计

- 分类切换：横向滑动 0.3s 缓动
- 购物车徽标：加入商品时缩小再弹回 0.2s
- 购物车展开：底部向上滑动 0.4s 弹性缓动
- 数量更新：数字淡入 0.1s
- 详情弹窗：从卡片位置放大展开
- 进度条更新：平滑过渡 0.5s
- 购物车图标抖动：添加商品时 0.2s
- 错误反馈：边框变红并闪烁 0.3s

## 1. 产品概述

在线协作食谱管理及智能采购清单生成应用，让家庭成员或室友能共同维护食谱库，并根据每周选定的菜单自动生成按超市区域分组的采购清单。

- 核心目标：解决家庭/室友群体在餐饮规划、食材采购协同上的效率问题，减少重复采购和遗漏
- 目标用户：家庭成员、合租室友、小型团体用户

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家庭成员/室友 | 邀请码加入房间 | 浏览食谱、创建/编辑食谱、规划每周菜单、查看并勾选采购清单 |

### 2.2 功能模块
1. **食谱主页 (RecipeHome)**：食谱网格展示、搜索筛选、卡片悬浮交互
2. **食谱详情页 (RecipeDetail)**：食材列表、烹饪步骤、评论评分、加入本周菜单
3. **菜单规划 (MealPlanner)**：7天3餐网格、食谱添加/拖拽排序、WebSocket实时同步
4. **采购清单 (ShoppingList)**：按超市区域分组、勾选已购、价格估算、强制同步

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| RecipeHome | 顶部搜索栏 | 300ms防抖搜索，按食谱名/主要食材过滤 |
| RecipeHome | 食谱卡片网格 | 展示食谱名、作者、时长、难度、缩略图，hover阴影加深上移 |
| RecipeDetail | 食材列表 | 类别色块区分（蔬菜/肉类/香料），带数量单位 |
| RecipeDetail | 烹饪步骤 | 有序列表，当前步骤高亮 |
| RecipeDetail | 评论评分区 | 1-5星评分组件，WebSocket实时推送新评论，下滑动画 |
| RecipeDetail | 加入菜单按钮 | 跨模块调用MealPlanner，自动填入下一空白餐次 |
| MealPlanner | 7x3网格 | 每行一天，每列早/午/晚餐，点击空白格弹出食谱选择器 |
| MealPlanner | 拖拽交互 | 单元格可拖拽到其他天同一餐次，半透明克隆跟随 |
| MealPlanner | WebSocket同步 | 所有菜单修改广播给房间内所有用户，toast提示 |
| ShoppingList | 区域分组清单 | 蔬菜水果/肉类海鲜/乳蛋/调味品/粮油干货/其他，缩进显示 |
| ShoppingList | 勾选交互 | 圆形复选框，勾选后文字删除线+变灰 |
| ShoppingList | 统计栏 | 已购买/总数，价格估算（若有均价数据） |
| ShoppingList | 同步按钮 | 强制拉取最新版本清单 |

## 3. 核心流程

用户进入房间 → 浏览食谱库（搜索/筛选） → 查看食谱详情 → 添加到本周菜单 → 在菜单规划中调整（拖拽/替换） → 系统自动生成采购清单 → 按超市区域分组展示 → 购物时逐项勾选 → 实时同步给所有家庭成员

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：#f5a623（暖橘色）- 温暖家庭厨房氛围
- **辅助色**：#4a90d9（蓝色）- 用于强调和交互元素
- **背景色**：#fff8f3（米白色）- 柔和不刺眼
- **卡片样式**：圆角12px，盒阴影 0 2px 8px rgba(0,0,0,0.08)，hover时 0 4px 16px rgba(0,0,0,0.12) 并上移2px
- **按钮样式**：主色填充+白色文字，hover时亮度提升10%
- **字体**：标题使用带手写感的衬线体提升温度感，正文使用现代无衬线体保证可读性
- **动效**：列表项0.3s淡入淡出，网格切换0.2s缩放，新评论下滑动画

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| RecipeHome | 顶部导航栏 | 暖橘色背景logo + 搜索框居中 + 右侧菜单规划/采购清单入口 |
| RecipeHome | 食谱卡片 | 12px圆角卡片，上方缩略图/渐变，下方信息区（名称+作者行+时长难度标签） |
| RecipeDetail | 大图区 | 全宽食谱图（无则隐藏），下方食谱名+评分概览 |
| RecipeDetail | 食材列表 | 每行左侧彩色类别方块 + 食材名 + 数量单位 |
| RecipeDetail | 步骤区 | 左侧大号圆形数字序号 + 右侧步骤描述 |
| RecipeDetail | 评论区 | 头像+昵称+星级+时间+内容，底部输入框+提交按钮 |
| MealPlanner | 顶部栏 | 当前周日期范围 + 生成采购清单按钮 |
| MealPlanner | 网格 | 7列3行，表头为星期，左列为餐次名，单元格12px圆角 |
| ShoppingList | 顶部 | 同步按钮 + 筛选/打印入口 |
| ShoppingList | 区域分组 | 区域标题加粗，区域内项目缩进20px |
| ShoppingList | 底部统计 | 进度条 + 已购/总数 + 预估金额 |

### 4.3 响应式设计
- 桌面优先，768px以下自动切换单列布局
- 网格变为垂直堆叠，搜索框和按钮宽度100%
- 触摸优化：点击热区≥44px，拖拽增加触觉反馈

### 4.4 视觉差异化
- 食谱卡片缩略图缺失时使用温暖的渐变色（橘→黄→粉）而非占位图
- 食材类别使用柔和色块而非硬边框，增强"食材本真"感
- 菜单规划单元格内食谱名使用稍带手写风格的字体，增添家庭感
- 整体微交互：卡片hover有2px上移+阴影加深，按钮hover轻微发光

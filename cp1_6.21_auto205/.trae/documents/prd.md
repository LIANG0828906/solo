## 1. 产品概述
动态代码演示沙盒是一款面向技术博客作者和前端培训讲师的可交互代码展示平台。读者无需跳转到 Codepen 或 CodeSandbox 等外部站点，即可在线查看、编辑和运行代码片段，并实时看到输出结果。

- 核心价值：降低技术内容创作门槛，提升读者学习体验，实现代码演示的无缝集成
- 目标用户：技术博客作者、前端培训讲师、编程学习者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 编辑代码、运行代码、保存/加载/删除代码片段 |

### 2.2 功能模块
1. **代码编辑模块**：支持 HTML、CSS、JavaScript 三种代码块的分别编辑，提供语法高亮、行号显示、自动补全
2. **实时预览模块**：iframe 沙盒环境运行代码，实时渲染页面效果，控制台输出日志和错误信息
3. **片段管理模块**：保存代码片段到后端，支持片段列表展示、加载、删除操作
4. **布局交互模块**：可拖动分隔条调整编辑区和预览区比例，响应式适配多端设备

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主页面 | 代码编辑区 | Monaco Editor 编辑器，支持 HTML/CSS/JS 三种语言切换，语法高亮、行号、自动补全 |
| 主页面 | 实时预览区 | iframe 沙盒渲染运行结果，下方显示控制台输出区域 |
| 主页面 | 运行按钮 | 点击后将代码注入 iframe 执行，200ms 内完成渲染 |
| 主页面 | 可拖动分隔条 | 3px 宽，支持拖拽调整左右宽度比例，悬停变色效果 |
| 主页面 | 代码片段列表 | 左侧展示已保存的片段列表，支持加载和删除 |
| 主页面 | 保存片段弹窗 | 输入名称（最多40字符）和描述（最多200字符），保存到后端 |

## 3. 核心流程

用户进入页面 → 编辑 HTML/CSS/JS 代码 → 点击运行按钮 → 代码在 iframe 沙盒中执行 → 预览区实时展示效果 → 控制台显示输出 → 点击保存按钮 → 输入名称和描述 → 片段保存到后端 → 片段列表新增记录 → 点击列表项加载代码 → 继续编辑运行

    H --> I["保存到后端 API"]

    K --> B

## 4. 用户界面设计

### 4.1 设计风格
- 主题：深色主题，专业技术感
- 主色调：#0F172A（背景）、#1E293B（编辑区背景）、#6366F1（品牌色/按钮）
- 辅助色：#818CF8（链接/按钮文字）、#F87171（错误信息）、#475569（分隔条）
- 字体：代码区使用 monospace 字体，界面使用无衬线字体
- 布局：左右分栏（桌面）、上下堆叠（移动端）
- 圆角：统一使用 8px-12px 圆角，营造现代感
- 动效：所有交互元素 0.2s-0.3s 过渡动画，cubic-bezier(0.4, 0, 0.2, 1) 缓动函数

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|---------|----------|----------|
| 主页面 | 整体布局 | 深色背景 #0F172A，左侧片段列表（280px）+ 中间编辑区 + 右侧预览区 |
| 主页面 | 代码编辑区 | 背景 #1E293B，圆角 12px，内边距 16px，Monaco Editor 编辑器 |
| 主页面 | 实时预览区 | 背景 #FFFFFF，圆角 12px，内边距 20px，iframe + 控制台输出区（150px 高，#0F172A） |
| 主页面 | 分隔条 | 3px 宽，#475569，悬停变 #6366F1，0.2s 过渡 |
| 主页面 | 运行按钮 | 120x40px，#6366F1，圆角 8px，文字 #FFFFFF，悬停下移 2px 阴影加深，点击闪缩 0.1s |
| 主页面 | 片段列表 | 宽度 280px，背景 #111827，圆角 12px，每行 48px，悬停 #1F2937，0.2s 过渡 |
| 主页面 | 可点击元素 | 悬停放大 1.05 倍，阴影过渡 0.3s cubic-bezier(0.4, 0, 0.2, 1) |

### 4.3 响应式
- 设计原则：桌面优先，移动端自适应
- 断点：< 768px 时左右分栏变为上下堆叠
- 移动端：编辑区和预览区各占 50% 高度，分隔条变为水平方向
- 触摸优化：按钮最小触控区域 44x44px，列表项适当增加内边距

### 4.4 性能要求
- 代码运行后 iframe 渲染时间 ≤ 200ms
- 编辑器加载时间 ≤ 500ms
- 片段保存/加载响应时间 ≤ 300ms
- 界面动画帧率 ≥ 60fps

## 1. 产品概述

在线食谱创意组合与分享应用——让用户从现有食材中智能发现可制作的菜肴，将个人创意菜品组合保存为公开食谱，并获得其他用户的点赞和评论反馈。

- 核心目标：帮助用户根据现有食材快速找到可制作的菜肴，激发烹饪创意
- 目标用户：家庭烹饪爱好者、美食博主、喜欢尝试新菜谱的人群

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览食谱、搜索食材、查看详情、点赞和评论 |
| 创作者 | 无需注册 | 在普通用户基础上可发布创意版食谱 |

### 2.2 功能模块

1. **首页**：轮播推荐、食材输入与匹配、食谱网格展示、搜索与筛选
2. **食谱详情页**：食谱步骤展示、创意修改与发布、点赞与评论互动

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 轮播推荐区 | 展示点赞数最多的3份食谱，4秒自动轮播，左右箭头切换，底部圆点指示器 |
| 首页 | 食材输入区 | 支持中文模糊搜索自动补全，最多8种食材，"发现食谱"按钮触发匹配 |
| 首页 | 搜索与筛选栏 | 搜索框（食谱名称/食材关键词）+ 菜系/难度/时间筛选 + 重置按钮 |
| 首页 | 食谱网格区 | 三列网格卡片，展示封面图、名称、用户信息、点赞数、匹配度进度条 |
| 食谱详情页 | 步骤展示 | 展示食材清单、烹饪步骤、烹饪时间、难度等级 |
| 食谱详情页 | 创意修改 | 替换/增加食材、调整步骤顺序、缩短烹饪时间，发布创意版（名称+描述+封面图） |
| 食谱详情页 | 互动区 | 点赞按钮（心形图标，可取消）+ 评论区（最近10条，头像占位符+昵称+内容+时间） |

## 3. 核心流程

**食材匹配流程**：用户输入食材 → 模糊搜索自动补全 → 点击"发现食谱" → 后端匹配计算 → 按匹配度降序返回 → 前端高亮展示（>70%绿色边框，<30%灰色+标注"需补充食材"）

**创意发布流程**：查看食谱详情 → 修改食材/步骤/时间 → 点击"发布创意版" → 填写名称+描述+封面图 → 后端保存并关联原食谱 → 首页展示紫色"创意版"标签

**互动流程**：查看食谱详情 → 点赞/取消点赞 → 评论（140字限制）→ 实时更新数据库 → 刷新不丢失

## 4. 用户界面设计

### 4.1 设计风格

- 主色：暖色调，背景米白 #FAF5EB，头部导航深褐 #4A3728，文字深灰 #2C2C2C
- 强调色：橙色 #E67E22（按钮/链接，hover变深 #D35400）
- 卡片风格：圆角12px，浅灰阴影，hover上浮4px（0.3s过渡）
- 字体：正文14-16px，食谱名称18px粗体
- 布局：三列网格（卡片宽度300px），导航栏顶部固定

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 深褐#4A3728背景，白色文字，橙色按钮 |
| 首页 | 轮播推荐区 | 全宽轮播，左右箭头，底部圆点，4秒间隔 |
| 首页 | 食材输入区 | 搜索框+下拉自动补全+食材标签列表+发现食谱按钮 |
| 首页 | 筛选栏 | 搜索框+菜系/难度/时间下拉+重置按钮 |
| 首页 | 食谱网格 | 三列卡片，封面图16:9，名称18px粗体，用户头像+昵称，爱心点赞数，匹配度进度条（红到绿渐变） |
| 食谱详情页 | 信息区 | 封面图+食谱名称+作者+烹饪时间+难度+食材清单 |
| 食谱详情页 | 步骤区 | 有序步骤列表，支持拖拽排序修改 |
| 食谱详情页 | 创意发布 | 橙色"发布创意版"按钮，弹出表单（名称+描述+封面图上传） |
| 食谱详情页 | 互动区 | 心形点赞按钮（缩放1.2倍0.2s动画），评论区（10条，下滑入0.3s动画），输入框+140字计数 |

### 4.3 响应式设计

- 桌面端（>1024px）：三列网格布局
- 平板端（768-1024px）：两列网格布局
- 移动端（<768px）：单列布局，轮播图自适应宽度

### 4.4 动效设计

- 页面切换：内容区域0.3秒淡入效果
- 点赞动画：爱心缩放1.2倍再恢复（0.2秒）
- 新评论插入：从顶部下滑入（0.3秒）
- 卡片悬浮：上浮4px，0.3秒过渡
- 轮播切换：4秒间隔自动轮播

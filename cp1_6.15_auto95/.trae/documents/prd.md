## 1. 产品概述

幻境裂隙是一款基于浏览器的3D Roguelike地牢探索游戏，玩家操控角色在随机生成的地牢中探索战斗、拾取装备并挑战Boss。采用Three.js实现3D渲染，React构建游戏UI覆盖层，Express后端管理地图种子与掉落逻辑。

- 目标用户：喜欢Roguelike和暗黑奇幻风格的玩家
- 核心价值：每次游戏体验不同的随机地牢、紧张刺激的实时战斗与装备驱动的成长系统

## 2. 核心功能

### 2.1 功能模块

1. **标题屏幕**：动态粒子星空背景，开始新游戏/继续游戏按钮
2. **地牢生成**：基于后端种子的随机房间与走廊布局，Three.js 3D渲染
3. **玩家控制与战斗**：WASD移动、翻滚闪避、近战/远程攻击
4. **敌人AI系统**：巡逻/追击/攻击三种状态，类型随层数变化
5. **装备与道具系统**：武器/护甲/饰品，背包界面，装备对比
6. **Boss战**：每5层Boss房间，三种攻击模式
7. **游戏HUD**：生命/能量环形条、层数、金币、动作按钮

## 3. 核心流程

玩家打开游戏 → 标题屏幕 → 点击开始新游戏 → 后端生成地牢种子 → 前端渲染第1层地牢 → 玩家探索房间、战斗、拾取装备 → 找到楼梯进入下一层 → 每5层Boss战 → 击败Boss获得稀有装备 → 继续深入 → 生命归零游戏结束

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深紫(#2a0845)和暗金(#b8860b)
- 辅助色：暗红(受击闪烁)、金色(掉落光柱/宝箱光)
- 字体：衬线字体（控制台风格），标题使用金属质感金色渐变
- 布局：3D场景全屏，UI元素覆盖层
- 动画：脉冲发光2秒周期、悬停放大1.05倍+金色描边0.2秒、点击弹性回弹0.1秒

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 标题屏幕 | 星空背景 | 100粒子，2-5px随机大小，缓慢旋转 |
| 标题屏幕 | 标题文字 | 金属质感金色渐变，2秒脉冲发光 |
| 标题屏幕 | 菜单按钮 | 暗金边框按钮，悬停放大+描边 |
| 游戏HUD | 生命/能量环形条 | 左上角，环形从顶部顺时针递减，绿→红渐变 |
| 游戏HUD | 层数与金币 | 右上角，衬线字体 |
| 游戏HUD | 动作按钮 | 底部中央，技能冷却指示器 |
| 背包界面 | 装备卡片 | 左侧滑入0.3秒，半透明深色背景，网格排列 |
| Boss血条 | 顶部中央 | 红色渐变长条，随血量减少出现裂痕纹理 |

### 4.3 响应式设计

- 桌面优先设计，UI组件使用vw单位适配
- 确保1280x720以上分辨率完整显示
- 3D场景始终全屏

### 4.4 3D场景指引

- 环境：暗黑地牢，低光照氛围，火把点光源
- 灯光：环境光低强度 + 房间内点光源模拟火把
- 相机：第三人称俯视45度角，跟随玩家
- 材质：地板随机灰砖纹理，墙壁深色石纹，天花板暗色
- 粒子：掉落光柱(金色)、能量弹(发光球体)、传送门(蓝色漩涡)
- 性能：60FPS目标，同时活跃敌人≤8，粒子≤200

## 1. 产品概述
本应用是一款面向小型独立书店店主的主题书展策划工具，解决传统书展策划中选书逻辑不透明、主题搭配不直观、推广物料制作耗时的痛点。
- 核心目标：通过可视化拖拽方式，帮助书店主快速完成书展策划、预览和物料生成，提升策划效率与展示效果
- 目标用户：小型独立书店经营者、文化空间策展人

## 2. 核心功能

### 2.1 功能模块
1. **书库管理面板**：左侧320px宽面板，按分类筛选图书，卡片式展示
2. **中央展区**：暖黄色背景区域，支持多主题标签创建、书籍拖拽分类、标签排序
3. **海报生成器**：自动生成虚拟海报与瀑布流书单，支持PNG导出
4. **历史管理面板**：右侧滚动列表，管理草稿与已发布书展
5. **预览系统**：全屏预览、Canvas截图导出(1920x1440px)

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主工作台 | 左侧书库面板 | 按虚构/非虚构、文学/社科/艺术分类筛选图书，卡片展示封面/书名/作者/定价，支持拖拽 |
| 主工作台 | 中央展区 | 虚线网格背景，创建主题标签(8种预设色板)，拖拽书籍入标签，标签排序动画 |
| 主工作台 | 右侧操作面板 | 当前编辑主题列表、生成预览按钮、历史书展列表(草稿/已发布状态) |
| 预览弹窗 | 海报预览 | 800x600px自适应海报，衬线字体标题，瀑布流封面排列，底部Logo位 |
| 预览弹窗 | 书单预览 | 瀑布流书单(每行3-4本)，点击卡片展开完整介绍 |
| 预览弹窗 | 导出功能 | Canvas截图导出PNG(1920x1440px)，全屏查看 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：米白到暖黄渐变 (linear-gradient(135deg, #F9F6EE 0%, #FDF6E3 100%))
- **书库背景**：#F8F4E8 (纸张米色)
- **展区背景**：#FFF8E7 (淡暖黄) + 1px虚线网格 #E0D5C1
- **标签色板**：胭脂红#D85A5A、松石绿#36A2A2、薰衣草紫#9B72AA、槐米黄#D4A843、陶土棕#B5795B、茶白#F0E5D8、雾霾蓝#6C8EB2、淡丁香#C9A9C6
- **删除按钮**：#D32F2F (警告红)
- **字体**：整体无衬线体(系统默认)，海报标题衬线字体
- **卡片圆角**：8px，阴影 0 2px 8px rgba(0,0,0,0.06)
- **动画**：所有交互0.2-0.4s过渡，使用ease或cubic-bezier(0.34, 1.56, 0.64, 1)弹性曲线

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主工作台 | 三栏布局 | 左320px书库、中央自适应展区、右320px操作面板 |
| 主工作台 | 书籍卡片 | 封面缩略图、书名、作者、定价，拖拽时scale(1.05) rotate(2deg) + 3px抬升阴影 |
| 主工作台 | 主题标签 | pill形状圆角矩形，8种预设色，放入书籍时脉冲光效 box-shadow:0 0 0 4px rgba(255,255,255,0.4) |
| 主工作台 | 顶部导航 | <900px时显示汉堡按钮，切换左右抽屉 |
| 预览弹窗 | 海报区域 | 800x600px居中，标题衬线字体稍暗色，瀑布流封面排列 |
| 预览弹窗 | 书单区域 | 响应式网格(每行3-4本)，点击展开完整介绍 |
| 预览弹窗 | 操作栏 | 全屏按钮、导出PNG按钮、关闭按钮 |

### 4.3 响应式
- **桌面端**(≥900px)：经典三栏式布局，左右面板固定320px，中央自适应
- **移动端**(<900px)：左右面板收缩为320px抽屉，顶部汉堡按钮切换滑入/滑出动画，中央展区填满宽度
- **触摸优化**：react-dnd-touch-backend支持触摸拖拽

### 4.4 交互动画规范
1. 拖拽开始：书籍卡片 transform: scale(1.05) rotate(2deg) + 3px抬升阴影
2. 拖拽结束：书籍落入展区 0.3s cubic-bezier(0.34, 1.56, 0.64, 1) 弹性归位
3. 放入标签：标签放大 + 脉冲光效一次
4. 标签排序：0.4s流畅弹性动画重排
5. 所有按钮/卡片 hover：0.2s ease过渡

## 5. 性能要求
- 交互帧率 ≥40fps，拖拽与动画无卡顿
- 海报生成时间 ≤2秒(含Canvas截图)
- 导出分辨率 1920x1440px

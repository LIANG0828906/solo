## 2. 技术描述

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **3D引擎**：Three.js
- **React 3D绑定**：@react-three/fiber
- **3D辅助组件**：@react-three/drei
- **后期处理**：@react-three/postprocessing
- **状态管理**：React useState/useRef（轻量场景，无需复杂状态库

## 3. 核心数据流向

```
用户输入参数 → currentField 生成速度向量场 → particleSimulator 更新颗粒位置 → 可视化模块渲染颗粒与热力图
```

## 4. 类型定义

### 4.1 颗粒类型 Particle

```typescript
interface Particle {
  id: number;
  x: number;
  y: number;
  z: number;
  vx: number;
  vy: number;
  vz: number;
}
```

### 4.2 网格单元类型 GridCell

```typescript
interface GridCell {
  i: number;
  j: number;
  k: number;
  density: number;
  color: string;
}
```

### 4.3 洋流参数类型 CurrentParams

```typescript
interface CurrentParams {
  currentSpeed: number;
```

## 6. 性能优化策略

- 使用 InstancedMesh 渲染大量颗粒
- 热力图网格计算优化
- requestAnimationFrame 驱动模拟循环
- 帧历史记录使用环形缓冲区
- 颗粒数上限1000

## 1. 产品概述

奇幻炼金术工坊模拟器是一款沉浸式3D网页应用，让玩家扮演炼金术师，在暖色木纹工坊中通过拖拽组合基础材料来炼制魔法药水。炼制成功的药水以三维瓶子形态展示在架子上，支持拖动视角旋转观察。

- 目标用户：喜欢奇幻题材和模拟经营的休闲玩家
- 核心价值：沉浸式炼金体验 + 3D可视化药水收藏

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 炼金术师 | 无需注册 | 拖拽材料、炼制药水、查看药水详情、旋转3D场景 |

### 2.2 功能模块

1. **工坊主页**: 三维工坊场景 + 材料库 + 实验台 + 展示架 + 信息栏

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工坊主页 | 信息栏 | 顶部显示当前炼制次数和成功率 |
| 工坊主页 | 材料库 | 左侧可折叠面板，格子布局展示草药/矿物/药剂基底，支持拖拽到实验台 |
| 工坊主页 | 三维工坊场景 | 中央区域（占宽60%），Three.js渲染实验台、材料槽位、展示架、药水瓶，支持视角旋转 |
| 工坊主页 | 成品展示架 | 右侧可滚动面板，三层每层4瓶药水，点击药水瓶弹出详情面板 |
| 工坊主页 | 效果描述卡片 | 炼制成功时在实验台上方弹出，白底黑字圆角8px，淡入0.5s |
| 工坊主页 | 不相容提示 | 配方无效时红色背景弹窗提示"材料不相容" |
| 工坊主页 | 详情面板 | 悬浮半透明背景，显示药水名称/效果描述/材料清单/炼制时间戳，带呼吸光晕动画 |

## 3. 核心流程

玩家从左侧材料库拖拽材料到中央实验台的六边形槽位中（最多4种），点击炼制按钮后配方引擎计算结果。若配方有效则产生对应药水（恢复/强化/毒药/爆炸），药水瓶自动添加到右侧展示架，同时触发金色粒子爆发特效。若配方无效则弹出"材料不相容"提示。玩家可点击展示架上的药水瓶查看详情，也可拖动3D场景旋转观察。

```mermaid

    "配方引擎计算" --> "配方有效?"
    "配方有效?" -- 是 --> "显示效果卡片 + 粒子特效"
    "显示效果卡片 + 粒子特效" --> "药水添加到展示架"

```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深木色 #3e2723，金色点缀 #ffd54f，墨绿点缀 #2e7d32
- 按钮风格：圆角按钮，悬停时0.2s缩放效果
- 字体：标题使用衬线体（MedievalSharp），正文使用无衬线体
- 布局：三栏布局（左材料库 | 中央3D场景 | 右展示架），顶部信息栏
- 图标风格：奇幻主题图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工坊主页 | 信息栏 | 深木色背景，金色文字，显示炼制次数和成功率 |
| 工坊主页 | 材料库 | 每格60x60px，背景#263238，边框#455a64，可折叠（折叠时箭头向右图标） |
| 工坊主页 | 3D场景 | 暖色木纹背景#3e2723，复古地砖8x8网格（#5d4037/#4e342e交替），深色橡木实验台，六个六边形槽位（边长30px，金色边框#ffd54f） |
| 工坊主页 | 展示架 | 三层每层4瓶，药水瓶半透明玻璃材质，颜色随效果变化，可滚动 |
| 工坊主页 | 效果卡片 | 白底黑字，圆角8px，淡入0.5s |
| 工坊主页 | 不相容提示 | 红色背景#e53935，白色文字 |
| 工坊主页 | 详情面板 | 半透明#1a1a2e背景，宽300px，右上角红色圆形关闭按钮（直径24px，#c62828），呼吸光晕0.8s循环 |

### 4.3 响应式

- 桌面优先设计
- 窄屏时材料库和展示架转为上下排列
- 3D场景在窄屏下自适应缩放

### 4.4 3D场景指导

- 环境：暖色室内灯光，木纹墙壁#3e2723
- 灯光：环境光（暖色）+ 实验台上方点光源
- 相机：透视相机，可拖动旋转，限制俯仰角范围
- 构图：中央实验台为主焦点，左右对称（材料架与展示架）
- 交互：鼠标拖动旋转视角，点击药水瓶触发详情
- 特效：炼金成功时金色粒子爆发（100粒子，4px大小，持续1s）
- 性能预算：保持30FPS以上

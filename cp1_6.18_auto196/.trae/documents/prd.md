# 渐变调色盘 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
渐变调色盘是一个面向设计师、前端开发者和创意工作者的交互式渐变背景生成工具。用户可以通过直观的拖拽操作创建精美的渐变色，并将其应用于网页设计、UI界面和数字艺术创作中。

### 1.2 产品定位
- **核心价值**：降低渐变色彩设计的门槛，提供即时可视化的渐变创作体验
- **目标用户**：UI设计师、前端开发者、数字艺术家、配色爱好者
- **使用场景**：网站背景设计、APP界面配色、海报设计、品牌色彩系统搭建

### 1.3 设计理念
采用**精致简约**的设计语言，以暖灰色系为主基调，让色彩本身成为主角。界面克制而优雅，通过微动画和细腻的交互反馈提升用户体验。

---

## 2. 核心功能

### 2.1 色标编辑器
- **拖拽调整**：圆形色标（直径24px）可在渐变条上自由拖拽，位置以百分比（0-100%）计算
- **数量限制**：最少2个色标，最多8个色标
- **颜色选择**：点击色标弹出圆盘颜色选择器，支持HEX/RGB/HSL三种输入模式
- **实时预览**：拖拽和颜色调整时，渐变条背景实时更新

### 2.2 角度与步长控制
- **角度控制**：0-360度步进调节，配直观旋转箭头图标
- **步长控制**：10-100步进调节，控制生成颜色数量，数值越大过渡越平滑
- **视觉反馈**：旋钮和滑块带有状态变化动画

### 2.3 实时大屏预览
- **大尺寸展示**：自适应宽度，固定高度500px
- **实时渲染**：根据色标和角度实时更新渐变背景
- **颜色标签**：色块边界处显示对应颜色值标签（12px字体，带浅色阴影）
- **交互探索**：支持鼠标拖动平移和滚轮缩放查看细节

### 2.4 收藏与瀑布流画廊
- **收藏功能**：一键保存当前渐变配置到本地
- **瀑布流布局**：列宽280px，卡片间距16px，视觉错落有致
- **卡片信息**：渐变缩略图（圆角8px）、色标数量、CSS代码片段
- **卡片交互**：悬停上抬4px + 浅色阴影，点击展开查看详情，支持删除
- **代码导出**：点击代码片段一键复制CSS渐变代码

### 2.5 撤销/重做
- **历史记录**：最多保留20步操作历史
- **双向导航**：左侧撤销箭头(←)、右侧重做箭头(→)
- **状态同步**：切换历史时所有控件（色标位置、角度、步长）同步恢复

## 3. 界面需求

### 3.1 整体布局
- **左右两栏结构**：
  - 左栏：宽400px，背景#F5F3EF，右侧1px分隔线#E4DED6
  - 右栏：预览+画廊区域，背景#FAF9F7

### 3.2 左栏 - 控制面板
- **顶部控制区**：背景#FFFFFF，圆角12px，padding 24px，box-shadow 0 2px 12px rgba(0,0,0,0.06)
  - 色标编辑区：高度120px
  - 角度旋钮
  - 步长滑条
  - 收藏按钮

- **底部历史按钮区**：背景#2D2D2D，圆角8px，文字#FFFFFF
  - 悬停状态：背景#444444
  - 过渡动画：0.2s ease

### 3.3 右栏 - 预览与画廊
- **顶部预览区**：高度500px，圆角16px，margin 24px
- **底部画廊区**：padding 24px，瀑布流布局

### 3.4 视觉规范
- **主色调**：#F5F3EF、#FFFFFF、#2D2D2D、#E4DED6
- **过渡动画**：统一使用0.2s ease
- **卡片悬停**：上抬4px + box-shadow 0 8px 24px rgba(0,0,0,0.08)

---

## 4. 非功能需求

### 4.1 性能要求
- **实时渲染**：色标拖动和颜色调整时，预览刷新间隔≤16ms（60FPS）
- **布局计算**：瀑布流布局计算时间≤50ms
- **响应速度**：所有交互操作反馈延迟≤100ms

### 4.2 技术约束
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **状态管理**：Zustand
- **颜色选择器**：react-colorful

## 6. 验收标准

1. 色标拖拽流畅，无卡顿，渐变实时更新
2. 颜色选择器支持HEX/RGB/HSL三种模式切换
3. 角度和步长控制准确，数值变化与视觉效果一致
4. 预览区域自适应宽度，渐变渲染准确
5. 瀑布流画廊加载流畅，卡片布局正确
6. 收藏功能正常工作，刷新页面后数据不丢失
7. 撤销/重做功能正确恢复历史状态
8. CSS代码导出格式正确，可直接使用

## 1. 产品概述

赛博朋克风格的实时数据仪表盘，通过WebSocket连接数据源，以动态图表和指标卡的方式监控服务器性能或多维数据流。面向运维工程师和数据分析师，提供沉浸式的数据监控体验。

## 2. 核心功能

### 2.1 功能模块

1. **仪表盘主页**：WebSocket连接管理、四卡片图表区、顶部菜单栏、底部状态栏

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘主页 | WebSocket连接器 | 左上角输入框+连接按钮，连接中旋转加载动画，成功脉冲绿色光效，失败红色抖动提示框 |
| 仪表盘主页 | 底部状态栏 | 实时显示连接状态、延迟（ping统计）、最近数据时间戳，延迟数字更新时放大缩回动画 |
| 仪表盘主页 | CPU折线图卡片 | 左上卡片，折线图实时渲染CPU使用率，标题+图标+实时数值（变化时背景闪烁），可收起/展开，可拖拽 |
| 仪表盘主页 | 内存环形图卡片 | 右上卡片，环形图实时渲染内存使用率，同上交互 |
| 仪表盘主页 | 网络双轴柱状图卡片 | 左下卡片，双轴柱状图实时渲染网络流入流出，同上交互 |
| 仪表盘主页 | 磁盘读写面积图卡片 | 右下卡片，面积图实时渲染磁盘读写速率，同上交互 |
| 仪表盘主页 | 拖拽重排 | 卡片可自由拖拽交换位置，400ms ease-out网格重排过渡动画 |
| 仪表盘主页 | 顶部菜单-清空数据 | 清除所有图表历史数据，点击时爆炸扩散消失动画 |
| 仪表盘主页 | 顶部菜单-重置布局 | 卡片回到初始位置，弹跳归位动画 |
| 仪表盘主页 | 顶部菜单-导出报告 | 截图导出PNG，底部向上扫过遮罩效果，完成后右下角滑入下载链接提示（3秒自动滑出） |
| 后端服务 | WebSocket服务器 | Node.js Express + ws，每秒广播模拟CPU/内存/网络/磁盘数据 |

## 3. 核心流程

用户打开页面 → 输入WebSocket地址并点击连接 → 建立连接后底部状态栏显示实时状态 → 四卡片区域接收数据流并实时渲染图表 → 用户可拖拽重排卡片、收起/展开 → 使用顶部菜单清空数据/重置布局/导出报告

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景(#0a0e17) + 亮青(#00f0ff) + 洋红(#ff007f)
- **卡片风格**：半透明毛玻璃效果(backdrop-filter: blur(20px))，渐变边框(从青到洋红)
- **字体**：等宽字体 JetBrains Mono
- **布局**：12列CSS Grid，左侧菜单栏240px，主区域自适应
- **按钮交互**：悬停缩放1.05倍 + 微光扫过动画
- **整体风格**：赛博朋克/未来科技感，霓虹光效，暗黑底色配高饱和度色彩

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘主页 | WebSocket连接器 | 输入框(暗色背景+青色边框)、连接按钮(渐变背景+旋转加载)、绿色脉冲光效、红色抖动错误提示 |
| 仪表盘主页 | 图表卡片 | 毛玻璃卡片、渐变边框、图标标题+实时数值、收起/展开按钮、拖拽手柄、闪烁高亮背景 |
| 仪表盘主页 | 底部状态栏 | 连接状态指示灯、延迟数字(放大动画)、时间戳 |
| 仪表盘主页 | 顶部菜单栏 | 三个按钮：清空数据(爆炸动画)、重置布局(弹跳动画)、导出报告(遮罩扫描+滑入提示) |

### 4.3 响应式

- 桌面优先设计，12列CSS Grid布局
- 1024px以下切换为单列堆叠布局
- 卡片拖拽在触摸设备上同样可用

### 4.4 性能要求

- 图表每秒更新帧率不低于30fps
- 超过4个卡片时使用虚拟滚动渲染
- 拖拽动画持续400ms ease-out

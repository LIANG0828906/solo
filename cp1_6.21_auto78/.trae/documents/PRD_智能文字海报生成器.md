## 1. 产品概述

智能文字海报生成工具，为设计师和内容创作者提供快速、高效的社交媒体海报与博客卡片制作能力。用户通过简单的文案输入、布局选择和配色调整，即可即时预览并批量导出高质量图片。

- **核心价值**：解决手动排版调整费时费力的问题，提供多种预设风格的快速变体生成
- **目标用户**：设计师、内容创作者、社交媒体运营人员、博客作者

## 2. 核心功能

### 2.1 用户角色

本产品为单角色工具类应用，无需登录注册。

| 角色 | 权限说明 |
|------|----------|
| 普通用户 | 使用全部功能：文案输入、布局选择、配色调整、字体切换、图片导出 |

### 2.2 功能模块

1. **控制面板（左侧）**：文案输入区、布局选择区、配色方案区、字体选择区、导出按钮
2. **海报预览区（右侧）**：实时渲染600x800px海报，支持平滑过渡动画

### 2.3 页面详情

| 区域名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 控制面板 | 文案输入区 | 标题输入框（≤50字）、副标题输入框（≤100字），附实时字数统计，防抖300ms更新预览 |
| 控制面板 | 布局选择区 | 3种预设布局：居中、左对齐、环绕，点击切换，0.4s ease-out过渡 |
| 控制面板 | 配色方案区 | 6个圆形色块（深色、渐变、莫兰迪等），点击切换背景和文字颜色，0.5s渐变 |
| 控制面板 | 字体选择区 | 下拉框：Serif、Sans-serif、Monospace，切换时0.3s淡入淡出 |
| 控制面板 | 导出按钮 | 200x50px圆角按钮，点击使用html-to-image转PNG下载，显示加载动画 |
| 预览区 | 海报渲染 | 600x800px固定尺寸，box-shadow效果，纯CSS实现排版，无外部图片 |

## 3. 核心流程

用户打开应用 → 默认展示初始海报 → 输入/修改文案（实时更新预览）→ 选择布局样式 → 选择配色方案 → 选择字体 → 点击导出按钮 → 图片自动下载

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：蓝色 #4A90D9（导出按钮主色）
- **辅助色**：#357ABD（悬停加深色）、#F8F9FA（控制面板背景）、#DEE2E6（边框色）
- **按钮风格**：导出按钮为胶囊形（圆角25px），悬停上移3px并加深
- **圆角规范**：统一8px（导出按钮25px、配色色块50%除外）
- **过渡动画**：0.3-0.5s平滑过渡，避免生硬跳变
- **阴影**：海报box-shadow: 0 10px 30px rgba(0,0,0,0.15)

### 4.2 页面布局结构

| 区域 | 尺寸/样式 | UI元素 |
|------|-----------|--------|
| 主容器 | flex横向布局 | 左(控制面板) + 右(预览区) |
| 控制面板 | 宽280px，背景#F8F9FA，1px右框线#DEE2E6 | 输入框、布局卡片、色块、下拉框、导出按钮 |
| 预览区 | flex:1，居中对齐 | 600x800px海报，居中显示，带阴影 |
| 间距规范 | 控制面板内各元素间距24px | - |

### 4.3 配色方案详情（6种预设）

| 方案名称 | 背景样式 | 文字颜色 |
|----------|----------|----------|
| 深色主题 | #1A1A2E | 白色 #FFFFFF |
| 渐变紫蓝 | 线性渐变 #667eea → #764ba2 | 白色 #FFFFFF |
| 莫兰迪色系 | #C6B4B0 | #4A4A4A |
| 温暖橙黄 | 线性渐变 #F093FB → #F5576C | 白色 #FFFFFF |
| 清新薄荷 | #A8E6CF | #2D6A4F |
| 简约纯白 | #FFFFFF | #212529 |

### 4.4 布局方案详情（3种预设）

| 布局名称 | 排版规则 |
|----------|----------|
| 居中布局 | 标题位于画面中心，副标题在标题下方，两者整体居中对齐 |
| 左对齐布局 | 标题位于左侧1/3处，副标题在标题下方，整体左对齐 |
| 环绕布局 | 副标题围绕标题弯曲排列，形成环绕视觉效果 |

### 4.5 响应式设计

桌面端优先设计，海报固定600x800px。当窗口宽度不足时，预览区保持横向滚动查看完整海报，控制面板始终可见。

### 4.6 动画细节

- **布局切换**：0.4s ease-out，文字位置平滑移动
- **配色切换**：0.5s，背景和文字颜色渐变过渡
- **字体切换**：0.3s，文字透明度0→1→0淡入淡出
- **导出按钮悬停**：transform translateY(-3px)，背景色加深，transition 0.3s
- **加载图标**：旋转动画周期1s，linear无限循环

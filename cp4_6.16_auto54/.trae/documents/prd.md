## 1. 产品概述

VaultPass 是一款纯本地运行的个人密码与敏感信息托管应用，所有数据在客户端使用 AES-GCM 加密后持久化到 IndexedDB，无需后端服务器，保障用户隐私与数据安全。目标用户为注重隐私的个人用户，希望安全存储各类账号密码、证件号和私密备注。

- 解决密码分散、明文存储的安全隐患，提供统一加密管理方案
- 零后端架构，数据不离开设备，消除服务器泄露风险

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 单用户 | 本地设置主密码 | 完整的加密数据读写权限 |

### 2.2 功能模块

1. **主密码设置/验证页**：首次访问设置主密码，后续访问输入主密码解密数据
2. **主界面**：搜索栏、分类标签栏、密码条目列表、浮动添加按钮
3. **条目操作面板**：新增/编辑条目的底部弹出表单

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主密码设置/验证页 | 主密码设置模态窗口 | 首次访问弹出全屏模态窗口，要求设置至少8位包含大小写字母和数字的主密码，经PBKDF2派生AES-GCM密钥，盐值存入IndexedDB |
| 主密码设置/验证页 | 主密码验证模态窗口 | 非首次访问时弹出全屏模态窗口，输入主密码验证，验证通过后解密所有数据 |
| 主界面 | 固定搜索栏 | 顶部固定搜索输入框，输入时300ms防抖更新搜索词，实时过滤条目 |
| 主界面 | 分类标签栏 | 水平滚动标签（全部、社交、金融、工作、其他），选中标签圆角高亮+底部滑动条指示器，切换有平滑动画 |
| 主界面 | 密码条目列表 | 瀑布流两列布局展示卡片，每张卡片显示标题、脱敏用户名、加密状态图标；点击展开内联详情，高度过渡动画 |
| 主界面 | 浮动添加按钮 | 右下角圆形加号按钮，点击弹出底部面板 |
| 条目操作面板 | 新增/编辑表单 | 底部弹出面板，包含标题、用户名、密码（默认生成强随机密码）、分类选择器、备注文本框；提交时加密后存入IndexedDB，列表即时更新+淡入通知条 |
| 条目操作面板 | 条目菜单 | 卡片右下角三点菜单，包含编辑（复用底部面板）、移动分类、删除（确认对话框+缩小旋转动画移除） |

## 3. 核心流程

### 3.1 首次访问流程

用户打开应用 → 检测IndexedDB无密钥记录 → 弹出主密码设置模态窗口 → 用户输入符合规则的主密码 → PBKDF2派生密钥+生成盐值 → 存储盐值到IndexedDB → 进入主界面

### 3.2 后续访问流程

用户打开应用 → 检测IndexedDB有盐值记录 → 弹出主密码验证模态窗口 → 用户输入主密码 → PBKDF2派生密钥 → 尝试解密验证数据 → 成功则进入主界面，失败则提示错误

### 3.3 密码条目管理流程

```mermaid
flowchart TD
    "A[点击浮动按钮]" --> "B[弹出底部面板]"
    "B" --> "C[填写表单/自动生成密码]"
    "C" --> "D[提交]"
    "D" --> "E[AES-GCM加密数据]"
    "E" --> "F[存入IndexedDB]"
    "F" --> "G[列表即时更新]"
    "G" --> "H[显示通知条]"
```

### 3.4 搜索与筛选流程

用户输入关键词 → 300ms防抖 → 更新Store搜索词 → 模糊匹配标题/用户名 → 未匹配卡片渐隐消失 → 匹配卡片弹跳显现

## 4. 用户界面设计

### 4.1 设计风格

- **主背景**：深蓝色 #0f172a
- **卡片风格**：半透明深色玻璃质感，背景 rgba(255,255,255,0.05)，边框 1px rgba(255,255,255,0.1)，backdrop-filter:blur(12px)
- **主色调**：青蓝色渐变 #38bdf8 → #818cf8，用于按钮和高亮元素
- **字体**：系统无衬线字体，字号14px，行高1.6
- **布局**：卡片列表左右2%内边距，两列间隔12px
- **动画**：所有交互反馈0.2-0.3s慢入慢出过渡
- **图标风格**：线性图标，与青蓝色主色调一致

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主密码设置/验证页 | 模态窗口 | 全屏半透明遮罩，居中卡片表单，主密码输入框，强度提示，确认按钮渐变背景 |
| 主界面 | 搜索栏 | 固定顶部，深色背景输入框，搜索图标左侧，输入时右侧清除按钮 |
| 主界面 | 分类标签栏 | 水平滚动，圆角标签，选中态青蓝渐变背景+底部滑动条，未选中态半透明 |
| 主界面 | 密码条目卡片 | 玻璃质感卡片，标题+脱敏用户名+锁图标，展开详情有高度过渡，眼睛图标切换明文/密文 |
| 主界面 | 浮动添加按钮 | 右下角固定，圆形青蓝渐变背景，白色加号图标，点击缩放反馈 |
| 条目操作面板 | 底部弹出面板 | 从底部滑入，深色背景，表单字段，分类选择器下拉，提交按钮渐变背景 |
| 条目操作面板 | 三点菜单浮层 | 卡片右下角弹出，深色半透明背景，三个操作项行排列 |
| 条目操作面板 | 删除确认对话框 | 居中模态，警示文字，取消/确认按钮 |

### 4.3 响应式设计

- 桌面优先设计，移动端（<640px）自动切换为单列布局
- 移动端搜索栏和分类标签固定非滚动
- 底部弹出面板在移动端全宽展示
- 卡片间距在小屏适当调整

### 4.4 性能要求

- 搜索输入防抖300ms，不导致输入卡顿
- 列表超过50个卡片时启用虚拟列表（WindowedList）
- 滚动帧率不低于55fps

## 1. 产品概述

Markdown代码块与图表联动排版编辑器，解决个人博客文章中代码块与图表样式混乱、缺乏语义关联的问题。面向技术博主和文档作者，提供所见即所得的Markdown编辑体验，自动建立代码块与可视化图表的语义关联。

核心价值：提升技术文章的阅读体验，让代码与图表形成视觉联动，增强内容表达力。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 编辑Markdown内容、预览渲染效果、复制代码、切换主题 |

### 2.2 功能模块

1. **编辑器模块**：左侧Markdown文本编辑区，支持实时输入
2. **渲染器模块**：右侧实时渲染区，展示排版后的文章内容
3. **代码高亮模块**：代码块语法高亮、行号显示、语言标签
4. **图表联动模块**：mermaid图表渲染、代码块与图表展开/收拢联动
5. **主题切换模块**：暗色/亮色主题切换，平滑过渡动画
6. **代码复制模块**：一键复制代码，复制反馈动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主页面 | 顶部导航栏 | 应用标题、主题切换按钮 |
| 主页面 | 左侧编辑区 | Markdown文本编辑、实时输入、滚动同步 |
| 主页面 | 可拖动分隔条 | 调整左右栏比例、悬停高亮 |
| 主页面 | 右侧渲染区 | Markdown渲染、代码高亮、图表展示、联动动画 |

## 3. 核心流程

用户打开应用 → 看到左侧编辑区（含示例内容）和右侧渲染区 → 在编辑区输入或修改Markdown内容 → 300ms防抖后右侧实时更新渲染 → 点击代码块旁的箭头展开/收拢图表 → 使用右上角按钮切换主题 → 点击代码块右上角复制按钮复制代码

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：蓝色 #1890ff
- **背景色**：浅灰 #f0f2f5（亮色），深灰 #1e1e1e（暗色代码块）
- **布局风格**：左右分栏，中间可拖动分隔条
- **字体**：系统默认字体（-apple-system, BlinkMacSystemFont），代码字体 Fira Code 16px
- **动效风格**：平滑过渡，0.3s cubic-bezier(0.4, 0, 0.2, 1)
- **图标风格**：简洁线条风，箭头符号

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 主页面 | 顶部导航栏 | 标题文字、主题切换开关（滑动式）、右侧对齐 |
| 主页面 | 左侧编辑区 | 白色卡片、无边框textarea、内边距16px、代码字体 |
| 主页面 | 分隔条 | 6px宽、#d9d9d9、悬停变#1890ff、可拖动光标 |
| 主页面 | 右侧渲染区 | 白色卡片、内边距24px、文章排版样式 |
| 主页面 | 代码块 | 圆角8px、语言标签左上角、复制按钮右上角、行号左侧 |
| 主页面 | 联动线 | #1890ff 1px水平线、右侧三角箭头、可点击 |

### 4.3 响应式

- 桌面端：左右分栏布局（左40%，右60%）
- 移动端（<768px）：上下布局（编辑区在上，渲染区在下）
- 触摸优化：按钮增大点击区域，支持手势滚动

### 4.4 微交互细节

- 渲染区更新：opacity 0→1，200ms ease-in 淡入
- 复制按钮：默认透明，悬停opacity:1，0.2s过渡
- 复制成功：按钮文本变"已复制"持续2s，涟漪扩散动画（半径30px，#52c41a，0.4s）
- 主题切换：background-color和color过渡0.5s ease
- 展开/收拢：高度从0到原始高度平滑变化，0.3s cubic-bezier

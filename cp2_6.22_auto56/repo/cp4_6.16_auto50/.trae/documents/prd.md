## 1. 产品概述

SwapTrail是一个面向可持续消费社区的闲置物品交换记录与可视化平台，帮助社区成员记录、分享闲置物品的交换历史，并追踪每件物品的流转故事，促进资源循环利用和社区互动。

- 核心价值：通过可视化的物品流转链条，赋予闲置物品独特的故事价值，鼓励可持续消费行为
- 目标用户：关注可持续生活方式的社区成员、环保组织、共享经济爱好者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 本地使用无需注册 | 添加物品、记录交换、查看流转链条、搜索筛选 |
| 管理员 | 本地模式切换 | 编辑/删除物品、管理社区数据 |

### 2.2 功能模块

1. **仪表盘首页**：社区统计概览、月度交换趋势图、最新交换动态
2. **物品管理**：添加物品、编辑物品、删除物品、物品列表展示
3. **交换记录**：记录交换事件、查看物品流转时间线
4. **搜索筛选**：关键词搜索、标签筛选、动态过滤动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 仪表盘首页 | 统计卡片 | 展示总交换次数、活跃物品数、参与人数 |
| 仪表盘首页 | 趋势图表 | 使用recharts绘制月度交换趋势柱状图，带进入动画 |
| 仪表盘首页 | 最新动态 | 显示最近的交换事件列表 |
| 物品列表页 | 物品卡片 | 展示物品信息，hover上浮动画，点击展开详情 |
| 物品列表页 | 搜索筛选 | 关键词搜索框和预设标签筛选栏 |
| 物品详情页 | 流转时间线 | 纵向时间线展示物品完整流转历史，渐变连线 |
| 物品详情页 | 交换记录 | 显示每次交换的双方、日期和附言 |

## 3. 核心流程

```mermaid
flowchart LR
    A["用户进入首页"] --> B["查看社区统计"]
    B --> C["浏览物品列表"]
    C --> D["搜索/标签筛选"]
    D --> E["点击物品卡片"]
    E --> F["查看流转时间线"]
    F --> G["记录新交换"]
    G --> H["数据存入IndexedDB"]
    H --> I["更新UI展示"]
```

用户在首页查看社区统计数据，浏览物品列表，通过搜索或标签筛选找到感兴趣的物品。点击物品卡片进入详情页，查看该物品的完整流转时间线。用户可以添加新物品或为现有物品记录新的交换事件，所有数据自动保存到IndexedDB，刷新页面不丢失。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#E8A87C（暖橙色，代表温暖和可持续）
- **辅助色**：#41B3A3（青绿色，代表清新和环保）
- **背景色**：#FFF1D0（米黄色，营造温馨氛围）
- **按钮风格**：圆角8px，悬停时轻微上浮，点击反馈
- **字体**：标题使用具有人文感的衬线或半衬线字体，正文使用清晰易读的无衬线字体
- **布局风格**：卡片式布局，左侧固定导航栏，右侧内容区
- **图标风格**：简洁线性图标，使用lucide-react图标库

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 仪表盘首页 | 统计卡片 | 3个并排数据卡片，渐变色背景，数字动画 |
| 仪表盘首页 | 趋势图表 | 柱状图从底部升起弹跳动画，柱子颜色随数值渐变 |
| 仪表盘首页 | 最新动态 | 时间线式列表，最新项高亮显示 |
| 物品列表页 | 物品卡片 | 网格布局，hover时上浮+阴影加深，标签颜色平滑过渡 |
| 物品列表页 | 筛选栏 | 横向排列标签按钮，选中状态下划线动画 |
| 物品详情页 | 流转时间线 | 纵向节点布局，节点间渐变连线，切换时滑动淡入 |
| 全局 | 导航栏 | 左侧固定，选中项下方下划线动画，移动端汉堡菜单 |

### 4.3 响应式设计

- **设计策略**：桌面优先，移动端自适应
- **断点**：768px为主要响应式断点
- **桌面端**（>768px）：左侧240px固定导航栏，右侧内容区
- **移动端**（≤768px）：导航栏收为顶部汉堡菜单，点击展开全屏侧滑
- **触摸优化**：按钮最小点击区域44px，列表项增加垂直间距，支持下拉刷新

## 5. 动画与交互

### 5.1 页面过渡
- 页面切换：左右滑动过渡，300ms内完成
- 元素入场：渐入+轻微上浮，使用framer-motion的stagger实现错落效果

### 5.2 微交互
- 卡片悬停：translateY(-4px) + 阴影加深，150ms平滑过渡
- 标签切换：颜色渐变过渡，下划线滑动动画
- 按钮点击：scale(0.96)反馈，快速回弹

### 5.3 数据可视化动画
- 柱状图：从y=0位置升起，末端带轻微弹跳效果
- 时间线节点：按顺序依次淡入，间隔50ms

## 1. 产品概述
音律视界是一款基于Web Audio API的实时音频可视化应用，将用户上传的音频文件转化为动态、美观的频谱柱状图和波形图展示。
- 主要用途：为音乐爱好者、音频创作者提供直观的音频频谱和波形可视化体验
- 目标用户：音乐爱好者、音频工程师、内容创作者

## 2. 核心特性

### 2.1 功能模块
1. **主页面**：文件上传区域、频谱柱状图、波形图、播放控制、时间显示

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 文件上传 | 支持点击或拖拽上传mp3、wav、ogg格式音频文件（≤10MB） |
| 主页面 | 频谱柱状图 | 64个频率柱，高度映射0-300px，青紫渐变，带呼吸感动画 |
| 主页面 | 波形图 | 256采样点，白色半透明线条，渐进式扫描绘制动画 |
| 主页面 | 播放控制 | 播放/暂停按钮，音量滑块（0-100线性控制） |
| 主页面 | 时间显示 | 右上角显示当前时间/总时长（mm:ss格式，等宽字体） |

## 3. 核心流程
用户通过点击或拖拽上传音频文件 → 应用使用Web Audio API解析音频 → 实时获取频谱和波形数据 → Canvas以60FPS渲染动态图形 → 用户可控制播放/暂停和音量

```mermaid
flowchart LR
    A["用户上传音频"] --> B["Web Audio API解析"]
    B --> C["实时获取频率/波形数据"]
    C --> D["Canvas 60FPS渲染"]
    D --> E["播放/暂停/音量控制"]
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色调：深色背景 #0D1117，容器背景 #161B22，边框 #30363D
- 强调色：频谱渐变（青色 #00FFFF → 紫色 #FF00FF），按钮渐变（#238636 → #2EA043）
- 文字色：时间显示 #B0B0B0，波形线条 #FFFFFF（透明度0.8）
- 按钮样式：圆角8px，hover亮度提升10%
- 字体：时间显示使用等宽字体（monospace）
- 布局：居中圆角矩形容器，最大宽度900px，内边距40px

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 文件上传 | 渐变绿色按钮，圆角8px，hover效果 |
| 主页面 | 频谱柱状图 | Canvas绘制，64柱，青紫渐变，0.2px垂直抖动 |
| 主页面 | 波形图 | Canvas绘制，白色半透明线条，2px宽度，扫描动画 |
| 主页面 | 播放控制 | 播放/暂停按钮，音量滑块 |
| 主页面 | 时间显示 | 右上角，等宽字体，浅灰色 |

### 4.3 响应性
桌面端优先设计，容器最大宽度900px，居中显示。

## 1. 产品概述

墨迹心情日记是一款水墨风格的情绪日记应用，将日常文字记录与水墨艺术相结合，让每篇日记都成为一幅独一无二的数字水墨画。用户通过水墨画布表达情绪，以竖排楷体文字记录心境，用色彩传递情感。

- 核心目标：打造沉浸式的书写体验，让情绪记录变成一种艺术创作行为
- 目标用户：喜欢书写、注重情绪表达、追求美学体验的用户群体
- 市场价值：填补传统日记应用缺乏艺术表达的空白，提供独特的情绪可视化体验

## 2. 核心功能

### 2.1 功能模块

1. **日记编辑页**：水墨画布、竖排文字编辑区、情绪调色盘
2. **时间轴列表页**：日记卡片时间轴、搜索与情绪筛选、卡片悬停效果
3. **日记详情页**：完整墨迹画布、日记内容展示

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面（编辑区） | 水墨画布 | 支持鼠标/触控涂抹，根据速度力度生成浓淡墨迹，实时渲染 |
| 主页面（编辑区） | 竖排文字编辑区 | 固定右侧35%宽度，楷体24px，行高1.8，米白背景竖线格纹 |
| 主页面（编辑区） | 情绪调色盘 | 五种预设情绪色选择，作为主色调混入微量随机色 |
| 主页面（编辑区） | 保存按钮 | 保存当前日记，生成墨迹缩略图和时间轴卡片 |
| 时间轴页 | 顶部导航栏 | 64px高度，半透明亚麻色背景，0.5px底部分隔线 |
| 时间轴页 | 搜索与筛选 | 顶部居中60%宽度搜索框，下方情绪筛选按钮组 |
| 时间轴页 | 日记卡片 | 240px宽圆角卡片，悬停磨砂玻璃简介效果 |
| 时间轴页 | 卡片列表 | 倒序排列时间轴 |

## 3. 核心流程

用户启动应用 → 进入主页 → 选择情绪色调 → 在画布涂抹水墨 → 竖排书写日记 → 保存日记 → 自动生成时间轴卡片 → 查看/搜索/筛选历史日记 → 点击卡片查看详情

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：宣纸米白#FAF5E8向下渐变到#F0E8D0
- **情绪色**：宁静淡蓝#8FBFFF、喜悦暖橙#FFB347、忧伤浅灰#A0A0A0、愤怒深褐#6B4226、平静墨绿#5A7A5A
- **边框色**：深灰色#4A4A4A
- **按钮风格**：扁平设计，hover时底部浮现微弱阴影
- **字体**：楷体(KaiTi)用于日记内容，整体衬线字体
- **布局风格**：宣纸画卷式，左右竖向装饰线，卡片轻微阴影悬浮感
- **装饰元素**：页面左右细长竖向装饰线，从顶部延伸到页脚

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 主页面 | 整体布局 | 宣纸渐变背景，左右装饰线 |
| 主页面 | 水墨画布 | 全宽或65%宽画布，水墨渲染效果 |
| 主页面 | 文字编辑区 | 米白#F5F0E8背景，细竖线格纹，楷体竖排 |
| 主页面 | 情绪调色盘 | 五色圆形色块，选中高亮 |
| 时间轴页 | 顶部导航 | 半透明亚麻色rgba(245,240,232,0.8) |
| 时间轴页 | 搜索框 | 居中60%宽，浅灰占位符 |
| 时间轴页 | 筛选按钮 | 80x36px圆角8px，选中填充情绪色+checkmark |
| 时间轴页 | 日记卡片 | 240px宽16px圆角，2.5px边框，hover磨砂玻璃简介 |

### 4.3 响应式设计

- **桌面端(≥768px)**：画布65% + 编辑区35%横向布局
- **移动端(<768px)**：画布全宽，编辑区移至画布下方
- 触控操作优化，支持手指涂抹

### 4.4 动画与交互

- 卡片悬停：0.2s过渡，半透明磨砂玻璃效果简介浮出
- 墨迹渲染：实时更新，延迟≤50ms
- 按钮交互：扁平到阴影过渡

## 1. 产品概述

读书会管理应用，旨在帮助小型读书会组织者高效管理书籍交流、成员阅读进度追踪和讨论话题管理。解决线下读书会活动组织混乱、成员阅读进度不透明和讨论缺乏深度的核心痛点。目标用户为5-30人规模的小型读书会组织者和成员。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 加入方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 创建读书会 | 管理书籍、调整活动排期、发起投票 |
| 成员 | 邀请加入 | 浏览书籍、打卡进度、参与讨论、投票 |

### 2.2 功能模块
1. **书籍管理面板**：添加书籍、排序浏览、卡片动画展示
2. **书籍详情页**：完整信息展示、阅读进度追踪、讨论话题、打卡记录
3. **讨论话题模块**：创建话题、聊天气泡回复、@成员通知、滑动切换动画
4. **阅读打卡功能**：记录进度与感想、时间线展示、API同步
5. **活动排期模块**：自动推荐讨论时间、日历卡片、投票柱状图实时更新

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书籍管理面板 | 添加书籍表单 | 填写书名、作者、封面图URL、简介和ISBN，提交后卡片淡入上浮出现 |
| 书籍管理面板 | 排序筛选 | 支持按书名、作者、添加时间排序 |
| 书籍管理面板 | 书库列表 | 卡片展示封面缩略图、书名、在读人数，悬停上浮加深阴影 |
| 书籍详情页 | 信息区 | 完整书籍信息和在读成员头像列表 |
| 书籍详情页 | 阅读进度区 | 横向时间轴、渐变进度条、圆点标记、头像点击弹出进度气泡 |
| 书籍详情页 | 讨论话题区 | 话题卡片列表、发起人头像、标题、回复数、最后回复时间 |
| 书籍详情页 | 阅读打卡 | 填写当前章节和感想，API提交，时间线展示，右侧滑入+抖动动画 |
| 话题详情页 | 聊天回复区 | 气泡样式回复、新回复底部淡入、@成员触发通知 |
| 活动排期页 | 日历推荐 | 基于进度自动推荐章节讨论时间，日历卡片展示 |
| 活动排期页 | 投票区 | 管理员调整时间并发起投票，柱状图实时更新 |

## 3. 核心流程

用户打开应用 → 浏览书库列表 → 点击书籍卡片进入详情 → 查看成员进度和讨论 → 打卡记录进度 → 参与话题讨论 → 查看活动排期并投票

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：米白(#FAF7F2)、浅咖啡(#D4C5B2)、深棕(#5C3D2E)
- 强调色：墨绿色(#2D6A4F)用于按钮和进度条
- 按钮风格：圆角12px、柔和阴影
- 字体：Noto Serif SC（标题）+ Noto Sans SC（正文）
- 布局风格：顶部固定导航栏 + 左侧侧边栏 + 主内容区
- 卡片风格：圆角12px、柔和阴影层

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 书籍管理面板 | 书库列表 | 暖色卡片网格、封面缩略图、淡入上浮动画(300ms)、悬停上浮+阴影加深 |
| 书籍详情页 | 进度追踪 | 横向时间轴、渐变进度条(墨绿→浅绿)、圆点标记、气泡弹出 |
| 书籍详情页 | 打卡时间线 | 垂直时间线、新记录右侧滑入+抖动 |
| 话题详情页 | 聊天气泡 | 气泡样式、左右滑动切换、新回复底部淡入 |
| 活动排期页 | 投票柱状图 | 墨绿色柱状图、实时轮询更新 |

### 4.3 响应式设计
- 桌面优先设计，最小宽度768px
- 平板适配：侧边栏可折叠
- 桌面适配：固定侧边栏+宽主内容区

### 4.4 骨架屏
- 加载状态使用浅灰色渐变脉冲动画占位
- 首屏渲染时间目标 < 1.5秒

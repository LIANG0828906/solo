## 1. 产品概述
皮影戏模拟器是一款在浏览器中运行的交互式文化体验游戏，让用户化身古代皮影戏师傅，在半透明戏台后方操纵影人进行即兴表演。

- 核心目标：通过直观的拖拽交互，让用户体验传统皮影戏的魅力，支持自由创作表演
- 目标用户：对中国传统文化感兴趣的普通用户、教育场景下的师生
- 产品价值：以数字化形式传承皮影戏艺术，提供沉浸式的创作体验

## 2. 核心功能

### 2.1 功能模块
1. **戏台主界面**：皮影戏台、光源效果、流苏帘幕装饰
2. **影人系统**：4个可拖拽影人（书生、武将、花旦、小丑），支持关节动画
3. **道具系统**：6种可拖拽道具，支持吸附到影人身上
4. **编钟配乐系统**：7枚编钟（Do-Si），支持实时点击、30秒录制、自动回放
5. **舞蹈联动系统**：编钟回放时影人根据音高做相应舞蹈动作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 戏台区域 | 1000x600px暗红色#400000背景，木质边框，流苏帘幕，两侧灯笼光源 |
| 主页面 | 表演区 | 200x150px半透明矩形框，影人进入后触发背光剪影效果 |
| 主页面 | 影人区 | 右侧4个影人（80x120px），单色剪影，支持拖拽和关节动画 |
| 主页面 | 道具区 | 左侧6种道具（60x40px），支持拖拽吸附到影人 |
| 主页面 | 编钟区 | 下方7枚编钟排列，支持点击、录制、回放，悬停显示音高名称 |

## 3. 核心流程

### 3.1 影人表演流程
```mermaid
flowchart LR
    A["用户拖拽影人"] --> B["进入表演区"]
    B --> C["背光剪影效果（#ffaa00发光）"]
    C --> D["点击手臂/腿部"]
    D --> E["关节动画（0.3s）+ 头部微转（5度）"]
```

### 3.2 道具吸附流程
```mermaid
flowchart LR
    A["拖拽道具"] --> B["接触影人"]
    B --> C["吸附到指定位置"]
    C --> D["闪光粒子特效（10-15个#ffd700粒子）"]
    D --> E["Web Audio音效（0.8s）"]
```

### 3.3 编钟录放流程
```mermaid
flowchart LR
    A["点击录制按钮"] --> B["编钟变红（录制状态）"]
    B --> C["用户点击编钟生成旋律"]
    C --> D["点击停止（30秒自动停止）"]
    D --> E["点击回放"]
    E --> F["编钟自动触发 + 影人舞蹈动作"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：中国红#800000，金色#d4af37
- **背景**：粗麻布纹理（CSS生成），底色#3d2b1f
- **字体**：Google Fonts Ma Shan Zheng（马善政毛笔字体）
- **边框**：木质效果，暗黄色渐变#a67c4f到#8b5e3c
- **动效风格**：弹性曲线cubic-bezier(0.25, 0.1, 0.25, 1)，中国风优雅过渡

### 4.2 页面设计概述

| 模块 | UI元素 | 设计细节 |
|-----------|-------------|-------------|
| 戏台 | 半透明幕布 | 暗红色#400000，上沿流苏帘幕，左右灯笼光源光晕 |
| 影人 | 单色剪影 | 书生（白）、武将（黑）、花旦（红）、小丑（蓝），80x120px |
| 道具 | 简约图标 | 钱袋、长剑、团扇、酒杯、书信等，60x40px |
| 编钟 | 古铜色钟体 | 7枚从小到大排列，悬停显示金色宋体音高名称 |
| 按钮 | 红金配色 | 录制/回放按钮，中国风边框，hover状态缩放效果 |

### 4.3 响应式设计
- 适配分辨率：1280x720以上
- 布局方式：整体居中，固定尺寸戏台，两侧区域自适应
- 触控优化：拖拽区域足够大，支持触摸设备

### 4.4 性能指标
- 编钟音效延迟：≤50ms
- 动画帧率：≥45fps
- 内存占用：稳定无泄漏

## 1. 产品概述
在线音乐播放器与歌单协作应用，让用户搜索音乐、创建和分享歌单，并支持好友协作编辑歌单。纯前端实现，数据存储在localStorage中。

- 主要用途：提供音乐搜索、播放、歌单管理和协作分享功能
- 目标用户：音乐爱好者，需要与朋友共享和协作编辑歌单的用户
- 产品价值：轻量级、无需后端、支持实时协作的音乐播放与歌单管理工具

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 搜索音乐、播放歌曲、创建/编辑/分享歌单、协作编辑他人歌单 |

### 2.2 功能模块
1. **首页**：推荐歌单展示、热门歌曲排行榜
2. **搜索页**：歌曲搜索、结果展示
3. **歌单详情页**：歌曲列表、协作编辑、分享功能
4. **播放器组件**：播放控制、进度条、音量控制、歌词显示、循环模式

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 推荐歌单 | 横向滚动卡片组展示预设推荐歌单，悬停显示播放按钮 |
| 首页 | 热门歌曲 | 排行榜展示热门歌曲列表，支持点击播放 |
| 搜索页 | 搜索框 | 输入关键词搜索歌曲，放大镜图标动态变色 |
| 搜索页 | 搜索结果 | 卡片形式展示歌曲，支持点击播放 |
| 歌单详情页 | 歌单信息 | 四宫格封面、歌单名称、歌曲数量 |
| 歌单详情页 | 歌曲列表 | 展示歌单内歌曲，支持添加/删除 |
| 歌单详情页 | 分享功能 | 复制分享链接，支持协作编辑 |
| 播放器组件 | 播放控制 | 播放/暂停、上一首/下一首、循环模式切换 |
| 播放器组件 | 进度条 | 拖拽进度条，显示时间提示 |
| 播放器组件 | 音量控制 | 滑动条调节音量 |
| 播放器组件 | 歌词显示 | 封面卡片翻转显示歌词 |

## 3. 核心流程

### 主要用户流程
1. 用户进入首页，浏览推荐歌单和热门歌曲
2. 用户点击搜索，输入关键词搜索歌曲
3. 用户在搜索结果中点击歌曲播放
4. 用户创建新歌单，添加喜欢的歌曲
5. 用户分享歌单链接给好友
6. 好友通过链接进入歌单，协作编辑添加/删除歌曲
7. 播放器在底部持续播放，支持各种控制操作

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝 #0a0e27，青色 #00d4ff
- **背景**：深色渐变，营造沉浸感
- **卡片**：毛玻璃效果 backdrop-filter，圆角设计
- **按钮**：圆角按钮，悬停有过渡动画
- **字体**：现代无衬线字体，层级分明
- **图标**：线性图标，与主色调一致

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 推荐歌单 | 横向滚动、卡片网格、悬停上浮动画、播放按钮 |
| 首页 | 热门歌曲 | 排行榜序号、歌曲信息、播放按钮、交替背景色 |
| 搜索页 | 搜索框 | 圆角输入框、放大镜图标动画、青色高亮 |
| 搜索页 | 搜索结果 | 卡片布局、颜色封面、歌曲信息、点击播放 |
| 歌单详情页 | 歌单封面 | 四宫格颜色拼图、歌单信息、分享按钮 |
| 歌单详情页 | 歌曲列表 | 列表布局、交替背景色、添加/删除按钮 |
| 播放器组件 | 控制区 | 播放/暂停、上一首/下一首、循环模式（旋转动画） |
| 播放器组件 | 进度条 | 渐变填充、青色高亮、时间提示、拖拽功能 |
| 播放器组件 | 音量控制 | 滑动条、渐变填充 |
| 播放器组件 | 歌词显示 | 卡片翻转动画 |

### 4.3 响应式
- 桌面端：内容区左右留白，清晰的网格布局
- 平板端：自适应布局，适当调整间距
- 移动端：导航栏折叠为汉堡菜单，单列布局，触摸友好的按钮尺寸

### 4.4 动画效果
- 页面切换：淡入淡出
- 歌曲切换：封面淡入淡出
- 卡片悬停：轻微上浮 + 阴影变化
- 循环模式切换：图标旋转动画
- 搜索图标：输入时灰色变青色
- 歌词显示：封面3D翻转

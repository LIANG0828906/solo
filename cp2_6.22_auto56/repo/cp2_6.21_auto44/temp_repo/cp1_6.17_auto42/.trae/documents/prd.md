## 1. 产品概述

互动剧情侦探学习游戏是一款面向在线教育平台的沉浸式剧情学习应用。学员在虚构的侦探剧情中通过选择对话和搜查线索推进故事，系统后台自动记录决策路径和学习成果，实现"玩中学"的教育目标。

- 目标用户：在线教育平台学员（18-35岁青年群体）
- 核心价值：通过分支剧情和即时反馈提升学习参与度与知识留存率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学员 | 输入昵称（字母数字2-10位）创建档案 | 游玩剧情、查看个人档案与进度 |
| 教师（未来扩展） | 管理员分配 | 查看学员数据、编辑剧情 |

### 2.2 功能模块

1. **剧情入口页**：学员昵称输入、创建档案、开始游戏
2. **剧情场景页**：左侧场景插画、右侧对话文本、底部选项按钮
3. **学员仪表盘**：进度环、历史决策时间线、侧边滑出面板
4. **总结页面**：最终得分、完成用时、决策列表、分支树状图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 剧情入口页 | 昵称输入 | 输入2-10位字母数字昵称，后端生成UUID，进入游戏 |
| 剧情场景页 | 场景插画 | 左侧640×400圆角插画，背景色随场景自动变换 |
| 剧情场景页 | 对话区域 | 右侧对话文本，字号18px，行高1.8，颜色#333 |
| 剧情场景页 | 选项按钮 | 底部2-4个选项，宽200px高48px，紫色主色#7E57C2，悬停变深+放大1.05倍，点击0.3s缩放反馈 |
| 剧情场景页 | 进度环 | 顶部导航栏圆形进度环，直径60px，描边#26A69A，背景#E0F2F1 |
| 剧情场景页 | 决策面板 | 右侧滑出面板宽320px，历史决策时间线，颜色区分类型 |
| 学员仪表盘 | 进度展示 | 进度百分比、得分、历史决策树状图 |
| 总结页面 | 得分展示 | 选择加成累加得分、完成用时、决策完整列表、Canvas分支树状图 |

## 3. 核心流程

学员进入游戏 → 输入昵称创建档案 → 加载初始剧情节点 → 阅读对话并选择选项 → 触发剧情推进+决策记录同步 → 循环选择直到剧情结束 → 显示总结页面（得分、用时、决策列表、分支树状图）

## 4. 界面设计

### 4.1 设计风格

- **主背景**：深蓝紫渐变（#1A0B2E → #2D1B4E）
- **玻璃拟态**：半透明卡片，背景rgba(255,255,255,0.08)，背光模糊12px，边框1px rgba(255,255,255,0.2)
- **主色**：紫色#7E57C2，悬停#5E35B1
- **强调色**：进度绿#26A69A，关键紫#7E57C2，对话蓝#42A5F5，得分金#FFA726
- **字体**：显示字体选用 Crimson Pro（神秘优雅），正文选用 Noto Sans SC（中文友好）
- **按钮风格**：圆角8px，白色文字，悬停变色+放大，点击缩放反馈
- **布局**：顶部固定导航栏，剧情内容居中聚焦

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 剧情入口页 | 昵称输入 | 深色渐变背景，中央玻璃卡片，放大镜Logo，输入框+开始按钮 |
| 剧情场景页 | 场景插画 | 左侧640×400圆角插画，渐变背景色随场景ID自动映射 |
| 剧情场景页 | 对话区域 | 右侧居中文本，18px/1.8行高，0.5s渐变过渡(opacity+缩放1.02→1) |
| 剧情场景页 | 选项按钮 | 底部2-4按钮，200×48px，紫色#7E57C2，悬停#5E35B1+1.05倍放大，点击0.3s缩放 |
| 剧情场景页 | 导航栏 | 固定顶部，放大镜Logo+"侦探学院"，昵称显示，进度环60px |
| 剧情场景页 | 决策面板 | 右侧320px滑出，#FAFAFA背景，阴影8px，竖线时间线+彩色圆点 |
| 总结页面 | 分支树 | Canvas树形结构，节点小圆点，已走路径紫色#7E57C2，未走浅灰#BDBDBD |

### 4.3 响应式

- 桌面端：左右分栏布局（场景图640px + 对话区域）
- 移动端（<768px）：单列布局，场景图缩放至全宽，对话和选项区域置于下方
- 所有交互元素在移动端保持可用，按钮增大点击区域

### 4.4 动画与交互反馈

- 场景切换：0.5s渐变过渡（opacity 0→1 + 缩放 1.02→1）
- 选项悬停：背景变深 + 轻微放大1.05倍
- 选项点击：0.3s缩放反馈
- 面板滑出：右侧滑入动画
- 加载状态：脉冲动画
- 决策记录同步：10秒内保存至后端

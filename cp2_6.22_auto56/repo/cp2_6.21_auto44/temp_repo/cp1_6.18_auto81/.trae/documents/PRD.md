## 1. 产品概述

星海棋局是一款科幻主题的回合制策略对战游戏，玩家指挥舰队在随机生成的星图上移动，与AI敌方舰队交战，争夺资源点并摧毁敌方母舰获胜。

- 核心玩法：回合制策略，舰队移动与战斗，资源点争夺，AI对战
- 目标用户：策略游戏爱好者，科幻题材受众
- 产品价值：提供轻量化、快节奏的太空战略对战体验

## 2. 核心功能

### 2.1 功能模块

1. **星图系统**：随机生成星图节点与连线，资源点与母舰位置
2. **舰队系统**：舰队部署、移动、属性管理
3. **战斗系统**：舰队交火、伤害计算、战报生成
4. **AI系统**：AI状态机决策，自动行动
5. **UI界面**：星图渲染、信息面板、结算弹窗

### 2.2 功能详情

| 模块名称 | 功能点 | 功能描述 |
|---------|--------|----------|
| 星图生成 | 节点生成 | 随机生成6-8个圆形节点，半径50px，背景#2D2D44，边框#4ECDC4 |
| 星图生成 | 连线生成 | 节点间连接线，线宽2px，颜色#4A4A6A |
| 星图生成 | 资源点 | 2个资源点带脉冲光晕动画，周期1秒，提供+1攻击修正 |
| 星图生成 | 母舰位置 | 敌方母舰红色高亮，玩家基地为初始舰队位置 |
| 舰队部署 | 初始舰队 | 1艘巡洋舰(HP8/攻击3/移动2/射程1) + 2艘护卫舰(HP4/攻击2/移动3/射程2) |
| 舰队移动 | 选中交互 | 点击己方舰队节点选中，边框金色#FFD700，缩放1.1倍 |
| 舰队移动 | 移动逻辑 | 点击射程内节点移动，粒子流动画(绿色#6BCB77，0.3秒) |
| AI决策 | 状态机 | 战力占优则攻击，资源被占则争夺，否则推进 |
| AI决策 | 动画 | AI移动粒子动画(红色#FF6B6B) |
| 战斗结算 | 伤害计算 | 基于总攻击力、总血量、地形修正计算损伤 |
| 战斗结算 | 战报展示 | 格式："巡洋舰攻击护卫舰，造成3点伤害，护卫舰剩余1HP" |
| 胜负判定 | 胜利条件 | 摧毁敌方母舰或消灭所有敌方舰队 |
| 胜负判定 | 失败条件 | 己方基地被摧毁或所有舰队被消灭 |
| 胜负判定 | 结算面板 | 背景#1A1A2E，圆角16px，金色标题，绿色按钮 |

## 3. 核心流程

游戏主流程：开始游戏 → 生成星图 → 玩家回合(选择舰队→移动/攻击) → 战斗结算 → AI回合(决策→移动/攻击) → 战斗结算 → 检查胜负 → 下一回合/游戏结束

```mermaid
flowchart TD
    A["开始游戏"] --> B["随机生成星图"]
    B --> C["部署初始舰队"]
    C --> D["玩家回合"]
    D --> E["选择己方舰队"]
    E --> F["点击目标节点移动"]
    F --> G{"目标有敌方舰队?"}
    G -->|是| H["触发战斗结算"]
    G -->|否| I["完成移动"]
    H --> J["检查胜负"]
    I --> J
    J -->{"游戏结束?"}
    -->|否| K["AI回合"]
    -->|是| L["显示结算面板"]
    K --> M["AI状态机决策"]
    M --> N["AI移动/攻击"]
    N --> O["战斗结算(如有)"]
    O --> P["检查胜负"]
    P -->{"游戏结束?"}
    -->|否| Q["下一回合"]
    -->|是| L
    Q --> D
```

## 4. 用户界面设计

### 4.1 设计风格

- 主题：深空科幻风格
- 主背景：深空渐变(#0B0B2A → #1A1A3E)
- 主色调：深紫蓝底色 + 青色高光(#4ECDC4) + 金色选中(#FFD700)
- 点缀色：绿色玩家(#6BCB77)、红色敌方(#FF6B6B)
- 字体：现代无衬线字体，清晰易读
- 按钮：圆角设计，悬停放大1.05倍加深阴影，点击缩小0.95倍，过渡0.15s

### 4.2 页面布局

| 区域 | 模块 | UI元素 |
|------|------|--------|
| 主区域 | 星图画布 | 节点、连线、舰队图标、粒子效果、移动动画 |
| 左侧 | 信息面板 | 选中节点详情、舰队信息、回合日志，宽320px，背景#1E1E2E，圆角16px |
| 浮层 | 结算面板 | 游戏结果、重新开始按钮，居中显示 |

### 4.3 动效设计

- 节点悬停：外发光阴影(0 0 12px #4ECDC4)
- 选中节点：脉动动画(缩放1.05~1.1，周期0.5s)
- 资源点：脉冲光晕动画(周期1s)
- 星点粒子：随机漂浮，速度0.5px/s，透明度0.3~0.6
- 舰队移动：粒子流动画(持续0.3s)

### 4.4 响应式

- 桌面端：左侧固定信息面板，右侧星图区域
- 移动端(<768px)：InfoPanel折叠为右侧抽屉，星图缩放适配屏幕
- 触控优化：增大点击热区，支持触摸手势

## 5. 性能约束

- 回合处理：AI决策+战斗结算+渲染在单次交互后500ms内完成
- 动画帧率：星图动画保持稳定60FPS，无卡顿
- 优化策略：CSS动画优先，requestAnimationFrame控制粒子，避免重排重绘

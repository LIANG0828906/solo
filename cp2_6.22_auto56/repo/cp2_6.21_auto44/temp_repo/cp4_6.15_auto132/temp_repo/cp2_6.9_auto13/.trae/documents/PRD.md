## 1. 产品概述

明代茶马互市交易管理与运输模拟系统，让用户以茶马司官员身份管理茶叶收购、马匹估价和商队编组，并模拟从产地到边境驿站的运输过程。

* 主要用途：模拟古代茶马古道的贸易运作，提供沉浸式的历史商业管理体验

* 目标用户：历史爱好者、策略游戏玩家、教育领域使用者

* 产品价值：融合历史文化与策略模拟，通过交互式界面重现古代茶马互市的运作机制

## 2. 核心功能

### 2.1 用户角色

| 角色    | 登录方式       | 核心权限                         |
| ----- | ---------- | ---------------------------- |
| 茶马司官员 | 直接进入（模拟登录） | 管理茶叶与马匹交易、编组商队、监控运输路线、处理驿站补给 |

### 2.2 功能模块

1. **交易市集**：实时价格走势折线图、交易记录表格、价格波动模拟
2. **商队编组**：骡马列表拖拽分配、物资装载、负载监控
3. **运输路线**：拓扑地图展示、商队行进动画、驿站补给决策

### 2.3 页面详情

| 页面名称 | 模块名称  | 功能描述                       |
| ---- | ----- | -------------------------- |
| 主面板  | 标签页切换 | 三个功能模块的水平滑动切换，显示时间戳和库存总量   |
| 交易市集 | 价格走势图 | 茶叶与马匹实时价格波动，每2秒更新，带抖动动画    |
| 交易市集 | 交易记录表 | 显示交易时间、物品、数量、单价、总价，新记录淡入效果 |
| 商队编组 | 骡马列表  | 展示骡马品种、负重上限、速度，支持拖拽        |
| 商队编组 | 编组网格  | 3x3网格展示已编组骡马，负载进度条从绿到红渐变   |
| 商队编组 | 物资库   | 茶包和铁器拖拽装载，超载时橙色进度条和抖动提示    |
| 运输路线 | 拓扑地图  | 雅安→康定→理塘→巴塘→拉萨，连线随海拔渐变     |

| 全局   | 通知栏   | 系统消息滚动展示，紧急/警告/信息分级配色      |

## 3. 核心流程

```mermaid
flowchart LR
    A["登录系统"] --> B["查看交易市集"]
    B --> C["观察价格走势"]
    C --> D["进行茶叶/马匹交易"]
    D --> E["进入商队编组"]
    E --> F["拖拽骡马到编组区"]
    F --> G["装载茶包/铁器物资"]
    G --> H["进入运输路线"]
    H --> I["点击启程按钮"]
    I --> J["商队沿路线行进"]
    J --> K["到达驿站，弹出补给窗口"]
    K --> L{"30秒内决策"}
    L -->|补充物资| M["更新物资后继续行程"]
    L -->|不补充| N["继续行程"]
    M --> O["到达下一站/终点"]
    N --> O
```

用户以茶马司官员身份进入系统后，首先在交易市集观察茶叶和马匹的实时价格波动，选择合适时机进行交易。完成采购后，进入商队编组模块，将骡马拖拽到编组区，并装载茶包和铁器等物资，注意控制负载避免超载。编组完成后，在运输路线页面点击启程，商队将沿茶马古道从雅安出发，经过康定、理塘、巴塘等驿站，最终到达拉萨。每到达一个驿站会弹出补给窗口，用户需在30秒内决定是否补充物资。

## 4. 用户界面设计

### 4.1 设计风格

* **主色调**：米黄(#f4e8c1)作为羊皮纸背景，赭石(#8b5a2b)作为主色，墨绿(#2d5016)作为强调色

* **按钮风格**：木纹纹理背景(#a0785a)，金色边框(#c8963e)，0.5秒悬停放大效果

* **字体**：思源宋体，营造古籍氛围

* **布局**：左窄右宽两栏结构，左侧240px固定导航栏，右侧自适应内容区

* **卡片效果**：0.6s从底部向上浮入动画，伴随纸张翻页声效（CSS keyframes）

* **图标**：线描风格，茶壶、马蹄、商队旗帜等历史元素

### 4.2 页面设计概述

| 页面名称 | 模块名称  | UI元素                                    |
| ---- | ----- | --------------------------------------- |
| 主面板  | 标签导航  | 三个标签页水平滑动切换（0.3s动画），顶部显示时间戳和库存总量        |
| 交易市集 | 价格走势图 | Recharts折线图，x轴时辰（辰时到酉时），y轴银两数，价格更新时线条抖动 |
| 交易市集 | 交易记录表 | 表格展示交易详情，新行从底部淡入                        |
| 商队编组 | 骡马列表  | 卡片展示骡马信息，拖拽时半透明效果                       |
| 商队编组 | 编组网格  | 3x3网格，每格显示马匹缩略                          |

## 1. 产品概述

线上虚拟乐队协作与音乐创作平台，让乐队成员远程同步创作音乐，包括上传音轨、混音调整、片段循环和最终混音导出。目标用户为远程协作的音乐人和乐队成员，核心价值在于打破地理限制，实现实时音频协作与混音创作。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 注册后创建乐队 | 管理乐队、邀请/移除成员、管理音轨和混音 |
| 成员 | 被邀请加入乐队 | 上传音轨、调整混音、导出混音文件 |

### 2.2 功能模块

1. **登录/注册页**：用户注册和登录，JWT认证
2. **乐队列表页**：展示所有乐队卡片，支持搜索和新建乐队
3. **乐队详情页**：音轨列表 + 混音控制台，左右分栏布局

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录/注册页 | 认证表单 | 用户注册/登录，JWT Token管理 |
| 乐队列表页 | 导航栏 | 应用名称（渐变发光）、用户头像下拉菜单、退出按钮 |
| 乐队列表页 | 乐队卡片网格 | 封面渐变色、乐队名、成员头像缩略图（前3+N）、悬停上浮阴影 |
| 乐队列表页 | 创建乐队表单 | 字段逐个出现动画、名称描述、邀请用户搜索选择 |
| 乐队详情页 | 乐队信息栏 | 乐队名称描述、成员横向滚动列表、管理员移除成员 |
| 乐队详情页 | 音轨列表 | 添加音轨（底部滑入模态框）、波形缩略图、音量/平移/静音/删除、拖拽排序 |
| 乐队详情页 | 混音控制台 | 播放/暂停（脉冲动画）、循环模式切换、全局音量+VU电平表、导出按钮 |
| 乐队详情页 | 效果器面板 | 混响/延迟开关和干湿比滑块，平滑过渡动画 |

## 3. 核心流程

用户注册/登录 → 进入乐队列表页 → 创建或加入乐队 → 进入乐队详情页 → 上传音轨 → 调整音量/平移/效果 → 播放预览混音 → 导出最终混音文件

```mermaid
    "调整混音参数" --> "添加效果器"
    "添加效果器" --> "播放预览"
    "播放预览" --> "导出混音"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：靛蓝（#1E1E2E）和电光蓝（#00D2FF）
- 文字色：白色（#F5F5F5）
- 强调色：电光蓝（#00D2FF）和紫罗兰（#8B5CF6）
- 深色主题，赛博朋克科技感风格
- 毛玻璃效果（backdrop-filter: blur(10px)）
- 发光线条边框、霓虹光晕按钮
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- 字体：Orbitron（标题）+ Rajdhani（正文）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 乐队列表页 | 卡片网格 | 渐变封面、悬停上浮阴影、成员头像缩略图 |
| 乐队详情页 | 左右分栏 | 音轨列表左侧、控制台右侧毛玻璃 |
| 乐队详情页 | 音轨卡片 | Canvas波形图、渐变音量滑块、旋转平移旋钮 |
| 乐队详情页 | 控制台 | 脉冲播放按钮、VU电平表、导出进度条 |

### 4.3 响应式适配

- >1024px：左右分栏布局
- 768-1024px：音轨列表在上、控制台在下
- <768px：控制台折叠为底部固定面板可展开

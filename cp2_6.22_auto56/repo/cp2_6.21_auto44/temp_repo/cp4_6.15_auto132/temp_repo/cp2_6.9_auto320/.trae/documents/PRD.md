## 1. 产品概述

一款基于浏览器的宋代香道大师调香与品香模拟游戏，让用户体验古代制香师的完整工艺流程，包括选料、研磨、成型、晾晒、品香全流程。

- 主要目的：通过沉浸式交互体验，让用户了解中国传统香道文化，享受调香创作的乐趣
- 目标用户：对中国传统文化、香道文化感兴趣的玩家，以及喜欢休闲模拟类游戏的用户
- 市场价值：传承传统文化的数字化呈现，具有文化传播与娱乐价值

## 2. 核心 Features

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 制香师 | 无需注册 | 选择香料、研磨混合、成型晾晒、品香评测、保存配方 |

### 2.2 功能模块

1. **香室工作台：香料架、研磨钵、成型模具、晾晒区
2. **品香评测：烟雾动画、嗅觉脑图、星级评分
3. **配方记录册：卷轴展开、配方列表、重新品香
4. **全局倒计时：制香过程计时提醒

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 香室工作台 | 香料架 | 18种香料罐展示，点击播放音效，显示香料信息卡片 |
| 香室工作台 | 研磨钵 | 拖拽香料，鼠标研磨动作，颗粒变小变均匀 |
| 香室工作台 | 成型模具 | 三种香型选择（线香、塔香、香丸），粉末填充动画 |
| 香室工作台 | 晾晒区 | 开窗通风按钮，阳光照射动画，干燥纹理变化 |
| 品香评测 | 香炉品香 | 烟雾动画，雷达图展示，星级评定 |
| 配方记录册 | 卷轴记录 | 配方保存，卷轴展开，金边动画，重新品香 |

## 3. 核心流程

用户进入香室 → 选择香料拖拽至研钵 → 鼠标研磨至合格 → 选择香型模具 → 填充粉末 → 开窗晾晒 → 品香评测 → 保存配方

```mermaid
flowchart TD
    A["进入香室"] --> B["选择香料"]
    B --> C["拖拽至研钵"]
    C --> D["鼠标研磨"]
    D --> E{"研磨合格?"]
    E -->|否| D
    E -->|是| F["选择香型模具"]
    F --> G["粉末填充动画"]
    G --> H["开窗晾晒"]
    H --> I["点击香炉品香"]
    I --> J["烟雾动画"]
    J --> K["雷达图评测"]
    K --> L["保存配方至记录册"]
```

## 4. 用户界面设计

### 4.1 设计风格

**主色调：
- 主背景：米色绢布纹理 #f5ebd4
- 案几：深褐色硬木 #5d4037
- 香料罐：青瓷色 #5a7d6b
- 研钵：青铜色 #8d6e63
- 香炉：铜绿色 #4a7c59
- 金边：金色 #d4a017

**按钮样式：**
- 悬停时颜色加深10%并缩放1.05倍
- 点击时有瞬间压暗效果（0.1s）
- 过渡动画0.3s缓动

**字体选择：**
- 标题字体："Noto Serif SC"（宋体风格）
- 正文字体："ZCOOL XiaoWei"（雅致古风）
- 字号层级：标题24px，副标题18px，正文14px

**布局风格：**
- 宋式文人书房风格
- 仿古卷轴、书页设计
- 褶皱纸纹背景
- 硬木纹理案几

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 香室工作台 | 案几区域 | 650x450px深褐色案几，左侧香料架三排六列 |
| 香室工作台 | 香料罐 | 青瓷罐体，不同颜色罐盖区分香料类型 |
| 香室工作台 | 研钵 | 直径180px青铜研钵，玻璃颗粒效果 |
| 香室工作台 | 成型区 | 三种模具，流动填充动画 |
| 香室工作台 | 晾晒区 | 阳光渐变光线，干燥纹理变化 |
| 品香评测 | 烟雾动画 | 蜿蜒向上烟雾，3秒消散 |
| 品香评测 | 评测卡片 | 三环雷达图，星级评定 |
| 配方记录册 | 卷轴展开 | 从左向右展开，仿古书页，金边动画 |

### 4.3 响应式设计

- **桌面端（960px以上）：横向两栏布局，左侧工作区60%，右侧记录册40%
- **平板端（768px-959px**：左右两栏，左70%右30%
- **移动端（768px以下）：纵向堆叠，工作区在上，记录册在下

### 4.4 交互动效

- 香料罐点击：瓷盖碰撞音效（CSS动画+AudioContext）
- 研磨拖拽：60fps流畅度，颗粒缩小变色
- 模具填充：0.6s流动动画
- 烟雾动画：3s蜿蜒向上
- 卷轴展开：0.3s-0.6s缓动
- 雷达图生成：0.5s动画
- 金边扫过：1s缓动

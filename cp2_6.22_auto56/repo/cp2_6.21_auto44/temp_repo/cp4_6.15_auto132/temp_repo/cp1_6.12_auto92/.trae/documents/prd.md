## 1. 产品概述

建筑光影预览工具（LightShadow Studio）是一款面向建筑系学生和室内设计师的实时光影预览与对比应用，解决设计早期阶段难以直观感受不同窗户朝向、不同时段日照对室内墙面和家具产生的阴影与色彩变化的问题。用户可通过配置房间参数、窗户属性和时间段，实时查看室内光影效果，并支持多方案对比和PNG导出。

- 目标用户：建筑系学生、室内设计师、建筑设计初学者
- 核心价值：在设计早期即可直观预览和对比不同光照方案，避免最终渲染后才发现环境光不理想

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 直接访问 | 所有功能均开放 |

### 2.2 功能模块

1. **主页面**：房间参数配置面板 + 实时渲染区域 + 方案对比区

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 房间参数配置 | 矩形房间尺寸滑块（长宽3-12米，步长0.5米）、墙面色12色卡选择、地板材质三选一（木地板/瓷砖/地毯），调整后实时生成俯视和透视双视图 |
| 主页面 | 光源模拟模块 | 添加1-3个窗户，每个窗户设定朝向（东南西北）、尺寸宽高（0.5-2.5米）、窗台高度（0.3-1.0米）、玻璃透光率（0.6-0.9）；根据时间计算太阳高度角和方位角，投射动态阴影（硬阴影模糊2px，软阴影模糊12px） |
| 主页面 | 时间控制 | 支持手动设定任意时间（24小时制，步长15分钟），切换时场景光照和阴影以15分钟关键帧插值渐变过渡（0.5秒） |
| 主页面 | 对比与导出 | 保存最多4个方案，四宫格并列展示，每格显示标签和参数摘要，点击切换全屏；一键导出PNG含参数水印（灰色12px半透明） |

## 3. 核心流程

用户打开应用 → 配置房间尺寸、墙面色、地板材质 → 添加窗户并设定朝向/尺寸/透光率 → 设定时间查看光影效果 → 调整参数实时预览 → 保存为对比方案 → 四宫格查看对比 → 导出PNG

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：品牌蓝 #4A90D9，hover 亮蓝 #5BA3EC
- 背景色：深灰 #1E1E1E，面板底色 #2D2D2D
- 按钮：圆角矩形，品牌蓝底色，0.2秒过渡
- 字体：系统等宽字体用于数值显示，无衬线字体用于标签
- 布局：左侧配置面板320px + 右侧自适应渲染区
- 动画：面板折叠0.3秒滑动，时间切换0.5秒渐变，视图切换0.3秒淡入淡出

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 配置面板 | 可折叠侧边栏（320px宽，折叠按钮三角形箭头旋转180°），滑块带刻度线和跟随气泡数值，12色卡色块选择器，地板材质单选按钮 |
| 主页面 | 渲染区域 | Canvas画布居中，俯视图和透视图双视图切换按钮，时间滑块和手动输入框 |
| 主页面 | 方案对比区 | 四宫格布局，每格含渲染缩略图+标签+参数摘要，点击全屏按钮，导出PNG按钮 |

### 4.3 响应式设计

- 1440px以上：两栏布局（左配置面板 + 右渲染区）
- 768-1440px：配置面板变为顶部可折叠条
- 768px以下：配置面板变为全屏抽屉从左侧滑出

### 4.4 性能要求

- 滑块调整到场景更新延迟 ≤ 50ms
- 阴影计算更新帧率 ≥ 30fps
- 多方案四宫格首屏加载 ≤ 1秒

### 4.5 技术约束

- TypeScript + 纯Canvas渲染（不使用第三方3D库）
- 文件结构：package.json、index.html、tsconfig.json、vite.config.js、src/main.ts、src/renderer.ts、src/ui.ts
- 运行方式：npm install && npm run dev

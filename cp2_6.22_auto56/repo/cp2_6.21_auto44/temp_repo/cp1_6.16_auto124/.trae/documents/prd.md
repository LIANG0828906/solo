## 1. 产品概述

量子纠缠态粒子对与自旋交互模拟3D可视化项目，通过沉浸式3D场景展示量子纠缠的核心概念——两个粒子之间的超距关联效应。用户可通过交互操作直观理解量子自旋、叠加态和纠缠度等抽象物理概念。

- 核心目的：将抽象的量子物理概念转化为可视化、可交互的3D体验
- 目标用户：物理爱好者、学生、科普教育工作者
- 市场价值：量子物理科普教育工具，降低量子力学的理解门槛

## 2. 核心功能

### 2.1 功能模块

1. **3D场景主界面**：双粒子系统、缠绕光带、背景星尘粒子、相机控制
2. **自旋交互系统**：点击粒子弹出控制面板、三种自旋状态切换、纠缠响应
3. **纠缠度仪表盘**：实时纠缠度显示、指针动画、状态指示
4. **视觉动效系统**：球体脉动、光带旋转、脉冲动画、状态过渡

### 2.2 功能详情

| 模块名称 | 功能描述 |
|---------|----------|
| 双粒子系统 | 两个半透明发光球体（紫色#9B59B6、青色#1ABC9C），半径0.8单位，表面正弦波脉动（2Hz，振幅0.03） |
| 缠绕光带 | 两球之间半透明螺旋光带，紫青渐变，两股螺旋，0.5圈/秒旋转 |
| 相机控制 | OrbitControls鼠标拖拽旋转视角，滚轮缩放 |
| 自旋控制面板 | 点击球体弹出，半透明毛玻璃背景，三按钮：自旋向上（红↑）、自旋向下（蓝↓）、叠加态（黄波浪） |
| 自旋状态 | 向上=顺时针旋转，向下=逆时针旋转，叠加态=半透明闪烁+双速随机切换 |
| 纠缠响应 | 另一粒子瞬时（≤0.1s）以相反方向或相同叠加态响应，伴随膨胀脉冲动画（1.1倍，0.3s） |
| 纠缠度仪表盘 | 左下角Canvas绘制半圆刻度盘，0-100指针，红→金渐变，80%+闪烁光晕 |
| 纠缠度算法 | 行为一致上升，不一致下降，easeOutCubic缓动更新（0.5s） |
| 背景环境 | 深空径向渐变（#0B0C10→#1F2833），500个浮动光点粒子（1-3px，透明度0.2-0.6） |

## 3. 核心流程

用户进入页面 → 查看初始3D场景（两粒子、缠绕光带、星尘背景）→ 鼠标拖拽旋转视角/滚轮缩放 → 点击任一球体 → 弹出自旋控制面板 → 选择自旋状态（向上/向下/叠加态）→ 当前粒子状态改变 → 另一粒子瞬时纠缠响应 → 纠缠度数值动画更新 → 可继续交互或旋转观察

## 4. 用户界面设计

### 4.1 设计风格

- **设计主题**：深空科技风 / 赛博量子美学
- **主色调**：深黑蓝背景（#0B0C10 - #1F2833），紫色（#9B59B6），青色（#1ABC9C）
- **点缀色**：红色（自旋向上）、蓝色（自旋向下）、黄色（叠加态）、金色（高纠缠度）
- **材质质感**：半透明发光、毛玻璃效果、能量光晕
- **动效风格**：平滑缓动（easeInOut 0.3s）、脉动呼吸、螺旋旋转、脉冲闪烁

### 4.2 页面设计概览

| 区域 | 元素 | UI细节 |
|------|------|--------|
| 中央区域 | 双粒子+缠绕光带 | 悬浮居中，发光材质，表面脉动 |
| 背景 | 深空渐变+星尘粒子 | 径向渐变底色，随机浮动光点 |
| 左下角 | 纠缠度仪表盘 | Canvas绘制半圆刻度盘，指针动画 |
| 球体点击 | 控制面板 | 毛玻璃背景（rgba(255,255,255,0.1)），2px半透明边框，圆角10px |
| 全局交互 | 过渡动画 | 所有状态变化0.3s easeInOut平滑过渡 |

### 4.3 响应式

- Desktop-first设计，全屏3D场景
- 移动端支持触摸拖拽和双指缩放
- 仪表盘尺寸自适应屏幕
- 控制面板在小屏幕上调整位置避免遮挡

### 4.4 3D场景指引

- **环境**：深空径向渐变背景，无外部HDRI，自发光粒子营造氛围
- **光照**：点光源配合粒子自发光，柔和环境光
- **相机**：PerspectiveCamera，初始距离适中，OrbitControls限制极角
- **构图**：双粒子水平对称分布，缠绕光带连接，视觉重心居中
- **交互**：点击检测使用Raycaster，状态切换有明显视觉反馈
- **后处理**：粒子自发光效果，可选Bloom辉光增强视觉
- **性能**：球体细分≤64x64面，粒子≤500个，目标60FPS

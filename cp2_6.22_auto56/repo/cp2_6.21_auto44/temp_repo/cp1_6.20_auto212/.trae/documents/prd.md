## 1. 产品概述

户外探险俱乐部活动策划与装备库存管理应用，帮助领队统一协调徒步/露营活动的成员报名、装备分配和天气预警信息，解决信息分散在不同渠道、难以统一管理的问题。

- 目标用户：小型户外探险俱乐部的领队和成员
- 核心价值：一站式活动策划、报名管理、装备分配和天气预警，提升活动组织效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 领队 | 系统预设 | 创建活动、管理装备库存、查看报名信息、查看天气预警 |
| 成员 | 报名时填写 | 报名参加活动、选择装备需求 |

### 2.2 功能模块

1. **主面板**：四区域布局（导航栏、活动列表、活动详情、装备清单），点击活动列表联动更新详情和装备面板
2. **装备清单**：按类别分组显示库存占用率，超80%脉冲高亮动画预警
3. **创建活动**：全屏浮层表单，含活动名称、日期、难度、描述、路线图上传、最大人数
4. **成员报名**：模态框报名，含姓名、手机号、装备需求选择，报名后联动更新装备库存
5. **天气预警**：导航栏天气图标下拉面板，三天天气预报，降水概率>70%蓝色闪烁边框，雨天黄色警告条

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主面板 | 导航栏 | 显示应用名称、用户头像、天气图标（点击展开天气面板） |
| 主面板 | 活动列表 | 按日期倒序显示活动，含名称、日期、报名人数/上限、难度彩色标签；虚拟滚动 |
| 主面板 | 活动详情 | 展示活动描述、路线图、报名成员名单、报名按钮、雨天警告条；点击活动时从左滑入 |
| 主面板 | 装备清单 | 按类别分组，每项显示名称、总库存、已分配数量、彩色进度条；超80%脉冲高亮 |
| 创建活动浮层 | 活动表单 | 全屏浮层从底部滑入，含完整表单字段，提交后新活动淡入列表顶部 |
| 报名模态框 | 报名表单 | 居中模态框（背景模糊），含姓名、手机号、装备勾选，提交后成员飞入动画 |

## 3. 核心流程

**活动策划流程**：领队打开应用 → 查看活动列表 → 点击创建新活动 → 填写活动信息并提交 → 新活动出现在列表顶部 → 查看天气预警确认安全 → 等待成员报名

**成员报名流程**：成员浏览活动列表 → 选中感兴趣的活动 → 查看详情 → 点击报名 → 填写姓名手机号并选择装备 → 提交报名 → 装备库存自动更新

**装备管理流程**：领队查看装备面板 → 发现库存占用率过高（脉冲高亮） → 安排补充库存 → 更新库存数据

## 4. 用户界面设计

### 4.1 设计风格

- 主色：森林绿 #2d6a4f，辅色：大地棕 #9c6644
- 背景：灰白 #f8f9fa，卡片：白色 #ffffff，阴影：rgba(0,0,0,0.08)
- 文字主色：深灰 #212529，次要文字：灰 #6c757d
- 按钮风格：主色填充圆角按钮，hover变浅 #40916c，点击缩放 scale(0.96)
- 字体：Noto Sans SC（中文）+ DM Sans（英文/数字）
- 布局风格：四区域卡片式布局，hover上浮4px加深阴影
- 图标风格：Lucide线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主面板 | 导航栏 | 森林绿背景，白色文字，左侧应用名称+logo，右侧天气图标+用户头像 |
| 主面板 | 活动列表 | 白色卡片列表，左侧边框颜色表示难度（绿/黄/红），虚拟滚动 |
| 主面板 | 活动详情 | 白色卡片，顶部天气警告条（#fff3cd），路线图网格，成员头像列表 |
| 主面板 | 装备清单 | 白色卡片按类别分组，彩色进度条（绿/黄/红），超80%脉冲动画 |
| 创建活动浮层 | 活动表单 | 全屏半透明浮层从底部滑入，白色表单区域，难度三选一带彩色圆点 |
| 报名模态框 | 报名表单 | 居中模态框，backdrop-filter: blur(4px)，装备勾选列表含剩余数量 |

### 4.3 响应式设计

- 桌面优先，四区域并排布局
- 移动端（<768px）：左侧列表和中间详情区上下堆叠，右侧装备清单通过底部固定标签页切换
- 触摸优化：按钮最小44px点击区域，卡片点击反馈

### 4.4 动画规范

- 活动切换：内容从左滑入 0.3s ease-out
- 创建浮层：从底部滑入/滑出
- 报名成员：从右向左飞入 0.4s cubic-bezier
- 新活动卡片：淡入动画
- 装备超80%：脉冲高亮 1s 周期
- 降水>70%：蓝色闪烁边框 0.8s 周期
- 雨天警告条：从右向左滚动进入 0.5s ease-in-out
- 卡片hover：上浮4px加深阴影

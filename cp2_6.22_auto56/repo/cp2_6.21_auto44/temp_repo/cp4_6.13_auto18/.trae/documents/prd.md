## 1. 产品概述

奇幻生物融合进化模拟养成页游——玩家通过收集和组合不同属性的基础生物，生成全新融合形态，并让它们参与自动战斗来验证进化路径的有效性。目标用户为喜欢养成、融合、策略对战的休闲游戏玩家，提供"融合+收集+对战"三位一体的游戏体验。

## 2. 核心功能

### 2.2 功能模块

1. **融合工坊（/home）**：基础生物卡片展示、拖拽入合成槽、融合生成新生物
2. **生物图鉴（/bestiary）**：已解锁生物列表、详细属性面板、融合溯源
3. **自动竞技场（/arena）**：组队选择、AI匹配、Canvas回合制自动对战、经验升级

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 融合工坊 | 基础生物卡片区 | 展示6张基础生物卡片（火元素、水元素、植物精灵、雷灵、暗影兽、岩石巨人），每张卡片显示元素图标、名称、HP、ATK、被动技能图标，点击卡片可拖入合成槽 |
| 融合工坊 | 合成槽区域 | 左右两个合成槽，支持拖拽放入和点击移出，两个槽都有生物时激活融合按钮 |
| 融合工坊 | 融合动画 | 两道光束汇聚成光球→爆炸→生成新卡片，动画不超过2秒 |
| 融合工坊 | 融合结果展示 | 新生物卡片飞入展示，显示融合后的属性和随机继承的技能 |
| 生物图鉴 | 图鉴列表 | 按获得时间倒序展示已解锁生物，融合变异品种有特殊彩色边框 |
| 生物图鉴 | 详细属性面板 | 右侧弹出面板，展示完整属性、技能描述、随机背景故事、融合来源 |
| 自动竞技场 | 队伍选择 | 玩家从已拥有生物中选择3只组队，展示生物缩略卡片 |
| 自动竞技场 | 对战场景 | Canvas回合制自动对战，生物自动攻击/概率触发技能，血条实时动画下降，底部能量流动光效 |
| 自动竞技场 | 对战结果 | 胜负判定，获胜队伍生物获得经验值，经验条可视化累积，升级提升基础属性 |

## 3. 核心流程

**融合流程**：玩家在融合工坊浏览基础生物卡片 → 拖拽两张卡片到合成槽 → 点击融合按钮 → 触发融合动画 → 获得新生物卡片 → 自动收录图鉴

**对战流程**：玩家进入竞技场 → 选择3只生物组队 → 系统匹配AI队伍 → Canvas自动回合制对战 → 胜负判定 → 获胜获得经验 → 升级提升属性

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暗紫 (#2D1B4E) + 星空蓝 (#0A1628)，强调色：琥珀金 (#D4A017)
- **卡片风格**：羊皮纸质感圆角边框，内层微黄底 (#F5E6C8)，外层暗紫描边
- **字体**：标题使用 MedievaSharp 或 Cinzel 风格的奇幻手绘体，正文使用 Noto Serif SC
- **布局**：顶部导航栏 + 内容区，融合工坊采用左右分栏（卡片池 + 合成区），图鉴采用左列表右详情
- **图标风格**：手绘风格元素图标，火焰/水滴/叶子/闪电/暗影/岩石
- **融合变异卡片**：特殊彩色渐变边框（紫→蓝→金流光动画）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|---------------------|
| 融合工坊 | 基础生物卡片区 | 卡片网格3列，每张卡片：羊皮纸背景、元素图标左上角、名称居中、HP/ATK数值底部、技能图标右下角，悬停放大+发光，点击拖拽效果 |
| 融合工坊 | 合成槽区域 | 两个圆形凹槽（虚线边框），拖入后显示生物缩略图，融合按钮居中（琥珀金色3D按钮），空槽有"+"提示 |
| 融合工坊 | 融合动画 | Canvas全屏覆盖层，两道光束从左右合成槽射出→中间汇聚成光球→光球膨胀→爆炸粒子扩散→新卡片从光芒中浮现 |
| 生物图鉴 | 图鉴列表 | 左侧滚动列表，卡片小尺寸，变异品种有流光边框动画，点击高亮选中态 |
| 生物图鉴 | 详细属性面板 | 右侧滑出面板，顶部大尺寸生物卡片，下方属性条（HP/ATK渐变进度条），技能描述区域，背景故事区域（卷轴样式），融合来源标签 |
| 自动竞技场 | 队伍选择 | 上方3个槽位选择框，下方生物卡片滚动列表，选中的卡片有发光边框 |
| 自动竞技场 | 对战场景 | Canvas画布，上方我方3只生物（头像+血条），下方敌方3只生物，攻击动画（弧线飞行→命中闪光），技能特效（元素粒子），底部能量流动光效 |
| 自动竞技场 | 对战结果 | 胜利/失败大字+粒子庆祝/灰色效果，经验条增长动画，升级闪光特效 |

### 4.3 响应式设计

- **桌面端**（≥1024px）：融合工坊左右分栏，图鉴左右布局，竞技场宽幅Canvas
- **平板端**（768-1023px）：融合工坊上下布局，图鉴切换模式，竞技场适中Canvas
- **手机竖屏**（<768px）：所有页面单列布局，卡片1-2列网格，竞技场全宽Canvas，底部固定导航栏替代顶部导航

### 4.4 性能目标

- 对战Canvas动画稳定60FPS
- 融合动画完成时间≤2秒
- 星云粒子背景控制在200个粒子以内
- 首次加载时间<3秒

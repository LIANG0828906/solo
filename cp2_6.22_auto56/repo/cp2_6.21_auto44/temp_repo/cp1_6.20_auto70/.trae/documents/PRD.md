## 1. 产品概述

阅读书签（ReadMark）是一款面向阅读爱好者的书单管理与读书笔记应用，解决纸质书和电子书阅读记录分散、无法快速回顾核心内容的痛点。目标用户为日常阅读者，产品价值在于将阅读进度、思维导图和读书笔记整合到统一界面中，构建完整的阅读知识管理体系。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 管理个人书单、笔记、思维导图，导入导出数据 |

### 2.2 功能模块
1. **首页**：图书卡片网格、搜索筛选、添加图书、空状态展示
2. **详情页**：书籍信息栏、思维导图编辑器、笔记列表与编辑器

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 图书卡片网格 | 展示所有图书卡片，卡片包含封面缩略图/渐变色块、阅读状态标签、页数进度条、最近阅读日期 |
| 首页 | 搜索筛选栏 | 按书名/作者关键词搜索，按阅读状态筛选，300ms防抖实时更新 |
| 首页 | 添加图书弹窗 | 填写书名、作者、总页数、封面URL（可选）、阅读状态 |
| 首页 | 空状态 | 无匹配结果时显示书本轮廓插画和"添加新书"按钮 |
| 详情页 | 左栏-书籍信息 | 封面大图、作者、页数、添加时间、阅读统计（今日时长、总天数、完成百分比） |
| 详情页 | 中栏-思维导图 | 可添加中心/子节点、拖拽位置、双击编辑文字、右键删除、贝塞尔曲线连接线 |
| 详情页 | 右栏-笔记列表 | 按时间倒序排列，含创建时间、字数、多选标签、正文，支持收藏置顶 |
| 详情页 | 笔记编辑器 | Markdown语法编辑+实时预览，保存后返回列表 |

## 3. 核心流程

用户打开应用 → 首页查看图书卡片 → 搜索或筛选图书 → 点击卡片进入详情 → 查看书籍信息/编辑思维导图/阅读笔记 → 新建Markdown笔记 → 收藏笔记 → 返回首页继续管理 → 导入导出数据备份

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：米白（#FAF6F1）、淡棕（#D4C5B2）、深褐（#5C3D2E）
- 强调色：低饱和度蓝色（#6B8DAE）
- 按钮样式：圆角，悬浮时渐变背景+1.05倍缩放
- 字体：标题使用衬线体（Playfair Display），正文使用无衬线体（Source Sans 3）
- 布局风格：卡片网格、顶部固定毛玻璃导航栏、三栏详情布局
- 图标风格：线性图标（Lucide）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 毛玻璃效果，半透明白色背景+模糊，固定顶部，圆角风格 |
| 首页 | 图书卡片 | 封面缩略图/渐变色块+首字母、阅读状态彩色标签、进度条、日期，悬停上浮-4px+阴影加深+0.2s缓动 |
| 首页 | 空状态 | 书本透明轮廓插画、提示文字、添加按钮 |
| 详情页 | 左栏 | 封面大图、统计面板 |
| 详情页 | 中栏 | Canvas思维导图，贝塞尔连线 |
| 详情页 | 右栏 | 笔记卡片列表，星标收藏图标 |
| 详情页 | 笔记编辑器 | 分屏Markdown编辑+预览 |

### 4.3 响应式
- 桌面优先设计，手机端（<768px）三栏变为单栏上下堆叠
- 手机端卡片通栏，缩小字体和间距
- 触控优化：按钮最小44px点击区域

### 4.4 动效
- 加载状态：旋转书本SVG动画
- 卡片悬停：translateY(-4px) + 阴影加深，0.2s ease过渡
- 按钮：hover渐变+scale(1.05)
- 页面切换：淡入淡出过渡

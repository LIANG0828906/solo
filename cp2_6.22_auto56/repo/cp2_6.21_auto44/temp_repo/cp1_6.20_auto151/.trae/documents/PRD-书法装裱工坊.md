## 1. 产品概述

面向书法爱好者的线上作品装裱模拟与风格对比应用，解决书法作品拍照上传后难以直观预览不同装裱样式视觉效果的痛点，支持卷轴、镜框、扇面三种装裱模板的实时预览与多风格对比。

- 核心目标：让书法爱好者在实体装裱前，通过数字化方式预览并对比多种装裱方案，降低决策成本
- 目标用户：书法爱好者、书画从业者、艺术品收藏者
- 市场价值：填补书法作品数字化装裱预览的细分空白，提升线上书画交流平台的用户体验

## 2. 核心功能

### 2.1 功能模块

1. **图片上传与裁剪模块**：拖拽/点击上传JPG/PNG，自动1:1.4比例裁剪，支持手动调整裁剪框
2. **装裱样式切换模块**：卷轴、镜框、扇面三种模板，Canvas分层绘制渲染
3. **装裱参数调整模块**：各样式独立参数面板（颜色、纹理、宽度、材质等）
4. **多风格并排对比模块**：2-4种装裱样式同屏展示，统一参数调整联动更新
5. **导出分享模块**：PNG格式导出，含进度条动画与自动下载

## 3. 核心流程

用户进入应用 → 上传书法图片（拖拽或点击）→ 自动进入裁剪模式，调整1:1.4裁剪区域 → 确认裁剪后生成预览 → 选择装裱样式（卷轴/镜框/扇面）→ 调整对应装裱参数（颜色、纹理、宽度等）→ 实时Canvas渲染预览 → 可选开启多风格并排对比（2-4种）→ 统一调整参数联动更新 → 点击导出按钮，展示进度条动画 → 自动下载PNG图片

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白（#f5f0e8）背景、深褐（#4a3520）导航/文字、朱红（#c23b22）强调色
- **按钮样式**：圆角设计，点击时 scale(0.95) 下沉100ms，滑块轨道4px高度渐变、滑块16px圆形朱红色
- **字体方案**：楷体风格（SimKai / Noto Serif SC / 系统楷体回退），应用名称使用艺术化楷体
- **布局风格**：卡片式参数面板（白底+淡褐阴影box-shadow），桌面端左右分栏，移动端上下堆叠
- **动画体系**：
  - 样式切换：200ms淡入淡出过渡
  - 选中样式按钮：朱红下划线从中心向两侧200ms ease-out展开
  - 拾色器弹出：300ms缩放动画
  - 参数变化预览：300ms交叉淡入
  - 样式卡片选中：蓝色边框+scale(1.1)放大
  - 上传拖拽：边框变朱红+波纹动画
  - 手风琴展开：max-height过渡动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 顶部导航 | 深褐#4a3520背景，左对齐楷体应用名，右侧导出按钮（朱红） |
| 主页面 | 上传区 | 居中圆形虚线边框（#4a3520），米白#f5f0e8背景，提示文字楷体深褐，拖拽时边框朱红+波纹动画 |
| 主页面 | 样式选择器 | 三个圆形图标按钮水平排列，选中底部朱红下划线中心展开动画（200ms ease-out） |
| 主页面 | 参数卡片 | 白色背景圆角，淡褐色box-shadow阴影，参数标签楷体深褐，拾色器缩放弹出动画 |
| 主页面 | 预览画布 | Canvas居中，外层淡褐边框，缩放拖拽光标，200ms淡入切换，移动端100%宽 |
| 主页面 | 并排对比 | 等宽CSS Grid布局，每列圆角卡片，顶部样式名标签，底部缩略图，联动300ms交叉淡入 |
| 主页面 | 导出控件 | 朱红按钮楷体文字，点击后内部进度条（白）从左至右1秒填满，完成后触发下载 |

### 4.3 响应式设计

- **桌面端（≥768px）**：UploadPanel + StyleSelector 左侧堆叠，PreviewPanel 右侧主区域，参数面板默认展开
- **平板/移动端（<768px）**：UploadPanel + StyleSelector 上半部分垂直排列，PreviewPanel 下半部分全宽，画布宽度自适应
- **小屏手机（<480px）**：参数面板改为可折叠手风琴（默认收起），点击展开时max-height过渡动画，裁剪控件缩小适配触控操作
- **触控优化**：滑块拖拽区域增大至44px高度，按钮最小触控区域44×44px

### 4.4 Canvas渲染分层策略

- **卷轴样式**（3层）：底层绫布纹理渐变 → 中层上下轴头（矩形+圆角，颜色可调）→ 上层裁剪后书法图片（居中）
- **镜框样式**（4层）：底层外框（颜色可调，宽度3-20px矩形描边）→ 中层白色卡纸（颜色可调，内阴影投影）→ 留白内边距 → 上层裁剪后书法图片
- **扇面样式**（3层）：底层扇形渐变背景底色 → 中层扇骨（径向渐变+放射线条，材质：竹/木/铜）→ 上层书法图片裁剪为扇形区域clip-path

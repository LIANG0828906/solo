## 1. 产品概述

技能进化论是一款在线角色技能进化与对战模拟器，让用户创建角色、配置技能树，并在模拟对战中测试不同技能搭配的效果。目标用户为游戏爱好者、RPG策略研究者和游戏设计学习者，核心价值在于提供直观的技能搭配策略实验环境与可视化战斗模拟体验。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接进入 | 创建角色、编辑技能树、发起战斗模拟、查看战斗报告 |

### 2.2 功能模块

1. **角色创建页**：职业选择（战士/法师/刺客）、初始属性展示、基础技能预览
2. **技能树编辑器**：横向展开树状图、技能节点交互、技能点分配
3. **战斗模拟器**：回合制自动对战、技能特效渲染、战斗日志实时展示
4. **战斗报告页**：战斗统计、MVP技能评选、技能使用对比图表、日志联动

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 角色创建页 | 职业选择卡片 | 展示三个职业（战士红褐色、法师蓝紫色、刺客墨绿色），点击选择后显示初始属性（HP/ATK/DEF/SPD）和3个基础技能 |
| 技能树编辑器 | 树状图渲染区 | 横向展开的技能树，已解锁节点金色边框发光动画，未解锁灰色半透明，点击弹出技能详情浮层 |
| 技能树编辑器 | 技能点分配 | 初始5点技能点，点击节点消耗1点解锁，支持重新分配 |
| 战斗模拟器 | 玩家信息面板 | 左侧展示角色圆形头像（边框色随职业）、已选技能列表及快捷键1-4 |
| 战斗模拟器 | 敌方AI面板 | 右侧展示随机生成的AI角色信息 |
| 战斗模拟器 | 战斗场地 | 中间俯视角方形网格，边缘LED灯带装饰，技能释放闪光特效+伤害数字跳字渐隐 |
| 战斗模拟器 | 战斗日志 | 底部滚动区域，新条目从底部滑入，回合间0.5秒延迟 |
| 战斗报告页 | 统计卡片 | 总回合数、双方总伤害、技能命中率、MVP技能（奖杯图标），从中心放大入场动画 |
| 战斗报告页 | 技能使用对比图 | 横向柱状图对比双方各技能使用次数，柱状图用角色主题色填充，点击高亮联动日志条目 |

## 3. 核心流程

用户进入应用后，首先在角色创建页选择职业（战士/法师/刺客），系统根据职业预设初始属性和3个基础技能。随后进入技能树编辑器，以横向树状图展示该职业的全部技能节点，用户使用初始5个技能点点击解锁技能节点，点击节点可查看技能详情浮层。编辑完成后进入战斗模拟器，系统随机生成一个敌方AI角色，点击开始战斗后自动进行回合制模拟，技能释放时有颜色闪光特效和伤害数字动画，底部实时滚动战斗日志。战斗结束后自动跳转到战斗报告页，展示详细统计数据、MVP技能和技能使用对比图表，支持点击图表项联动高亮对应日志条目。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝紫渐变（#0a0e27 → #1a1040）太空主题背景，微弱星空粒子动画
- 强调色：金色（#d4a843）用于高亮、边框、装饰
- 职业色系：战士红褐色（#c0392b / #a0522d）、法师蓝紫色（#8e44ad / #6c3483）、刺客墨绿色（#1a6b3c / #0d4f2b）
- 卡片：毛玻璃效果（backdrop-filter: blur(10px)，边框 1px rgba(255,255,255,0.1)）
- 按钮：职业色系渐变，悬停时亮度和饱和度提升
- 字体：Orbitron（标题/数字）+ Exo 2（正文/描述）
- 布局：居中卡片式布局，顶部导航栏
- 图标风格：几何线条风格图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 角色创建页 | 职业选择卡片 | 三列卡片布局，每张卡片顶部圆形职业图标、职业名、属性条、基础技能列表，深蓝紫背景+毛玻璃卡片 |
| 技能树编辑器 | 技能树画布 | 横向展开树状图，SVG/Canvas连线，圆形节点（60px），已解锁金色发光边框，未解锁灰色半透明，点击弹出浮层（深灰蓝#1a2332底+金色点缀） |
| 技能树编辑器 | 技能点计数器 | 右上角显示剩余技能点，金色数字 |
| 战斗模拟器 | 玩家面板 | 左侧竖排，圆形头像（职业色边框）、角色名、HP条、技能列表（快捷键1-4） |
| 战斗模拟器 | 战斗场地 | 中间方形网格（8x8），深色纹理，边缘LED灯带装饰（渐变呼吸动画） |
| 战斗模拟器 | 敌方面板 | 右侧竖排，同玩家面板布局 |
| 战斗模拟器 | 战斗日志 | 底部半透明滚动区域，日志条目左侧带职业色竖线 |
| 战斗报告页 | 结果卡片 | 居中大卡片，从中心放大入场动画，金色MVP奖杯图标 |
| 战斗报告页 | 对比柱状图 | 横向柱状图，玩家侧用玩家职业色，AI侧用AI职业色，可点击联动日志 |

### 4.3 响应式设计

- 桌面优先设计（1280px+最佳体验）
- 平板端（768px-1279px）战斗场地缩小，面板水平排列
- 移动端（<768px）面板折叠为顶部标签页切换

### 4.4 性能目标

- 所有动画帧率 ≥ 30fps
- 页面初始加载 ≤ 2秒
- 战斗动画回合间延迟0.5秒

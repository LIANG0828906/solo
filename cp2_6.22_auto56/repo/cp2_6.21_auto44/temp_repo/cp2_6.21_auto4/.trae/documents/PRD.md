## 1. 产品概述

一款面向博客作者和小团队的浏览器端电子杂志制作与翻页预览工具，解决图文内容低成本编排为具有纸质阅读体验的交互式杂志的问题。用户可在可视化编辑器中拖拽添加文字、图片和色块，自动生成封面与目录，并以纸张卷曲翻页动画全屏预览，支持JSON导入导出与封面缩略图生成。

- 目标用户：博客作者、自媒体人、小团队编辑
- 核心价值：零成本、零安装，浏览器中即可完成从编辑到预览的完整杂志制作流程

## 2. 核心功能

### 2.1 功能模块

1. **编辑器页面**：页面管理面板 + A4画布编辑区 + 顶部工具栏
2. **预览页面**：全屏翻页预览模式

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 左侧页面面板 | 添加最多12页杂志页面（白色圆角卡片），支持拖拽排序，点击选中进入编辑模式 |
| 编辑器页面 | 画布编辑区 | A4比例画布（1:1.414），可自由拖拽添加文字块、图片、矩形色块，支持选中后变换位置/大小/旋转角度，独立z-index层级 |
| 编辑器页面 | 顶部工具栏 | 毛玻璃效果工具栏，胶囊按钮控制添加元素、预览、导入导出等操作 |
| 编辑器页面 | 封面生成 | 指定某页为封面，自动居中显示杂志名称和作者名（白色粗体+深色半透明遮罩） |
| 编辑器页面 | 目录生成 | 自动生成目录页，列出每页标题（截取前18字符）和缩略图，点击跳转对应页 |
| 预览页面 | 翻页预览 | 全屏模式，左右半透明翻页水印区域，悬停翻页，键盘左右箭头翻页，纸张卷曲CSS动画 |
| 预览页面 | 进度条 | 底部进度条显示当前页码/总页数 |
| 编辑器页面 | 数据导入导出 | 保存/加载JSON文件，导出时生成封面缩略图PNG（400×566px） |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖灰(#f8f5f0)背景，纯白(#ffffff)面板，墨水蓝(#2c3e50)按钮与文字
- **高亮色**：暖橙(#e67e22)，用于当前选中页的页码标记
- **选中态**：1.5px虚线边框(#3498db) + 四角8px圆形手柄(#3498db)
- **按钮样式**：圆角8px墨水蓝胶囊按钮，悬停变深(#1a252f)
- **工具栏**：半透明毛玻璃效果（rgba(255,255,255,0.75)，blur 10px）
- **字体**：编辑器UI使用系统字体，杂志内容文字块支持5种Google Fonts（Noto Serif SC、ZCOOL QingKe HuangYou、Ma Shan Zheng、ZCOOL XiaoWei、Dancing Script）
- **布局**：左侧320px面板 + 右侧自适应编辑区，极简杂志风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 编辑器页面 | 左侧页面面板 | 白色背景，页面缩略图卡片（圆角8px），拖拽排序手柄，页码标签（选中态暖橙），添加页面按钮 |
| 编辑器页面 | 画布编辑区 | A4比例白色画布居中，元素拖拽变换手柄，选中虚线边框，暖灰背景 |
| 编辑器页面 | 顶部工具栏 | 毛玻璃半透明条，胶囊墨水蓝按钮（添加文字/图片/色块、封面、目录、预览、导入、导出） |
| 编辑器页面 | 元素属性面板 | 字体选择下拉、字号滑块(12-72px)、颜色选择器 |
| 预览页面 | 翻页区域 | 深灰(#2a2a2a)背景，A4页面居中，左右半透明翻页水印区域 |
| 预览页面 | 进度条 | 底部页码指示条（当前页/总页数） |

### 4.3 响应式

- 桌面优先设计，浏览器宽度<768px时左侧面板折叠为顶部可展开托盘
- 编辑区占满剩余高度
- 翻页预览始终全屏展示

### 4.4 动效

- 页面元素添加/删除：0.3s透明度过渡动画
- 翻页动画：CSS perspective + rotateY，0.5s ease-in-out，模拟纸张卷曲
- 元素拖拽：延迟不超过50ms，60fps流畅
- 按钮悬停：颜色加深过渡

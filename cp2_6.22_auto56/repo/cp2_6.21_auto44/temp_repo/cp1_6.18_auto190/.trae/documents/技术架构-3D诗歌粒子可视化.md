## 1. 架构设计

```mermaid
graph TD
    A["React UI层 (AppUI.tsx)"] -->|用户事件| B["Zustand全局状态 (appStore.ts)"]
    B -->|状态更新| A
    B -->|诗句/主题| C["TextParser.ts 文本解析"]
    C -->|粒子初始化数据| D["ParticleEngine.ts 粒子引擎"]
    D -->|粒子快照| E["SceneRenderer.ts Three.js渲染"]
    E -->|交互事件(悬停/拖拽)| D
    B -->|保存/加载/点赞/评论| F["GalleryEngine.ts 画廊引擎"]
    F -->|画廊数据| B
    F <-->|持久化| G["IndexedDB 本地存储"]
    F -->|缩略图| H["OffscreenCanvas 渲染"]
```

## 2. 技术说明

- **前端框架**：React@18 + TypeScript
- **构建工具**：Vite@5 + @vitejs/plugin-react
- **3D渲染引擎**：Three.js（原生，不使用@react-three/fiber，按用户要求直接在engine层管理）
- **状态管理**：Zustand
- **唯一ID**：uuid
- **数据库**：IndexedDB（通过原生API封装）
- **无后端**：纯前端应用，画廊数据本地持久化

## 3. 文件组织与路由

本项目为单页应用（SPA），无多路由，通过UI组件切换显示主场景与画廊视图。

| 文件路径 | 职责说明 |
|----------|----------|
| `package.json` | 依赖声明与启动脚本 |
| `vite.config.js` | Vite + React插件配置 |
| `tsconfig.json` | TypeScript严格模式，target ES2020 |
| `index.html` | 入口HTML，全屏应用容器，Google Fonts引入 |
| `src/main.tsx` | React入口 |
| `src/engine/ParticleEngine.ts` | 粒子系统管理：粒子生成、位置/颜色/大小更新、动画状态、悬停扩散/恢复逻辑 |
| `src/engine/TextParser.ts` | 文本解析：汉字笔画数估算、粒子初始位置生成（扁球体随机分布）、渐变颜色映射 |
| `src/engine/SceneRenderer.ts` | Three.js场景构建：相机、光源（环境光0.4+方向光0.6）、粒子Points渲染、Raycaster交互、场景淡入淡出 |
| `src/engine/GalleryEngine.ts` | 画廊CRUD、IndexedDB交互、OffscreenCanvas缩略图生成、点赞/评论数据管理 |
| `src/ui/AppUI.tsx` | React主组件：左侧工具栏、底部控制栏、画廊瀑布流、全局UI状态协调 |
| `src/stores/appStore.ts` | Zustand全局状态：当前诗句、主题ID、粒子快照、画廊数据、交互状态 |

### 4.2 状态Store结构（Zustand）

```typescript
interface AppState {
  // 场景状态
  currentPoem: string;
  currentThemeId: string;
  particles: Particle[];
  charClouds: CharCloud[];
  
  // 交互状态
  hoveredCharIndex: number | null;
  isTransitioning: boolean;
  
  // 画廊
  galleryItems: GalleryItem[];
  
  // Actions
  setPoem: (text: string) => void;
  setTheme: (themeId: string) => void;
  setHoveredChar: (index: number | null) => void;
  saveSnapshot: () => Promise<void>;
  loadGallery: () => Promise<void>;
  likeItem: (id: string) => void;
  addComment: (snapshotId: string, username: string, text: string) => void;
  loadSnapshot: (id: string) => void;
}
```

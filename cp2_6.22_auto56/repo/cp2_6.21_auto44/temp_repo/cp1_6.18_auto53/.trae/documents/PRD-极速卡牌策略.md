## 1. 产品概述

极速卡牌策略是一款在线双人对战卡牌游戏，玩家与AI对手轮流从手牌中选择卡牌进行战斗。游戏包含攻击、防御和特殊效果卡牌，每回合限时30秒，目标是将对手生命值降为零。

- 主要目的：提供快速、刺激的卡牌策略对战体验
- 目标用户：休闲卡牌游戏爱好者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 说明 |
|------|------|
| 玩家 | 真人玩家，控制手牌进行对战 |
| AI对手 | 模拟对手，根据简单策略自动出牌 |

### 2.2 功能模块
1. **对战主界面**：玩家信息区、对手信息区、战场区、手牌区
2. **卡牌系统**：攻击牌、防御牌、特殊效果牌（抽牌、弃牌）
3. **回合系统**：30秒限时、回合切换动画、能量点管理
4. **AI系统**：模拟AI对手思考和出牌逻辑
5. **胜负判定**：生命值归零触发胜利界面

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 对战主界面 | 对手信息区 | 显示头像、名字、弧形生命值条、能量点 |
| 对战主界面 | 战场区 | 放置打出的卡牌，2秒后进入墓地动画 |
| 对战主界面 | 玩家信息区 | 显示头像、名字、弧形生命值条、能量点 |
| 对战主界面 | 手牌区 | 横向排列卡牌，最多7张，拖拽出牌 |
| 对战主界面 | 倒计时条 | 顶部显示回合剩余时间，最后5秒变红闪烁 |
| 胜利界面 | 全屏覆盖 | 显示获胜者信息，金色渐变背景 |

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色主题背景 #0A0B1E
- 攻击卡牌：#FF6B6B 到 #CC4444 渐变
- 防御卡牌：#4ECDC4 到 #3BA3A0 渐变
- 特殊卡牌：#FFD93D 到 #CCA800 渐变
- 倒计时正常：#4ECDC4，最后5秒：#FF6B6B 闪烁
- 胜利背景：金色 #FFD700 渐变
- 卡牌尺寸：宽120px，高170px，圆角8px，边框1px #2A2A44
- 字体：现代无衬线字体，突出游戏感

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 对战主界面 | 对手信息区 | 头像、名字、弧形生命值条、能量点指示器 |
| 对战主界面 | 战场区 | 网格纹理背景（透明度0.1）、卡牌放置区域、墓地动画 |
| 对战主界面 | 玩家信息区 | 头像、名字、弧形生命值条、能量点指示器 |
| 对战主界面 | 手牌区 | 横向卡牌排列、拖拽交互、滚动指示器 |
| 对战主界面 | 倒计时条 | 顶部进度条、颜色变化、闪烁动画 |
| 胜利界面 | 全屏覆盖 | 获胜者文字、金色渐变背景、重新开始按钮 |

### 4.3 响应式设计
- 桌面优先设计，适配主流分辨率
- 手牌区支持触摸操作和鼠标拖拽
- 卡牌悬停放大效果（1.2倍缩放，阴影半径20px）

### 4.4 动画效果
- 卡牌打出：缩放与位移动画（0.3秒 ease-out）
- 卡牌悬停：放大1.2倍，阴影增强
- 回合切换：卡牌褪色（透明度0.3），"对手回合"文字+3秒倒计时
- 超时跳过：卡牌返回手牌并抖动0.5秒
- 墓地动画：卡牌打出后停留2秒，然后淡出移动

## 1. 产品概述

在线多人协作食谱管理与智能配餐平台，让美食爱好者创建、分享食谱，并根据冰箱现有食材智能推荐菜品，同时支持家庭成员或朋友之间协作编辑购物清单。
- 目标用户：家庭烹饪爱好者、合租室友、需要协同采购的家庭成员
- 核心价值：减少食材浪费、简化每日配餐决策、提升购物协作效率

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 创建/编辑食谱、智能配餐、协作编辑购物清单 |

### 2.2 功能模块
1. **食谱列表页**：网格卡片展示食谱、搜索筛选、悬停动效
2. **食谱详情页**：步骤逐步展开动画、食材列表、封面图
3. **配餐页**：输入食材、智能匹配推荐食谱、匹配度进度条动画
4. **购物清单页**：多人实时协作编辑、分类折叠、勾选消失动画

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 食谱列表页 | 搜索栏 | 按名称搜索食谱，300ms防抖反馈 |
| 食谱列表页 | 筛选器 | 按难度等级、烹饪时间筛选 |
| 食谱列表页 | 食谱卡片网格 | 网格布局展示食谱卡片，悬停上浮阴影效果，显示烹饪时间徽章 |
| 食谱详情页 | 封面区 | 食谱封面图（上传或渐变色占位图） |
| 食谱详情页 | 信息区 | 名称、准备时间、烹饪时间、难度星级 |
| 食谱详情页 | 食材列表 | 食材名称、用量、单位 |
| 食谱详情页 | 步骤区 | 文本+图片混合排版，逐步展开动画 |
| 配餐页 | 食材输入区 | 搜索或手动添加冰箱已有食材，标签式展示 |
| 配餐页 | 推荐结果区 | 瀑布流展示匹配食谱，匹配度进度条动画，差缺食材红色波浪线标记 |
| 配餐页 | 待做清单 | 一键添加推荐食谱到待做清单 |
| 购物清单页 | 分类折叠组 | 按食材类别分组（蔬菜、肉类、调味品等），组标题点击展开/收起，平滑高度过渡动画 |
| 购物清单页 | 食材项 | 添加、勾选已购买（划掉-消失动画）、删除，支持多人实时协作 |
| 购物清单页 | 协同指示 | 显示在线协作者，实时同步编辑状态 |

## 3. 核心流程

**主流程**：用户浏览食谱 → 选择食谱查看详情 → 根据食材智能配餐 → 生成购物清单 → 多人协作采购

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：温暖橙色（#E8722A）与米白色（#FFF8F0）
- 强调色：橄榄绿（#6B8E4E）
- 背景色：奶油白（#FFFAF5）
- 卡片色：纯白（#FFFFFF）带柔和阴影
- 按钮风格：圆角（12px），橙色主按钮配白色文字，橄榄绿次按钮
- 字体：Playfair Display（标题）+ Noto Sans SC（正文），标题24-32px，正文14-16px
- 布局风格：卡片式网格布局，顶部导航栏
- 图标风格：线性图标（Lucide），2px描边

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 食谱列表页 | 搜索栏 | 圆角输入框，搜索图标，橙色聚焦边框，300ms防抖 |
| 食谱列表页 | 食谱卡片 | 白色圆角卡片，悬停上浮8px + 阴影加深，右上角烹饪时间橙色徽章，封面渐变色占位图 |
| 食谱详情页 | 步骤区 | 步骤卡片逐步展开（高度0→auto过渡），步骤编号圆形橙色标记 |
| 食谱详情页 | 食材标签 | 橄榄绿圆角标签，显示用量 |
| 配餐页 | 食材标签 | 圆角标签，×删除按钮，添加输入框 |
| 配餐页 | 匹配度进度条 | 渐变进度条（橙→绿），从0%到目标值动画过渡 |
| 配餐页 | 差缺食材 | 红色文字 + 红色波浪下划线 |
| 配餐页 | 推荐卡片 | 瀑布流布局，圆角卡片，匹配度+食谱名+差缺食材 |
| 购物清单页 | 分类组标题 | 橄榄绿文字，右侧展开/收起箭头，点击平滑高度过渡 |
| 购物清单页 | 食材项 | 圆形勾选框，勾选后文字划掉+淡出消失动画 |
| 购物清单页 | 协同指示 | 头像圆形，在线绿点，实时编辑高亮 |

### 4.3 响应式设计
- 桌面端（≥1024px）：食谱卡片3-4列网格，配餐结果瀑布流3列，购物清单居中宽版
- 平板端（768-1023px）：食谱卡片2列网格，配餐结果2列，食谱详情步骤两栏并排
- 手机端（<768px）：食谱卡片1列，配餐结果单列，食谱详情步骤纵向滑动卡片式布局，购物清单全宽

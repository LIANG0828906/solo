## 1. 产品概述

病毒传播与变异模拟器——一个基于蜂窝网格地图的实时疫情推演可视化应用，用于直观展示不同病毒株在空间中的竞争、传播速率差异以及环境因素对传播路径的影响。面向疫情策略分析人员及策略游戏设计者，提供交互式的病毒传播推演与干预模拟能力。

## 2. 核心功能

### 2.1 功能模块

1. **主界面（蜂窝网格地图）**：30x30蜂窝网格地图，每格可呈三种状态——健康（浅绿#e8f5e9）、感染（深红#c62828→橙渐变）、免疫（淡蓝#bbdefb）；地图右侧实时模拟时间轴（0-1000代），含播放/暂停按钮与代次跳转滑块
2. **病毒变异系统**：感染格子0.5%几率变异，变异后颜色变紫#9c27b0，传播率+20%、免疫恢复时间+50%；触发时地图中央紫色圆环扩散动画，右侧面板显示变异简报
3. **用户干预系统**：点击接种（单格免疫+两圈50%概率扩散免疫）、拖拽框选批量接种（矩形选区确认后全部免疫）、免疫格白色光晕脉冲动画
4. **控制面板**：传播率/免疫恢复时效/变异率/初始感染格数四组滑块，带实时数值显示与渐变色条
5. **实时统计图表**：三条折线（健康绿、感染红、免疫蓝），X轴代数0-1000，Y轴格子数0-900，支持悬停提示框

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主界面 | 蜂窝网格地图 | Canvas渲染30x30蜂窝网格，处理点击/拖拽交互，渲染变异圆环动画 |
| 主界面 | 时间轴控制 | 0-1000代模拟进度条，播放/暂停按钮，代次跳转滑块 |
| 主界面 | 变异简报面板 | 右侧信息面板，实时显示变异事件文本 |

| 主界面 | 统计图表 | 地图下方Canvas折线图，三条曲线实时更新，悬停数值提示 |

## 3. 核心流程

用户打开应用 → 设置参数（传播率、变异率等） → 点击播放 → 病毒从初始感染点向周围扩散 → 每代更新网格状态 → 可随时暂停并干预（接种疫苗） → 感染格有概率变异 → 变异触发动画与简报 → 模拟持续至1000代或用户停止 → 图表实时反映各状态格子数量变化

```mermaid
flowchart TD
    A["应用启动"] --> B["参数设置"]
    B --> C["点击播放"]
    C --> D["Web Worker计算传播"]
    D --> E{"有变异事件?"}
    E -->|是| F["触发变异动画+简报"]
    E -->|否| G["更新网格状态"]
    F --> G
    G --> H["Canvas渲染地图"]
    H --> I["更新折线图"]
    I --> J{"用户干预?"}
    J -->|点击接种| K["免疫扩散"]
    J -->|拖拽批量接种| L["批量免疫"]
    J -->|否| M{"达到1000代?"}
    K --> M
    L --> M
    M -->|否| D
    M -->|是| N["模拟结束"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：健康绿#e8f5e9、感染红#c62828、免疫蓝#bbdefb、变异紫#9c27b0
- **背景**：深色主题#121212
- **卡片样式**：毛玻璃效果（backdrop-filter: blur(16px)，背景rgba(30,30,30,0.7)），白色内发光边框1px solid rgba(255,255,255,0.1)
- **按钮**：主按钮#1976d2深蓝，悬停变亮；按下缩小反馈
- **字体**：系统无衬线字体，字号14px
- **滑块渐变**：传播率红橙渐变、免疫时效蓝紫渐变、变异率紫红渐变
- **动画**：感染格脉冲动效（10%放大）、变异圆环扩散（1秒）、免疫光晕脉冲（0.8s循环）、滑块数值跳动0.1s ease

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主界面 | 蜂窝网格地图 | Canvas画布，蜂窝六边形格子，三色状态填充，脉冲放大动画，变异紫色圆环3D扩散 |
| 主界面 | 时间轴 | 进度滑块0-1000，播放/暂停图标按钮，当前代数显示 |
| 主界面 | 变异简报 | 右侧面板滚动列表，紫底文本卡片，格式"第X代：区域(Y,Z)发生变异" |
| 主界面 | 控制面板 | 左侧毛玻璃卡片，四组渐变色滑块+数值标签，初始感染数选择器 |
| 主界面 | 统计图表 | 下方毛玻璃卡片内Canvas折线图，三色曲线，悬停十字线+数值提示框 |

### 4.3 响应式适配

- 桌面端：左控制面板 + 中央地图 + 右时间轴/简报，下方图表
- 宽度<768px：控制面板、地图、图表自动上下堆叠布局
- 图表高度自适应容器宽度

### 4.4 3D场景指引

- **变异圆环**：Three.js渲染紫色圆环从中心向外扩散，半径0→地图边缘，1秒持续，透明度渐淡
- **脉冲动效**：感染格更新时短暂10%放大，使用Three.js缩放动画
- **免疫光晕**：免疫格白色脉冲光环，0.8s循环，Three.js粒子或环形光效

## 1. 产品概述

三维分子结构交互浏览器是一款面向学生和化学爱好者的 Web 应用，用户可以在浏览器中导入或选择常见分子模型（甲烷、水、苯、乙醇、葡萄糖），在三维空间中旋转、缩放、拆解并查看原子与化学键的详细信息。

- 解决传统二维化学教学无法直观展示三维分子结构的痛点
- 目标用户为中学生、大学生及化学爱好者，提供沉浸式分子探索体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览和交互所有分子模型 |

### 2.2 功能模块

1. **主场景页面**：分子选择、三维渲染、模式切换、拆解视图、信息展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景页面 | 分子选择栏 | 顶部/侧边分子预设列表，点击切换分子 |

| 主场景页面 | 悬浮信息标签 | 鼠标悬停原子时显示元素符号和原子序数 |
| 主场景页面 | 模式切换面板 | 球棍/空间填充/线框三种模式切换，含动画过渡 |

| 主场景页面 | 键角标注 | 模型周围细线标注键角数值 |

| 主场景页面 | 控制面板 | 毛玻璃效果面板，含模式切换、重置视角、拆解开关 |

## 3. 核心流程

用户打开应用 → 从预设列表选择分子 → 三维场景中央展示球棍模型 → 用户可旋转/缩放查看 → 悬停原子查看信息 → 切换显示模式（带动画）→ 点击拆解按钮查看爆炸图 → 点击单个原子查看扩展信息 → 点击空白区域组合回原位 → 选择其他分子继续探索

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色主题（主背景 #0F0F23，辅助色 #2A2A4A，文字色 #E0E0FF）
- 按钮样式：圆角矩形，选中蓝紫色填充，未选中透明描边
- 字体：科学感等宽字体配合无衬线体
- 布局：全屏三维场景为主，控制面板浮于右下角
- 图标风格：简洁线条图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主场景页面 | 三维渲染区 | 全屏深色背景，中央分子模型，菲涅尔光泽原子 |
| 主场景页面 | 分子名称标签 | 左上角分子名称+当前模式标签 |
| 主场景页面 | 控制面板 | 右下角圆角半透明毛玻璃卡片，蓝紫渐变边框 |
| 主场景页面 | 模式切换按钮组 | 三个圆角矩形按钮，选中蓝紫填充 |
| 主场景页面 | 拆解开关 | 滑动式开关，滑动时背景灰变紫 |
| 主场景页面 | 重置视角按钮 | 带旋转图标的圆角按钮 |

### 4.3 响应式适配

- 桌面端：主场景全屏，右下角浮层控制面板
- 竖屏平板：控制面板缩小为底部横条
- 交互优化：鼠标悬停高亮 + 触摸点击兼容

### 4.4 3D 场景指导

- 环境：深色空间背景，微弱环境光 + 方向光营造科学感
- 灯光设置：环境光 + 主方向光 + 补光，保证原子表面菲涅尔效果可见
- 相机设置：透视相机，初始距离自适应分子大小，OrbitControls 旋转缩放
- 构图焦点：分子始终居中，相机环绕旋转
- 交互与动画：模式切换时球体膨胀/收缩动画、键消失/出现动画、拆解时原子沿径向飞出动画
- 后处理效果：菲涅尔光泽（自定义 Shader）、微弱辉光效果
- 性能预算：45FPS+ 交互帧率，最复杂分子（葡萄糖24原子）加载 < 2秒

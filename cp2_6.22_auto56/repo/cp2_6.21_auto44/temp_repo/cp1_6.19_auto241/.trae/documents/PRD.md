## 1. 产品概述
交互式水文模拟沙盘，用户可通过画笔绘制地形并模拟雨水在不同地形上的流动、渗透与蒸发过程，用于教育演示和科学可视化。
- 主要用途：水文现象可视化教学、地理科普展示
- 目标用户：学生、地理教师、科普爱好者
- 产品价值：通过交互式模拟直观展示水文循环过程

## 2. 核心 Features

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 完整使用所有模拟功能 |

### 2.2 功能模块
1. **主画布区域**：地形绘制、粒子模拟、轨迹渲染
2. **控制面板**：参数调节（渗透率、蒸发速率、降雨强度）
3. **统计面板**：实时数据展示、重置功能
4. **视图切换**：高度图、流速矢量图切换显示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 地形绘制 | 画笔工具绘制山地、平原、洼地三种地形 |
| 主页面 | 降雨模拟 | 点击按钮生成雨滴粒子，模拟物理运动 |
| 主页面 | 参数控制 | 滑块调节全局渗透率、蒸发速率、降雨强度 |
| 主页面 | 数据统计 | 实时显示降水量、蒸发量、渗透量、存留粒子数 |
| 主页面 | 视图切换 | 右键切换高度图/流速矢量图显示 |

## 3. 核心流程
用户打开页面 → 用画笔在画布上绘制三种地形区域 → 点击降雨按钮 → 雨滴粒子从顶部落下 → 粒子根据地形坡度流动、渗透或蒸发 → 实时查看统计数据 → 右键切换视图模式查看高度图或矢量图 → 可点击重置按钮清空重新开始。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰背景 #1A1A2E，蓝色系 #4A90D9/#3498DB/#7EC8E3，强调色 #1ABC9C，危险色 #E74C3C
- 按钮：圆角设计，悬停状态有明暗变化
- 字体：现代无衬线字体，标题 20px 粗体，标签 14px 半粗体，数值 12px 常规
- 布局：左右对称面板，中央画布区域，响应式折叠
- 图标：简洁的功能图标（雨滴、重置等）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主页面 | 标题栏 | 高 48px，居中标题，深色背景 |
| 主页面 | 控制面板 | 左侧深色半透明面板，圆角 16px，三个滑块组件 |
| 主页面 | 统计面板 | 右侧深色半透明面板，圆角 12px，数据展示和重置按钮 |
| 主页面 | 画布区域 | 浅沙色背景，2px 虚线边框，占宽 85% 高 75% |
| 主页面 | 图例条 | 切换视图时左侧出现，垂直渐变色条和数值标注 |

### 4.3 响应式设计
- 桌面端（≥768px）：左右面板分列画布两侧，画布占宽 85% 高 75%
- 移动端（<768px）：两个面板折叠到底部横向排列，每个宽 45%，间距 10px，画布高度缩减至 60%
- 触控优化：支持捏合手势操作，滑块触控区域增大

### 4.4 动画与交互
- 粒子动画：使用 requestAnimationFrame 实现流畅的 30fps+ 动画
- 滑块交互：实时响应数值变化，数值显示同步更新
- 悬停效果：按钮悬停时颜色变化，提供视觉反馈
- 过渡动画：面板折叠/展开使用 framer-motion 实现平滑过渡

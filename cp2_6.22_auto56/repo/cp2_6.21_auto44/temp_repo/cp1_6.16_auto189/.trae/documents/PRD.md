## 1. 产品概述

多人虚拟密室逃脱协作平台——一组玩家（2-4人）通过浏览器进入同一主题密室房间，各自在屏幕前通过点击、拖拽和组合道具来协作解开谜题，房间内的机关状态实时同步给所有成员。

- 目标用户：喜欢解谜和团队协作的休闲游戏玩家
- 核心价值：沉浸式复古暖色调氛围 + 实时多人协作 + 道具组合机制

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 输入房间号加入 | 浏览场景、拾取道具、拖拽组合、查看队友操作 |

### 2.2 功能模块

1. **大厅页面**：创建/加入房间，显示房间号和在线玩家列表
2. **密室场景页面**：区域切换、交互点点击、道具拾取、协作槽位拖拽、组合动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 大厅页面 | 房间创建/加入 | 输入房间号创建或加入房间，显示当前房间玩家列表 |
| 大厅页面 | 等待准备 | 显示已加入玩家，所有玩家点击"准备"后进入密室 |
| 密室场景 | 区域导航 | 左侧/顶部区域切换按钮，切换书房/实验室/阁楼/密室 |
| 密室场景 | 场景渲染 | 手绘暖色背景，交互点高亮，点击弹出局部放大视图 |
| 密室场景 | 道具拾取 | 点击放大视图中的道具图标拾取道具，触发叮当音效 |
| 密室场景 | 协作槽位 | 底部固定栏，拖拽道具到槽位，相邻道具触发组合 |
| 密室场景 | 组合动画 | 道具融合闪烁光效1.5秒，生成新道具并解锁新区域/答案 |
| 密室场景 | 实时同步 | 所有玩家操作实时广播，道具状态和区域解锁同步 |

## 3. 核心流程

玩家打开页面 → 输入房间号加入 → 等待队友 → 所有玩家准备 → 进入密室（默认书房为主视角）→ 点击交互点探索 → 拾取道具 → 拖拽道具到协作槽位 → 至少两名玩家将特定道具拖入相邻槽位 → 触发组合动画 → 生成新道具/解锁新区域 → 继续探索直到通关

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖色复古风 — 主色 `#F5E6CA`（米黄），深褐色 `#5D4037`（线条勾勒），浅蓝 `#B3E5FC`（道具图标），浅金 `#FFD54F`（悬停高亮）
- **按钮风格**：圆角矩形，悬停变浅金 `#FFD54F` 并上浮3px
- **字体**：衬线体标题 + 无衬线体正文，营造复古书房氛围
- **布局风格**：全屏场景 + 底部固定协作栏 + 顶部区域导航
- **动画风格**：所有页面/区域切换1秒淡入淡出（opacity 0→1），道具悬停弹跳放大1.2倍

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 大厅页面 | 房间创建/加入 | 居中卡片式布局，暖色背景，输入框+按钮，玩家列表 |
| 密室场景 | 区域导航 | 顶部/侧边标签页按钮，当前区域高亮 |
| 密室场景 | 场景渲染 | 全屏手绘风背景 `#F5E6CA`，深褐 `#5D4037` 线条家具，交互点脉冲光晕 |
| 密室场景 | 局部放大视图 | 点击交互点弹出模态框，展示道具圆形图标（50px直径，浅蓝 `#B3E5FC` 背景） |
| 密室场景 | 协作槽位 | 底部固定栏80px高，半透明白色背景，虚线边框 `#9E9E9E` 2px |
| 密室场景 | 组合动画 | 道具融合闪烁光效1.5秒 |
| 密室场景 | 玩家状态 | 右上角显示在线玩家头像和当前区域 |

### 4.3 响应式适配

- 桌面端：底部横向协作槽位栏（高80px）
- 移动端/平板：协作槽位改为侧边栏（宽120px），区域导航改为底部标签栏
- 交互点点击区域在移动端放大，确保触控友好

### 4.4 音效反馈

- 道具拾取：Web Audio API 生成清脆正弦波叮当音效
- 道具组合：组合音效（上升音阶）
- 区域解锁：低沉的机关开启音效

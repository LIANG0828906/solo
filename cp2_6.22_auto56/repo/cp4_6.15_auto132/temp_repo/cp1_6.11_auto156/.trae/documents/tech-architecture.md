## 2. 技术说明
- 前端：TypeScript + 原生 JavaScript（无框架）
- 构建工具：Vite
- 初始化工具：Vite 原始模板（vanilla-ts）
- 后端：无
- 数据库：无（纯前端本地状态）

## 3. 路由定义

## 4. 文件结构

```
├── package.json          # 依赖：typescript、vite；启动脚本：npm run dev
├── index.html            # 入口页面，包含布局和样式
├── vite.config.js        # 构建配置，入口index.html，端口3000
├── tsconfig.json         # 严格模式，target ES2020
└── src/
    ├── main.ts           # 应用主模块，初始化网格、预览面板、工具按钮，管理全局状态
    ├── grid.ts           # 网格逻辑：创建、点击切换颜色、缩放、平移、矩形填充、对称镜像，导出导入数据
    └── preview.ts        # 预览面板逻辑：三种预览模式的渲染和切换，图案映射到不同形状
```

## 5. 模块职责

### 5.1 main.ts — 主模块
- 初始化应用，创建DOM结构
- 管理全局状态：当前工具、当前颜色、网格数据
- 绑定工具栏事件（铅笔、矩形、镜像、颜色选择器、导入/导出）
- 协调 grid.ts 和 preview.ts 的交互

### 5.2 grid.ts — 网格模块
- 网格数据结构：二维数组，每个元素为颜色索引（0=空, 1=色A, 2=色B, 3=色C）
- Canvas 渲染：绘制网格线、填充格子、纸纹理
- 交互：点击切换颜色、矩形区域填充、对称镜像
- 缩放与平移：以鼠标位置为中心缩放（0.5x-3x），拖拽平移
- 导出：生成包含网格数据的JSON文件
- 导入：读取JSON文件并替换当前网格数据

### 5.3 preview.ts — 预览模块
- 平面预览：4x4重复平铺，每块25px，块间1px分割线
- 围巾环绕预览：贝塞尔曲线路径映射，深灰背景#3A2E24
- 毛衣平铺预览：人体上半身轮廓（肩宽120px、衣长180px），图案缩放填充
- 预览模式切换：0.3秒淡入淡出过渡

## 6. 性能要求
- 网格操作响应时间 ≤ 50ms
- 预览模式切换渲染延迟 ≤ 200ms
- 缩放平移帧率 ≥ 30fps

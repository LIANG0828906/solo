## 1. 产品概述

代码依赖关系可视化工具——帮助前端开发者在重构项目时快速理解模块之间的引用关系。用户通过上传单文件或粘贴代码片段，工具自动解析 import/require 语句，以力导向图形式在 Canvas 上展示依赖关系，支持节点交互、布局切换和 JSON 导出，所有逻辑客户端运行，无需后端。

- 核心问题：前端开发者重构时搞不清模块引用关系，现有工具要么太复杂要么需要上传整个仓库
- 目标用户：前端开发者，需要快速、轻量地理解代码依赖结构

## 2. 核心功能

### 2.1 功能模块

1. **代码输入与解析**：支持上传文件或粘贴代码，自动解析 import/require/export 语句
2. **依赖图可视化**：力导向图展示模块依赖，节点可拖拽，连线带箭头按类型着色
3. **节点交互**：点击节点高亮并显示浮动信息卡（完整导入路径、被引用列表、导出成员列表）
4. **布局切换**：力导向布局 / 同心圆布局 / 树状布局，切换带过渡动画
5. **控制面板**：可折叠侧边栏，包含布局选择、标签开关、力导向强度滑块
6. **JSON 导出**：导出包含 nodes 和 edges 的依赖关系数据文件

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 导航栏 | Logo、代码输入框（上传/粘贴）、导出按钮、帮助图标 |
| 主页面 | Canvas 画布 | 力导向图渲染，节点拖拽、缩放、点击交互 |
| 主页面 | 控制面板 | 侧边栏，布局模式切换、标签开关、力导向强度滑块 |
| 主页面 | 信息卡 | 点击节点弹出浮动卡片，显示模块详情 |

## 3. 核心流程

用户输入代码 → 解析器提取依赖关系 → 构建节点和边数据 → 渲染力导向图 → 用户交互（拖拽/点击/切换布局）→ 导出 JSON

```mermaid
    "G" --> "L[导出 JSON]"
    "H" --> "E"
    "J" --> "D"
    "K" --> "D"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主题：深色科技风
- 主背景：#0f0f23 到 #1a1a2e 线性渐变
- 强调色：#6c5ce7（紫色）、#a29bfe（浅紫色）
- 节点连线颜色：本地文件 #3498db（蓝）、第三方包 #e74c3c（红）、内置模块 #2ecc71（绿）
- 导航栏：深紫色半透明 + 毛玻璃效果
- 控制面板：深色 #1a1a2e + 毛玻璃效果
- 字体：14px monospace（代码标签）、无衬线字体（UI 文本）
- 圆角：6px（输入框）、12px（信息卡）、圆角按钮

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 导航栏 | 高 60px，半透明深紫背景，毛玻璃模糊 15px，底部 1px #2d2d44 分隔线 |
| 主页面 | 代码输入框 | 高 40px，宽 50%，圆角 6px，聚焦时 #6c5ce7 发光边框 |
| 主页面 | 导出按钮 | 圆角，渐变 #6c5ce7→#a29bfe，悬浮右移 2px + 阴影 |
| 主页面 | Canvas 画布 | 全屏减去导航栏高度，深色渐变背景 |
| 主页面 | 节点 | 圆形，半径 30-80px（随被引用次数增大），可拖拽 |
| 主页面 | 连接线 | 带箭头有向曲线，宽 2px，箭头 8px，按类型着色 |
| 主页面 | 信息卡 | 半透明白色 rgba(255,255,255,0.95)，圆角 12px，模糊背景，X 关闭按钮 |
| 主页面 | 控制面板 | 宽 240px，深色 #1a1a2e，毛玻璃模糊 20px，可折叠 |

### 4.3 响应式

- 桌面优先设计，Canvas 自适应窗口大小
- 控制面板可折叠，不遮挡画布内容

### 4.4 动效规范

- 布局切换过渡：500ms ease-out
- 节点高亮：放大 1.2 倍 + 外发光
- 滑块数值：50ms 抖动反馈
- 导出按钮：悬浮右移 2px + 阴影
- 力导向布局：requestAnimationFrame 增量更新，每帧计算 1/5 节点

## 1. 产品概述
斗茶雅集是一款还原明代文人斗茶品鉴的沉浸式交互网页应用。用户可在虚拟茶寮中通过点击、拖拽等交互方式，完成选茶、碾茶、点茶、击拂等传统茶艺流程，最终获得茶汤品评成绩单，体验中华茶文化之美。

- 面向茶文化爱好者、古典美学追求者，提供寓教于乐的茶艺体验
- 以游戏化方式传承和推广宋代点茶技艺，具有文化传播价值

## 2. 核心 Features

### 2.1 Feature Module
1. **选茶面板**：四种名茶卡片展示与选择
2. **制作流程**：碾茶、点茶、击拂三步交互操作
3. **结果面板**：色香味评分与古文评语展示
4. **步骤进度**：实时显示当前进度与操作指引

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 主页面 | 茶寮背景装饰 | CSS绘制竹帘、屏风图案，营造古典氛围 |
| 主页面 | 选茶面板 | 水墨风格茶叶卡片，点击选中放大效果 |
| 主页面 | 碾茶交互 | 点击茶饼研磨，茶粉粒子坠落动画 |
| 主页面 | 点茶交互 | 按住壶柄倾倒热水，SVG水面涟漪扩散 |
| 主页面 | 击拂交互 | 快速点击茶筅搅动，茶沫渐起效果 |
| 主页面 | 结果面板 | 卷轴样式展示色香味评分与总分，淡入动画 |
| 主页面 | 步骤进度条 | 右侧显示当前步骤与完成状态 |

## 3. 核心 Process
用户进入页面后，首先浏览四种茶叶介绍并选择一种，然后依次完成碾茶、点茶、击拂三个制作步骤，每个步骤根据操作速度和精准度打分，最后系统综合评分生成斗茶成绩单，显示色香味各项分数与古文评语。

```mermaid
flowchart LR
    A["进入茶寮"] --> B["选择茶叶"]
    B --> C["碾茶"]
    C --> D["点茶"]
    D --> E["击拂"]
    E --> F["生成品评结果"]
    F --> G["查看成绩单"]
    G --> H["重新体验"]
```

## 4.  User Interface Design

### 4.1 Design Style
- **主色调**：米白#faf0e6、墨黑#2c2c2c、茶绿#7d946c
- **风格定位**：宋代极简美学，素雅宁静，留白写意
- **字体**：标题使用书法风格字体，正文使用宋体类衬线字体
- **布局**：三栏式布局，左侧茶寮背景，中间操作区，右侧进度预览
- **装饰元素**：水墨纹理、印章图标、卷轴样式、竹帘图案

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 主页面 | 顶部标题 | 古风"斗茶雅集"标题 + 印章图标 |
| 主页面 | 选茶卡片 | 水墨插画、茶叶名称、简介，选中放大淡墨背景 |
| 主页面 | 碾茶区域 | 茶饼图形、研磨动画、茶粉粒子效果 |
| 主页面 | 点茶区域 | 茶壶SVG、水面波浪路径、涟漪扩散效果 |
| 主页面 | 击拂区域 | 茶筅摆动动画、泡沫粒子效果 |
| 主页面 | 结果卷轴 | 古卷轴样式、三项评分、总分、古文评语 |
| 主页面 | 进度侧栏 | 步骤节点、完成标记、当前状态指示 |

### 4.3 Responsiveness
- **桌面端（横屏）**：三栏布局，操作区居中便于操作
- **移动端（竖屏）**：上下滚动布局，操作区靠下便于拇指点击
- **触控优化**：按钮最小尺寸48px，点击热区扩大，长按反馈清晰

### 4.4 Interaction Details
- 选茶卡片切换使用framer-motion布局动画
- 所有点击反馈<100ms，无卡顿
- 动画使用CSS动画和SVG，避免重排重绘
- 保持60fps流畅体验

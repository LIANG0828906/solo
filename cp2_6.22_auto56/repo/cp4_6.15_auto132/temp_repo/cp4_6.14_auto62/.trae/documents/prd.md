## 1. 产品概述

在线招聘管理平台，为分布式团队提供职位发布、简历筛选、面试安排和团队协作评估的一站式招聘流程管理解决方案。目标用户为小型创业公司的HR和面试官，旨在提升招聘效率、减少沟通成本、实现候选人全生命周期可视化管理。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| HR | 系统分配 | 创建/管理职位、筛选简历、安排面试、查看看板 |
| 面试官 | 系统分配 | 查看候选人、安排面试、评分评价 |
| 候选人 | 无需注册 | 查看职位、提交简历 |

### 2.2 功能模块

1. **职位管理页**：职位列表展示、创建新职位、职位详情查看
2. **简历库页**：简历上传（拖拽）、简历列表展示、简历详情查看
3. **面试安排页**：日历选择面试时段、面试时间线展示、面试模态框
4. **看板页**：四泳道候选人状态看板、拖拽移动候选人卡片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 职位管理 | 职位卡片列表 | 展示所有职位，卡片280x180px，白色圆角8px，阴影2px，悬停阴影加深至6px上移4px，过渡0.3s ease-out |
| 职位管理 | 创建职位表单 | 填写职位标题、部门、工作地点、薪资范围滑块（1万-10万，步长5000）、富文本描述（2000字以内） |
| 简历库 | 简历上传区 | 拖拽上传PDF/Word（最大5MB），蓝色渐变进度条#3b82f6到#8b5cf6，2秒动画 |
| 简历库 | 简历卡片列表 | 每行3个卡片，含姓名、应聘职位、上传时间，24小时内新简历右上角绿色发光圆点 |
| 面试安排 | 面试模态框 | 半透明黑色遮罩，0.3s中心放大进入，月历组件+时段选择，已占用时段灰色不可选 |
| 面试安排 | 评分评价 | 星级评分1-5（悬停金色#f59e0b），textarea评价500字，显示平均分和评价数 |
| 看板 | 四泳道看板 | 待筛选/已面试/已录用/已淘汰，拖拽移动（0.4s ease-in-out），半透明克隆跟随 |
| 看板 | 泳道背景色 | 浅灰#f1f5f9、浅蓝#eff6ff、浅绿#f0fdf4、浅红#fef2f2 |

## 3. 核心流程

**招聘主流程**：HR创建职位 → 候选人查看职位并提交简历 → HR筛选简历 → HR安排面试 → 面试官评分评价 → 更新候选人状态（录用/淘汰）

```mermaid
flowchart TD
    "HR创建职位" --> "职位发布到JobBoard"
    "候选人查看职位" --> "提交简历"
    "提交简历" --> "简历进入简历库(待筛选)"
    "HR筛选简历" --> "安排面试"
    "安排面试" --> "面试官评分评价"
    "面试官评分评价" --> "已录用"
    "面试官评分评价" --> "已淘汰"
    "HR筛选简历" --> "已淘汰"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：蓝色#3b82f6（按钮、链接、关键高亮）
- 辅助色：紫色#8b5cf6（标签、徽章）
- 背景色：极浅灰色#f8fafc
- 卡片色：白色#ffffff
- 文字主色：深灰#1e293b
- 按钮风格：圆角6px，主色填充，点击涟漪效果（0.6s从点击点扩散）
- 字体：系统字体栈，标题加粗
- 布局：固定导航栏56px + 侧边栏240px + 主内容区

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 顶部导航栏 | 高56px，白色背景，底部1px阴影，左侧Logo+右侧用户信息 |
| 全局 | 侧边栏 | 宽240px，背景#f1f5f9，4个导航项，激活项左侧3px蓝色竖条 |
| 职位管理 | 职位卡片网格 | 卡片280x180px，白色圆角8px，阴影2px悬停加深，网格布局 |
| 职位管理 | 创建职位弹窗 | 模态框表单，薪资滑块，富文本编辑器 |
| 简历库 | 上传区域 | 虚线边框拖拽区，蓝色渐变进度条 |
| 简历库 | 简历卡片 | 每行3列，姓名/职位/时间，绿色新标记 |
| 面试安排 | 面试日历 | 月视图，当天蓝色圆形高亮，时段30分钟格子 |
| 面试安排 | 评分组件 | 5颗星，悬停金色，点击固定，评价textarea |
| 看板 | 四泳道 | 水平排列，不同背景色，拖拽卡片移动 |

### 4.3 响应式设计

- 桌面优先（≥768px）：侧边栏常驻240px，卡片列表每行3个
- 移动端（<768px）：侧边栏折叠为抽屉式（汉堡菜单触发，左侧滑入，半透明遮罩），卡片列表每行1个
- 所有交互操作带0.3-0.4s平滑过渡动画
- 按钮点击涟漪效果持续0.6s

### 4.4 动画规范

| 交互类型 | 动画描述 | 持续时间 | 缓动函数 |
|----------|----------|----------|----------|
| 卡片悬停 | 阴影加深+上移4px | 0.3s | ease-out |
| 进度条 | 蓝紫渐变0%→100% | 2s | linear |
| 模态框弹出 | 中心放大进入 | 0.3s | ease-out |
| 拖拽放置 | 卡片滑入目标泳道 | 0.4s | ease-in-out |
| 按钮点击 | 涟漪扩散 | 0.6s | ease-out |
| 侧边栏展开 | 从左侧滑入 | 0.3s | ease-out |

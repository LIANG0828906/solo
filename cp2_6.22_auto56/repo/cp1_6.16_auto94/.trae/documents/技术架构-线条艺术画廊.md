## 4. 数据模型

### 4.1 线条数据结构

```typescript
interface Point {
  x: number;
  y: number;
  timestamp: number;
}

interface LineData {
  id: string;
  points: Point[];
  color: string;
  widths: number[];
  startTime: number;
  endTime: number;
}
```

### 4.2 状态定义

- lines: LineData[] - 所有已完成的线条
- currentLine: LineData | null - 当前正在绘制的线条
- isReplaying: boolean - 是否正在回放
- replayIndex: number - 回放线索引
```

每条线条包含：
- id: 唯一标识
- points: 控制点坐标序列（含时间戳）
- color: 线条颜色
- widths: 动态线宽序列
- startTime/endTime: 绘制起止时间戳

## 5. 核心算法

### 5.1 贝塞尔曲线绘制
使用二次贝塞尔曲线，通过鼠标移动时以上一点和当前点的中点作为控制点，实现平滑曲线。

### 5.2 动态线宽计算
基于相邻点间距计算鼠标速度，速度越快线宽越细，范围 2-10px。

### 5.3 粒子拖尾效果
每条线末端 20 个粒子，随机扩散半径 15px，透明度 0.3 到 0 递减。

### 5.4 回放控制
按时间戳逐帧还原，使用 requestAnimationFrame 控制帧率。

## 6. 性能优化

- 使用 requestAnimationFrame 进行绘制循环
- 离屏 Canvas 预渲染已完成的线条
- devicePixelRatio 适配高清屏适配
- 粒子数量控制，避免过度绘制

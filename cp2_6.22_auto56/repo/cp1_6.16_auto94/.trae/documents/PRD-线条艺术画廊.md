## 1. 产品概述

线条艺术画廊是一个基于 Canvas 的交互式抽象艺术创作应用，用户通过鼠标拖拽在虚拟画布上创建流动线条构成的艺术作品，并支持回放创作过程。

- 主要用途：创意绘画、艺术创作、放松解压
- 目标用户：艺术爱好者、创意工作者、普通用户
- 产品价值：提供沉浸式的线条艺术创作体验，支持回放功能可用于教学、展示创作过程

## 2. 核心功能

### 2.1 功能模块

1. **画布绘制模块：全屏 Canvas 画布，支持鼠标绘制贝塞尔曲线
2. **工具栏模块：清空、撤销、回放三大核心操作
3. **历史记录模块：保存所有线条数据，支持撤销和回放
4. **回放动画模块：按原速重现创作过程，带光晕扩散动画

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 画布区域 | 全屏深灰色画布，鼠标绘制贝塞尔曲线，动态线宽，粒子拖尾 |
| 主页面 | 工具栏 | 清空画布、撤销上一步、回放创作过程三个操作按钮 |
| 主页面 | 线条计数 | 左上角显示当前线条数量 |

## 3. 核心流程

用户进入页面 → 鼠标按下开始绘制 → 鼠标移动实时绘制贝塞尔曲线 → 鼠标释放完成一条线 → 可点击撤销/清空/回放

```mermaid
flowchart TD
    A[进入页面] --> B[显示空画布]
    B --> C{鼠标操作}
    C -->|按下开始绘制
    按下开始绘制 -->|移动| 实时绘制贝塞尔曲线
    实时绘制贝塞尔曲线 -->|释放| 保存线条数据
    保存线条数据 --> D[更新线条计数
    D --> E{工具栏操作}
    E -->|清空| 清空画布和历史
    E -->|撤销| 删除最后一条线
    E -->|回放| 按原速重绘所有线条
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：霓虹蓝紫渐变（#4a00e0 到 #00b4d8）
- 背景色：深色调径向渐变（#1a1a2e 到 #16213e）
- 线条颜色池：靛蓝#4a00e0、青绿#00b4d8、紫色#7209b7、粉色#f72585
- 按钮风格：圆角按钮，悬停放大，点击弹性缩放
- 字体：系统无衬线字体
- 整体风格：深色科技感、霓虹光效、磨砂玻璃

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 画布区域 | 全屏 Canvas，径向渐变背景，实时绘制曲线，粒子拖尾 |
| 主页面 | 工具栏 | 右侧浮动磨砂玻璃面板，三个功能按钮，白色图标+文字 |
| 主页面 | 线条计数 | 左上角白色小字，透明度 0.6 |

### 4.3 响应式

- 桌面端（≥768px）：右侧浮动工具栏（宽 220px，右边缘 20px，圆角 16px
- 移动端（<768px）：底部固定工具栏（高 80px，按钮横向排列）
- devicePixelRatio 适配，确保高 DPI 屏幕清晰显示

## 5. 性能要求

- 绘制时 FPS ≥ 55
- 回放时 FPS ≥ 50
- 支持连续绘制 500 条线条无卡顿

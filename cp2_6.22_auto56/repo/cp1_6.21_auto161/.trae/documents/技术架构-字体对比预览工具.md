## 2. 技术描述

- **前端框架**：React 18 + TypeScript 5 + Vite 5

- **状态管理**：React Context + useReducer（用户指定）

- **HTTP请求**：Axios

- **样式方案**：原生CSS（CSS Modules / styled-components，内联样式

- **图标库**：lucide-react

- **构建工具**：Vite

- **后端**：无后端，纯前端应用

## 4. 核心类型定义

```typescript
interface FontConfig {
  id: string;
  family: string;
  family: string;
  size: number;      // 12-72
  weight: number;    // 100-900
  lineHeight: number; // 1.0-2.0
  color: string;   // 颜色
}

interface FontContextType {
  fonts: FontConfig[];
  previewText: string;
  focusedFontId: string | null;
  addFont: (family: string) => void;
  removeFont: (id: string) => void;
  updateFontConfig: (id: string, config: Partial<Omit<FontConfig, 'id' | 'family'>) => void;
  setPreviewText: (text: string) => void;
  setFocusedFontId: (id: string | null) => void;
  clearAll: () => void;
}
```

## 5. 数据持久化方案

### 5.1 localStorage 键名

- `font-comparer-configs`：字体配置列表（JSON数组）

- `font-comparer-text`：预览文本

- `google-fonts-cache`：Google Fonts列表缓存（含时间戳）

### 5.2 缓存策略

- 字体列表缓存24小时过期机制

- 配置变化时自动写入

- 页面加载时自动恢复

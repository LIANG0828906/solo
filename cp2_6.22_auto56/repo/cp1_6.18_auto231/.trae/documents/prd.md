## 1. 产品概述

几何拼图互动应用——一款基于浏览器的拖拽式几何形状拼图游戏，解决在线拼图游戏创作自由度低、形状识别不准确及缺少动态视觉反馈的问题。用户可从形状库中拖拽圆形、矩形、三角形到预设槽位，系统智能识别并吸附，配合丰富的动画反馈和多人关卡模式，提供沉浸式拼图体验。

- 目标用户：休闲游戏玩家、儿童教育、交互设计爱好者
- 核心价值：创作自由度 + 精准形状识别 + 动态视觉反馈

## 2. 核心功能

### 2.1 功能模块

1. **拼图主界面**：左侧形状库 + 中央拼图区 + 右侧工具栏 + 底部关卡选择
2. **形状拖拽与智能吸附**：拖拽几何图形，进入正确槽位自动吸附，触发弹性动画与闪烁效果
3. **拼图完成检测与反馈**：全部吸附后金色光效边框 + 磨砂玻璃完成弹窗
4. **错误放置提示**：错误槽位红色警告边框 + 形状滑回原位动画
5. **重置与随机模式**：重置形状到初始位置 / 随机打乱，带旋转飞行动画
6. **多关卡支持**：3个关卡，不同形状数量和位置，缩放淡入切换过渡

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 拼图主界面 | 形状库 | 展示可拖拽的圆形(#FF6B6B)、矩形(#4FC3F7)、三角形(#FFD93D)，带1px同色阴影 |
| 拼图主界面 | 拼图板 | 600x400px区域，2x3虚线槽位网格，吸附判定(误差半径25px) |
| 拼图主界面 | 工具栏 | 重置按钮 + 随机按钮，圆角8px深蓝底色 |
| 拼图主界面 | 关卡选择 | 底部3个60x60px缩略图，当前关卡高亮#FFD700 |
| 拼图主界面 | 完成弹窗 | 磨砂玻璃背景blur:12px，"恭喜完成"文字+"再来一局"按钮 |

## 3. 核心流程

1. 用户从左侧形状库拖拽几何图形
2. 拖拽过程中图形跟随鼠标，放大1.05倍、透明度0.85
3. 松开时系统判定是否进入正确槽位（误差半径25px）
4. 正确：自动吸附对齐中心 → 弹性动画(0.3s scale 1.1→1.0) → 槽位闪烁3次
5. 错误：红色警告边框(0.5s) → 形状滑回原位(1s ease-in-out)
6. 所有形状吸附完成 → 金色光效边框(2s无限循环) → 完成弹窗
7. 可通过重置/随机按钮操作，或切换关卡

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色赛博风格
- **主色**：深蓝渐变背景 #1A1A2E → #16213E
- **强调色**：#4FC3F7（天蓝）、#FFD700（金色）、#FF6B6B（珊瑚红）、#FFD93D（明黄）
- **按钮风格**：圆角8px，深蓝底色#0F3460，悬停上浮2px+蓝色投影
- **字体**：系统字体栈，清晰可读
- **布局**：三栏式（左250px形状库 + 中央600px拼图区 + 右200px工具栏）
- **动画**：全局0.2-0.3s ease-out过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 拼图主界面 | 形状库 | 宽250px，背景#0F3460圆角12px，内距20px，列表项间距16px |
| 拼图主界面 | 拼图区 | 600x400px，背景#16213E，4px #0F3460空心边框，6px内阴影，虚线槽位 |
| 拼图主界面 | 工具栏 | 宽200px，背景#0F3460圆角12px，按钮悬停#1A5276+上浮2px |
| 拼图主界面 | 关卡选择 | 底部3个缩略图60x60px圆角8px，当前高亮#FFD700 |
| 拼图主界面 | 完成弹窗 | 磨砂玻璃blur:12px圆角16px |

### 4.3 响应式

- 桌面优先设计
- 视口宽度<768px时，左右面板折叠为底部图标栏
- 拼图区域自适应缩放

### 4.4 动画规格

| 交互 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 拖拽中 | scale(1.05) + opacity(0.85) | 即时 | - |
| 正确吸附 | scale(1.1)→1.0弹性 | 0.3s | ease-out |
| 槽位闪烁 | 白色↔原色 x3 | 0.3s | - |
| 错误放置 | 滑回原位 | 1s | ease-in-out |
| 错误边框 | 红色2px | 0.5s | 渐隐 |
| 完成光效 | 金色渐变循环 | 2s | ease-in-out无限 |
| 关卡切换 | 缩放淡入 | 0.4s | ease-out |
| 重置/随机 | 飞回/飞至+旋转15° | 0.5s间隔依次 | ease-out |

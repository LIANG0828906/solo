## 1. 产品概述

基于语境的智能歌词自动生成器，用户输入心情描述或场景关键词，系统通过马尔可夫链算法和押韵模板自动生成个性化歌词，支持编辑、收藏、分享及社区互动。

- **目标用户**：音乐爱好者、创作者、需要灵感的音乐人
- **核心价值**：降低歌词创作门槛，通过AI辅助快速生成富有意境的歌词内容
- **市场定位**：轻量级创意工具，结合社区分享功能的歌词创作平台

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 生成歌词、编辑、收藏、浏览集锦广场、点赞评论 |

### 2.2 功能模块

1. **创作首页**：沉浸式入口，动态粒子背景，心情标签选择，场景输入
2. **歌词展示与编辑**：打字机效果逐行展示，频谱可视化，逐词编辑，押韵调整
3. **收藏与分享**：心形收藏动画，社交平台分享，重新生成
4. **集锦广场**：瀑布流卡片展示，搜索筛选，详情模态框，点赞评论，自动刷新

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|

## 4. 用户界面设计

### 4.1 设计风格

**设计方向**：沉浸式音乐创作体验，融合赛博朋克与艺术化表达
- **主色调**：深色背景为基底，根据心情动态切换
  - 忧郁：蓝紫色调（#6366F1 → #A855F7）
  - 快乐：橙粉色调（#F97316 → #EC4899）
  - 浪漫：玫红色调（#EC4899 → #F472B6）
  - 热血：红橙色调（#EF4444 → #F97316）
- **字体**：展示字体使用 "Playfair Display"，正文字体使用 "Noto Sans SC"
- **动效**：流畅的贝塞尔曲线过渡，微交互动画，粒子系统
- **布局**：沉浸式全屏设计，非对称布局，视觉焦点集中

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 创作首页 | 粒子背景 | Canvas粒子系统，色调随心情切换，流体运动 |
| 创作首页 | 输入区域 | 居中大输入框，玻璃拟态，占位文字动态轮播 |
| 创作首页 | 心情标签 | 4个渐变色按钮，圆角设计，hover缩放+阴影动效 |
| 歌词展示页 | 歌词区域 | 左侧滚动区域，行高宽松，打字机光标 |
| 歌词展示页 | 频谱区域 | 右侧Canvas，动态波形，渐变色彩 |
| 歌词展示页 | 操作按钮 | 收藏（心形）、分享、换一批，浮动布局 |
| 集锦广场 | 瀑布流 | 不规则卡片，阴影层叠，悬停遮罩 |
| 集锦广场 | 模态框 | 底部滑入，毛玻璃背景，内容区圆角 |

### 4.3 响应式

- **桌面优先**：1200px以上为主要设计断点
- **平板适配**：768-1200px，两列布局调整为单列
- **移动端**：768px以下，全屏单栏，触控优化
- **触摸交互**：按钮最小44px触控区域，滑动手势支持

### 4.4 动画与动效规范

| 动效类型 | 参数 | 应用场景 |
|---------|------|---------|
| 页面过渡 | cubic-bezier(0.4, 0, 0.2, 1), 500ms | 页面切换 |
| 悬停缩放 | scale(1.05), 200ms | 按钮、卡片 |
| 粒子运动 | 随机速度0.5-2px/帧 | 背景粒子 |
| 打字机效果 | 每个字符50ms延迟 | 歌词展示 |
| 辉光脉冲 | opacity 0.5-1, 2s循环 | 押韵词高亮 |
| 模态滑入 | translateY(100%) → 0, 400ms | 详情弹窗 |

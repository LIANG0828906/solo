## 1. 产品概述
山谷风电布局优化模拟器，帮助风电项目开发者在浏览器中直观模拟山谷地形中的气流流动轨迹，评估不同机位的风速影响，优化风力发电机组布局。
- 核心解决痛点：风电项目前期选址缺乏直观流体可视化工具，难以评估不同机位的尾流影响和发电效率
- 目标用户：风电项目规划师、能源工程师、环境评估人员

## 2. 核心 Features

### 2.1 功能模块
1. **3D场景主页面**：地形展示、风机布局、粒子气流模拟、发电量统计、控制面板

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主页面 | 三维地形模块 | 200x200单位山谷地形，起伏幅度10-50可调，绿色渐变着色 |
| 主页面 | 风机布局模块 | 点击放置8-12台风机，拖拽调整位置，显示实时风速标签 |
| 主页面 | 尾流可视化模块 | 半透明锥体显示相邻机位尾流影响，拖拽实时更新 |
| 主页面 | 气流粒子系统 | 800-1200个粒子沿地形流动，遇风机产生湍流效果 |
| 主页面 | 发电量统计面板 | 柱状图显示各机组瞬时功率，全场总功率和平均风速 |
| 主页面 | 布局优化模块 | 自动推荐3个最佳新增机位，显示预期发电量提升 |
| 主页面 | 控制面板 | 地形起伏、粒子密度、风机数量滑块，优化按钮，可折叠 |

## 3. 核心流程
用户进入页面 → 查看默认山谷地形与气流模拟 → 调整地形起伏参数 → 点击放置风机 → 拖拽调整风机位置 → 观察尾流影响和发电量变化 → 点击优化按钮获取布局建议 → 根据建议调整布局

```mermaid

    A["进入页面"] --> B["查看默认地形与气流"]
    B --> C["调整地形参数"]
    C --> D["放置/拖拽风机"]
    D --> E["观察尾流与发电量"]
    E --> F["点击优化按钮"]
    F --> G["获取推荐机位"]
    G --> H["调整布局"]
```

## 4. User Interface Design

### 4.1 设计风格
- 主色调：暗色调背景 #121212，绿色主题 #4caf50
- 辅色：蓝色粒子 #64b5f6→#1e88e5，橙色尾流 #ff9800，黄色功率 #ffeb3b→#f44336，金色推荐 #ffd700
- 字体：白色标题14px，深色半透明面板 rgba(30,30,30,0.85)
- 圆角：控制面板和统计面板圆角12px
- 动效：面板切换0.3s fade，柱状图更新0.5s动画

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI Elements |
|---------|---------|-------------|
| 主页面 | 左上角LOGO区 | 白色字体14px，LOGO+标题 |
| 主页面 | 左侧3D场景区 | 占70%宽度，最小高度600px，Three.js渲染 |
| 主页面 | 右侧控制面板 | 占30%宽度，半透明深色，可折叠展开，自定义圆形滑块 |
| 主页面 | 发电量统计面板 | 右侧固定，半透明深色，柱状图，实时数据 |

### 4.3 响应式
- 桌面优先，适配1366px以上分辨率
- 场景区域最小高度600px

### 4.4 3D场景指引
- 环境：暗色背景，方向光模拟太阳光，环境光提供基础照明
- 相机：PerspectiveCamera，初始视角俯瞰山谷
- 交互：OrbitControls控制视角，DragControls拖动机组
- 动画：风机叶片持续旋转，粒子每帧更新位置
- 性能：粒子模拟60fps，尾流计算1秒内完成，优化计算3秒内

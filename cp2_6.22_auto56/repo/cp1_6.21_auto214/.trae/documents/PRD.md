## 1. 产品概述

CSS 动画组合预览工具 —— 帮助前端开发者直观理解多 CSS 属性协同作用下的复杂过渡和关键帧动画效果，解决现有文档和在线工具仅展示单一属性动画、难以预览组合效果的问题。

- **目标用户**：前端开发者、UI 设计师、动画爱好者
- **核心价值**：可视化配置多属性关键帧动画，实时预览 + 代码生成 + 一键导出

## 2. 核心功能

### 2.1 功能模块

1. **样式编辑面板**：属性选择下拉框、目标值输入、关键帧步骤的增删改和拖拽排序
2. **动画预览区**：实时播放预览元素、播放控制器（播放/暂停/重置/速度）、时间进度条
3. **代码预览区**：实时生成 @keyframes CSS 代码、语法高亮、一键复制
4. **预设动画库**：弹跳、渐入、旋转+缩放、模糊+位移四个模板卡片
5. **配置导入导出**：本地 JSON 文件保存与恢复

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 样式编辑面板 | 选择 CSS 属性（transform/opacity/filter/clip-path/border-radius/background-color），填写目标值，最多 8 个关键帧步骤，支持拖拽排序 |
| 主页面 | 动画预览区 | 300×300px 预览元素，默认 2s 循环播放，播放/暂停/重置按钮，1x-3x 速度滑块，时间进度条 |
| 主页面 | 代码预览区 | 实时生成 @keyframes 代码，属性名高亮 #82AAFF，属性值高亮 #C3E88D，一键复制（绿色 toast 2s） |
| 主页面 | 预设动画库 | 4 个动画模板卡片，140×100px，悬浮上浮 2px，点击即加载并自动播放 |
| 主页面 | 导入导出 | 下载 animation-config.json，从本地 JSON 文件导入恢复配置 |

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色主题（背景 #0F172A，文字 #E2E8F0）
- **主色**：#6366F1（选中态、悬浮高亮）、#A78BFA（进度条填充）
- **辅助色**：#475569（控件背景）、#334155（边框/虚线框）、#3B82F6（预览元素）
- **圆角**：统一使用 8px-16px 圆角（面板 12px，预览区 16px，小卡片 8px）
- **字体**：代码区 monospace 14px/1.6，其他使用系统无衬线字体
- **动效**：所有控件 0.2s-0.3s ease-out 过渡，拖拽时卡片缩至 95%（0.15s），悬浮上浮 2px
- **阴影**：细微阴影增强层次感

### 4.2 页面布局

| 区域 | 尺寸/占比 | 背景色 | 圆角 | 内边距 |
|-----|----------|--------|------|--------|
| 左侧编辑面板 | 固定 360px（25%） | #1E293B | 12px | 20px |
| 中央预览区 | 弹性 50% | #111827 | 16px | - |
| 右侧代码预览区 | 25% | #0F172A | 12px | 16px |

### 4.3 响应式适配

- **≥1024px**：三列并排布局（左 25% / 中 50% / 右 25%）
- **768px-1023px**：上下两行（上方预览区，下方编辑+代码并列）
- **<768px**：单列纵向排列（预览区 → 编辑面板 → 代码预览区）

### 4.4 性能要求

- 动画播放帧率稳定 60fps
- 代码实时预览更新延迟 ≤ 50ms


## 1. 架构设计
前端单页应用架构，分为编辑器模块、渲染器模块、触发器模块、配置导出模块四大核心模块。

## 2. 技术描述
- 前端：React@18 + TypeScript + Vite
- 拖拽库：react-dnd + react-dnd-html5-backend
- 图标库：lucide-react
- 状态管理：React useState/useReducer
- 唯一ID：uuid
- 构建工具：Vite

## 3. 核心文件结构
```
src/
├── main.tsx                 # 应用入口
├── types/
│   └── popup.ts             # 类型定义
├── modules/
│   ├── editor/
│   │   ├── App.tsx          # 主布局组件
│   │   ├── ComponentPanel.tsx  # 左侧组件面板
│   │   ├── PreviewCanvas.tsx # 右侧预览画布
│   │   └── PropertyPanel.tsx # 属性配置面板
│   ├── renderer/
│   │   └── PopupRenderer.tsx  # 弹窗渲染引擎
│   │   └── popups/          # 具体弹窗组件
│   │       ├── CountdownPopup.tsx
│   │       ├── SurveyPopup.tsx
│   │       └── AnnouncementPopup.tsx
│   ├── triggers/
│   │   └── EventDispatcher.ts # 事件调度模块
│   └── configExporter/
│       └── ConfigExporter.ts # 配置导出模块
```

## 4. 数据模型

### 4.1 弹窗配置类型
```typescript
interface PopupConfig {
  id: string;
  type: 'countdown' | 'survey' | 'announcement';
  x: number;
  y: number;
  width: number;
  height: number;
  backgroundColor: string;
  borderRadius: number;
  trigger: {
    type: 'delay' | 'click' | 'scroll';
    delay?: number;
    scrollPosition?: number;
  };
  content: any;
}
```

## 5. 模块调用关系
- App.tsx 作为主容器，管理弹窗配置状态
- ComponentPanel 提供拖拽源
- PreviewCanvas 接收拖拽目标 + 渲染弹窗实例
- PropertyPanel 编辑选中弹窗的属性
- PopupRenderer 根据配置渲染具体弹窗
- EventDispatcher 处理触发事件（延迟、点击、滚动）
- ConfigExporter 序列化配置为JSON

## 1. 产品概述
零售门店库存与销售管理系统——帮助小型独立杂货店老板摆脱纸质账本，实时掌握商品库存、销售情况及供应商信息，实现商品入库、销售收银、库存预警和销售报表一站式管理。

- 主要用途：为杂货店提供数字化的库存管理、POS收银和销售分析工具
- 目标用户：小型独立杂货店经营者，需要简洁直观的操作界面

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 店主 | 无需注册（单用户系统） | 全部功能：商品管理、销售收银、库存预警、销售报表 |

### 2.2 功能模块
1. **库存管理面板**：商品列表展示、新增/编辑/删除商品、实时搜索过滤、库存预警
2. **销售收银面板**：条码/名称搜索商品、购物车管理、总价实时计算、结账与库存扣减
3. **销售报表面板**：最近30笔销售记录、日期/类别筛选、记录删除与撤销

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 库存管理面板 | 商品列表 | 两列网格展示商品卡片，含名称、条码、类别、进价、售价、库存量、供应商，支持搜索和类别筛选 |
| 库存管理面板 | 商品表单 | 新增/编辑商品表单，含名称、条码、类别、进价、售价、库存量、供应商、预警阈值，提交带动画反馈 |
| 库存管理面板 | 库存预警 | 低于阈值的商品以红色背景脉冲动画展示，导航栏角标闪烁提示 |
| 销售收银面板 | 商品搜索 | 输入条码或搜索商品名称快速添加到购物车 |
| 销售收银面板 | 购物车 | 显示商品名、单价、数量、小计，支持修改数量和移除，总价实时更新 |
| 销售收银面板 | 结账 | 点击结账扣减库存并生成销售记录，成功弹出星星散落动画确认对话框 |
| 销售报表面板 | 销售记录列表 | 滚动卡片展示最近30笔记录，含时间、商品清单、总金额，加载时淡入上浮动画 |
| 销售报表面板 | 筛选与操作 | 按日期和类别筛选，删除记录带滑动消失和5秒撤销按钮 |

## 3. 核心流程

**商品入库流程**：店主打开库存面板 → 点击新增商品 → 填写商品信息（名称、条码、类别、价格、库存、供应商）→ 提交 → 系统校验条码唯一性 → 保存成功绿色闪烁/失败红色抖动

**销售收银流程**：店主打开收银面板 → 扫码或搜索商品 → 添加至购物车 → 调整数量 → 点击结账 → 系统扣减库存并生成销售记录 → 星星散落确认弹窗

**库存预警流程**：商品库存低于阈值 → 导航栏和库存面板同时显示红色角标闪烁 → 点击进入预警列表 → 红色背景脉冲动画突出显示

**销售报表流程**：打开报表面板 → 浏览最近30笔记录 → 按日期/类别筛选 → 可删除记录（5秒内可撤销）

## 4. 用户界面设计

### 4.1 设计风格
- 主色：木质棕金色 #D2A679，深棕色 #6B4423，柔和米白底色 #FAF5EE
- 按钮风格：圆角8px，点击缩放0.95→1.0过渡效果
- 字体：使用 Playfair Display 作为标题字体，Noto Sans SC 作为正文字体
- 布局风格：固定顶部导航栏，两列网格卡片布局，卡片带圆角12px和细微投影
- 图标风格：使用 lucide-react 线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 顶部导航栏 | 深棕色#6B4423背景，选中项淡金色下划线动画，库存预警红色角标闪烁 |
| 库存管理面板 | 商品列表 | 两列网格，卡片圆角12px阴影，悬停上浮3px加深阴影，类别标签筛选 |
| 库存管理面板 | 商品表单 | 模态对话框，成功绿色高亮闪烁，失败红色抖动动画 |
| 库存管理面板 | 预警列表 | 红色背景脉冲动画卡片，醒目预警标识 |
| 销售收银面板 | 购物车 | 商品添加时从右侧滑入，数量调整数字跳动效果 |
| 销售收银面板 | 结账确认 | 星星散落动画弹窗，确认信息展示 |
| 销售报表面板 | 记录卡片 | 加载时从底部淡入上浮，删除时滑动消失，撤销按钮5秒倒计时 |

### 4.3 响应式设计
- 桌面优先设计，小屏设备（<768px）网格变为单列
- 导航栏在移动端变为汉堡菜单
- 购物车在移动端自适应缩窄
- 触摸优化：按钮最小点击区域44px

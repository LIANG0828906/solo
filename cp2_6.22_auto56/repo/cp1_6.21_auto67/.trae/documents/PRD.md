## 1. 产品概述

智能食验室是一款基于AI图像识别的个性化食谱推荐应用，解决用户"冰箱有菜不知道做什么"的痛点。用户上传食材图片，系统智能识别并推荐可用食材覆盖率60%以上的食谱。

- **核心价值**：将闲置食材转化为美味食谱，减少食物浪费，激发烹饪灵感
- **目标用户**：家庭主妇、上班族、烹饪爱好者等需要快速解决用餐问题的人群
- **市场定位**：轻量级、智能化的厨房助手工具

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传图片、智能识别、浏览食谱、查看详情 |

### 2.2 功能模块
1. **主页面**：图片上传区、识别按钮、食谱结果瀑布流展示
2. **食谱详情面板**：烹饪步骤列表、食材清单、难度评级

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 图片上传区 | 支持拖拽/点击上传，虚线边框400x240px，波纹动画反馈，缩略图预览120x120px |
| 主页面 | 智能识别按钮 | 200x48px圆角按钮，渐变背景，悬停上浮动画，触发识别请求 |
| 主页面 | 食谱瀑布流 | 260px宽卡片，悬停阴影加深上浮，按覆盖率和难度排序 |
| 详情面板 | 步骤展示 | 右侧滑入占35%宽度，带序号圆圈的步骤列表 |

## 3. 核心流程

用户上传食材图片 → 前端校验图片格式大小 → 生成缩略图预览 → 点击识别按钮 → 后端模拟物体识别返回食材列表 → 后端匹配食谱库筛选60%以上覆盖率 → 返回排序后的食谱 → 瀑布流渲染展示 → 用户点击卡片 → 右侧滑出详情面板展示完整步骤

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：蓝紫渐变 #667eea → #764ba2，绿色渐变 #43e97b → #38f9d7
- **背景色**：浅灰 #F0F2F5，卡片白色，分隔线 #E0E0E0
- **文字色**：深灰 #333333，中灰 #9E9E9E，绿色 #2E7D32
- **按钮风格**：圆角胶囊形，渐变背景，悬停上浮+亮度提升，点击缩放回弹
- **字体**：现代无衬线字体，标题24px加粗，正文16px，标签14px
- **布局风格**：顶部导航栏+居中内容区，卡片式瀑布流，右侧滑出面板
- **图标风格**：简洁线性图标，使用lucide-react库

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|----------|
| 主页面 | 顶部导航 | 56px高白色背景，Logo+标题24px，右侧40px圆形头像占位 |
| 主页面 | 上传区 | 400x240px虚线边框圆角，浅灰背景#F8F9FA，拖入时蓝紫渐变边框+波纹动画 |
| 主页面 | 缩略图 | 120x120px圆角8px，带删除按钮，交错淡入动画 |
| 主页面 | 识别按钮 | 200x48px圆角24px，绿色渐变，悬停亮度+10%上浮2px，点击缩放0.95回弹 |
| 主页面 | 食谱卡片 | 260px宽白色圆角12px，1px边框8px阴影，悬停16px阴影上浮4px，0.3s过渡 |
| 主页面 | 食材标签 | 24px高圆角12px，已识别浅绿#E8F5E9，未识别浅灰#F5F5F5 |
| 详情面板 | 步骤序号 | 28px直径圆圈，蓝紫渐变背景，白色14px文字 |

### 4.3 响应性
- 桌面端优先，最大宽度1200px居中，左右边距自适应最小20px
- 瀑布流列数随屏幕宽度自适应：大屏3-4列，中屏2列，小屏1列
- 详情面板移动端改为底部滑入，占80%高度
- 触摸设备优化点击区域，确保可点击元素不小于44px

### 4.4 动画效果
- 页面加载：内容区交错淡入（0.1s间隔）
- 图片拖入：边框渐变+背景波纹扩散（0.3s淡入）
- 按钮交互：悬停上浮+亮度变化，点击缩放回弹（cubic-bezier弹性曲线）
- 卡片悬停：阴影加深+上浮4px（0.3s过渡）
- 详情面板：右侧滑入（cubic-bezier(0.25,0.1,0.25,1)，0.3s）
- 缩略图生成：缩放淡入（0.2s内完成）

## 1. 产品概述

TerrainTactics 是一款基于地形元素和资源博弈的回合制对战游戏，两位玩家在8x8网格地图上轮流行动，通过部署单位、占领资源点、利用地形优势进行战斗，最终占领对方基地或全歼对方单位获胜。

- 目标用户：线上策略游戏社区玩家
- 核心价值：将地形策略与资源管理深度结合，提供丰富的战术选择和博弈深度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 玩家1 | 创建房间 | 操作己方单位、部署单位、购买单位 |
| 玩家2 | 加入房间 | 操作己方单位、部署单位、购买单位 |

### 2.2 功能模块

1. **游戏大厅页**：房间创建、加入、玩家匹配
2. **对战主页面**：地图渲染、单位操作、战斗动画、资源管理、回合控制

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏大厅 | 房间列表 | 显示可用房间，创建/加入房间 |
| 游戏大厅 | 玩家信息 | 显示玩家编号和连接状态 |
| 对战主页面 | 地图渲染区 | Canvas渲染8x8网格地图、地形纹理、单位图标、选中高亮、可移动/攻击范围 |
| 对战主页面 | 单位操作 | 选中单位→显示可移动格子→移动或攻击，部署新单位 |
| 对战主页面 | 战斗动画 | 攻击冲刺、击中白闪、伤害数字弹出、死亡碎裂动画 |
| 对战主页面 | UI面板 | 回合信息、金币数量、单位卡片、购买单位、游戏日志、结束回合按钮 |
| 对战主页面 | 回合计时 | 60秒倒计时进度条，最后10秒闪烁警告 |

## 3. 核心流程

### 3.1 游戏流程

玩家创建/加入房间 → 双方就绪后游戏开始 → 初始化地图（随机地形+4资源点+双方基地）→ 玩家1先手回合 → 选择操作（移动单位+攻击/部署单位/购买单位）→ 结束回合或超时 → 切换到对方回合 → 循环直到胜负判定

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：中古奇幻风格
  - 主背景：深棕色 #3D2B1F
  - 地图区域：暖黄色（暗调）
  - UI面板：羊皮纸纹理 #F5DEB3 带暗色斑点
  - 木质边框：#8B7355，3px粗，轻微内阴影
- 按钮风格：木质边框圆角按钮，hover时发光
- 字体：标题使用MedievalSharp或UnifrakturMaguntia，正文使用Cinzel，日志使用Courier New
- 布局风格：左侧地图为主，右侧UI面板，顶部回合计时条
- 图标风格：Canvas绘制简化中世纪图标（盾牌、弓、剑）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏大厅 | 房间列表 | 中世纪风格卡片列表，木质边框，羊皮纸底色 |
| 游戏大厅 | 操作按钮 | 木质边框按钮"创建房间"/"加入房间" |
| 对战主页面 | 地图区域 | Canvas 8x8网格，地形颜色纹理，单位图标，选中绿色光环脉冲，可移动蓝色半透明高亮闪烁，攻击红色半透明高亮 |
| 对战主页面 | 回合计时条 | 顶部水平进度条，绿→红渐变，最后10秒红色闪烁 |
| 对战主页面 | 金币与资源 | 金币图标+数字，资源点占领状态 |
| 对战主页面 | 单位卡片 | 8px圆角矩形，木质边框，Canvas绘制图标（盾/弓/剑），选中时边框发光脉冲0.8秒 |
| 对战主页面 | 购买面板 | 3种单位卡片，显示属性和价格，点击购买部署 |
| 对战主页面 | 游戏日志 | 等宽字体Courier New，时间戳灰色小字+行动图标，最近5条 |
| 对战主页面 | 地形提示框 | 鼠标悬停0.2秒渐入气泡，显示地形名称和效果说明 |

### 4.3 地形视觉设计

| 地形 | 颜色 | 纹理图案 | 效果说明 |
|------|------|----------|----------|
| 平原 | 浅绿 | 无特殊纹理 | 标准地形，1点移动力 |
| 森林 | 深绿 | 树状图案 | +20%防御，2点移动力 |
| 山脉 | 棕灰 | 等高线图案 | +30%攻击，3点移动力（飞行无视） |
| 河流 | 蓝色 | 波浪纹 | 消耗全部移动力，+15%闪避 |
| 沙漠 | 黄色 | 点状纹理 | 无加成，移动消耗减半 |

### 4.4 响应式设计

- 桌面优先设计
- 屏幕宽度 < 768px 时：缩小网格单元格尺寸，隐藏部分UI细节（游戏日志折叠）
- 最小操作区域不小于44px
- Canvas自适应缩放

### 4.5 动画规格

| 动画 | 时长 | 实现方式 |
|------|------|----------|
| 选中光环脉冲 | 1秒周期 | Canvas绘制 |
| 可移动格闪烁 | 1秒周期 | Canvas alpha动画 |
| 攻击冲刺 | 0.3秒 | Canvas直线移动 |
| 击中白闪 | 0.2秒 | Canvas白色叠加 |
| 伤害数字弹出渐隐 | 1秒 | Canvas + CSS opacity |
| 死亡碎裂 | 0.5秒 | Canvas缩小白光消失 |
| 资源点旋转 | 持续 | Canvas旋转动画 |
| 旗帜飘动 | 持续 | Canvas旗帜动画 |
| 地形提示渐入 | 0.2秒 | CSS opacity transition |
| 倒计时闪烁 | 0.1秒间隔 | CSS animation |
| 单位卡片发光 | 0.8秒脉冲 | CSS box-shadow animation |

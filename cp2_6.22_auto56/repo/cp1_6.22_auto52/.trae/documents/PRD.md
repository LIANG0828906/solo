# 线稿上色与配色灵感生成器 — 产品需求文档

## 1. 项目概述

### 1.1 项目背景
线稿上色是数字绘画和设计工作中的常见环节，传统方式需要用户手动调配颜色并逐区域填充，过程繁琐且配色经验不足的用户难以获得和谐的视觉效果。本项目旨在提供一个轻量级的在线工具，帮助用户快速为黑白线稿上色，并通过智能配色推荐降低配色门槛。

### 1.2 产品定位
一款面向插画师、设计师和创意爱好者的在线线稿上色工具，主打"简单上手 + 智能配色"，让用户在几分钟内完成线稿上色并获得专业级配色方案。

### 1.3 目标用户
- 数字绘画初学者
- 需要快速产出配色方案的设计师
- 喜欢涂鸦和填色的休闲用户

---

## 2. 核心功能需求

### 2.1 图片上传与画布展示

**功能描述**：用户从本地上传黑白线稿图片，在画布中央显示。

**详细需求**：
- 支持常见图片格式（PNG、JPG、JPEG）
- 画布背景为浅灰色网格线，方便定位
- 图片上传后以淡入缩放动画出现（流畅过渡）
- 左上角显示图片原始尺寸信息（如 "800 × 600 px"）
- 图片居中显示，保持原始比例

### 2.2 调色板与区域填充

**功能描述**：用户从左侧调色板选择颜色，点击线稿内封闭区域进行填充上色。

**详细需求**：
- 调色板包含 12 种预设常用颜色
- 提供自定义颜色吸色器（HTML color picker）
- 点击封闭区域后，以点击点为中心向外波纹扩散填充动画
- 扩散动画持续 300ms
- 填充边缘自动做 1px 柔和抗锯齿处理
- 颜色不会溢出到相邻黑色线条外（像素级 flood fill 算法）
- 当前选中的颜色有明显高亮状态

### 2.3 配色推荐模块

**功能描述**：根据用户已使用的颜色，自动生成和谐的配色方案。

**详细需求**：
- 收集用户当前已使用的所有颜色
- 生成 3 套互补色方案
- 生成 3 套类似色方案
- 每套方案以横向色带卡片展示
- 色带从左到右渐变过渡平滑自然
- 点击任一方案后，整幅画所有已填色区域同步变换为新方案对应颜色
- 颜色变换采用整体淡出再淡入的过渡动画（持续 500ms）

### 2.4 撤销 / 重做

**功能描述**：支持操作撤销和重做，最多记录 20 步。

**详细需求**：
- 撤销按钮：恢复上一步操作前的画布状态
- 重做按钮：恢复被撤销的操作
- 最多记录 20 步历史
- 撤销时填充区域以反向波纹收缩动画还原
- 重做时重新扩散填充动画
- 按钮可用时显示彩色图标，不可用时灰色半透明并禁用点击

### 2.5 导出功能

**功能描述**：一键导出当前上色后的作品为 PNG 图片。

**详细需求**：
- 导出按钮位于画布上方工具栏
- 点击后显示 300ms 旋转齿轮加载动画
- 动画结束后自动下载 PNG 文件
- 文件名包含当前时间戳（如 `artwork_20240622_143025.png`）

---

## 3. 交互与动效需求

### 3.1 按钮交互
- 所有按钮（上传、撤销、重做、导出）鼠标悬停时底部出现 2px 宽彩色下划线
- 点击时按钮整体缩小为 90%，再弹回原大小
- 反馈清晰，有按压感

### 3.2 性能要求
- 点击填充到动画完成渲染 ≤ 400ms
- 配色推荐计算时间 ≤ 200ms
- 画布交互流畅，无明显卡顿

---

## 4. 视觉设计

### 4.1 整体风格
- 现代简约风格，注重功能与美感平衡
- 柔和中性色调为主，强调画布区域的视觉焦点
- 精致的微交互动效提升使用愉悦感

### 4.2 布局结构
- 左侧：调色板区域
- 中央：画布区域（含顶部工具栏）
- 右侧：配色推荐区域
- 三栏布局，画布区域占据最大比例

### 4.3 色彩系统
- 主色调：柔和的蓝紫色系，传达创意与艺术感
- 辅助色：暖橙色用于强调和交互反馈
- 中性色：多层次的灰色用于界面框架
- 画布背景：浅灰网格（#f5f5f5 底色 + 1px 网格线）

### 4.4 字体
- 标题：现代无衬线字体，清晰有力
- 正文：易读性良好的无衬线字体

---

## 5. 非功能性需求

### 5.1 技术约束
- 使用 TypeScript 和 React 构建
- Vite 作为构建工具
- 配色推荐算法为独立纯计算模块，与 UI 分离
- 数据通过 Props 和回调函数传递

### 5.2 浏览器兼容性
- 支持主流现代浏览器（Chrome、Firefox、Safari、Edge）

### 5.3 代码质量
- TypeScript 严格模式
- 模块职责清晰，代码可维护性良好
- 关键算法有性能优化

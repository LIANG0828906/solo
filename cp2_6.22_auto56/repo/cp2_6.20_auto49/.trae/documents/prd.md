## 1. 产品概述

数字花园笔记管理与知识图谱可视化应用，旨在解决个人笔记分散、难以形成知识关联和结构化探索的问题。目标用户为知识工作者、学生和终身学习者，通过双向链接和图谱可视化帮助用户构建个人知识网络。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册（本地使用） | 创建、编辑、删除笔记；查看图谱；搜索筛选 |

### 2.2 功能模块

1. **笔记列表主页**：笔记卡片列表、标签过滤、浮动操作按钮
2. **笔记编辑器**：富文本/Markdown编辑、标签管理、双向链接语法、自动摘要
3. **图谱页面**：力导向知识图谱可视化、节点交互、双击跳转
4. **搜索页面**：模糊搜索、标签与日期组合筛选、结果高亮

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 笔记列表主页 | 笔记卡片列表 | 展示所有笔记的标题、摘要、标签胶囊、创建时间，支持长按拖拽排序 |
| 笔记列表主页 | 浮动操作按钮 | 青绿色圆形按钮，展开为四个子菜单（新笔记、图谱、搜索、回收站），带旋转缩放动画 |
| 笔记列表主页 | 标签过滤栏 | 按标签类别过滤笔记列表 |
| 笔记编辑器 | 富文本编辑区 | 支持Markdown语法、插入图片和代码块，暖白背景(#faf8f5)，衬线字体Georgia，行距1.6倍 |
| 笔记编辑器 | 标签胶囊管理 | 添加多个标签，以圆角彩色胶囊形式显示（浅蓝、浅绿、浅紫） |
| 笔记编辑器 | 双向链接 | 通过[[笔记标题]]语法引用其他笔记，保存后自动建立双向链接 |
| 笔记编辑器 | 反向引用列表 | 笔记详情底部展示反向引用，来源标题和代码高亮引用片段，底部向上滑入渐显动画 |
| 笔记编辑器 | 自动摘要 | 编辑保存后自动生成前100字摘要 |
| 图谱页面 | 力导向图谱 | d3-force力导向图，节点大小根据引用次数动态变化，颜色按标签类别区分 |
| 图谱页面 | 节点交互 | 节点可拖拽重新布局，悬停显示标题和标签列表，双击跳转笔记详情 |
| 图谱页面 | 连线样式 | 引用关联用实线，标签关联用虚线 |
| 搜索页面 | 搜索框 | 模糊匹配笔记标题和内容 |
| 搜索页面 | 搜索结果 | 折叠列表，匹配部分浅黄色高亮 |
| 搜索页面 | 过滤面板 | 右侧面板勾选标签、日期范围进行组合筛选，实时更新 |

## 3. 核心流程

用户打开应用后，可在主页查看笔记列表，通过浮动按钮创建新笔记或切换到图谱/搜索视图。在编辑器中编写笔记，使用[[笔记标题]]建立双向链接。保存后笔记自动生成摘要，图谱自动更新节点和连线关系。在图谱视图中直观浏览知识网络，双击节点进入笔记详情，底部可查看反向引用。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖白(#faf8f5)、深灰(#1e1e24)、青绿(#2a9d8f)
- **标签颜色**：科技类青蓝色#2a9d8f、生活类金橙色#e9c46a、学习类粉紫色#b5838d
- **按钮样式**：圆角8px，hover时上浮2px并加深阴影
- **字体**：标题用无衬线(Inter)，正文用衬线(Instrument Serif/Georgia)
- **布局风格**：桌面端左右分栏（30%笔记列表+70%图谱），平板端上下分栏，手机端全屏+底部滑动面板
- **标签胶囊**：圆角彩色小胶囊，背景色柔和（浅蓝#dbeafe、浅绿#dcfce7、浅紫#ede9fe）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 笔记列表主页 | 笔记卡片列表 | 卡片圆角8px，标题无衬线粗体，摘要衬线体，标签胶囊，hover上浮阴影 |
| 笔记列表主页 | 浮动操作按钮 | 青绿色圆形，展开子菜单带旋转缩放动画，每个子项有文字标签 |
| 笔记编辑器 | 编辑区域 | 暖白背景#faf8f5，Georgia衬线体，行距1.6倍，顶部固定工具栏，全屏沉浸式 |
| 笔记编辑器 | 文本选择动画 | 光标位置出现浅色放大镜环状动画 |
| 笔记编辑器 | 反向引用列表 | 底部向上滑入渐显动画，代码高亮引用片段 |
| 图谱页面 | 力导向图谱 | 深灰背景#1e1e24，节点柔和光晕动画，节点大小动态变化（最大40px），连线实线/虚线区分 |
| 搜索页面 | 搜索结果 | 折叠列表，匹配部分浅黄色高亮，右侧过滤面板 |

### 4.3 响应式设计

- **桌面端（>1024px）**：图谱自动占据右侧70%区域，左侧30%为笔记列表
- **平板端（768px-1024px）**：切换为上下分栏布局
- **手机端（<768px）**：图谱全屏展示，通过底部滑动面板查看笔记详情

### 4.4 性能目标

- 图谱视图200个节点以内帧率稳定30fps以上
- 搜索响应不超过200ms

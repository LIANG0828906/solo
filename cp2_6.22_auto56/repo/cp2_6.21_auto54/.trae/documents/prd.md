## 1. 产品概述

Cell Explorer 是一个在线 3D 细胞结构探索与交互可视化应用，让用户能够在三维空间中像在显微镜下一样探索放大的动物细胞模型，点击不同细胞器获取详细知识卡片，支持结构图/剖面图模式切换和截图导出功能。面向生物学教育、科普展示和医学学习场景。

## 2. 核心功能

### 2.1 功能模块

1. **3D 细胞模型展示页**：程序化生成动物细胞三维模型，包含细胞膜、细胞核、线粒体、内质网、高尔基体、溶酶体六种细胞器
2. **交互与知识卡片**：点击细胞器触发高亮发光效果，右侧滑入知识卡片面板
3. **视图模式切换**：结构图模式（完整半透明细胞膜）与剖面图模式（平面裁切暴露内部）
4. **截图保存**：将当前 3D 视角导出为 PNG 图片下载

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 细胞探索 | 细胞模型渲染 | 程序化生成细胞膜（半透明球体）、细胞核（自发光球体）、线粒体（椭球体）、内质网（扁平环状）、高尔基体（堆叠圆盘）、溶酶体（小球体），所有细胞器缓慢漂浮旋转 |
| 3D 细胞探索 | 悬浮与点击交互 | 鼠标悬浮检测（射线检测），悬浮时细胞器微微放大，点击时高亮发光（白色边缘光，闪烁动画），更新 store 状态 |
| 3D 细胞探索 | 知识卡片面板 | 右侧滑入面板（360px），显示细胞器名称、功能描述、示意图标、详情链接按钮，0.3s ease-out 滑入动画 |
| 3D 细胞探索 | 工具栏 | 顶部居中，包含"结构图"/"剖面图"切换按钮和截图按钮，圆角图标按钮风格 |
| 3D 细胞探索 | 剖面模式 | 通过 clipping plane 沿 Z 轴裁切前半部分，细胞膜变为全透明，内部细胞器完整保留，裁切面动画过渡 0.5s |
| 3D 细胞探索 | 截图功能 | 临时隐藏 UI，使用 WebGLRenderer 导出 PNG，文件名 cell-snapshot-{timestamp}.png |

## 3. 核心流程

用户打开应用 → 3D 细胞模型自动加载并缓慢漂浮旋转 → 用户可自由旋转/缩放视角 → 悬浮细胞器时高亮反馈 → 点击细胞器触发知识卡片滑入 → 切换结构图/剖面图模式观察内部结构 → 点击截图按钮保存当前视角

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色实验科技风，背景 #13172a 到 #0c0f1e 径向渐变
- **辅助色**：科技蓝 #4a90d9 作为主强调色，各细胞器独立配色
- **按钮风格**：圆形图标按钮（40x40px），背景 #2a2f52，hover #3a3f62，圆角 8px
- **字体**：'Inter', sans-serif，标题白色带微弱发光描边
- **布局风格**：全屏 3D 沉浸式，工具栏浮动顶部，知识卡片右侧滑入
- **图标风格**：简洁线性图标 + emoji 细胞器标识

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 3D 细胞探索 | 应用标题 | 左上角 "Cell Explorer"，白色，text-shadow: 0 0 8px rgba(100,200,255,0.5) |
| 3D 细胞探索 | 工具栏 | 居中浮动，背景 rgba(20,25,50,0.7)，backdrop-blur 10px，圆角 12px |
| 3D 细胞探索 | 知识卡片 | 宽 360px，背景 rgba(255,255,255,0.92)，圆角 16px，box-shadow 0 8px 32px rgba(0,0,0,0.2) |
| 3D 细胞探索 | 3D 场景 | 全屏 WebGL 画布，OrbitControls 自由旋转缩放 |

### 4.3 响应式设计

- 桌面优先设计，768px 以上完整体验
- 宽度 < 768px：工具栏按钮缩小为 32x32px，知识卡片宽度改为 260px
- 触控优化：支持触控旋转/缩放/点击

### 4.4 3D 场景指导

- **环境与氛围**：深色空间感，微弱环境光营造显微镜观察感
- **光照设置**：环境光（低强度）+ 定向光 + 点光源，突出细胞器自发光效果
- **摄像机设置**：透视相机，FOV 50°，初始距离 5，OrbitControls 限制缩放范围
- **构图与焦点**：细胞模型居中，各细胞器围绕细胞核分布
- **交互与动画**：细胞器漂浮旋转（0.005rad/s，幅度 0.05），悬浮放大，点击高亮闪烁
- **后处理效果**：可选 bloom 效果增强发光感
- **性能预算**：FPS 稳定 45+，交互延迟 < 100ms，截图导出 < 1s

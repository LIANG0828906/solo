# 星图演算 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 产品背景
考古团队在遗址现场发现了一组古星图残片，需要借助三维技术将不同年代的星图数据融合，观察星座在历史长河中的位移与演变。

### 1.2 产品目标
构建一个沉浸式的三维星图可视化应用，让用户能够：
- 直观对比三个历史年代（公元前2000年、公元元年、公元1500年）的星空差异
- 观察星体在三千年间的运动轨迹
- 探索15个星座在不同年代的形态演变
- 通过交互深入了解每个星体的详细信息

### 1.3 目标用户
- 考古研究人员
- 天文学爱好者
- 历史学者
- 科普教育工作者

## 2. 功能需求

### 2.1 星图加载与年代切换
| 需求项 | 详细描述 |
|--------|----------|
| 内置数据 | 三组星图数据：公元前2000年、公元元年、公元1500年 |
| 数据规模 | 每个年代包含15个星座，每个星座3-6颗星 |
| 切换方式 | 页面底部滑块（范围0-2，步长1） |
| 过渡效果 | 星体位置在1秒内平滑插值过渡 |

### 2.2 星座连线与标签展示
| 需求项 | 详细描述 |
|--------|----------|
| 星座连线 | 金色半透明线条（#FFD700, opacity 0.4）连接星体 |
| 连线跟随 | 线条端点跟随星体位置变化 |
| 星座标签 | TextGeometry，字体大小0.3，颜色#FFF，始终面向相机 |
| 标签背景 | 半透明黑色圆角矩形（rgba(0,0,0,0.5)，内边距4px） |

### 2.3 星体详情面板
| 需求项 | 详细描述 |
|--------|----------|
| 触发方式 | 点击任意星体 |
| 面板位置 | 左侧固定弹出（桌面）/ 底部弹出（移动端） |
| 面板样式 | 宽240px，背景#1A1A2E，圆角8px，内边距16px |
| 显示内容 | 星体名称（16px粗体，#FFD700）、所属星座、星等（保留2位小数）、三年代坐标偏移量（弧秒） |
| 过渡动画 | 左侧滑入，0.3秒 ease-out |

### 2.4 时间轴动画控制
| 需求项 | 详细描述 |
|--------|----------|
| 控制按钮 | 圆形，直径36px，颜色#FFD700 |
| 图标状态 | 播放时∥，暂停时▶ |
| 动画节奏 | 每0.5秒前进一个年代，循环播放 |
| 滑块同步 | 动画过程中滑块同步移动 |

### 2.5 星座路径轨迹
| 需求项 | 详细描述 |
|--------|----------|
| 轨迹样式 | 淡蓝色圆环（RingGeometry），颜色#87CEEB，opacity 0.2，wireframe |
| 轨迹半径 | 星体在三个年代中最大位移距离×1.2 |
| 更新时机 | 年代切换时自动更新 |

## 3. 界面设计规范

### 3.1 视觉主题
- **设计风格**：深空沉浸式主题
- **主色调**：深蓝 #0A0A2A（背景）
- **强调色**：金色 #FFD700（交互元素、高亮）
- **辅助色**：淡蓝 #87CEEB（轨迹、辅助信息）
- **文字色**：白色 #FFFFFF

### 3.2 场景背景
- 深空渐变色：从 #0A0A2A 到 #000011

### 3.3 星体渲染
- **材质**：Points + Sprite发光效果
- **颜色映射**：根据星等从白#FFFFFF渐变到蓝#1A237E
- **尺寸范围**：1~3像素
- **发光效果**：Sprite材质 + 圆形渐变纹理

### 3.4 UI组件样式

#### 操作提示浮层（左下角常驻）
- 文字：鼠标拖拽旋转·滚轮缩放·点击星体
- 字体：14px，颜色#AAA
- 背景：rgba(0,0,0,0.3)，圆角6px，内边距8px

#### 右侧控制面板
- 毛玻璃效果：背景rgba(255,215,0,0.1)
- backdrop-filter: blur(10px)
- 圆角12px，内边距12px

#### 底部年代滑块
- 宽度：300px
- 轨道色：#333
- 滑块头直径：20px

### 3.5 响应式适配
| 断点 | 适配规则 |
|------|----------|
| ≥768px（桌面） | 星体标签显示，信息面板左侧弹出 |
| <768px（移动） | 星体标签隐藏，信息面板底部弹出（宽度100%，高度自适应） |

### 3.6 交互动画
- 所有交互过渡时间：0.3秒，ease曲线
- 点击反馈：平滑过渡
- 年代切换：1秒插值动画

---

## 4. 性能约束

| 约束项 | 指标 |
|--------|------|
| 目标帧率 | ≥30 FPS |
| 每帧插值计算 | ≤2ms |
| 状态更新 | 使用requestAnimationFrame批次处理 |
| 重渲染优化 | 避免不必要的组件重渲染 |

---

## 5. 非功能需求

- **兼容性**：现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
- **可访问性**：基础键盘操作支持
- **可维护性**：模块化代码结构，清晰的数据流向

---

## 6. 数据模型

### 6.1 星体数据 (StarData)
- id: string
- name: string
- constellation: string
- magnitude: number (星等)
- position: { x, y, z } (三维坐标)
- positionsByEra: [{ x, y, z }] (三个年代的坐标数组)

### 6.2 星座数据 (Constellation)
- id: string
- name: string
- starIds: string[]
- connections: [[starIdA, starIdB]]

### 6.3 时间轴状态 (TimelineState)
- currentEra: number (0-2)
- isPlaying: boolean
- selectedStarId: string | null

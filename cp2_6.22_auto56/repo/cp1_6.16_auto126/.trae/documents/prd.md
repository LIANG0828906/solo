## 1. 产品概述

时光邮局与手写字体动画生成器——一款让用户在线创作复古手写风格电子信笺，并以逐字书写动画预览和导出为GIF/视频分享的Web应用。面向追求文艺表达和情感传递的用户群体。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 创建、编辑、预览、导出手写信笺 |

### 2.2 功能模块
1. **信笺编辑页**：信纸编辑区、字体选择、颜色板、行距调节、实时手写体预览
2. **动画预览页**：逐字书写动画播放/暂停、速度调节、钢笔图标沿轨迹移动
3. **控制面板**：背景纸样切换、日期戳、签名样式、导出GIF/视频

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 信笺编辑页 | 信笺编辑区 | 600×800px固定区域，仿古牛皮纸纹理背景，撕纸边缘，横线格，支持中英文输入实时渲染手写体 |
| 信笺编辑页 | 字体与样式选择 | 至少两种手写体（潇洒行书、优雅斜体），字体大小16-32px可调，复古色板（墨黑、深棕、锈红、靛蓝） |
| 动画预览页 | 手写动画生成器 | 逐字拆分路径，贝塞尔曲线模拟书写，墨迹晕染扩散效果，钢笔图标沿轨迹移动 |
| 动画预览页 | 播放控制 | 播放/暂停/继续，速度0.5x/1x/2x调节，总时长自适应字数 |
| 控制面板 | 背景纸样 | 三种选择：牛皮纸、水纹纸、花草纸，切换时平滑过渡1秒 |
| 控制面板 | 日期戳与签名 | 右上角日期戳（YYYY年MM月DD日），底部手写签名（输入或预设库） |
| 控制面板 | 导出功能 | GIF导出（gif.js，12fps，循环），视频导出（canvas.captureStream，webm格式），进度条+剩余时间 |

## 3. 核心流程

用户打开应用 → 在信笺编辑区输入文字 → 选择字体/颜色/大小 → 选择背景纸样 → 添加日期戳/签名 → 点击播放预览手写动画 → 调整动画速度 → 导出为GIF或视频 → 下载分享

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暖灰#E6DCC8（页面背景）、深棕#4A3728（标题文字）
- 辅助色：金色#C9A96E（木质纹理、钢笔图标）、浅米色#F5E6CA（信纸）
- 按钮风格：圆角8px，悬停上浮阴影+背景色渐变过渡0.2s
- 字体：Georgia（标题）、手写体（信笺内容）、衬线体（日期戳）
- 布局风格：居中单页布局，顶部仿木质标题栏，信纸居中，底部版权脚注

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 信笺编辑页 | 标题栏 | 高120px仿木质纹理，Georgia 36px深棕标题居中 |
| 信笺编辑页 | 信笺编辑区 | 600×800px，撕纸边缘，横线格，手写体实时预览 |
| 信笺编辑页 | 字体选择器 | 下拉菜单，潇洒行书/优雅斜体 |
| 信笺编辑页 | 颜色板 | 四色复古色板圆点选择 |
| 信笺编辑页 | 字体大小 | 滑块16-32px |
| 动画预览页 | 播放控制栏 | 播放/暂停按钮，速度切换0.5x/1x/2x |
| 动画预览页 | 钢笔图标 | 金色矢量钢笔头沿书写轨迹移动 |
| 控制面板 | 背景切换 | 三种纸样缩略图切换 |
| 控制面板 | 日期戳 | 右上角小号衬线体日期 |
| 控制面板 | 签名输入 | 文本输入+预设签名库 |
| 控制面板 | 导出按钮 | GIF/视频双按钮，进度条显示 |

### 4.3 响应式适配
- 桌面端：信纸居中显示，工具栏侧边展开
- 平板端（768px以下）：信纸缩小为80%，字体自适应
- 手机端（480px以下）：信纸宽度95%，工具栏折叠为底部抽屉菜单

### 4.4 性能要求
- 动画播放帧率稳定30fps以上
- GIF导出单帧生成不超过50ms
- 视频导出内存占用不超过200MB

## 1. 产品概述

本产品是一款面向在线教育者的分层式课程内容管理应用，旨在解决教师手动编排多章节课程、评估学生作业以及跟踪学习进度时操作繁琐、反馈滞后的问题。通过直观的界面设计和流畅的交互体验，帮助教师高效组织课程内容、快速批改作业、实时掌握学生学习动态。

### 产品核心价值
- **高效课程编排**：可视化章节树结构，支持无限嵌套、拖拽排序、内联编辑
- **作业全流程管理**：发布-提交-批改-反馈一站式作业处理
- **实时进度追踪**：多维度数据展示，及时发现学习瓶颈
- **流畅交互体验**：精心设计的动效和反馈机制，提升用户操作愉悦感

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 教师 | 课程创建者和管理者 | 创建/编辑课程章节、发布作业、批改评分、查看全班进度 |
| 学生 | 课程学习者 | 查看课程内容、提交作业、查看批改结果、追踪个人进度 |

### 2.2 功能模块

1. **课程管理模块**：左侧章节树面板，支持折叠/展开、拖拽排序、内联编辑
2. **作业管理模块**：作业发布、提交列表、状态标识、时间刷新
3. **批改评分模块**：作业详情查看、评语输入、分数评定
4. **进度追踪模块**：章节完成度环形图、作业时间线展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 教师主界面 | 课程面板 | 左侧320px宽课程树，章节可折叠、可拖拽、可编辑 |
| 教师主界面 | 作业列表 | 右侧主内容区展示当前章节下的作业列表，区分已批改/待批改状态 |
| 批改详情页 | 批改视图 | 滑入动画展示学生提交内容，下方评语和评分输入区 |
| 学生主页 | 进度仪表板 | 圆形进度环展示各章节完成度，时间线展示最近批改作业 |

## 4. 用户界面设计

### 4.1 设计风格

- **色彩方案**：明亮柔和风格
  - 主色：#5D9CEC（清新蓝）
  - 辅色：#A29BFE（优雅紫）
  - 背景：#FFFFFF（纯净白）
  - 文字：#2C3E50（深灰蓝）
  - 成功：#2ECC71（翠绿）
  - 待处理：#F39C12（暖橙）
  - 面板背景：#F5F7FA（浅灰蓝）

- **排版层次**：
  - 标题：18px 600权重
  - 正文：14px 400权重
  - 辅助文字：12px 400权重
  - 字体选择：采用现代无衬线字体，确保清晰易读

- **组件风格**：
  - 卡片圆角：10px，悬浮阴影2px提升层次感
  - 按钮圆角：6px，悬浮时颜色加深10%，点击波纹扩散效果
  - 输入框：底部边框聚焦变主色，0.2s过渡动画
  - 动效标准：统一0.3s ease过渡时间

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 教师主界面 | 课程面板 | 320px宽侧边栏，#F5F7FA背景，圆角8px；章节卡片带旋转箭头动画，拖拽时透明度0.6，目标位置虚线占位符 |
| 教师主界面 | 作业列表 | 卡片式布局，状态色标（绿色/橙色圆点），右下角相对时间显示（定时刷新） |
| 批改详情页 | 批改视图 | 学生内容从顶部滑入（0.3s cubic-bezier），500字限制评语框，1-100打分器带加减按钮 |
| 学生主页 | 进度仪表板 | 圆形进度环（渐变#3498DB到#9B59B6，1s动画），时间线竖线+圆点节点（间距24px），卡片从左侧滑入（stagger 0.1s延迟） |

### 4.3 响应式设计

- **桌面端（≥768px）**：左侧320px导航面板 + 弹性主内容区
- **移动端（<768px）**：顶部固定汉堡菜单（白色背景），主内容区全宽显示，点击汉堡展开/收起导航
- **触摸优化**：按钮最小触控区域44x44px，拖拽操作支持触摸事件

### 4.4 动效与交互细节

| 交互类型 | 动效参数 | 触发场景 |
|----------|----------|----------|
| 折叠箭头旋转 | 0.3s ease-in-out | 点击章节展开/收起 |
| 内容滑入 | 0.3s cubic-bezier(0.4, 0, 0.2, 1) | 打开批改详情页 |
| 进度环动画 | 1s ease-out | 加载学生进度页 |
| 时间线卡片 | stagger 0.1s each | 时间线卡片依次入场 |
| 波纹扩散 | 0.4s linear（伪元素实现） | 按钮点击 |
| 加载状态 | spin动画，20px直径，0.8s/圈 | 提交、评分、拖拽操作 |
| 成功提示 | 0.5s弹跳绿色勾号 | 操作成功反馈 |

## 1. 产品概述

智能书签管理与内容归档工具，帮助用户解决浏览器收藏夹内容杂乱无章、难以检索利用的问题。通过自动解析、智能分类和全文搜索，将收藏夹转化为高效的数字知识库。

- **核心价值**：将浏览器导出的HTML书签自动解析、智能分类，提供卡片化浏览和全文搜索能力
- **目标用户**：有大量网页收藏需求，需要高效管理和检索书签内容的用户
- **解决问题**：收藏内容杂乱、搜索低效、缺乏智能分类、难以长期利用

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，纯前端应用 | 导入书签、管理分类规则、浏览搜索书签、删除恢复书签 |

### 2.2 功能模块

1. **书签导入与解析**：支持上传HTML书签文件，实时解析并提取标题、URL等信息
2. **智能分类引擎**：基于预设规则自动将书签归类到不同目录，支持自定义规则
3. **卡片网格浏览**：以响应式卡片网格展示所有书签，支持毛玻璃效果和悬停动画
4. **全文搜索与筛选**：实时模糊匹配标题和URL，支持按分类筛选，匹配文字高亮
5. **详情面板**：点击卡片滑出详情面板，显示完整信息并支持打开链接
6. **撤销删除**：删除卡片时提供撤销功能，防止误操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主应用页 | 顶部导航栏 | 显示应用名称、导入按钮、设置齿轮图标 |
| 主应用页 | 搜索筛选区 | 搜索输入框、分类下拉筛选器 |
| 主应用页 | 卡片网格区 | 响应式书签卡片网格，支持点击查看详情 |
| 主应用页 | 详情面板 | 右侧滑出面板，展示书签完整信息 |
| 主应用页 | 设置面板 | 管理分类规则（新增、编辑、删除） |
| 主应用页 | 撤销提示 | 删除书签后底部显示撤销按钮 |

## 3. 核心流程

### 3.1 主要用户流程

用户打开应用 → 点击导入按钮 → 选择HTML书签文件 → 系统自动解析并分类 → 卡片网格展示 → 用户搜索/筛选 → 点击卡片查看详情 → 可删除书签（支持撤销）

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#f8f9fa（背景）、#e9ecef（次级背景）
- **强调色**：#4dabf7（蓝色系）
- **标签色**：背景#e7f5ff，文字#1971c2
- **字体**：使用优雅的无衬线字体，主标题字号18px，正文字号14px，辅助文字12px
- **布局风格**：顶部导航 + 卡片网格 + 右侧滑出面板
- **动效风格**：卡片悬停上浮（translateY(-4px) 0.2s ease-out）、删除滑出（0.3s ease）、面板滑入（0.25s ease-out）
- **视觉效果**：毛玻璃效果（backdrop-filter: blur）、柔和阴影、圆角设计

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 主应用页 | 顶部导航栏 | 高度56px，白色背景，底部淡灰色阴影，左侧应用名称，右侧导入按钮和设置图标 |
| 主应用页 | 搜索筛选区 | 搜索框（聚焦蓝色发光描边）、分类下拉选择器 |
| 主应用页 | 卡片网格区 | CSS Grid布局，minmax(280px, 1fr)，间距16px，卡片圆角12px，毛玻璃背景 |
| 主应用页 | 详情面板 | 宽度300px，白色背景，左侧1px分割线，滑入动画 |
| 主应用页 | 分类标签 | pill样式小圆角，背景#e7f5ff，文字#1971c2 |

### 4.3 响应式设计

- **桌面端（≥768px）**：三栏布局，详情面板从右侧滑出
- **移动端（<768px）**：详情面板转为全屏蒙层模式，卡片Grid变为单列
- **触摸优化**：增大点击区域，支持触摸滑动关闭面板

## 5. 性能约束

- 解析2000个书签的HTML文件，整个解析与分类过程在1秒内完成
- 界面无卡顿，搜索筛选实时响应
- 使用高效的DOM操作和状态更新策略

## 1. 产品概述
智能食谱规划器是一款帮助用户根据冰箱现有食材和饮食偏好，自动推荐合理食谱并生成一周购物清单的移动端Web应用。
- 解决用户面对冰箱食材不知做什么菜、购物容易遗漏的问题
- 面向注重饮食健康和效率的都市用户，提升日常烹饪规划体验

## 2. 核心功能

### 2.1 功能模块
1. **食材管理页**：食材模糊搜索与自动补全、数量设置（克/个/杯）、添加/删除/修改、列表平滑收缩展开动画
2. **食谱推荐页**：基于食材匹配度与营养目标打分、按分数从高到低展示前5推荐、点击展开详细步骤/热量/补购食材
3. **购物清单页**：汇总缺失食材、按类别分栏（蔬菜/肉类/调味品）、勾选已购买、勾选淡出动画

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 食材管理页 | 搜索输入框 | 模糊搜索食材，自动补全下拉列表，聚焦时外发光动画 |
| 食材管理页 | 食材卡片列表 | 展示已选食材及数量，支持修改数量和删除，删除/修改时列表平滑收缩或展开 |
| 食材推荐页 | 饮食偏好选择 | 支持选择低卡、高蛋白、低碳水等偏好标签 |
| 食谱推荐页 | 推荐列表 | 按匹配分数从高到低展示前5个食谱卡片 |
| 食谱推荐页 | 食谱详情 | 点击卡片展开详细步骤、热量信息、所需补购食材 |
| 购物清单页 | 分类清单 | 按蔬菜、肉类、调味品分栏展示缺失食材 |
| 购物清单页 | 勾选功能 | 每项可勾选已购买，勾选时淡出动画效果 |

## 3. 核心流程
用户打开应用 → 在食材管理页输入冰箱现有食材并设置数量 → 选择饮食偏好 → 切换到食谱推荐页查看推荐结果 → 点击食谱查看详情和补购食材 → 切换到购物清单页查看所有缺失食材汇总 → 逐一勾选已购买食材

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暖橙色（#F97316），辅色调：米白色（#FFF8F0）
- 按钮风格：柔和圆角（12px），轻微投影
- 字体：正文16px，标题20px，使用系统字体栈
- 布局风格：卡片式布局，CSS Grid栅格
- 图标风格：简洁线条图标（Lucide Icons）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 食材管理页 | 搜索输入框 | 米白背景、圆角输入框、聚焦暖橙外发光、自动补全下拉 |
| 食材管理页 | 食材卡片 | 暖橙边框圆角卡片、轻微投影、数量输入、删除按钮、添加/删除动画 |
| 食谱推荐页 | 偏好标签 | 暖橙色胶囊标签、选中态加深 |
| 食谱推荐页 | 推荐卡片 | 米白圆角卡片、分数徽章、展开/收起动画 |
| 食谱推荐页 | 详情区域 | 步骤列表、热量标签、补购食材清单 |
| 购物清单页 | 分类标题 | 暖橙色粗体标题 |
| 购物清单页 | 勾选项 | 复选框+食材名+数量、勾选淡出动画 |
| 全局 | 底部Tab栏 | 三个Tab图标+文字、选中态暖橙色、未选中灰色 |
| 全局 | 页面切换 | 内容区域水平滑动过渡效果 |

### 4.3 响应式
- 移动端优先设计，适配375px-428px宽度手机屏幕
- CSS Grid栅格布局，单列展示
- 触控优化：按钮最小点击区域44px

### 4.4 3D场景指引
- 不适用

## 1. 产品概述

基于体素渲染的乐高积木虚拟搭建空间，让用户像搭积木一样在三维网格中自由放置和移除彩色方块，自动生成光影效果和碰撞检测，支持多视角观察作品全貌。

- 面向创意爱好者、设计师和儿童教育场景的3D积木搭建工具
- 核心价值：零门槛的三维创作体验，即开即用，浏览器内完成全部交互

## 2. 核心功能

### 2.1 功能模块

1. **主场景页面**：三维网格坐标系、方块放置与移除、光影效果、视角切换
2. **工具栏**：色板选择、撤销/重做、清空场景、视角切换菜单、方块统计

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景 | 三维网格坐标系 | 透明三维网格，每格边长32单位，暖色(#3a3a3a)网格线，深灰(#1a1a1a)背景 |
| 主场景 | 方块放置 | 点击网格面放置彩色方块，8种基础颜色，0.1秒放大回弹动画(0.8→1.0) |

## 3. 核心流程

### 3.1 方块放置流程
用户选择颜色 → 鼠标悬停网格面高亮 → 点击网格面 → 碰撞检测(是否重叠/超出边界) → 通过则创建方块(0.1秒放大回弹动画) → 失败则红色弱提示

### 3.2 方块移除流程
按R键进入移除模式 → 点击方块 → 方块缩小淡出(0.15秒) → 删除完成 / 或按住左键拖拽框选 → 松开后选区内方块高亮闪烁3次 → 批量删除

### 3.3 视角切换流程
用户点击视角下拉菜单 → 选择视角类型 → 相机平滑过渡到目标位置

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰色(#1a1a1a)背景 + 暖色(#3a3a3a)网格线
- 方块描边：0.5px黑色边缘线增强立体感
- 工具栏：半透明浮动(rgba 0,0,0,0.6)，毛玻璃效果
- 字体：系统无衬线字体，小巧精致
- 交互反馈：网格面高亮、方块放置闪烁、操作失败红色提示

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主场景 | 三维网格 | 深灰背景、暖色网格线、透明坐标系 |
| 主场景 | 方块 | 8色实色立方体、0.5px黑色描边、浅色边缘线 |
| 主场景 | 鼠标交互 | 网格面悬停高亮、放置闪烁、失败红色提示 |
| 主场景 | 框选 | 半透明矩形选区、高亮闪烁边框 |
| 工具栏 | 容器 | 左侧浮动、半透明黑底、毛玻璃、圆角 |
| 工具栏 | 色板 | 8个圆形色块、当前选中高亮 |
| 工具栏 | 按钮 | 撤销/重做/清空图标按钮 |
| 工具栏 | 下拉菜单 | 视角切换6选项 |
| 工具栏 | 统计 | 底部方块数量文字 |

### 4.3 响应式设计
- 桌面优先，3D视口占满浏览器窗口
- 工具栏绝对定位悬浮，不影响3D渲染区域
- 最小分辨率1024x768

### 4.4 3D场景指引
- 环境：深灰色无环境纹理，简洁空间感
- 光照：环境光 + 方向光，增强方块立体感
- 相机：自由视角默认(120, 100, 120)看向原点，阻尼系数0.95平滑跟随
- 交互：鼠标拖拽旋转、滚轮缩放、点击放置/移除
- 动画：放置放大回弹(0.1秒)、移除缩小淡出(0.15秒)、框选闪烁3次
- 性能：最多2000方块保持30FPS+，操作响应<50ms

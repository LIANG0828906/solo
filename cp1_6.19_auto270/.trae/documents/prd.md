## 1. 产品概述
「书信留痕」是一款面向手写信件爱好者的数字信件美化工具，解决拍摄手写信件时背景干扰、文字模糊及缺乏优雅数字展示效果的问题。用户可上传信件照片，自动消除背景、增强文字、套用精美信封装帧模板，最终导出高品质数字信件图像。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 上传、编辑、导出信件 |

### 2.2 功能模块
1. **编辑主页**：信件照片上传与背景消除、文字增强与自动校正、画布操作
2. **模板面板**：数字信封装帧与主题模板选择、签名编辑
3. **导出功能**：PNG/JPG格式导出、加载动画

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑主页 | 照片上传区 | 支持JPG/PNG上传（最大12MB），拖拽或点击上传 |
| 编辑主页 | 画布编辑区 | 800px宽白色画布，实时显示信件与滤镜效果 |
| 编辑主页 | 操作按钮栏 | 上传、旋转左、旋转右、增强文字、重置、撤销、重做 |
| 编辑主页 | 调节滑块 | 亮度/对比度（-50~+50）、微调角度（-15°~+15°） |
| 编辑主页 | 手动裁剪 | 识别失败时矩形框选信件区域（2px虚线#E67E22） |
| 模板面板 | 模板卡片列表 | 4种信封风格预览，选中高亮 |
| 模板面板 | 签名编辑 | 可编辑签名文字，字体/字号选择 |
| 编辑主页 | 导出弹窗 | PNG透明/300dpi 或 JPG白底/92%质量，1200px宽度 |

## 3. 核心流程

用户打开应用 → 上传信件照片 → 系统自动识别纸张区域并消除背景 → 若识别失败，手动框选裁剪 → 调节亮度/对比度 → 点击"增强文字"自动优化 → 旋转/微调角度 → 选择信封装帧模板 → 编辑签名 → 导出下载

```mermaid
    "F" --> "G[增强文字/旋转/微调]"
    "G" --> "H[选择信封模板]"
    "H" --> "I[编辑签名]"
    "I" --> "J[导出PNG或JPG]"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖色调，浅奶油色背景（#FFF8F0）
- **强调色**：橙色系（#E67E22），按钮及选中态
- **辅助色**：深橙（#D35400）按钮悬浮态
- **按钮风格**：圆角6px胶囊形状，白色文字，悬浮加深，0.2s缩放回弹动画
- **字体**：标题使用优雅衬线体，正文使用清晰无衬线体
- **布局**：页面居中，最大宽度1280px，左70%编辑区+右30%模板面板
- **阴影**：信件边角柔和阴影（5px偏移，15px模糊，rgba(0,0,0,0.15)）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 编辑主页 | 画布编辑区 | 800px白色画布，信件居中，阴影效果，CSS滤镜实时渲染 |
| 编辑主页 | 操作按钮栏 | 胶囊按钮（#E67E22背景），7个功能按钮横排 |
| 编辑主页 | 调节滑块 | 自定义range输入，标签+数值显示 |
| 编辑主页 | 手动裁剪覆盖层 | 2px虚线矩形框（#E67E22），拖拽调整 |
| 模板面板 | 模板卡片 | 120×160px卡片，缩略预览，选中橙色边框 |
| 模板面板 | 签名编辑区 | 文字输入框，字体下拉，字号滑块 |
| 编辑主页 | 导出遮罩 | 半透明遮罩+环形进度条，颜色匹配模板主色 |

### 4.3 响应式
- 桌面优先设计，1280px最大宽度居中
- 小屏幕时模板面板折叠到底部
- 触控优化：滑块和按钮适配触控操作

### 4.4 动效
- 按钮点击：scale 1→0.95→1，0.2s回弹
- 模板卡片选中：透明度70%→100%，边框出现
- 导出加载：旋转环形进度条
- 画布操作：实时滤镜渲染，响应<100ms

## 1. 产品概述

个人知识笔记管理Web应用，解决笔记分散在多个平台、难以分类检索和形成知识关联的问题。目标用户为知识工作者、学生和研究人员，通过Markdown编辑、标签体系和实时搜索，提供一站式知识管理体验。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册 | 新建、编辑、删除、搜索、过滤笔记 |

### 2.2 功能模块

1. **主页面**：搜索栏、标签过滤器、笔记卡片网格、浮动新建按钮
2. **笔记详情页**：笔记完整内容展示、编辑和删除操作
3. **笔记编辑器**：Markdown编辑、浮动工具栏、标签输入

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 搜索栏 | 输入时300ms防抖触发实时搜索，匹配标题和标签，结果以淡入动画逐条显示 |
| 主页面 | 标签过滤器 | 搜索框下方可点击标签列表，选中标签高亮对应卡片，未选中以0.3秒淡出隐藏 |
| 主页面 | 笔记卡片网格 | 卡片左上角标签徽章（颜色按标签哈希生成）、右下角相对时间、悬停上浮阴影加深 |
| 主页面 | 浮动新建按钮 | 右下角FAB按钮，点击打开笔记编辑器模态窗 |
| 笔记编辑器 | 浮动工具栏 | 加粗、斜体、标题、列表按钮，悬停提示文字+图标150%放大 |
| 笔记编辑器 | Markdown编辑区 | 支持Markdown语法编辑 |
| 笔记编辑器 | 标签输入 | 最多5个标签，输入时自动补全已存在标签 |
| 笔记详情页 | 内容展示 | Markdown渲染后的完整内容 |
| 笔记详情页 | 操作按钮 | 顶部编辑和删除按钮，编辑时原内容可编辑并保存 |

## 3. 核心流程

用户打开应用后，主页展示所有笔记卡片网格。用户可通过搜索框搜索或通过标签过滤器筛选笔记。点击卡片进入详情页查看完整内容。点击右下角浮动按钮打开编辑器新建笔记，编辑器顶部有Markdown工具栏，支持标签输入和自动补全，提交后卡片从底部飞入列表。详情页支持编辑和删除操作。

## 4. 用户界面设计

### 4.1 设计风格

- **风格**：极简毛玻璃（Glassmorphism）
- **主色**：白色半透明 `rgba(255,255,255,0.85)` 作为卡片背景
- **背景**：模糊渐变（从 `#f0f4ff` 到 `#e2e8f0`）
- **阴影**：柔和阴影 `box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1)`
- **圆角**：16px
- **字体**：系统字体 `-apple-system, BlinkMacSystemFont`，标题加粗，正文常规
- **标签颜色**：按标签文字哈希值生成HSL颜色
- **动画**：所有交互 `transition: all 0.25s ease`

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 搜索栏 | 毛玻璃背景、圆角输入框、搜索图标、300ms防抖 |
| 主页面 | 标签过滤器 | 横向可点击标签列表、选中态高亮、胶囊样式 |
| 主页面 | 笔记卡片网格 | 响应式网格、卡片悬停上浮+阴影加深、标签徽章、相对时间 |
| 主页面 | 浮动按钮 | 右下角圆形按钮、毛玻璃效果、悬停放大 |
| 笔记编辑器 | 模态窗 | 居中弹出、毛玻璃背景遮罩、白色半透明内容区 |
| 笔记编辑器 | 浮动工具栏 | 编辑区顶部固定、图标按钮、悬停150%放大+提示文字 |
| 笔记详情页 | 内容区 | Markdown渲染、标签徽章、创建时间 |

### 4.3 响应式设计

- **桌面端**：留白边距，卡片网格每行3-4张
- **平板端**：每行2张卡片
- **移动端**：单列全宽布局
- 所有交互均有 `transition: all 0.25s ease` 动画

### 4.4 性能要求

- 首页50张以内笔记无卡顿
- 搜索响应在输入后300ms内显示结果
- 页面FPS稳定在30以上

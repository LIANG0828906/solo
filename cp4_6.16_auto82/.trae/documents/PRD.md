## 1. 产品概述

CritiqueCircle 是一个面向大学文学社的匿名作品互评与评选平台，让社员围绕每周主题提交原创短篇作品、互相匿名点评并自动评选每周最佳。核心价值在于消除作者身份偏见，促进真实、公正的文学交流。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 社员 | 无需注册，直接使用 | 浏览主题、提交作品、盲评打分 |
| 管理员 | 无需注册，内置标识 | 创建主题、设置起止日期 |

### 2.2 功能模块

1. **主题列表页**：展示所有历史主题卡片，支持创建新主题，按时间倒序排列
2. **作品提交与盲评页**：左栏展示匿名作品摘要并可盲评打分，右栏提交新作品
3. **周报统计页**：下拉选择周次，展示评选结果、排名列表和统计概览

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主题列表页 | 主题卡片列表 | 瀑布流布局展示主题名、起止日期、作品数，点击进入详情 |
| 主题列表页 | 新建主题弹窗 | 管理员输入标题、描述、起止日期创建新主题 |
| 作品提交与盲评页 | 匿名作品摘要列表（左栏） | 显示作品编号和摘要，点击进入盲评模式 |
| 作品提交与盲评页 | 盲评模式 | 显示作品编号和正文，1-5星评分+评语提交，每人每作品限评一次 |
| 作品提交与盲评页 | 新作品提交区（右栏） | 输入标题和正文（限2000字），提交后显示为匿名编号 |
| 周报统计页 | 周次选择器 | 下拉选择要查看的主题周 |
| 周报统计页 | 主题信息 | 显示选中周的主题名称和起止日期 |
| 周报统计页 | 排名列表 | 按均分排序显示作品编号、平均分、评语数，顶部高亮"本周最佳" |
| 周报统计页 | 统计卡片 | 显示总作品数、总评分数、参与人数（去重作者数） |

## 3. 核心流程

用户浏览主题列表 → 选择主题进入详情 → 在开放期内提交匿名作品 → 浏览其他匿名作品 → 对作品进行1-5星盲评打分 → 主题截止后系统自动计算均分排名 → 在周报页查看"本周精选"和统计数据

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：羊皮纸色 #F5E6C8
- 文字色：深褐色 #4A3B32
- 强调色：墨绿色 #3A6B47
- 按钮风格：圆角半透明按钮，墨绿色主按钮
- 字体：正文使用衬线字体 Georgia，行高1.8，首行缩进2字符；标题使用无衬线字体
- 布局风格：卡片式布局，瀑布流主题列表，左右分栏详情页
- 图标风格：简洁线条图标（lucide-react）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主题列表页 | 主题卡片 | 阴影+悬停上浮动画(translateY -2px+阴影加深)，瀑布流4列布局 |
| 作品提交与盲评页 | 盲评作品卡片 | 居中对齐，衬线字体排版，首行缩进 |
| 作品提交与盲评页 | 星级评分组件 | 半透明圆角按钮，点击时星星旋转填充动画(0.2s ease) |
| 周报统计页 | 金色奖杯动画 | CSS keyframes左右摇摆 |
| 周报统计页 | 获奖作品卡片 | 渐变边框，高亮"本周最佳"标签 |

### 4.3 响应式设计

- 桌面优先，768px断点适配移动端
- 主题卡片列数：桌面4列 → 移动1列
- 作品提交与盲评页：桌面左右分栏 → 移动端上下堆叠
- 触摸优化：按钮最小44px触摸区域

### 4.4 动画设计

- 主题卡片悬停：translateY(-2px) + box-shadow加深，0.2s ease
- 星星评分点击：rotate(360deg) 旋转填充动画，0.2s ease
- 金色奖杯：keyframes左右摇摆动画
- 页面过渡：淡入效果

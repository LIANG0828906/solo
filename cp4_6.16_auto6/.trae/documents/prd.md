## 1. 产品概述

SubTracker 是一款个人订阅服务管理工具，帮助用户在多个平台间追踪订阅费用和到期日期，自动统计每月总支出，避免遗忘扣款和重复订阅。

- 目标用户：拥有多个订阅服务（视频网站、音乐App、云存储等）的个人用户
- 核心价值：一站式管理所有订阅，自动汇总支出，智能提醒到期

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册，本地使用 | 管理个人订阅数据、查看统计、导入导出 |

### 2.2 功能模块
1. **主页面**：三栏布局，左侧订阅列表、中间续费日历、右侧统计面板
2. **订阅管理**：添加、编辑、删除订阅条目
3. **续费日历**：月历视图展示扣款日期，点击日期展开详情
4. **月度统计**：支出汇总、分类饼图、即将到期提醒
5. **到期通知**：定时检查并弹出非阻塞通知
6. **数据导入导出**：JSON格式的导入导出功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 订阅列表 | 展示所有订阅卡片，支持添加/编辑/删除，按类别筛选，搜索过滤 |
| 主页面 | 订阅表单 | 弹窗形式，包含服务名、价格、计费周期、下次扣款日期、类别、备注字段 |
| 主页面 | 续费日历 | 月历视图标记扣款日，同日多订阅叠加，点击日期弹出当日订阅清单 |
| 主页面 | 统计面板 | 当月总支出、分类饼图、即将到期（7天内）订阅数量和列表 |
| 主页面 | 到期通知 | 页面右上角非阻塞通知，3天内到期订阅提醒，5秒自动消失，带音效 |
| 主页面 | 数据导入导出 | 导出为JSON文件，从JSON导入并合并（重复项覆盖） |

## 3. 核心流程

### 3.1 订阅管理流程
用户点击"添加订阅"→ 填写表单（服务名、价格、周期、日期、类别、备注）→ 保存至IndexedDB → 列表/日历/统计联动更新

### 3.2 到期提醒流程
系统每8小时检查一次 → 发现3天内到期订阅 → 右上角弹出通知 + 音效 → 5秒自动消失或手动标记已读

### 3.3 数据导入导出流程
导出：点击导出 → 从IndexedDB读取所有数据 → 生成JSON文件下载
导入：选择JSON文件 → 解析数据 → 与现有数据合并（重复项覆盖）→ 保存至IndexedDB → 界面刷新

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝(#1A1A2E)背景 + 灰白(#F0F0F5)文字和卡片
- 强调色：橙红(#E94560)用于紧急/警示/删除等操作
- 按钮风格：圆角按钮，带hover上浮2px和阴影加深效果
- 字体：Inter（Google Fonts），正文14-16px，标题20-24px
- 布局风格：卡片式设计，三栏布局（>768px），单栏滚动（<768px）
- 图标风格：使用lucide-react图标库，线条风格

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 订阅列表 | 深蓝背景，灰白卡片，hover上浮+阴影加深，类别标签带颜色 |
| 主页面 | 订阅表单 | 弹窗模态框，输入框聚焦时底部边框变色（橙红），类别下拉选择 |
| 主页面 | 续费日历 | 月历网格，扣款日标注橙红圆点，月份切换淡入淡出300ms，点击日期弹窗 |
| 主页面 | 统计面板 | 总支出数字突出显示，饼图（娱乐-紫/工具-蓝/存储-绿/其他-灰），扇区hover缩放 |
| 主页面 | 到期通知 | 右上角浮动通知卡片，5秒自动消失，轻微音效 |
| 主页面 | 数据导入导出 | 底部工具栏，导出/导入按钮 |

### 4.3 响应式设计
- 桌面端（≥768px）：三栏布局，左侧订阅列表(30%)、中间日历(40%)、右侧统计(30%)
- 移动端（<768px）：单栏垂直滚动，订阅列表 → 日历 → 统计面板依次排列
- 骨架屏：每个功能模块加载时展示骨架屏动画

### 4.4 动效规范
- 卡片hover：translateY(-2px) + box-shadow加深
- 日历月份切换：opacity过渡300ms淡入淡出
- 饼图扇区hover：scale(1.05)缩放提示
- 表单输入框聚焦：底部border从灰变为橙红(#E94560)
- 通知弹出：从右侧滑入，5秒后淡出
- 骨架屏：脉冲动画(pulse)

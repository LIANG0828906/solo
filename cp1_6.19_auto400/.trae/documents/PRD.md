# 在线歌曲创作协作与和弦进行推荐应用 - 产品需求文档

## 1. 产品概述

### 1.1 项目背景
面向音乐爱好者的浏览器端在线协作创作工具，解决音乐创作过程中多人实时协作、和弦进行灵感获取、以及快速试听的痛点。

### 1.2 目标用户
- 音乐爱好者
- 业余词曲创作者
- 小型音乐创作团队

### 1.3 核心价值
- **实时协作**：支持2-4人同时编辑同一首歌曲，所见即所得
- **智能推荐**：基于音乐理论的和弦进行推荐，降低创作门槛
- **即时试听**：MIDI合成播放，快速验证创作想法
- **零配置**：浏览器即可使用，无需安装任何软件

## 2. 功能需求

### 2.1 歌曲项目管理模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 歌曲列表展示 | 首页展示用户参与的所有歌曲，显示歌曲名、最后编辑时间、成员数 | P0 |
| 新建歌曲 | 输入歌曲名和拍号（4/4, 3/4），创建后自动加入 | P0 |
| 歌曲编辑入口 | 点击歌曲卡片进入编辑页面 | P0 |
| 并发限制 | 最多同时操作10首歌曲，基于服务器内存存储 | P1 |
| 卡片悬停效果 | 悬停时卡片上浮4px并显示编辑按钮 | P2 |

### 2.2 和弦编辑与推荐模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 和弦轨道展示 | 右侧展示和弦轨道，每行代表一个小节 | P0 |
| 和弦选择器 | 点击小节空位弹出半透明浮动面板，选择常用和弦（C, Am, F, G等） | P0 |
| 和弦插入 | 在指定位置插入1-4个和弦/小节 | P0 |
| 智能推荐 | 基于调性与已选和弦，实时推荐下一个和弦，虚线框提示 | P0 |
| 推荐算法 | 基于和弦功能进行（如C大调G→C为五级到一级），考虑前后一致性 | P0 |
| 推荐响应 | 推荐计算响应时间 ≤ 200ms | P1 |

### 2.3 实时协作模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| WebSocket实时同步 | 编辑操作（插入和弦、修改歌词）实时广播 | P0 |
| 协作光标显示 | 显示其他用户的编辑位置，带用户名标签 | P0 |
| 光标样式 | 细竖线（宽2px）+ 半透明圆角用户名标签 | P0 |
| 冲突处理 | 最近修改优先原则处理编辑冲突 | P1 |
| 同步延迟 | 编辑操作同步延迟 ≤ 500ms | P1 |
| 人数限制 | 支持2-4人同时编辑 | P0 |

### 2.4 歌词与和弦对齐模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 歌词输入区 | 左侧歌词编辑区，等宽字体 | P0 |
| 上下对齐 | 歌词与和弦轨道按音节上下对齐 | P0 |
| 联动滚动 | 点击歌词行，和弦轨道自动滚动到对应小节 | P0 |
| 富文本支持 | 支持粗体(Ctrl+B)、斜体样式 | P0 |
| 音节分行 | 歌词按音节自动分行 | P1 |

### 2.5 MIDI播放与试听模块

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 播放控制条 | 固定底部，包含播放/暂停、进度条、速度滑块 | P0 |
| 进度显示 | 显示当前小节/总小节 | P0 |
| BPM调节 | 速度滑块 60-200 BPM | P0 |
| MIDI合成 | 琶音形式播放，每个和弦2拍 | P0 |
| 小节高亮 | 播放时当前小节高亮(#2D2D44)，带脉冲动画 | P0 |
| 帧率要求 | 播放时帧率 ≥ 30fps | P1 |

## 3. 界面设计规范

### 3.1 整体风格
- **设计方向**：深色专业音乐创作工具风格，带有科技感与艺术感的结合
- **主背景**：#121212
- **次要背景**：#1E1E2E
- **文字颜色**：#E0E0E0
- **强调色**：#BB86FC（紫色）
- **字体方向**：显示字体使用有个性的衬线或几何字体，正文使用清晰的无衬线字体

### 3.2 组件样式规范

#### 歌曲卡片
- 宽度：320px
- 圆角：16px
- 背景：#1E1E2E
- 边框：#BB86FC 渐变边框
- 悬停：上浮4px，显示编辑按钮
- 布局：卡片网格布局

#### 编辑页面布局
- 左侧歌词区：40% 宽度，背景 #2A2A3E，圆角 12px，内边距 20px
- 右侧和弦区：60% 宽度
- 移动端（<768px）：上下排列

#### 和弦小节
- 尺寸：120px × 40px
- 圆角矩形
- 背景：#3A3A5E
- 文字：白色和弦名称

#### 播放控制条
- 高度：60px
- 固定底部
- 背景：#1E1E2E
- 顶部分隔线：1px #BB86FC

#### 协作光标
- 细竖线：宽2px
- 颜色：随机高饱和度颜色
- 用户名标签：圆角6px，半透明背景

### 3.3 交互效果
- 所有按钮悬停颜色加深
- 点击涟漪效果
- 页面加载过渡动画
- 小节播放脉冲动画
- 卡片悬停上浮动画

---

## 4. 非功能需求

### 4.1 性能指标
- 和弦推荐响应：≤ 200ms
- 编辑同步延迟：≤ 500ms
- MIDI播放帧率：≥ 30fps
- 页面加载时间：≤ 2s

### 4.2 兼容性
- 支持现代浏览器（Chrome, Firefox, Safari, Edge 最新版本）
- 响应式设计，支持移动端

### 4.3 数据存储
- 服务器内存存储（Map数据结构）
- 最多同时10首歌曲
- 无持久化数据库需求

## 5. 技术栈选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | React 18 + TypeScript | 组件化开发 |
| 状态管理 | Zustand | 轻量级状态管理，支持时间旅行 |
| 构建工具 | Vite | 快速开发构建 |
| 后端框架 | Express | RESTful API |
| 实时通信 | WebSocket | 双向实时同步 |
| HTTP客户端 | Axios | API请求 |
| MIDI生成 | midi-writer-js | MIDI数据生成 |
| 音频播放 | Web Audio API | 浏览器端音频合成 |
| 唯一ID | uuid | 资源标识 |

## 1. 产品概述

智能书架与阅读日历是一款帮助用户管理待读书目、制定每日阅读计划并追踪阅读进度的Web应用。通过书架网格管理书籍、阅读计时器记录阅读时长与翻页、日历视图回顾阅读历史，并可根据阅读速度智能推荐下次阅读安排。

- 目标用户：有阅读习惯、希望量化阅读进度和建立阅读规律的个人读者
- 核心价值：将阅读从碎片行为转化为可视化、可追踪的习惯养成系统

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 模拟登录（内存存储） | 管理书架、记录阅读、查看日历 |

### 2.2 功能模块

1. **书架页面**：书籍网格展示、创建新书、进度追踪、状态标签
2. **阅读计时器页面**：计时阅读、翻页记录、自动更新进度
3. **日历页面**：月度阅读日历、每日阅读明细、目标完成状态

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书架页面 | 书籍网格 | 展示所有书籍卡片，每张卡片含封面缩略图（自定义颜色）、书名、作者、阅读进度百分比、状态标签（未读/在读/读完） |
| 书架页面 | 创建新书弹窗 | 输入书名、作者、总页数，调色板选取封面颜色，生成卡片时带右侧滑入动画和阴影悬浮效果 |
| 书架页面 | 智能推荐 | 根据用户历史阅读速度，推荐下次应读的书籍和预计阅读时长 |
| 阅读计时器页面 | 计时面板 | 圆角矩形居中面板，显示当前书籍信息、计时器（每500ms更新）、翻页按钮（每页模拟300字） |
| 阅读计时器页面 | 阅读记录 | 点击结束后自动更新书籍进度条，弹出恭喜提示，进度条平滑过渡动画 |
| 日历页面 | 月度网格 | 展示当月日历，每日含阅读时长饼图和目标完成标识，周末背景浅灰色，当日高亮浅蓝色边框 |
| 日历页面 | 日期详情 | 点击日期弹出阅读明细（读了哪本书、多少页、耗时多少） |
| 日历页面 | 月份切换 | 左右切换月份，淡入过渡效果 |

## 3. 核心流程

用户打开应用后进入书架页面，可浏览已有书籍或创建新书。点击某本书进入阅读计时器，开始计时并翻页阅读，阅读结束后进度自动更新回书架。日历页面汇总每日阅读数据，帮助用户回顾和坚持阅读习惯。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：淡蓝（#B3D4FC）与白色（#FFFFFF），辅助色为深蓝（#4A90D9）
- 按钮：渐变蓝到浅蓝（#4A90D9 → #7CB9F0），圆角8px，悬停放大1.05倍
- 字体：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- 布局：卡片式网格布局，底部固定导航栏（书架/阅读/日历三个图标）
- 图标风格：线性图标（lucide-react）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 书架页面 | 书籍网格 | 网格卡片（圆角12px、阴影0 2px 8px rgba(74,144,217,0.15)）、进度条、状态标签、悬浮效果 |
| 书架页面 | 创建新书弹窗 | 居中模态框、输入框、调色板选择器、确认/取消按钮 |
| 阅读计时器页面 | 计时面板 | 居中圆角矩形面板、大号计时显示、翻页+按钮、开始/暂停/结束按钮 |
| 阅读计时器页面 | 进度更新 | 平滑过渡进度条、恭喜弹窗 |
| 日历页面 | 月度网格 | 7列日历网格、日期单元格（含饼图）、周末浅灰背景、当日浅蓝边框 |
| 日历页面 | 日期详情 | 底部滑出面板、阅读记录列表 |
| 全局 | 底部导航 | 固定底部3图标导航栏、当前页高亮 |

### 4.3 响应式适配

- 桌面（≥1024px）：书架4列，日历7列正常展示
- 平板（768px-1023px）：书架3列
- 手机（<768px）：书架2列，日历单元格紧凑显示，底部导航图标缩小

## 1. 产品概述

光影城境是一款基于Web的3D交互式城市光影演示工具，通过Three.js渲染低多边形微缩城市，让用户直观感受不同时间段的光影变化与环境氛围切换。目标用户为建筑设计师、城市规划师、教育工作者及展览展示场景的观众。产品价值在于以可视化方式呈现时间维度下的空间光影效果，弥补静态3D模型在时间演进表现上的不足。

## 2. 核心功能

### 2.1 功能模块
1. **主场景页**: 3D微缩城市场景渲染、时间控制面板、预设快照管理

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景页 | 3D城市场景 | 渲染10+栋低多边形建筑、地面、树木，支持动态阴影与光照变化 |
| 主场景页 | 时间控制面板 | 时间滑块（0-24h）、6个预设时间卡片、当前时间显示、保存/加载快照 |
| 主场景页 | 光照系统 | 随时间线性插值的太阳方向、光色、环境光、阴影模糊度 |
| 主场景页 | 天空系统 | 随时间平滑变化的天空盒颜色（清晨/正午/黄昏/夜晚） |
| 主场景页 | 快照管理 | 保存当前场景状态到后端、加载已保存预设列表并还原 |

## 3. 核心流程

用户打开应用 → 加载3D城市场景（显示进度条）→ 场景以淡入动画呈现 → 
用户拖动滑块或点击预设卡片 → Context更新time值 → 场景模块监听变化 → 
光照/天空/阴影实时更新 → 用户点击"保存预设" → 调用后端API存储快照 → 
从后端加载预设列表 → 用户点击快照 → 场景还原至对应时刻

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**: 深空灰渐变（#111827 → #1F2937）作为控制面板背景，搭配蓝色强调色#3B82F6
- **辅助色**: 绿色保存按钮#10B981，暖色系建筑#E2C9A0/#C29B62，冷色系建筑#7BB3D9/#4A7C9B
- **按钮风格**: 圆形保存按钮（36px，50%圆角），预设卡片（80x40px，8px圆角），统一卡片圆角12px
- **字体**: #E2E8F0文字色，14px主体字号，16px时间显示字号，字距0.5px
- **布局风格**: 左右分栏布局（左侧控制面板340px + 右侧全屏3D场景），卡片式UI
- **图标**: 白色磁盘SVG图标用于保存按钮

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |

### 4.3 响应式
桌面端优先布局，控制面板固定宽度340px；触摸设备优化滑块拖拽体验。

### 4.4 3D场景指引
- **环境氛围**: 四个时段（清晨/正午/黄昏/夜晚）各有独特光照与天空颜色
- **光照设置**: DirectionalLight模拟太阳（方向随时间变化）+ AmbientLight环境光 + 夜间PointLight月光
- **阴影配置**: 阴影贴图2048x2048，阴影相机范围30单位，阴影模糊度随时间插值（0.1-0.8）
- **相机设置**: 初始透视相机位置，可通过OrbitControls交互旋转查看
- **场景构成**: 10+栋2-10单位高度低多边形建筑（冷暖色交替）、绿色渐变地面、0.2-0.8单位球形树木随机散布
- **天空系统**: Scene.background随时间线性插值在#FFE4B5/#FED7AA/#87CEEB/#FF7B54/#F97316/#0B0D17之间过渡

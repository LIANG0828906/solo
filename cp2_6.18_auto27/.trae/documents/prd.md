## 1. 产品概述

StyleSeed 是一个设计令牌管理应用，旨在解决团队中设计师与前端开发者因组件风格不统一而反复返工的问题。通过集中管理颜色、间距、圆角、阴影等设计令牌，并实时预览组件外观，使设计决策可视化和可导出，从而实现设计与开发的高效协同。

- 目标用户：UI/UX 设计师、前端开发者、设计系统维护者
- 核心价值：统一设计语言，消除设计与开发之间的风格偏差，一键导出可用的 CSS 代码

## 2. 核心功能

### 2.1 功能模块

1. **令牌编辑面板**：管理 20 个设计令牌（8 颜色 + 4 间距 + 4 圆角 + 4 阴影），支持色相环选色、滑块调参
2. **组件预览区**：实时渲染按钮、输入框、卡片、导航栏等组件，展示令牌应用效果
3. **预设与重置**：3 个预设风格模板（柔和、现代、复古）+ 重置按钮
4. **导出功能**：生成 CSS 自定义属性代码，支持一键复制

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 令牌编辑面板 | 左侧面板，含色相环、滑块、折叠区域，实时写入 store |
| 主页面 | 组件预览区 | 右侧区域，两列三行网格展示 5 个标准组件，从 store 读取令牌动态渲染 |
| 主页面 | 预设与重置 | 顶部工具栏，3 个预设按钮 + 重置按钮，批量更新 store |
| 主页面 | 导出模态框 | 居中弹出，显示 CSS 代码，含复制按钮和成功提示 |

## 3. 核心流程

用户打开应用 → 左侧面板调整设计令牌 → 右侧预览区实时更新组件样式 → 满意后点击导出 → 模态框显示 CSS 代码 → 复制到剪贴板

## 4. 用户界面设计

### 4.1 设计风格

- 整体风格：深色编辑面板 + 浅色预览区的双色调设计，专业工具感
- 主色调：由主色令牌动态决定，默认为蓝紫色调
- 编辑面板背景：#1E1E2E（深色），预览区背景：#FAFAFA（浅灰）
- 字体：Fira Code（代码区域），系统无衬线字体（UI 文本）
- 布局：左右分栏，左侧 340px 固定宽度编辑面板，右侧自适应预览区
- 组件卡片：白色背景，8px 圆角，hover 时阴影上浮动效

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主页面 | 令牌编辑面板 | 深色背景，4 个折叠区域（颜色/间距/圆角/阴影），色相环 180px，自定义滑块 |
| 主页面 | 组件预览区 | 浅灰背景，2×3 网格布局，5 个组件卡片，hover 阴影动效 |
| 主页面 | 预设与重置工具栏 | 预设按钮组（柔和/现代/复古）+ 重置按钮 |
| 主页面 | 导出模态框 | 半透明遮罩，深色代码区域（#1E1E1E），等宽字体，复制按钮+成功提示 |

### 4.3 响应式设计

- 桌面端（≥900px）：左右分栏布局，左侧 340px 固定宽度编辑面板
- 移动端（<900px）：上下布局，编辑面板转为可折叠顶部面板（默认收起）
- 编辑面板可折叠，折叠按钮位于左上角

### 4.4 交互动效

- 折叠区域箭头旋转：展开时顺时针 90°，折叠时回 0°，动画 0.2s ease
- 组件卡片 hover：阴影从 `0 1px 3px rgba(0,0,0,0.12)` 变为 `0 4px 8px rgba(0,0,0,0.16)`，Y 偏移 -2px，过渡 0.2s ease
- 色相环交互：点击/拖拽选色，白色圆点指示器（6px）
- 滑块自定义：轨道填充色为主色令牌（透明度 0.3），thumb 直径 16px，hover/active 时缩放 1.2 倍
- 所有令牌更改在 16ms 内反映到预览组件

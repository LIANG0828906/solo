## 1. 产品概述

像素农场与天气模拟是一款策略经营类网页游戏，核心创新在于将天气系统从背景装饰升级为直接影响作物生长和农场布局的关键机制。玩家需要根据动态天气变化调整种植计划，在晴天加速生长与浇水消耗、雨天自动灌溉与疾病风险、暴风雨破坏与修复之间做出策略抉择。

- 目标用户：喜欢经营类休闲游戏、追求策略深度的玩家
- 核心价值：通过天气-作物深度联动机制，为传统农场游戏注入策略层

## 2. 核心功能

### 2.1 功能模块

1. **农场主场景**：8x8像素风格农场网格，格子种植、湿度变色、作物4阶段生长与采收动画
2. **天气系统**：动态天气切换（晴朗/多云/小雨/暴雨/暴风），粒子效果，背景渐变过渡
3. **操作面板**：点击格子弹出操作面板（浇水/施肥/收获），体力值管理
4. **信息面板**：天气预报、金币、作物库存，响应式折叠/展开

| 页面名称 | 模块名称 | 功能描述 |

| 游戏主页面 | 天气系统 | 每30秒天气切换，5种天气状态，粒子效果（小雨斜线蓝点、暴雨白色竖线、暴风横线+落叶），0.5s渐变过渡 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：像素风格复古色调，土地褐#D2B48C/#8B4513，晴天#87CEEB，暴雨#4A4A4A
- **按钮风格**：像素风格圆角12px按钮，按压缩放0.9→1.0动画
- **字体**：像素风格字体（如 Press Start 2P / Silkscreen），8-16px
- **布局风格**：左侧70%农场区域 + 右侧30%信息面板，像素网格对齐
- **图标风格**：像素风格小图标（太阳☀️、云朵☁️、雨滴💧、闪电⚡）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主页面 | 农场网格 | Canvas渲染8x8网格，土地湿度变色，作物4阶段像素图，采收弹起动画+粒子消散0.3s |
| 游戏主页面 | 天气条 | 顶部天气指示，粒子效果叠加，背景色0.5s渐变过渡 |
| 游戏主页面 | 操作面板 | 浮动白色#FFFFFF圆角12px面板，水滴/叶子/镰刀图标按钮，0.15s按压动画 |
| 游戏主页面 | 体力显示 | 左上角❤️#FF4444心形图标+数值文字 |
| 游戏主页面 | 信息面板 | 右侧固定面板，天气详情+3天预报图标+金币+库存，移动端底部抽屉 |

### 4.3 响应式设计

- **桌面端**：左侧农场70% + 右侧信息面板30%
- **移动端竖屏**：农场全屏显示，信息面板折叠为底部抽屉，可上滑展开0.3s弹簧动画
- **触摸优化**：格子点击区域≥44px，按钮间距≥8px

### 4.4 性能目标

- 农场网格渲染帧率≥30fps
- 粒子系统50粒子以下不影响主交互帧率
- 天气切换0.5s平滑过渡

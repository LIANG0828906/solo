## 1. 产品概述

看板时光机是一款轻量级任务进度可视化应用，面向敏捷团队和项目协作场景。它结合看板管理与时间轴分析，让团队成员既能通过拖拽直观管理任务状态，又能通过时间轴和负载图表快速了解项目历史变更和每个人的负载变化。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 默认可用 | 创建看板、管理任务、查看时间轴和负载统计 |

### 2.2 功能模块

1. **看板管理页**：看板列表导航、任务三列看板、任务CRUD与拖拽、成员负载折线图
2. **时间轴页**：横向时间轴展示任务变更历史、悬停浮窗查看操作详情

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 看板管理页 | 看板列表 | 左侧深色导航区展示看板卡片列表，点击切换当前看板，支持创建新看板 |
| 看板管理页 | 任务看板 | 三列（待办/进行中/已完成）展示任务卡片，支持拖拽改变状态和排序 |
| 看板管理页 | 任务卡片 | 展示标题、描述、负责人、优先级，支持创建、编辑、删除 |
| 看板管理页 | 成员负载折线图 | 底部使用Recharts渲染折线图，展示每位成员每天的任务承担量 |
| 时间轴页 | 时间轴视图 | 横向时间轴展示变更日志，圆点按操作类型着色，悬停显示详情浮窗 |

## 3. 核心流程

用户进入应用后，在左侧导航区查看或创建看板，选择看板后在右侧三列看板中管理任务（创建、拖拽改变状态、删除），系统自动记录操作日志。切换到时间轴页面，可以以时间轴形式查看所有任务变更历史，了解项目演进过程。

```mermaid
flowchart TD
    "A[用户进入应用]" --> "B[查看/创建看板]"
    "B" --> "C[选择看板]"
    "C" --> "D[三列看板展示任务]"
    "D" --> "E[创建任务]"
    "D" --> "F[拖拽改变状态]"
    "D" --> "G[删除任务]"
    "E" --> "H[自动记录日志]"
    "F" --> "H"
    "G" --> "H"
    "H" --> "I[时间轴展示变更历史]"
    "H" --> "J[成员负载折线图更新]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：极简浅色主题，主背景#F8FAFC，深色导航#1E293B
- **强调色**：蓝色#3B82F6（创建/主要操作）、黄色#F59E0B（移动/进行中）、红色#EF4444（删除/待办）、绿色#10B981（已完成）
- **按钮风格**：圆角6px，悬停0.2s平滑过渡
- **字体**：系统默认字体（-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif）
- **布局风格**：左侧固定导航 + 右侧自适应内容区
- **卡片风格**：白色背景，8px圆角，12px内边距，2px底部边框#E2E8F0

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 看板管理页 | 看板列表 | 深色背景#1E293B，白色文字，280px宽，卡片式展示 |
| 看板管理页 | 三列看板 | 待办列#EF4444淡红、进行中列#F59E0B淡黄、已完成列#10B981淡绿 |
| 看板管理页 | 任务卡片 | 220px宽，白色背景，8px圆角，拖拽阴影8px rgba(0,0,0,0.15) |
| 看板管理页 | 负载折线图 | Recharts折线图，色环配色，网格线#E2E8F0 |
| 时间轴页 | 时间轴 | 背景#FAFAFA，轴线#CBD5E1 2px，圆点10px，浮窗#1E293B白字 |

### 4.3 响应式适配

- 桌面优先设计，屏幕宽度<768px时：
  - 左侧看板列表变为顶部横向选项卡
  - 三列任务改为垂直排列
- 所有可交互元素悬停时0.2s背景色或阴影平滑过渡

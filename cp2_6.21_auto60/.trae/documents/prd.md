## 1. 产品概述

智能饮食营养分析应用，帮助用户精确追踪每日食物营养摄入、评估膳食结构合理性，并获得个性化的饮食调整建议。主要解决用户在减肥或健康管理中难以精确掌握营养成分、缺乏科学评估手段的问题。

- 核心价值：通过食物库匹配、可视化分析和智能建议，让用户轻松掌握每日营养状况
- 目标用户：关注健康管理、体重控制、膳食均衡的人群

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 本地存储（无需注册） | 记录饮食、查看分析、生成报告 |

### 2.2 功能模块
1. **首页（食物记录）**：食物搜索与记录、当日营养概览、时间线展示
2. **历史页面**：30天热量趋势图、周度报告生成
3. **诊断面板**：7天营养分析、膳食改善建议、替代食物推荐

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 首页 | 食物搜索框 | 中英双语模糊搜索，支持拼音和汉字，实时匹配食物库 |
| 首页 | 食物选择器 | 显示每100克热量、蛋白质、脂肪、碳水、纤维素、钠含量 |
| 首页 | 克数滑块 | 10-500克范围，步长10克，实时计算营养值 |
| 首页 | 营养环形图 | 三大宏量营养素占比（蛋白/脂肪/碳水），Chart.js绘制 |
| 首页 | 六维雷达图 | 当日摄入vs WHO推荐量，超标红色高亮，不足蓝色闪烁 |
| 首页 | 时间线卡片 | 当日记录列表，悬停升影动画，左侧边框高亮 |
| 首页 | 诊断按钮 | 触发7天营养分析，右侧抽屉展示建议 |
| 历史页 | 热量折线图 | 30天热量趋势，绿色推荐区间，红色超标区域 |
| 历史页 | 周报生成 | 点击生成PDF周度营养报告（模拟下载） |
| 诊断抽屉 | 建议列表 | 3-5条具体改善建议，每条带替代食物按钮 |
| 替代弹窗 | 对比表格 | 5种推荐替代食物及其营养成分对比 |

## 3. 核心流程

**主要用户流程：记录一餐饮食 → 查看营养分析 → 获取改善建议**

用户在首页搜索食物名称 → 从搜索结果中选择目标食物 → 拖动滑块调整食用克数 → 点击记录按钮添加到当日日志 → 顶部图表实时更新 → 点击诊断按钮查看7天分析与建议 → 可点击查看替代食物 → 切换到历史页查看趋势与生成周报

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：蓝绿色系，清新健康风格
  - 主色：#4ecdc4（青绿）
  - 次色：#2a9d8f（深青绿）
  - 强调色：#ff6b6b（珊瑚红，用于警告/超标）
  - 辅助色：#ffe66d（柠檬黄，碳水）、#888888（浅灰文字）
- **背景**：径向渐变 #f0faf0 → #e0f5e9
- **按钮**：#4ecdc4 到 #2a9d8f 渐变，悬停亮度+10%，点击水波纹动画
- **字体**：现代无衬线字体，清晰易读
- **布局**：左侧固定220px侧边栏 + 右侧主内容区
- **动效**：卡片悬停升起、按钮水波纹、雷达图动画、侧边栏滑入

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|-------|
| 首页 | 顶部概览 | 宏量营养素环形图、六维雷达图、日期标签 |
| 首页 | 搜索区域 | 搜索输入框、搜索结果下拉、食物详情卡片、滑块、记录按钮 |
| 首页 | 时间线 | 垂直时间线、白色圆角卡片、日期右对齐、悬停动画 |
| 历史页 | 折线图 | 30天热量趋势、绿色推荐区间、红色超标填充 |
| 历史页 | 报告区 | 生成周报按钮、下载提示 |
| 诊断抽屉 | 建议列表 | 图标+标题+描述、替代食物按钮 |
| 替代弹窗 | 对比表 | 食物名称+营养素对比、高亮推荐 |

### 4.3 响应式设计
- **桌面端（>768px）**：左侧220px固定侧边栏，主内容区自适应
- **移动端（≤768px）**：侧边栏隐藏为汉堡菜单，卡片全宽展示，堆叠布局
- **触摸优化**：按钮最小高度44px，滑块触摸区域扩大

### 4.4 动效与性能
- 雷达图动画帧率 ≥ 55fps
- 月切换/热力图加载 ≤ 200ms
- 卡片悬停：6px阴影 + 3px左侧边框高亮（#4ecdc4）
- 按钮点击：水波纹扩散动画（0.3s）
- 营养不足项：蓝色闪烁动画提醒

# WindWalker 纸飞机飞行游戏 产品需求文档

## 1. 项目概述

### 1.1 产品定位
WindWalker 是一款横版卷轴 2D 飞行游戏，玩家操控像素风格纸飞机，借助风力穿越多种地形，收集萤火虫补充能量，挑战最远距离飞行记录。

### 1.2 目标用户
- 休闲游戏爱好者
- 喜欢飞行/探索类游戏的玩家
- 桌面端和移动端用户全覆盖

### 1.3 核心玩法
- 操控纸飞机在横版世界中滑翔
- 利用上升气流、顺风等风力区域节省能量
- 收集萤火虫恢复能量并触发加速技能
- 避免能量耗尽导致坠落
- 挑战最长飞行距离

## 2. 功能需求

### 2.1 核心游戏机制

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 纸飞机操控 | 鼠标/触控上下滑动调整仰角，松开自动滑翔下降，转弯有侧倾动画 | P0 |
| 能量系统 | 能量条每秒减少2%，收集萤火虫恢复10%，归零则坠落 | P0 |
| 风力系统 | 上升气流、下降气流、持续顺风三种区域，影响飞行轨迹 | P0 |
| 萤火虫收集 | 集群分布，靠近自动磁力吸附，收集20只触发2秒冲刺加速 | P0 |
| 地形生成 | 森林、峡谷、瀑布、火山口等多种地形随机生成 | P0 |
| 碰撞检测 | 与地形碰撞导致坠落 | P0 |

### 2.2 视觉表现

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 像素风纸飞机 | 主色淡黄 #FDE68A，翅膀半透明 | P0 |
| 分层滚动背景 | 前景树影、中景山峦、远景天空云海 | P0 |
| 渐变天空 | 日出橙红到深夜紫蓝的动态过渡 | P0 |
| 粒子尾迹 | 纸飞机尾部细碎光点，大小渐变消散 | P0 |
| 萤火虫动画 | 发光黄色圆点，呼吸闪烁频率0.5-1.5秒随机 | P0 |
| 坠落动画 | 翻滚、冒烟、碎片飘散 | P1 |
| 冲刺特效 | 金色光晕环绕纸飞机 | P1 |

### 2.3 UI 界面

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 能量条 | 左上角弧形半透明，绿到红渐变，发光边框 | P0 |
| 风向提示 | 屏幕边缘毛玻璃半透明条，背景模糊8px | P0 |
| 飞行数据 | 当前距离、萤火虫收集数 | P0 |
| 游戏结束界面 | 本次距离、收集数、新地形解锁、历史最高 | P0 |
| 排行榜 | 基于 IndexedDB 持久化存储 | P1 |

### 2.4 交互与适配

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 桌面端操控 | 鼠标上下移动调整仰角 | P0 |
| 移动端操控 | 触控上下滑动调整仰角 | P0 |
| 响应式布局 | flexible 布局，移动端全屏，桌面端居中留白 | P0 |
| 交互动画 | CSS transition/keyframe 平滑过渡 | P0 |

## 3. 非功能需求

### 3.1 性能要求
- 纸飞机移动和地形滚动保持 60FPS
- 粒子数量控制在 200 个以内，动态回收
- Canvas API 渲染保证轻量高效

### 3.2 技术约束
- TypeScript + React 18
- Vite 构建工具
- Zustand 状态管理
- Canvas API 渲染游戏画面
- IndexedDB (idb-keyval) 数据持久化

### 3.3 设计风格
- 清新手绘风格配色，主背景 #E0F2FE
- UI 元素圆角 12px
- 按钮微妙按压缩放动画 0.1s

## 1. 产品概述

虚拟乐队合奏排练室是一款浏览器端的实时音乐创作工具，让用户可以组合多个虚拟乐手（鼓手、贝斯手、吉他手、键盘手），通过直观的参数调节实时生成合成音乐，并可视化音频波形与频谱。

- 核心价值：无需音乐基础即可快速创作乐队风格音乐片段
- 目标用户：音乐爱好者、创意工作者、教育场景
- 产品定位：轻量级、即时反馈的Web音乐合成工具

## 2. 核心功能

### 2.1 功能模块

1. **乐队舞台视图**：4位乐手半身剪影图标排列展示，支持选中高亮与状态面板
2. **乐手详细编辑面板**：节奏型选择、和弦进行编辑、独奏开关
3. **主控台**：播放/暂停、BPM控制、总音量、波形显示、频谱分析
4. **实时音频合成引擎**：基于 Web Audio API 的多轨音频合成
5. **音频可视化**：时域波形 + 频域频谱双显示器

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主排练室 | 乐队舞台 | 4位乐手布局展示、选中交互、状态面板滑块 |
| 主排练室 | 详细编辑面板 | 右侧滑出面板，含节奏型、和弦编辑器、独奏开关 |
| 主排练室 | 主控台 | 底部播放控制、BPM滑块、总音量、波形+频谱可视化 |

## 3. 核心流程

用户进入排练室 → 默认播放预设乐队配置 → 点击乐手选中（金色描边高亮）→ 右侧弹出详细面板 → 调节滑块/选择节奏型/编辑和弦 → 实时听到音频变化 → 观察波形与频谱可视化 → 调整BPM与总音量 → 继续创作

    H --> I["调整BPM与总音量"]

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：暗色夜店风，科技感霓虹氛围
- **主背景**：垂直渐变 `#0a0a0f` → `#1a1a2e`
- **主色调**：霓虹色系 `#ff00aa`（品红）、`#00ffaa`（青绿）、`#ffaa00`（金橙）
- **高亮色**：金色 `#ffd700`（选中乐手描边）
- **波形颜色**：`#00ff88`（翠绿）
- **频谱渐变**：`#4488ff`（蓝）→ `#ff4488`（粉红）
- **字体**：无衬线细体，轻盈现代感
- **按钮风格**：霓虹发光边框，按下时 `transform: scale(0.9)` 弹跳 + 光晕扩散
- **滑块风格**：半透明灰色轨道，圆形霓虹色手柄带发光描边
- **交互反馈**：所有操作 100ms 内响应

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 主排练室 | 乐队舞台 | 4个乐手剪影（鼓后中、贝斯左前、吉他右前、键盘最前），每个下方4参数滑块，选中金色描边 |
| 主排练室 | 详细编辑面板 | 右侧滑入，6个圆形节奏型按钮，和弦编辑器（根音/类型/拍号下拉），独奏开关 |
| 主排练室 | 主控台 | 底部深色条带，播放/暂停按钮，BPM滑块(60-180)，总音量滑块，波形Canvas，频谱柱状图 |

### 4.3 响应式

- 桌面端优先设计，固定布局保证舞台视图比例
- 中等屏幕自适应缩放
- 乐手布局采用相对定位 + 百分比，保持舞台透视感

### 4.4 性能要求

- 音频合成 + 双可视化同时运行时 FPS ≥ 30
- 参数调整实时生效，延迟 < 150ms
- 所有交互反馈 < 100ms

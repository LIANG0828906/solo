# 社区图书馆管理系统 PRD

## 1. 产品概述

### 1.1 产品背景
小型社区图书馆目前依赖人工记录借还书、统计逾期情况、手动推荐图书，效率低下。需要一个数字化管理系统提升运营效率。

### 1.2 产品目标
- 实现图书流通全流程数字化管理
- 自动化逾期检测与提醒
- 基于读者偏好提供个性化图书推荐
- 提供数据看板直观展示运营状况

### 1.3 目标用户
社区图书馆员、管理员

## 2. 功能需求

### 2.1 图书管理与搜索
- **图书添加**：支持录入书名、作者、ISBN、分类、库存数量
- **图书搜索**：按书名/作者/分类模糊匹配搜索
- **结果展示**：卡片列表形式，卡片宽220px，圆角12px，背景#FDFCFA
- **交互效果**：悬浮时卡片上移8px，阴影加深，0.2秒过渡动画

### 2.2 读者与借阅管理
- **读者信息**：卡片展示，宽200px，圆角10px，背景#F4F7F9
- **借阅功能**：为读者借阅图书，自动记录借阅日期和应还日期（14天）
- **逾期检测**：定期检查，超期显示红色提醒
- **借阅记录**：完整展示借阅历史与状态

### 2.3 图书推荐
- **个性化推荐**：基于借阅历史（分类、作者偏好）
- **热门推荐**：按借阅次数排序展示热门图书
- **展示形式**：横滚轮播，推荐卡片宽180px

### 2.4 数据看板
- **统计指标**：总藏书量、总读者数、当前借出数、逾期未还数
- **卡片样式**：宽160px，高100px，渐变背景从#E8F4F8到#F0F8F0
- **数字样式**：36px加粗字体

## 3. UI/UX 设计规范

### 3.1 设计风格
极简图书馆风，温暖、知性、舒适

### 3.2 配色方案
- 主色调：暖木色 #C49A6C
- 背景色：米白色 #F5F0E8
- 页面背景：淡灰色 #F9F9F7
- 边框色：#E4DDD3、#D2E0E6

### 3.3 布局规范
- 顶部导航栏：高56px，固定定位，背景#C49A6C，白色文字，底部圆角8px
- 响应式布局：桌面端三列网格，平板双列，手机单列

### 3.4 动画规范
- 搜索结果淡入：0.3秒
- 卡片悬浮过渡：0.2秒缓动

## 4. 性能要求
- 搜索响应时间 ≤ 200ms（1000条模拟数据）
- 页面整体加载时间 ≤ 1秒

## 5. 技术约束
- 框架：React 18 + TypeScript
- 构建工具：Vite
- UI组件库：Ant Design（按需加载）
- 路由：React Router DOM

## 1. 产品概述

项目决策时间线（Decision Timeline）是一个面向远程协作团队的轻量级可视化工具，将项目中的关键决策、文件变更和讨论以时间轴形式串联，帮助团队像浏览历史故事一样回顾项目演化过程，解决信息碎片化与版本混乱导致的追溯困难问题。

- 目标用户：远程协作团队的项目经理、开发者、设计师
- 核心价值：让每一次决策的"为什么"和"谁做的"清晰可追溯，降低团队沟通成本

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 团队成员 | 直接访问 | 创建、编辑、删除决策条目；评论；查看时间线 |
| 访客 | 只读链接 | 浏览时间线、查看详情 |

### 2.2 功能模块

1. **时间线主视图**：垂直滚动布局，决策卡片沿时间轴排列，支持筛选和搜索
2. **决策创建/编辑**：表单弹窗，支持标题、描述、类型、附件上传
3. **决策详情展开**：评论区、附件预览、操作日志
4. **视图切换**：垂直时间线 ↔ 瀑布流卡片视图
5. **交互手势**：双击置顶、长按菜单、右滑快速删除
6. **项目侧边栏**：项目meta信息、选中条目快捷预览

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：浅蓝灰背景 #F0F4F8，主色蓝 #2196F3
- 辅助色：技术决策绿 #4CAF50、设计决策橙 #FF9800、管理决策蓝 #2196F3
- 风格：Google Material Design 3，圆角8px统一，阴影层级分明
- 字体：标题 #212121、正文 #424242、辅助 #757575
- 按钮风格：胶囊形筛选按钮（圆角20px）、方形操作按钮（圆角8px）
- 布局：时间线居中，垂直轴贯穿，两侧留白60px

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 时间线主视图 | 筛选栏 | 胶囊按钮组 + 搜索输入框，浅灰背景 |
| 时间线主视图 | 决策卡片 | 白色卡片，1px边框#E0E0E0，圆角8px，类型圆点12px |
| 时间线主视图 | 时间轴线 | 2px宽#D0D0D0垂直线贯穿 |
| 时间线主视图 | 悬浮按钮 | 圆形48px，背景#2196F3，点击旋转90度动画 |
| 创建弹窗 | 表单容器 | 白色背景，宽500px，圆角12px，阴影0 8px 32px |
| 创建弹窗 | 输入框 | 高40px，焦点蓝色光晕，边框#CCCCCC |
| 创建弹窗 | 文本域 | 高120px，可拖拽调整 |
| 详情展开 | 展开区域 | 背景变#F5F5F5，高度动画400ms |
| 详情展开 | 评论区 | 评论滑入动画translateY 300ms |
| 视图切换 | 切换按钮 | 右上角两个按钮，立方体旋转1s过渡 |
| 瀑布流视图 | 网格布局 | 浅灰#ECECEC背景，3列网格，间距16px |
| 侧边栏 | 项目信息 | 左侧项目meta，右侧快捷预览（>1200px显示） |

### 4.3 响应式适配

- **桌面优先设计**，核心体验在桌面端
- **>1200px**：时间线居中 + 左右双侧边栏
- **768px-1200px**：隐藏侧边栏，时间线居中占满
- **<768px**：卡片宽度90%视口宽，按钮/输入框尺寸不变

### 4.4 动画规范

- 所有交互过渡 duration 0.2s~0.5s，ease-out曲线
- 筛选切换：淡出300ms → 淡入300ms
- 卡片展开：高度0→自适应 400ms ease-out
- 评论滑入：translateY(20px)→0 300ms
- 视图切换：立方体旋转1s ease-in-out
- 悬浮按钮点击：旋转90度 0.2s ease-out
- 新条目滚动：平滑滚动0.5s

## 1. 产品概述

独立摄影师在线作品集管理与虚拟展览生成平台，解决摄影师作品展示形式单一、无法快速按客户需求生成不同风格展览页面、作品集数据散落难以集中管理的问题。目标用户为独立摄影师及其客户/观众，核心价值在于一键生成沉浸式3D虚拟展览厅，提升作品展示的专业感和互动性。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 摄影师 | 默认进入 | 上传作品、编辑详情、生成展览、管理标签 |
| 观众 | 无需注册 | 浏览展览、点赞、留言 |

### 2.2 功能模块

1. **作品管理页**：上传区域、作品网格预览、编辑面板、标签筛选与排序、数据看板
2. **虚拟展览厅页**：3/4透视展厅空间、画框展示、墙面切换、点赞与留言交互

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 作品管理页 | 上传区域 | 支持拖拽上传jpg/png/webp格式，最大10MB |
| 作品管理页 | 作品网格预览 | 每行4个卡片，宽240px，圆角8px，悬停上浮8px+阴影 |
| 作品管理页 | 编辑面板 | 弹出面板含标题、拍摄日期、相机参数、标签多选、保存删除 |
| 作品管理页 | 筛选排序 | 按标签筛选、按拍摄日期排序 |
| 作品管理页 | 数据看板 | 横向条形图展示各标签作品数量、作品总数、总获赞数 |
| 虚拟展览厅 | 展厅空间 | 暗色调3/4透视CSS模拟空间，三面墙壁各挂4幅画框作品 |
| 虚拟展览厅 | 画框组件 | 画框200x280px，颜色#B8860B，左右箭头切换墙面作品 |
| 虚拟展览厅 | 点赞交互 | 心形图标点击填充+1，0.3s缩放动画 |
| 虚拟展览厅 | 留言面板 | 右侧滑入320px宽，含留言框和列表，即时显示 |

## 3. 核心流程

摄影师上传作品 → 设置作品详情（标题/日期/相机参数/标签） → 选择至少4张作品 → 点击生成展览 → 系统生成暗色调3D虚拟展厅 → 观众浏览展厅、点赞、留言。

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色#FAFAFA，侧边栏和头部导航深灰#2C3E50，强调色#E67E22
- 简约留白搭配暖木色点缀，画框色#B8860B体现艺术质感
- 圆角统一8px或12px，卡片和面板带微弱box-shadow（rgba(0,0,0,0.05)）
- 表单元素边框色#E0E0E0，聚焦时变为#E67E22
- 所有交互按钮和卡片悬停0.3s~0.4s transition
- 展览厅暗色调渐变#1A1A2E到#16213E

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 作品管理页 | 上传区域 | 虚线边框拖拽区，居中图标+文字提示 |
| 作品管理页 | 作品网格 | 4列卡片网格，卡片240px宽，8px圆角，#F8F8F8背景 |
| 作品管理页 | 编辑面板 | 浮层#FFFFFF背景，12px圆角，表单字段+操作按钮 |
| 作品管理页 | 数据看板 | #F5F5F5背景12px圆角，横向条形图，标签颜色区分 |
| 虚拟展览厅 | 展厅空间 | CSS transform 3/4透视，暗色渐变背景 |
| 虚拟展览厅 | 画框 | 200x280px #B8860B画框，内部作品图 |
| 虚拟展览厅 | 导航箭头 | 40px直径圆形，#FFFFFF 15%不透明度，悬停30% |
| 虚拟展览厅 | 留言面板 | 右侧滑入320px，#2C3E50背景，12px圆角 |
| 虚拟展览厅 | 点赞按钮 | 心形图标，空心/填充#E74C3C，0.3s缩放动画 |

### 4.3 响应式设计

- 桌面优先设计，浏览器宽度<768px时内容区自动单列布局
- 侧边栏折叠为汉堡菜单
- 展览厅页面保持核心交互，简化3D效果
- 作品网格从4列降为2列再降为1列

### 4.4 3D场景指引

- 使用CSS transform perspective + rotateX模拟3/4俯视展厅空间
- 三面墙壁通过perspective和translateZ定位
- 墙面之间通过箭头按钮切换显示作品
- 切换动画0.4s过渡，确保延迟<100ms

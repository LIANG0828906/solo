## 1. 产品概述

情感记账（EmoLedger）是一款情感化消费记录与分析应用，旨在将日常消费与情绪关联，帮助用户洞察消费行为背后的情感驱动因素。目标用户为注重自我觉察与理财意识的年轻群体，通过情感标签+视觉化分析替代传统枯燥的数字记账方式。

- 核心问题：传统记账工具枯燥无味、缺乏情感温度，无法直观呈现消费行为与情绪的关联
- 产品价值：让每一笔消费都承载情感记忆，通过可视化分析帮助用户建立更健康的消费心智

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（本地应用） | 记录消费、查看分析、设置预算 |

### 2.2 功能模块
1. **账单列表页**：消费记录列表、情感标签筛选、新增/编辑账单
2. **仪表盘页**：情感圆环图分析、预算进度与提醒、时间范围切换

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 账单列表页 | 情感标签选择器 | 记账时选择情感标签（开心、焦虑、满足、后悔、平静），彩色磨砂玻璃徽章交互 |
| 账单列表页 | 账单编辑表单 | 输入金额、选择分类（8个类别）、选择情感标签、填写备注（最多50字） |
| 账单列表页 | 瀑布流账单卡片 | 展示每笔账单，左侧情感色竖条装饰，悬停上浮动画 |
| 仪表盘页 | 时间范围切换 | 日/周/月切换，控制数据分析范围 |
| 仪表盘页 | 情感圆环图 | 各情感消费总额占比，圆环中心显示总金额，点击展开详情，外圈流动小点动画 |
| 仪表盘页 | 预算进度条 | 水平渐变进度条显示预算使用情况，右侧沙漏倒计时动画 |
| 全局 | 预算警告横幅 | 周消费超预算80%时从右滑入红色警告横幅，显示百分比和剩余天数 |
| 全局 | 导航栏 | 固定顶部导航，应用名称、记账按钮、仪表盘切换，移动端变底部标签栏 |

## 3. 核心流程

用户打开应用 → 在账单列表页点击浮动记账按钮 → 填写金额、选择分类、选择情感标签、填写备注 → 保存后在列表中看到新账单卡片 → 切换到仪表盘查看情感圆环图和预算进度 → 发现焦虑消费占比偏高 → 调整后续消费行为

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色主题（#121212主背景、#1D1D1D卡片背景、#E0E0E0主文字色）
- 强调色：#BB86FC（浮动按钮）、各情感色（开心#FFD700、焦虑#FF6B6B、满足#6BCB77、后悔#A66CFF、平静#4FC3F7）
- 按钮风格：圆形浮动按钮56px，带脉冲光晕动画
- 字体：中文优先使用系统字体栈，数字使用等宽字体
- 布局风格：瀑布流卡片布局，圆角16px，内边距20px
- 图标风格：lucide-react线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 账单列表页 | 导航栏 | 固定顶部64px，#1D1D1D背景20%透明度，顶部装饰线随滚动渐隐渐显 |
| 账单列表页 | 浮动记账按钮 | 56px圆形，#BB86FC背景，3s脉冲光晕，点击0.8倍缩放弹回 |
| 账单列表页 | 瀑布流卡片 | 最大宽480px，左侧4px情感色竖条，悬停上浮6px+阴影#00000020，0.2s ease |
| 账单列表页 | 情感标签徽章 | 48px磨砂玻璃圆形，半透明渐变+21px模糊，选中1.2倍放大+光晕，0.3s cubic-bezier |
| 仪表盘页 | 情感圆环图 | recharts圆环图，中心总金额，外圈流动小点12s周期，点击高亮展开 |
| 仪表盘页 | 预算进度条 | #4CAF50→#FFC107→#F44336渐变，右侧沙漏旋转2s周期 |
| 全局 | 警告横幅 | #FF416C→#FF4B2B渐变，48px高，右向左滑入，0.5s震动动画 |
| 全局 | 页面转场 | 左右滑动0.3s ease-in-out |

### 4.3 响应式设计
- 桌面优先设计，宽度<768px时自适应
- 移动端瀑布流改为单列布局
- 移动端导航栏变为底部标签栏（56px高，4个图标项）
- 所有卡片和交互元素支持触摸操作

### 4.4 性能要求
- 账单列表滚动保持60FPS
- 情感分析图表渲染不超过200ms
- CSS动画使用transform/opacity确保GPU加速

# CloudWeaver 产品需求文档 (PRD)

## 1. 产品概述

CloudWeaver 是一个基于 WebGL 的三维云层生成与天气可视化工具，允许用户在浏览器中实时观察和操控由粒子系统模拟的各种类型云朵。

### 1.1 项目背景
- 为气象爱好者、教育工作者和创意人员提供一个直观的云层可视化工具
- 展示粒子系统在三维空间中的动态效果
- 提供交互式参数调节，实时观察云层变化

### 1.2 目标用户
- 气象教育工作者和学生
- 创意设计师和数字艺术家
- WebGL/Three.js 技术爱好者

### 1.3 关键特性
- 三种云类型粒子模拟（积云、层云、卷云）
- 实时参数调节面板（云类型、风速、降水强度）
- 360度自由视角观察
- 动态天气效果（风力、降水）
- 高性能粒子系统（10000粒子@60FPS）

## 2. 功能需求

### 2.1 主场景功能
| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 天空背景 | 浅蓝色渐变背景（#87CEEB 到 #B0E0E6） | 高 |
| 粒子云系统 | 数千个半透明白色粒子组成云朵 | 高 |
| 相机控制 | 鼠标拖拽旋转、滚轮缩放 | 高 |
| 云类型切换 | 积云（团簇状）、层云（扁平带状）、卷云（丝缕状） | 高 |

### 2.2 控制面板功能
| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 面板展开/收起 | 右下角半透明圆形按钮触发 | 高 |
| 云类型选项卡 | 三个预设按钮，选中时发光边框 | 高 |
| 风速滑块 | 0-20 m/s，渐变轨道 | 高 |
| 风向旋钮 | 0-360度旋转，带刻度线 | 高 |
| 降水强度滑块 | 0-100% | 高 |

### 2.3 动态效果功能
| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 风力平移 | 云朵沿风向整体移动 | 高 |
| 湍流抖动 | 粒子相对位置随机抖动，幅度随风速增大 | 高 |
| 边缘消散 | 云朵边缘粒子散开/聚合形成拖尾 | 中 |
| 降水粒子 | 蓝色半透明颗粒下落 | 高 |
| 降水摆动 | 下落轨迹带正弦波摆动 | 中 |

### 2.4 统计面板功能
| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 粒子总数 | 实时显示当前粒子数量 | 中 |
| 云类型 | 显示当前选中的云类型 | 中 |
| 风速值 | 显示当前风速（m/s） | 中 |
| 降水强度 | 显示当前降水强度（%） | 中 |

## 3. 非功能需求

### 3.1 性能要求
- 粒子数量 ≤ 10000 时，帧率稳定在 60 FPS
- 使用 `requestAnimationFrame` 驱动渲染循环
- 使用 `Float32Array` 存储粒子数据，避免垃圾回收抖动
- 参数调整后下一帧立即响应，无卡顿

### 3.2 技术约束
- 基于 TypeScript + Three.js + Vite 构建
- 使用 Three.js 0.160 版本
- 严格模式 TypeScript 配置

### 3.3 兼容性要求
- 支持现代浏览器（Chrome、Firefox、Safari、Edge 最新版本）
- 支持 WebGL 2.0

## 4. UI/UX 设计规范

### 4.1 视觉主题
- **设计风格**: 轻盈、通透的天空主题
- **背景色**: #F0F8FF（爱丽丝蓝）
- **卡片色**: #FFFFFF（纯白）
- **强调色**: #4A90D9（天蓝色）
- **字体颜色**: #333333（深灰）
- **圆角**: 8px
- **阴影**: 轻微柔和阴影
- **过渡动画**: 0.2s ease-out

### 4.2 交互规范
- 控制面板按钮：鼠标悬停时放大 1.1 倍，显示工具提示
- 云类型按钮：选中时背景填充对应颜色并带发光边框
- 风速滑块：轨道从浅蓝到深蓝渐变
- 风向旋钮：旋转时数值实时更新，周围有刻度线和指针动画
- 所有控件过渡动画：0.2s ease-out

### 4.3 排版规范
- 字体：系统无衬线字体（system-ui, -apple-system, sans-serif）
- 统计面板：右上角固定位置，清晰易读

## 1. 产品概述

风格印记是一款数字胶片风格照片编辑器，让用户上传照片后自动解析主色调与构图，生成多种风格的胶片边框效果，一键生成带有水印和日期的数字胶片作品，并支持虚拟相册收藏与翻页浏览。

- 核心价值：将普通照片转化为富有文艺气息的胶片风格作品，降低创作门槛
- 目标用户：摄影爱好者、文艺青年、社交媒体内容创作者
- 产品定位：轻量级、高颜值的在线胶片风格化工具

## 2. 核心功能

### 2.1 功能模块

1. **主编辑页**：上传区、预览区、边框模板选择区、操作工具栏
2. **相册抽屉**：翻页式虚拟相册、作品收藏管理、页码指示

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主编辑页 | 上传区 | 支持点击上传和拖拽上传，限制 jpg/png/webp 格式，上传后居中显示 |
| 主编辑页 | 预览区 | 实时显示照片裁剪并叠加边框后的效果，200ms 淡入切换动画 |
| 主编辑页 | 边框模板区 | 5 种胶片风格卡片（复古棕调、日系青蓝、赛博紫绿、黑白银盐等），悬停阴影加深，选中显示对勾 |
| 主编辑页 | 工具栏 | 下载作品按钮（涟漪动效）、添加到相册按钮（提示条反馈）、重置上传按钮（旋转动效） |
| 相册抽屉 | 翻页相册 | 底部滑入抽屉，可拖动手柄展开收起，每页 3 张作品卡片 |
| 相册抽屉 | 翻页交互 | 左右箭头切换，旧页右滑出新页左滑入（300ms ease-in-out），页码圆点同步 |
| 相册抽屉 | 作品卡片 | 点击放大预览，圆角 8px，内阴影效果 |

## 3. 核心流程

用户上传照片 → 系统自动解析并生成预览 → 选择边框风格 → 实时预览效果 → 下载作品 / 添加到相册 → 打开相册抽屉浏览收藏

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：低饱和度复古暖色调 —— #F5F2ED（浅米色背景）、#EEE7DD（抽屉背景）、#5D4E37（深棕主文字/按钮）
- **辅助色**：#D6CDC0（边框）、#B0A597（虚线边框）、#8B7355（拖拽高亮）、#6B8E23（选中对勾）、#C4B8A8（页码/手柄）
- **按钮样式**：圆角 8px，主按钮深棕底白字，悬停变深 #4A3D2B，微压感 transform: scale(0.96)
- **字体**：系统无衬线字体 'Segoe UI', 'Roboto', sans-serif
- **布局风格**：卡片式布局，圆角统一 8-16px，毛玻璃效果 backdrop-filter: blur(12px)
- **动效风格**：细腻流畅，200-300ms 过渡，ease-out / ease-in-out 缓动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |

### 4.3 响应式

- 桌面端（≥768px）：左右两栏布局，左预览区 650px，右模板区 300px
- 移动端（<768px）：上下堆叠布局，预览区宽 100%，模板区 90% 宽度，改为横向可滚动卡片条
- 触摸优化：按钮最小点击区域 44px，翻页支持左右滑动手势

### 4.4 性能要求

- 翻页动画及图片叠加操作保持 50fps 以上
- Canvas 合成操作优化，避免频繁重绘
- 使用 CSS transform 和 opacity 实现动画，触发 GPU 加速

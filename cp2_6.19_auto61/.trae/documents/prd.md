## 1. 产品概述

灵感画廊是一个3D虚拟房间应用，让用户可以将网页截图、设计稿以画框形式挂在3D房间的墙面上，自由排列布局，添加文字便签做注释，并生成可分享的3D链接。目标用户为设计师、创意工作者和灵感收集爱好者，帮助他们以沉浸式3D方式整理和展示灵感素材。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 创建房间、上传图片、挂画、添加便签、生成分享链接 |

### 2.2 功能模块

1. **3D房间主场景**：沉浸式3D房间，四面墙+地板+天花板，暖灰色调纹理，画框悬挂与交互
2. **左侧边栏**：图片缩略图列表，导入、拖拽挂画功能
3. **分享功能**：生成模拟短链接，复制分享

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主场景 | 3D房间 | Three.js构建3D房间，四面墙/地板/天花板有柔和纹理，默认暖灰色系；图片上传后自动生成白色边框画框挂到墙上 |
| 主场景 | 画框交互 | 画框可拖拽沿墙面移动，每面墙最多6幅；悬停时边框发光+轻微上浮；点击弹出模态框展示原图、文件名、添加日期 |
| 主场景 | 漫游控制 | WASD移动，鼠标拖拽旋转视角，Shift加速，碰撞检测不穿墙 |
| 主场景 | 迷你小地图 | 左上角俯视图，绿色圆点表示玩家位置，蓝色小方块表示画框位置 |
| 主场景 | 右键菜单 | 右键画框可删除或添加文字便签（黄色方形贴纸，手写风格字体） |
| 左侧边栏 | 导入面板 | 已上传图片缩略图列表，悬停放大1.2倍+阴影，点击缩略图挂到当前面向墙面 |
| 右上角 | 分享按钮 | 生成唯一短链接（模拟），点击复制后toast提示"已复制到剪贴板"，toast底部滑入2秒后消失 |
| 模态框 | 画框详情 | 半透明毛玻璃背景，展示原图+文件名+添加日期，关闭时缩放淡出 |

## 3. 核心流程

用户打开应用 → 进入3D房间 → 通过左侧边栏上传图片 → 图片自动挂到墙面 → 拖拽调整画框位置 → 右键添加便签注释 → WASD漫游查看 → 点击画框查看详情 → 完成布置后点击分享 → 生成短链接复制分享

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暖灰（#8C8279）、米白（#F5F0EB）、浅棕（#C4A882）
- 强调色：柔金色（#D4A574）
- 按钮风格：柔和圆角矩形（border-radius: 8-12px），暖色填充+悬停加深
- 字体：正文使用系统字体，便签使用手写风格字体（如 'Caveat' 或 'Ma Shan Zheng'）
- 布局：桌面端左侧边栏+3D主场景，移动端侧边栏收起为底部图标栏
- 图标：使用 lucide-react 图标库
- 动画：所有动画200-400ms，ease-out缓动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主场景 | 3D房间 | 全屏3D画布，暖色环境光，柔和阴影，纹理墙面 |
| 主场景 | 画框 | 白色边框画框，悬停发光+上浮动画300ms ease-out |
| 主场景 | 迷你小地图 | 左上角200x150半透明面板，俯视图，绿点+蓝方块 |
| 主场景 | 模态框 | 居中，毛玻璃背景（backdrop-filter: blur），缩放淡出关闭动画300ms |
| 左侧边栏 | 导入面板 | 280px宽，米白背景，缩略图网格，悬停放大1.2倍+阴影 |
| 右上角 | 分享按钮 | 圆角矩形按钮，图标+文字，悬停加深 |
| 全局 | Toast | 底部居中滑入，2秒后自动消失，300ms ease-out |
| 全局 | 右键菜单 | 圆角下拉菜单，删除+添加便签选项 |

### 4.3 响应式

- 桌面端（>768px）：左侧边栏常驻显示，3D场景占满剩余空间
- 移动端（≤768px）：侧边栏收起为底部图标栏（4-5个图标），3D场景全屏，触摸操控

### 4.4 3D场景指引

- 环境与氛围：暖色调室内场景，柔和环境光（#FFF5E6），微弱阴影
- 灯光设置：环境光（AmbientLight）+ 方向光（DirectionalLight），暖白色
- 相机：第一人称视角，PerspectiveCamera，FOV 60，近裁面0.1，远裁面100
- 碰撞检测：基于AABB的墙壁碰撞，防止穿墙
- 交互：Raycaster用于画框点击和悬停检测
- 性能目标：FPS ≥ 30

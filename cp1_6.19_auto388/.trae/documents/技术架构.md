## 2. 技术说明

- **前端**：React 18 + TypeScript + Vite
- **状态管理**：Zustand
- **后端**：Express 4 + LokiJS（内存数据库）
- **本地存储**：IndexedDB（浏览器端方案保存）
- **HTTP客户端**：Axios
- **开发服务器代理**：Vite server proxy → localhost:3001

## 4. API 定义

### 4.1 花材接口

**GET /api/flowers**
- 描述：获取所有花材列表
- 响应：
```typescript
interface Flower {
  id: string;
  name: string;
  price: number;
  stock: number;
  color: string;
  imageUrl: string;
}
```

### 4.2 订单方案接口

**GET /api/schemes**
- 描述：获取所有保存的方案
- 响应：`Scheme[]`

**POST /api/schemes**
- 描述：保存新方案
- 请求体：
```typescript
interface SaveSchemeRequest {
  name: string;
  flowers: { flowerId: string; quantity: number; position: { x: number; y: number } }[];
  packagingStyle: string;
  packagingFeeRatio: number;
  totalPrice: number;
}
```

**DELETE /api/schemes/:id**
- 描述：删除指定方案

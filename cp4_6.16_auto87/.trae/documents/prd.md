## 1. 产品概述

PodcastFlow 是一款面向独立播客创作者的集中化管理工具，旨在解决传统笔记与表格管理混乱、缺少统一进度追踪与自动预告生成的问题。目标用户为独立或小团队播客制作人，需要高效管理多集节目的制作状态、音频标注和嘉宾信息。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 播客创作者 | 无需注册，本地使用 | 管理节目、嘉宾、音频标注及预览页面生成 |

### 2.2 功能模块

1. **节目看板页**：卡片网格展示所有节目，状态色圆点标识，悬停编辑/删除，点击进入详情
2. **节目详情页**：水平四阶段进度条（带发光节点与工具提示）、倒计时组件、音频播放器与标注系统
3. **嘉宾管理页**：卡片网格展示嘉宾，头像/首字母占位，点击进入详情
4. **嘉宾详情页**：统计环（节目数、平均评分、最近参与日期）、关联节目列表
5. **预览页面**：深色主题公开预告页，SVG封面、嘉宾头像列表、倒计时动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 节目看板 | 节目卡片网格 | 展示所有节目卡片，左上角状态色圆点（灰/蓝/金/绿），悬停浮现编辑与删除按钮 |
| 节目看板 | 新建节目弹窗 | 表单输入标题、描述、录制日期、初始状态，提交后新增卡片 |
| 节目详情 | 进度条组件 | 水平四阶段进度条，当前阶段节点发光+工具提示，动画过渡 |
| 节目详情 | 倒计时组件 | 显示距发布日期剩余天数，归零时全屏纸屑动画 |
| 节目详情 | 音频播放器 | 播放/暂停、进度条拖动、音量控制，基于Web Audio API |
| 节目详情 | 标注系统 | Ctrl+M或按钮添加标注，彩色针图标，时间线上方可拖拽，导出JSON |
| 嘉宾管理 | 嘉宾卡片网格 | 展示姓名、头像/首字母占位、简介、关联节目数 |
| 嘉宾管理 | 新建嘉宾弹窗 | 表单输入姓名、简介、上传头像 |
| 嘉宾详情 | 统计环组件 | 圆形进度环展示节目数、平均评分、最近参与日期 |
| 嘉宾详情 | 关联节目列表 | 显示参与的节目列表，可点击跳转，可添加新关联 |
| 预览页面 | 节目预告 | 深色主题，SVG封面、概要、嘉宾头像列表、倒计时数字缩放动画 |

## 3. 核心流程

1. **节目管理流程**：用户创建节目 → 设置状态和发布日期 → 在看板查看进度 → 进入详情编辑 → 标记发布
2. **音频标注流程**：进入详情页 → 上传MP3 → 播放音频 → Ctrl+M添加标注 → 输入笔记 → 拖拽调整 → 导出JSON
3. **嘉宾关联流程**：创建嘉宾 → 在节目详情中关联嘉宾 → 嘉宾详情页查看参与节目
4. **预告发布流程**：选中待发布节目 → 自动生成预览页 → 复制短链接分享

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝侧边栏 #1e293b + 亮青色强调 #06b6d4 + 浅灰主区 #f1f5f9
- **状态色**：灰色草稿 #94a3b8、蓝色录制中 #3b82f6、金色剪辑中 #f59e0b、绿色发布 #22c55e
- **按钮风格**：圆角8px，hover亮度提升10%，点击scale(0.97)凹陷效果
- **字体**：标题使用 Outfit（几何感现代字体），正文使用 DM Sans
- **布局**：左侧固定侧边栏240px + 右侧主内容区
- **卡片风格**：白色背景、8px圆角、4px阴影（0 1px 3px rgba(0,0,0,0.12)）、16px内边距

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 节目看板 | 卡片网格 | 白色卡片、状态色圆点、悬停浮现按钮、fade-in 0.3s |
| 节目详情 | 进度条 | 四节点水平条、发光节点、工具提示弹出、过渡动画 |
| 节目详情 | 倒计时 | 大号数字、归零时全屏纸屑动画 |
| 节目详情 | 音频播放器 | 播放/暂停按钮、拖动进度条、音量滑块、标注针图标 |
| 嘉宾管理 | 卡片网格 | 头像/首字母圆形占位、随机柔和色背景、fade-in |
| 嘉宾详情 | 统计环 | 圆形进度环、数字居中、渐变描边 |
| 预览页面 | 深色预告 | 背景#1a1a2e、SVG封面、横向嘉宾头像、粗体倒计时缩放动画 |
| 全局 | 确认对话框 | 毛玻璃半透明背景、居中弹窗 |
| 全局 | 侧边栏 | 深蓝#1e293b、波形图标、亮青竖条高亮 |

### 4.3 响应式设计

- 桌面优先设计，600px以下断点适配移动端
- 移动端侧边栏收窄为底部导航栏
- 卡片网格从多列切换为单列
- 音频播放器全宽展示

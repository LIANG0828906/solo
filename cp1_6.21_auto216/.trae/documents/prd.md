## 1. 产品概述

弹幕风暴是一款俯视角弹幕躲避射击Web游戏，玩家操控飞船在星空中躲避多种样式的弹幕并发射子弹反击，通过连击系统累积分数。目标用户为独立游戏开发者和休闲玩家，核心价值在于提供一套高性能、模块化的弹幕生成与碰撞检测系统示范。

## 2. 核心功能

### 2.1 功能模块

1. **游戏主页面**：Canvas画布（800x600px）+ HUD覆盖层，包含完整的游戏体验
2. **开始/结束界面**：游戏启动和Game Over状态切换

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主页面 | 玩家操控 | WSAD/方向键移动飞船，鼠标左键射击，右键释放大招 |
| 游戏主页面 | 弹幕生成 | 每隔1-3s随机生成圆形扩散/扇形扫射/螺旋追击弹幕 |
| 游戏主页面 | 碰撞检测 | 子弹-弹幕碰撞产生爆炸粒子，弹幕-玩家碰撞扣血中断连击 |
| 游戏主页面 | HUD显示 | 生命值、连击数、分数、大招冷却进度条 |
| 游戏主页面 | 大招系统 | 右键释放扩散冲击波，8s冷却 |
| 开始界面 | 启动画面 | 游戏标题、开始按钮、操作说明 |
| 结束界面 | Game Over | 显示最终分数、重新开始按钮 |

## 3. 核心流程

玩家进入游戏→控制飞船移动和射击→弹幕随机生成→子弹击中弹幕产生粒子效果并累积连击→弹幕击中玩家扣血并中断连击→生命值归零显示Game Over和最终分数

## 4. 界面设计

### 4.1 设计风格

- 主色调：深蓝灰（#0F172A, #1E293B, #334155）+ 琥珀黄（#F59E0B, #FDE047）
- 飞船/玩家系：蓝色（#3B82F6, #60A5FA）
- 弹幕区分色：紫色（#A78BFA）、绿色（#34D399）、橙色（#FB923C）
- 大招色：红色（#EF4444）
- 圆角：8px，UI面板背景#1E293B，文字#E2E8F0
- 字体：'Press Start 2P'（像素风格），HUD文字16px白色
- 鼠标悬停：背景#334155，0.2s ease-out过渡
- 背景：深空渐变（#0F172A→#020617）+ 50颗静态星星

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 游戏主页面 | Canvas画布 | 800x600px，深空渐变背景，星星，飞船，弹幕，粒子 |
| 游戏主页面 | HUD-生命值 | 左上角3个红心24x24px，失去时灰色缩小旋转 |
| 游戏主页面 | HUD-连击数 | 左上角，字号24px #FDE047，10连击放大36px闪0.3s |
| 游戏主页面 | HUD-分数 | 左上角，白色16px |
| 游戏主页面 | HUD-大招冷却 | 右上角进度条120x8px，背景#334155填充#EF4444圆角4px |
| 开始界面 | 标题 | '弹幕风暴'大字，像素风格 |
| 开始界面 | 按钮 | 开始游戏，圆角8px背景#1E293B |
| Game Over | 最终分数 | 居中显示 |
| Game Over | 按钮 | 重新开始 |

### 4.3 响应式适配

- 画布在窗口宽度<800px时等比缩放至全宽，保持16:9比例
- HUD元素随画布缩放但保持绝对尺寸
- 桌面优先设计，支持键盘和鼠标双模式

## 1. 产品概述

CodeCanvas 是一个轻量级在线代码沙箱，让用户能在线编写 Python 和 JavaScript 代码片段并实时看到运行结果。采用暗色科幻风格界面，提供代码编辑、安全执行、结果预览和本地保存功能，面向开发者、学习者和编程爱好者。
- 解决快速试验代码片段的需求，无需搭建本地环境即可运行 Python 和 JavaScript
- 目标是成为开发者和学习者首选的在线轻量代码实验工具

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 编辑、运行、保存、管理代码片段 |

### 2.2 功能模块
1. **编辑器页面**：代码编辑区（CodeMirror）、语言切换、撤销重做
2. **预览页面**：运行结果展示（打字机效果）、错误信息高亮
3. **工具栏**：运行按钮、保存按钮、语言切换、状态信息
4. **侧边栏**：已保存片段列表、加载与删除

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 代码编辑区 | CodeMirror 编辑器，支持 Python/JS 语法高亮，语言切换时保留内容，Ctrl+Z/Ctrl+Shift+Z 撤销重做 |
| 编辑器页面 | 结果预览区 | 展示执行结果，打字机逐字渲染，错误信息红色高亮，运行前提示"请运行代码" |
| 编辑器页面 | 拖拽分隔条 | 左右分栏中间可拖拽分隔条，调整编辑区与预览区宽度比 |
| 工具栏 | 顶部工具栏 | 显示当前语言、最后保存时间，包含运行和保存按钮 |
| 侧边栏 | 片段列表 | 显示所有已保存片段名称和最后修改时间，支持点击加载和删除 |
| 命名对话框 | 保存模态框 | 居中弹窗，输入片段名称（最多20字符）后保存到 localStorage |

## 3. 核心流程

用户打开 CodeCanvas → 选择语言（Python/JavaScript）→ 在编辑器中编写代码 → 点击运行按钮 → 代码通过模拟后端发送（延迟500ms）→ 后端解析语言类型 → Python 使用 Brython 执行 / JavaScript 在沙箱 iframe 中 eval → 执行结果以打字机效果逐行出现在预览区 → 运行结束弹出 toast 提示 → 用户可点击保存按钮 → 输入名称 → 保存到 localStorage → 侧边栏更新片段列表

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：#1A1A2E（深蓝黑）和 #0F0F23（极深蓝黑）
- 强调色：#6C63FF（科技紫蓝），用于按钮、聚焦边框、悬停高亮
- 错误色：#FF6B6B（柔红），用于错误信息文字
- 成功色：#2ECC71（翠绿），用于 toast 提示背景
- 按钮风格：圆角6px，悬停放大1.02倍，点击缩小0.95倍，带加载旋转动画
- 字体：代码编辑器使用 Consolas 14px，行高1.6；预览区白色文字16px；时间标签灰色12px
- 布局风格：左右分栏（编辑器+预览），顶部工具栏，左侧侧边栏
- 图标风格：SVG 线性图标，软盘保存图标，汉堡菜单图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 编辑器页面 | 代码编辑区 | 背景#1E1E2E，Consolas 14px，行高1.6，边框1px solid #2D2D4A，聚焦边框#6C63FF，CodeMirror 语法高亮 |
| 编辑器页面 | 结果预览区 | 背景#0F0F23，白色文字16px，圆角8px，内边距16px，错误文字#FF6B6B |
| 编辑器页面 | 拖拽分隔条 | 宽6px，背景#3A3A5C，悬停#6C63FF，过渡0.2s |
| 工具栏 | 顶部工具栏 | 高48px，背景#1A1A2E，底边框1px solid #2D2D4A，运行按钮#6C63FF 120x40px，保存按钮软盘SVG |
| 侧边栏 | 片段列表 | 宽240px，背景#16162D，左边框1px solid #2D2D4A，列表项高48px，悬停#1E1E2E |
| 命名对话框 | 保存模态框 | 居中，背景#1A1A2E，圆角12px，边框1px solid #2D2D4A，输入框背景#2D2D3F |

### 4.3 响应式设计
- 桌面优先（>768px）：左右分栏布局，侧边栏常驻
- 移动端（≤768px）：上下布局（编辑器在上占50%，预览在下占50%），分隔条变水平方向，侧边栏收起，工具栏左侧显示汉堡菜单按钮（30x30，三条横线SVG，颜色#A0A0C0），点击侧边栏从左侧滑出（过渡0.3s ease-out）

### 4.4 性能要求
- 代码执行后结果预览响应时间不超过800ms（含500ms模拟网络延迟）
- 编辑器输入无卡顿，打字延迟不超过50ms
- 所有过渡动画统一使用 0.2s ease

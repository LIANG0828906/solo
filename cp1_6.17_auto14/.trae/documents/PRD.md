# 实时投票/问卷调查应用 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**QuickVote** - 轻量级实时投票与问卷平台

### 1.2 产品定位
面向小团队和活动策划者的浏览器端轻量级投票/问卷创建工具，解决传统在线表单工具配置复杂、结果反馈延迟、无法在活动场景下动态调整的问题。

### 1.3 目标用户
- 活动策划者：在会议、团建、线下活动中快速收集意见
- 小团队管理者：团队决策投票、满意度调研
- 教育工作者：课堂互动、课后反馈收集

### 1.4 核心价值
- **快速创建**：极简配置，分钟级完成问卷发布
- **实时反馈**：WebSocket推送，结果秒级可视化
- **动态调整**：活动进行中可随时添加题目或关闭投票
- **即开即用**：无需注册，浏览器直接访问

---

## 2. 功能需求

### 2.1 投票创建模块

#### 2.1.1 问卷创建表单
- 支持三种题型：
  - **单选题**：唯一选项选择
  - **多选题**：多个选项选择
  - **评分题**：1-10分滑块评分
- 支持拖拽调整题目顺序
  - 拖拽时题目卡片下移10px
  - 显示虚线占位提示
  - 松开鼠标后完成排序
- 题目卡片样式规范：
  - 背景色：`#FFFFFF`
  - 圆角：`8px`
  - 边框：`1px solid #E0E0E0`
  - 悬停阴影：`2px → 6px`，过渡 `0.2s`

#### 2.1.2 短码生成
- 创建完成后自动生成 **6位短码**（大写字母+数字组合，如 `A3F2B1`）
- 短码展示区域：
  - 位置：页面顶部
  - 背景：`#4CAF50`（绿色）
  - 文字：白色、粗体
  - 圆角：`6px`
  - 交互：点击自动复制到剪贴板，显示"已复制"提示

---

### 2.2 投票参与模块

#### 2.2.1 短码进入
- 首页提供短码输入框
- 输入6位短码后跳转到对应问卷页面
- 短码不存在时提示错误

#### 2.2.2 投票提交
- 提交时显示进度条：
  - 主色：`#2196F3`
  - 动画时长：`0.3s`
- 系统自动记录提交时间戳

#### 2.2.3 完成页面
- 提交成功后显示感谢页面，淡入动画 `1s`
- 页面中央显示旋转的绿色勾选图标：
  - 旋转动画：`0.5s`
  - 颜色：`#4CAF50`
- 投票已关闭状态：
  - 显示"投票已结束"
  - 背景：`#F44336`
  - 文字：白色，居中显示

---

### 2.3 实时仪表盘模块

#### 2.3.1 仪表盘概览页
- 全屏宽度布局
- 顶部导航栏：
  - 背景：`#3F51B5`
  - 高度：`56px`
- 卡片网格布局：
  - 每列最小宽度：`280px`
  - 响应式自适应列数
- 每张卡片显示：
  - 投票标题
  - 参与人数
  - 创建时间
  - 状态（进行中/已关闭）
- 卡片渐入动画：`fadeIn 0.5s`

#### 2.3.2 投票详情页
- 左右分栏布局：
  - **左侧**：问卷题目列表
  - **右侧**：实时结果图表
- 实时刷新机制：
  - 通过WebSocket接收新投票推送
  - 自动刷新图表（≤300ms重新渲染）
- 最后更新时间：
  - 位置：右下角
  - 样式：灰色 `12px` 文字

#### 2.3.3 图表类型

**单选题 - 横向条形图**
- 条形颜色：`#FF9800` 渐变
- 最大值条形：`#FF5722` 高亮短划线标记
- 动画：宽度从0到目标值 `1s`

**多选题 - 堆叠条形图**
- 颜色循环：`#4CAF50` → `#2196F3` → `#FF9800` → `#9C27B0`
- 每个选项对应一种颜色
- 显示各选项选择占比

**评分题 - 趋势折线图**
- 线条颜色：`#E91E63`
- 数据点：圆形，半径 `3px`
- 动画：自左向右绘制 `1.5s`
- 显示平均分趋势

---

### 2.4 管理操作模块

#### 2.4.1 动态添加题目
- 问卷进行中可添加新题目
- 新题目自动插入到末尾
- 高亮动画：脉冲效果闪烁 `3次`

#### 2.4.2 关闭投票
- 创建者可随时关闭投票
- 关闭后投票页面显示结束状态
- 结果数据保留30天后自动删除

#### 2.4.3 数据导出
- 支持CSV格式下载
- 导出按钮位置：详情页右上角
- 按钮样式：
  - 图标：向下箭头
  - 背景：`#607D8B`
  - 文字：白色
  - 悬停：`#455A64`

---

## 3. 界面设计规范

### 3.1 设计风格
- **风格**：Material Design
- **主色**：`#3F51B5`（靛蓝）
- **辅色**：`#FF4081`（粉红）
- **背景色**：`#F5F5F5`（浅灰）

### 3.2 布局结构
```
┌─────────────────────────────────────────────┐
│  侧边栏(240px)  │      内容区域              │
│                 │                           │
│  ┌───────────┐  │  ┌─────────────────────┐  │
│  │ Logo      │  │  │ 页面标题 + 操作按钮 │  │
│  ├───────────┤  │  ├─────────────────────┤  │
│  │ 投票列表  │  │  │                     │  │
│  │  项1      │  │  │   主要内容区域      │  │
│  │  项2      │  │  │                     │  │
│  │  ...      │  │  │                     │  │
│  └───────────┘  │  └─────────────────────┘  │
│                 │                           │
└─────────────────────────────────────────────┘
```

### 3.3 侧边栏规范
- 宽度：`240px`
- 背景：`#FFFFFF`
- 右边框：`1px solid #E0E0E0`
- Logo区域：紫色 `#673AB7` 圆角矩形，显示应用缩写
- 投票列表项：
  - 高度：`48px`
  - 悬停背景：`#ECEFF1`
  - 选中状态：左侧蓝色竖条指示，底色 `#E8EAF6`

### 3.4 按钮规范
- 圆角：`20px`
- 悬停态：颜色加深
- 点击态：Material Design涟漪动效

### 3.5 响应式适配
- **≥768px**：左侧固定侧边栏 + 右侧内容区（桌面布局）
- **<768px**：侧边栏变为底部导航栏
  - 高度：`56px`
  - 图标 + 文字标签
  - 水平排列

### 3.6 动效规范
| 动效名称 | 时长 | 应用场景 |
|---------|------|---------|
| `fadeIn` | `0.5s` | 卡片和组件渐入 |
| `pulse` | 3次循环 | 新添加题目高亮 |
| 条形增长 | `1s` | 单选题图表 |
| 折线绘制 | `1.5s` | 评分题图表 |
| 进度条 | `0.3s` | 投票提交 |
| 勾选旋转 | `0.5s` | 提交成功图标 |
| 感谢页淡入 | `1s` | 提交完成页面 |

---

## 4. 性能需求

| 指标 | 要求 |
|------|------|
| 仪表盘加载（100份投票） | ≤ 2秒 |
| 图表重新渲染（新投票推送后） | ≤ 300ms |
| WebSocket连接延迟 | ≤ 100ms |
| 短码跳转响应 | ≤ 500ms |
| CSV导出（1000条数据） | ≤ 1秒 |

---

## 5. 数据保留策略
- 投票结果保留期：`30天`
- 过期数据：自动删除
- 删除触发：后端定时任务检查过期记录

---

## 6. 术语表

| 术语 | 定义 |
|------|------|
| 短码 | 6位大写字母+数字组合，用于访问特定投票 |
| 创建者 | 创建和管理投票的用户 |
| 投票者 | 输入短码参与投票的用户 |
| 仪表盘 | 创建者查看所有投票的概览页面 |
| 实时推送 | 通过WebSocket即时推送新投票数据 |

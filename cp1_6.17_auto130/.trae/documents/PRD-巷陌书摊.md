## 1. 产品概述

巷陌书摊是一款模拟旧书摊主整理和交换图书日常的Web应用，让用户能够录入闲置书籍信息、管理书籍状态、记录摊主手记，并与其他书摊访客通过留言互动。

- 主要用途：管理个人闲置书籍、记录读书心得、模拟旧书摊社交体验
- 目标用户：爱书人士、二手书爱好者、喜欢怀旧氛围的用户
- 产品价值：为用户提供温馨的数字化旧书摊体验，让书籍管理变成一件有温度的事情

## 2. 核心功能

### 2.1 功能模块

1. **首页**：手绘书摊像素横幅、书籍展示区、翻找书摊按钮、添加书籍入口
2. **书籍卡片**：复古借书卡样式展示、悬浮动效、点击查看详情
3. **书籍详情弹窗**：完整书籍信息、可折叠摊主手记、留言板
4. **探索模式**：瀑布流布局、随机闪烁效果模拟翻找

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 手绘横幅 | CSS绘制的街角书摊像素画，营造复古氛围 |
| 首页 | 书籍展示区 | 网格布局展示书籍卡片，支持虚拟滚动 |
| 首页 | 翻找书摊按钮 | 切换到探索模式的入口按钮 |
| 首页 | 添加书籍 | 录入书名、作者、出版年份、简评等信息 |
| 详情弹窗 | 书籍信息 | 展示完整的书籍元数据和状态 |
| 详情弹窗 | 摊主手记 | textarea编辑，本地存储，可折叠展开 |
| 详情弹窗 | 留言板 | 手写风格评论，随机emoji头像 |
| 探索模式 | 瀑布流布局 | 卡片以瀑布流重新排列 |
| 探索模式 | 闪烁效果 | 每5秒随机一张卡片闪烁，模拟翻找 |

## 3. 核心流程

用户进入首页 → 浏览书籍卡片（网格布局/虚拟滚动）→ 点击卡片查看详情 → 编辑摊主手记/查看留言 → 返回首页 → 点击"翻找书摊"进入探索模式 → 瀑布流浏览+随机闪烁效果

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米黄色背景 #FDF6E3，复古棕色 #8B4513，金色渐变 #DAA520
- **辅助色**：卡片背景 #FFFEF5，装饰线 #D4C9A3，手写文字 #5D4037
- **按钮样式**：圆角50px的复古棕色按钮，hover时变为 #A0522D
- **字体**：正文使用优雅衬线字体，留言使用 Caveat 手写字体
- **布局风格**：卡片式网格布局，探索模式切换为瀑布流
- **动效风格**：温暖柔和的过渡动画（0.2s-0.3s ease/ease-out）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | 手绘横幅 | CSS像素画、暖色调、居中展示 |
| 首页 | 书籍卡片 | 260px宽、8px圆角、左右装饰线、渐变封面、悬浮上浮6px |
| 首页 | 翻找按钮 | 圆角50px、棕色背景、白色文字、过渡0.2s |
| 详情弹窗 | 遮罩层 | 半透明 #00000040、scale 0.85→1 动画0.3s |
| 详情弹窗 | 留言区 | Caveat手写字体、棕色文字、emoji头像 |
| 探索模式 | 瀑布流 | 多列布局、2px #D4C9A3 边框闪烁0.5s |

### 4.3 响应式

- Desktop-first 设计
- 移动端适配卡片尺寸和布局列数
- 确保弹窗在小屏幕上可正常滚动

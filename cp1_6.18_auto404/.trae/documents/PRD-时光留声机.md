## 1. 产品概述

时光留声机是一款复古照片处理Web应用，让用户将私藏老照片上传后，通过调节颗粒感、褪色、划痕、暗角等效果，模拟不同年代胶片相机的成像风格，最终生成带水印的数字明信片，支持在线分享与下载保存。

- 核心价值：让老照片焕发新生，通过数字技术重现胶片年代的质感与温度
- 目标用户：摄影爱好者、怀旧人群、文创创作者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 访客模式/注册 | 上传照片、使用基础滤镜、基础水印、下载图片 |
| 高级用户 | 付费升级 | 全部滤镜、自定义水印文字与颜色、高清下载 |

### 2.2 功能模块

1. **个人主页**：最近处理照片展示、照片上传入口、快速访问编辑
2. **照片编辑**：实时滤镜预览、五参数滑块调节、快速滤镜预设、水印设置
3. **导出下载**：格式选择（JPEG/PNG）、质量设置、一键下载

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 个人主页 | 顶部导航 | 应用Logo、用户头像、上传按钮 |
| 个人主页 | 最近照片 | 展示最近处理的3张照片，200x200px等比例缩略图，圆角16px |
| 个人主页 | 上传区域 | 拖拽/点击上传，支持JPG/PNG，单次最多5张，单张不超过5MB |
| 编辑页面 | 双栏预览 | 原图与效果图并排显示，最大宽800px，实时同步 |
| 编辑页面 | 参数滑块 | 颗粒感、褪色程度、划痕密度、暗角强度、色温偏移五个滑块 |
| 编辑页面 | 快速滤镜 | 复古棕、冷调蓝、暖调橙三种预设，一键应用 |
| 编辑页面 | 水印设置 | 基础版/高级版水印、位置选择、透明度调节（10%-50%） |
| 编辑页面 | 导出下载 | JPEG/PNG格式选择、150dpi输出、一键下载 |

## 3. 核心流程

```mermaid
flowchart LR
    A["进入主页"] --> B["上传照片"]
    B --> C["选择照片进入编辑"]
    C --> D["调节滑块/应用滤镜"]
    D --> E["实时预览效果"]
    E --> F["设置水印"]
    F --> G["选择格式下载"]
    G --> H["保存到本地/分享"]
```

用户上传照片后，可在主页看到最近处理的三张缩略图，点击进入编辑页。编辑页左右分栏显示原图与预览图，通过下方滑块实时调节效果，也可一键应用预设滤镜。完成编辑后设置水印样式，选择输出格式并下载。

## 4. 用户界面设计

### 4.1 设计风格

- **主背景色**：#F5E6C8（米白暖调）
- **卡片背景**：#FFF8E7（奶白色）
- **主色调**：#8B5E3C（深棕色）
- **滑块轨道**：#D4A76A（浅金棕）
- **强调色**：#FFD700（金色，用于光晕、高亮）
- **按钮**：圆角8px，底色#8B5E3C，悬停变为#A0704D
- **滑块**：平滑过渡动画 0.2s ease
- **光标特效**：可交互元素悬停时显示半径10px的淡金色光圈（#FFD700，透明度30%）
- **卡片圆角**：16px
- **字体风格**：复古衬线字体搭配现代无衬线字体，营造怀旧与现代的平衡感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 个人主页 | 顶部导航 | 深棕色Logo文字、上传按钮（金色描边）、用户头像 |
| 个人主页 | 最近照片区 | 三列卡片布局，悬停金色光晕渐显，照片等比例缩放 |
| 个人主页 | 上传区域 | 虚线边框卡片，拖拽上传提示文字，复古纸张纹理背景 |
| 编辑页面 | 预览区 | 左右双栏，原图与效果图对比，细金色分割线 |
| 编辑页面 | 滑块控制区 | 深棕色标签，金色滑块填充，平滑过渡动画 |
| 编辑页面 | 滤镜预设区 | 三个圆形色卡，点击选中态金色边框 |
| 编辑页面 | 水印设置区 | 切换按钮组、位置九宫格选择、透明度滑块 |
| 编辑页面 | 下载区 | 格式选择下拉、下载按钮（金色渐变） |

### 4.3 响应式设计

- **桌面端**：三列布局，每列照片宽280px
- **平板端**：两列布局
- **手机端**：单列布局，宽度自适应
- **触控优化**：按钮最小尺寸48px，滑块触控区域扩大

### 4.4 动效与交互

- **页面加载**：卡片渐入上浮动画（staggered）
- **滑块调节**：0.2s平滑过渡，预览图60fps实时更新
- **悬停效果**：卡片轻微上浮 + 金色光晕渐显
- **点击反馈**：按钮缩放 0.95 → 1.0 弹性动画
- **照片上传**：进度条金色填充动画

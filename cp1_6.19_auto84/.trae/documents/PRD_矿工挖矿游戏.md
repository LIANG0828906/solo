## 1. 产品概述

地下矿工挖矿与资源管理小游戏，玩家控制矿工在随机生成的矿洞中挖掘矿石、收集资源、熔炼金属、升级工具，解锁更深层级。

- 主要目的：提供休闲挖矿模拟体验，结合资源管理与工具升级的养成玩法
- 目标用户：喜欢休闲模拟类游戏玩家

## 2. 核心功能

### 2.1 功能模块

1. **矿洞场景模块**：2D俯视方格地图、随机矿道生成、矿工角色渲染
2. **挖掘交互模块**：WASD移动控制、碰撞检测、空格挖掘、矿石掉落
3. **资源背包模块**：矿石收集显示、资源计数管理
4. **熔炼系统模块**：熔炉动画、矿石拖拽熔炼、进度条、金属锭产出
5. **工具升级模块**：镐头/灯/背包升级、深层矿洞解锁

## 3. 核心流程

玩家进入游戏 → WASD控制矿工在矿洞中移动 → 走到可挖掘地块按空格挖掘矿石 → 矿石掉落聚拢到背包 → 打开熔炼面板拖拽矿石到熔炉 → 等待熔炼完成获得金属锭 → 使用金属锭升级工具 → 解锁更深层矿洞。

```mermaid
flowchart LR
    A["开始游戏"] --> B["WASD移动矿工"]
    B --> C["遇到矿石"]
    C --> D["按空格挖掘"]
    D --> E["矿石掉落聚拢"]
    E --> F["收集到背包"]
    F --> G["打开熔炼面板"]
    G --> H["拖拽矿石入熔炉"]
    H --> I["等待熔炼完成"]
    I --> J["获得金属锭"]
    J --> K["升级工具"]
    K --> L["解锁深层矿洞"]
    L --> B
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暗色调矿洞风格，背景深灰#212121到#37474F渐变
- 按钮/面板：深蓝灰#263238，圆角8px，微弱内阴影
- 动画：所有弹窗面板ease-out缓动0.3s，按钮点击scale(0.95) 0.1s
- 性能：60fps运行，交互响应延迟≤100ms

### 4.2 界面布局

| 区域 | 元素 | UI设计 |
|------|------|--------|
| 顶部 | 资源背包栏 | 水平排列4种矿石图标+计数，图标16x16像素风 |
| 中部 | 游戏主视区 | 矿洞地图+矿工角色+矿粒动画 |
| 底部 | 熔炼面板/升级面板 | 可展开面板，包含熔炉、进度条、升级按钮 |

### 4.3 响应式

桌面端优先，单屏游戏区域居中显示。

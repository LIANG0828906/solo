# 像素拓荒记 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
像素拓荒记是一款2D俯视视角的像素风生存建造小游戏，玩家在随机生成的像素世界中采集资源、建造基地、抵御夜晚怪物入侵，体验从一无所有到建立繁荣聚落的拓荒乐趣。

### 1.2 目标用户
- 喜欢像素风格游戏的休闲玩家
- 对资源管理和建造类游戏感兴趣的用户
- 喜欢轻量级、即开即玩的网页游戏用户

### 1.3 核心玩法循环
```
采集资源 → 建造建筑 → 强化基地 → 抵御怪物 → 扩张领地
```

## 2. 功能需求

### 2.1 地图生成与采集系统
| 需求编号 | 需求描述 | 优先级 |
|---------|---------|-------|
| FR-001 | 世界地图由程序生成，包含草地、泥地、岩石块三种地形 | P0 |
| FR-002 | 每个资源点包含可采集数量（木材5-15，石块3-10，食物10-20） | P0 |
| FR-003 | 玩家靠近资源点（距离<30px）并长按E键可开始采集 | P0 |
| FR-004 | 采集进度条出现在资源点上方（宽40px，高6px），背景深灰#444，填充色#4ecdc4 | P0 |
| FR-005 | 采集完成后资源数实时更新到HUD | P0 |

### 2.2 怪物入侵与防御系统
| 需求编号 | 需求描述 | 优先级 |
|---------|---------|-------|
| FR-101 | 游戏时间每过120秒进入夜晚，屏幕顶部叠加深蓝色半透明遮罩（透明度0→0.6，持续3秒） | P0 |
| FR-102 | 夜晚期间每10秒在玩家视野外生成2-3只怪物 | P0 |
| FR-103 | 怪物自动追逐玩家（速度1.2单位/秒），使用BFS寻路算法 | P0 |
| FR-104 | 怪物接触玩家时每帧扣2点生命值 | P0 |
| FR-105 | 玩家血条：满血绿色，每减少20点变为黄色，少于50点变为红色闪烁 | P0 |
| FR-106 | 防御塔攻击范围120px，攻击间隔1.5秒，自动攻击进入范围的怪物 | P0 |
| FR-107 | 防御塔弹丸为黄色圆形（直径4px），速度3单位/秒 | P0 |

### 2.3 基地建造系统
| 需求编号 | 需求描述 | 优先级 |
|---------|---------|-------|
| FR-201 | 玩家按B键打开建造菜单 | P0 |
| FR-202 | 可建造建筑：木墙（木材10）、石墙（木材5+石块5）、防御塔（木材15+石块10）、仓库（木材20+石块15） | P0 |

### 2.4 背包与资源管理系统

|---------|---------|-------|
| FR-401 | 玩家按Ctrl+S保存游戏状态到localStorage | P0 |

| FR-403 | 玩家按Ctrl+L加载最近一次保存 | P0 |

## 3. 界面设计规范

### 3.1 整体风格
- **像素风格**：所有元素采用像素化设计
- **配色方案**：
  - 背景色：深紫灰色 #1a1a2e
  - 主文字：白色 #ffffff
  - 强调色：青色 #4ecdc4
  - UI背景：深灰蓝 #2d2d3e
  - 悬停色：#3d3d5e
- **字体**：无衬线字体，粗细400

### 3.2 HUD信息栏
- 位置：屏幕底部固定
- 高度：40px
- 背景：半透明黑色（#000000，透明度80%）
- 内容：血量条、木材/石块/食物资源数量
- 每个资源区宽度：60px
- 字号：14px，白色像素风格

### 3.3 建造菜单
- 位置：弹出式，居中偏右
- 背景：#2d2d3e，圆角8px，宽度180px
- 建筑条目：每个高40px
- 悬停效果：背景变为#3d3d5e
- 选中效果：条目左侧出现青蓝色竖线

### 3.4 背包界面
- 覆盖全屏，背景半透明黑（透明度70%）
- 中央网格区域背景#2d2d3e，圆角12px，内边距16px
- 网格：8列4行，每格50x50px，边框1px虚线#555

### 3.5 夜晚效果
- 全屏覆盖遮罩，从边缘向中心渐变深蓝#0d1b2a

### 3.6 交互反馈
- 所有按钮点击时有缩放反馈（缩小0.95倍，持续0.1秒）

## 4. 性能要求

| 指标 | 要求 |
|-----|------|
| 目标帧率 | 60FPS |
| 平均帧率 | ≥55FPS |
| 最坏情况帧率（满建筑+10怪物） | ≥40FPS |
| 同时存在粒子数 | ≤50个 |
| Canvas drawImage调用次数 | ≤地图图块数+实体数 |
| 每实体绘制次数 | ≤5次 |
| localStorage单次保存大小 | ≤200KB |

---

## 5. 地图与数据规格

### 5.1 地图规格
- 地图大小：64×64图块
- 单图块尺寸：16×16像素
- 地形类型：草地、泥地、岩石

### 5.2 建筑蓝图
| 建筑类型 | 木材消耗 | 石块消耗 | 功能 |
|---------|---------|---------|-----|
| 木墙 | 10 | 0 | 基础防御，阻挡怪物 |
| 石墙 | 5 | 5 | 高耐久防御 |
| 防御塔 | 15 | 10 | 自动攻击怪物 |
| 仓库 | 20 | 15 | 存储功能 |

### 5.3 资源点数值
| 资源类型 | 数量范围 |
|---------|---------|
| 木材 | 5-15 |
| 石块 | 3-10 |
| 食物 | 10-20 |

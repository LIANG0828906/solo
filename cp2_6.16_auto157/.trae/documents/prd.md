## 1. 产品概述

CrystalMaze 是一款基于 Three.js 的3D水晶迷宫探索游戏。玩家在由半透明彩色水晶构成的迷宫中，控制发光小球从入口走到出口，同时采集散落的能量晶体触发光晕特效。目标用户为喜欢3D视觉体验和迷宫探索的玩家。

## 2. 核心功能

### 2.1 功能模块

1. **3D迷宫场景**：10x10随机生成的水晶迷宫，支持俯视与近景切换
2. **玩家控制**：WASD键盘控制发光小球移动，含碰撞检测与弹回反馈
3. **能量晶体系统**：5种不同颜色和形状的晶体，支持自转动画、悬停高亮、点击采集与粒子破碎效果
4. **HUD界面**：能量进度条、晶体计数、倒计时器、结算面板
5. **音效系统**：Web Audio API生成碰撞声与采集声
6. **胜利动画**：相机拉远、墙壁粒子消散、烟花特效
7. **隐藏区域**：集满能量后解锁通道与传送门

## 3. 核心流程

玩家进入游戏 → 迷宫随机生成 → 从入口（绿色光柱）出发 → WASD移动小球探索迷宫 → 发现并点击采集能量晶体 → 晶体破碎粒子特效 → 进度条增长 → 集满5颗解锁隐藏区域通道 → 到达出口（金色光柱）→ 触发胜利动画 → 结算面板展示

## 4. 用户界面设计

### 4.1 设计风格

- **主背景色**：#0F0F1A（深色科幻风）
- **面板色**：#1E1E30（带蓝色边框#2A2A50）
- **地面色**：#1A1A2E（深色磨砂）
- **HUD效果**：毛玻璃（背景模糊12px），半透明悬浮
- **字体**：科幻风格等宽字体
- **布局**：全屏3D场景 + 覆盖HUD

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主场景 | 迷宫墙壁 | 半透明彩色水晶块，随机颜色渐变，3秒呼吸光斑 |
| 游戏主场景 | 地面 | 深色磨砂材质 #1A1A2E |
| 游戏主场景 | 入口/出口 | 绿色/金色粒子柱旋转上升 |
| 游戏主场景 | 小球 | 直径0.5，发光，陨石噪点纹理 |
| HUD | 左上角 | 晶体计数（已采集/总数） |
| HUD | 右上角 | 3分钟倒计时 |
| HUD | 右下角 | 能量进度条 |
| HUD | 结算面板 | 采集数、用时、隐藏区域解锁 |

### 4.3 3D场景指引

- **环境**：深色太空感环境，低环境光+点光源
- **光照**：小球自带点光源，入口/出口光柱，晶体微弱自发光
- **相机**：支持俯视（远景）与近景（跟随小球）切换
- **后处理**：光晕效果（Bloom），粒子特效
- **性能预算**：稳定50fps以上，粒子爆发时无明显掉帧

## 1. 产品概述

平台跳跃关卡沙盒是一款面向独立游戏开发者的2D平台跳跃关卡原型工具，支持在浏览器中直接编辑、测试和分享关卡设计。通过可视化编辑器放置平台、敌人和收集品，一键切换至测试模式验证玩法手感，并通过后端API实现关卡数据的序列化保存与加载。

- 目标用户：独立游戏开发者、关卡设计师、游戏策划
- 核心价值：快速验证2D平台跳跃关卡的玩法手感，零门槛浏览器运行，一键分享原型

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 无需注册 | 编辑关卡、测试关卡、保存/加载关卡 |

### 2.2 功能模块

1. **编辑器页面**：关卡画布（Canvas）、实体选择面板、属性编辑浮层、保存/加载控制
2. **测试模式**：角色控制、物理模拟、碰撞检测、实时得分与生命值、游戏结束/胜利反馈

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 编辑器页面 | 关卡画布 | 32x32像素网格，支持实体放置与拖动，网格线#475569，编辑器背景#0F172A |
| 编辑器页面 | 控制面板 | 右侧320px宽面板，实体选择卡片（平台/敌人/收集品），保存/加载/测试切换按钮 |
| 编辑器页面 | 属性编辑浮层 | 选中实体后弹出，可编辑平台尺寸、敌人移动路径、收集品分数 |
| 编辑器页面 | 加载模态框 | 背景#1E2A3A，圆角16px，显示已保存关卡列表 |
| 测试模式 | HUD信息栏 | 顶部实时显示得分和生命值（3条命） |
| 测试模式 | 游戏结束遮罩 | 0.3s fade-in动画，"重新开始"按钮 |
| 测试模式 | 胜利动画 | 金色粒子从得分区域向上飘散，持续1s |

## 3. 核心流程

用户打开应用 → 在编辑器中选择实体类型 → 在画布网格上点击放置实体（带0.2s淡入） → 选中实体编辑属性 → 点击保存按钮调用API序列化关卡 → 点击测试按钮切换至测试模式（白色闪光过渡0.15s） → 控制角色移动跳跃 → 碰到敌人扣命/收集品加分 → 生命归零弹出游戏结束遮罩/收集全部显示胜利动画 → 点击重置回到起点 → 切回编辑器继续编辑

## 4. 界面设计

### 4.1 设计风格

- 主色调：深色主题，编辑器区域#0F172A，侧边栏#1E293B
- 次色调：网格线#475569，实体卡片渐变背景
- 按钮风格：圆角按钮，hover时0.2s背景色变换，cursor: pointer
- 字体：系统字体栈，标题加粗
- 布局：左侧70%画布 + 右侧320px控制面板（圆角左侧12px）
- 图标风格：彩色卡片式（宽80px高60px，圆角8px，渐变背景），hover时scale 1.05 + 阴影0 4px 12px rgba(0,0,0,0.3)

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 编辑器页面 | 关卡画布 | 深色背景#0F172A，32px网格线#475569，Canvas渲染 |
| 编辑器页面 | 控制面板 | 背景#1E293B，宽度320px，圆角左侧12px，实体选择卡片（80x60px，圆角8px，渐变背景），保存/加载/测试按钮 |
| 编辑器页面 | 属性浮层 | 半透明遮罩，居中卡片，输入框+确认按钮 |
| 编辑器页面 | 加载模态框 | 背景#1E2A3A，圆角16px，关卡列表项 |
| 测试模式 | HUD | 顶部固定栏，左侧得分，右侧生命值心形图标 |
| 测试模式 | 游戏结束 | 全屏遮罩层，0.3s fade-in，居中"重新开始"按钮 |
| 测试模式 | 胜利动画 | 金色粒子Canvas叠加层，1s向上飘散 |

### 4.3 响应式适配

- 桌面端（≥768px）：左侧70%画布 + 右侧320px控制面板
- 移动端（<768px）：编辑器占满高度，控制面板折叠为底部导航栏（高度80px）
- 实体选择卡片在移动端横向滚动排列

### 4.4 动效规范

- 实体放置：0.2s淡入动画（opacity 0→1）
- 模式切换：0.15s白色半透明遮罩闪光
- 按钮hover：0.2s背景色变换
- 实体卡片hover：scale 1.05 + 阴影0 4px 12px rgba(0,0,0,0.3)
- 游戏结束：0.3s fade-in遮罩
- 胜利：1s金色粒子向上飘散

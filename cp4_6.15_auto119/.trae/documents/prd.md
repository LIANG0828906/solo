## 1. 产品概述

面向独立手工艺人的作品集展示与在线报价生成应用。匠人可发布作品（多角度照片、材质、尺寸、工时、定价），客户浏览作品详情后选择定制选项（木料等级、雕刻复杂度、签名盒等），系统自动计算总价并生成可下载PDF报价单。解决手工艺人设计图纸分散、客户询价效率低下的核心痛点。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 访问方式 | 核心权限 |
|------|----------|----------|
| 匠人 | 直接访问 | 浏览作品、生成报价、下载报价单 |
| 客户 | 直接访问 | 浏览作品、选择定制选项、生成并下载报价单 |

### 2.2 功能模块

1. **首页画廊**：瀑布流布局展示所有已发布作品，含搜索与标签筛选
2. **作品详情页**：图片轮播、作品信息、定制选项、实时报价计算
3. **报价单生成**：PDF报价单生成与下载，含条形码与有效期脚注

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页画廊 | 瀑布流卡片 | 展示作品主图（模糊渐入加载）、名称、材料标签、起拍价；悬停上浮8px+暖色阴影，覆盖层浮现"查看详情"和"快速询价"按钮（波纹反馈） |
| 首页画廊 | 筛选标签栏 | 按材料标签筛选，切换时卡片错位依次淡入 |
| 作品详情页 | 图片轮播 | 左侧图片轮播，支持左右滑动和双指缩放，立方体翻转切换动画 |
| 作品详情页 | 作品信息 | 作品名称、材质说明、尺寸规格表 |
| 作品详情页 | 定制选项 | 下拉选择木料等级（5种）、雕刻复杂度（3种）、配件选项、加急选项；每次变更实时重算总价，价格数字滚动跳动动画 |
| 作品详情页 | 生成报价按钮 | 按钮色随总价区间变化：500以下浅绿、500~2000天蓝、2000以上琥珀渐变 |
| 报价生成 | 加载动画 | 木工锯片旋转图标，2-3秒后弹出下载对话框 |
| 报价生成 | PDF下载 | 含工作室Logo、作品大图、选项及单价明细、总价、UUID条形码、30天有效期脚注 |
| 报价生成 | Toast提示 | 下载完成后底部绿色toast"报价单已保存" |

## 3. 核心流程

用户打开首页浏览瀑布流作品卡片 → 点击卡片或"查看详情"进入作品详情页 → 左侧查看图片轮播 → 右侧查看作品信息 → 选择定制选项（木料等级/雕刻复杂度/配件/加急）→ 实时查看总价变化 → 点击"生成报价单"→ 显示锯片加载动画 → 系统调用后端计算并生成PDF → 弹出下载对话框 → 下载完成后显示绿色toast

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#D2A679（浅橡木色）、#8B5E3C（深胡桃色）、#F5F0E8（米白底色）
- **背景**：CSS重复线性渐变模拟木纹纹理
- **卡片**：圆角12px，底部柔和阴影
- **按钮**：圆角8px，过渡动画0.3s ease
- **字体**：标题使用衬线字体（Playfair Display），正文使用无衬线字体（Noto Sans SC）
- **风格**：暖色木纹质感，手工艺匠心氛围

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页画廊 | 瀑布流卡片 | 3-4列网格（大屏）、2列（平板）、1列（手机）；卡片圆角12px；图片模糊渐入；悬停上浮8px+暖色阴影；覆盖层半透明浮现按钮 |
| 首页画廊 | 筛选标签 | 横向滚动标签栏，选中态为深胡桃色填充 |
| 作品详情页 | 图片轮播 | 左侧占50%宽度，立方体翻转动画，缩放手势支持 |
| 作品详情页 | 定制选项 | 右侧表单区，下拉选择器，实时价格显示带滚动动画 |
| 作品详情页 | 生成按钮 | 全宽按钮，颜色随价格区间变化，0.3s过渡 |
| 报价预览 | 弹出层 | 半透明深色遮罩rgba(0,0,0,0.5)，底部滑入弹出卡片+弹性效果 |

### 4.3 响应式设计

- **大屏幕（>1024px）**：瀑布流3-4列，详情页左右分栏
- **平板（768-1024px）**：瀑布流2列，详情页左右分栏收窄
- **手机（<768px）**：瀑布流1列，详情页纵向单列，图片轮播占满宽度

### 4.4 动画与交互

- 卡片加载：模糊渐入效果（blur从10px到0，opacity从0到1）
- 卡片悬停：上浮8px + 暖色box-shadow
- 按钮波纹：点击时从点击位置扩散的半透明波纹
- 图片轮播：立方体3D翻转切换
- 价格变化：数字滚动跳动动画（旧数字上滑淡出，新数字下滑淡入）
- 报价弹出：底部滑入+弹性贝塞尔曲线
- 筛选切换：卡片错位依次淡入（stagger动画，每张延迟50ms）

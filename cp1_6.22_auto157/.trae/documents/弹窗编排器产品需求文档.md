
## 1. 产品概述
可视化弹窗编排与行为配置应用，帮助前端开发者快速编排和预览营销活动弹窗组件。
- 目标用户：前端开发者、营销活动运营人员
- 核心价值：减少重复劳动，提升弹窗配置效率，可视化预览弹窗效果

## 2. 核心功能

### 2.2 功能模块
1. **组件面板**：左侧展示可拖拽的弹窗类型列表（倒计时弹窗、问卷弹窗、公告弹窗）
2. **预览画布**：右侧可视化编排区域，支持放置、选中、缩放、拖拽移动弹窗实例
3. **属性配置面板**：右侧浮出面板，配置弹窗的尺寸、样式、触发行为等属性
4. **行为模拟**：预览模式下模拟弹窗触发效果（延迟弹出、滚动触发、点击触发）
5. **配置导出**：将所有弹窗配置导出为JSON格式

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 主编辑器 | 组件面板 | 展示3种弹窗类型卡片，支持拖拽到画布 |
| 主编辑器 | 预览画布 | 展示已放置的弹窗实例，支持选中、缩放、移动 |
| 主编辑器 | 属性面板 | 编辑选中弹窗的样式和行为属性 |
| 主编辑器 | 预览模式切换 | 切换编辑模式和预览模式 |
| 主编辑器 | 导出配置 | 导出JSON配置，支持复制和下载 |

## 3. 核心流程
用户从左侧组件面板拖拽弹窗类型到右侧画布 → 点击选中弹窗实例 → 在属性面板配置样式和触发行为 → 切换到预览模式测试弹窗效果 → 导出JSON配置

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：紫色系（#6366F1 到 #8B5CF6 渐变）
- 背景：浅灰背景（#F8FAFC），画布白色带浅灰棋盘格
- 卡片：白色圆角卡片，悬停效果
- 字体：现代无衬线字体
- 动效：0.2-0.3秒平滑过渡动画

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主编辑器 | 组件面板 | 240px宽，浅灰背景，卡片列表 |
| 主编辑器 | 预览画布 | 棋盘格背景，弹窗实例可拖拽缩放 |
| 主编辑器 | 属性面板 | 300px宽浮出面板，标签+输入控件布局 |
| 主编辑器 | 底部操作栏 | 预览模式切换按钮、导出配置按钮 |

### 4.3 响应式
桌面端优先，左右两栏布局

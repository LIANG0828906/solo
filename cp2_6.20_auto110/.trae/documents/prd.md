## 1. 产品概述

个人阅读书单管理与阅读轨迹可视化应用，帮助读者系统记录阅读进度、整理书摘和笔记，并直观回顾年度阅读统计与时间线。目标用户为热爱阅读的个人读者，解决阅读记录碎片化、缺乏系统性回顾与数据可视化的问题。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册，本地使用 | 全部功能（离线可用） |

### 2.2 功能模块

1. **图书管理页**：图书搜索添加、标签筛选、状态筛选、卡片网格展示
2. **图书详情页**：章节目录、笔记与书摘管理、阅读计时器、进度同步
3. **年度统计页**：月度阅读柱状图、标签分布环形图、累计阅读天数折线图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 图书管理页 | 添加图书 | 通过ISBN或书名手动添加，填充封面、作者、出版社、页数 |
| 图书管理页 | 筛选列表 | 按标签（小说、历史、科幻等）和阅读状态（想读、在读、已读）筛选 |
| 图书管理页 | 卡片网格 | 每张卡片含封面、书名、进度条、更新时间，悬停上浮动画 |
| 图书详情页 | 章节目录 | 左侧固定280px目录，可自定义章节名，浅灰色分隔线 |
| 图书详情页 | 笔记管理 | 每章节下添加笔记，支持Markdown基本格式，时间倒序，左侧彩色竖条 |
| 图书详情页 | 书摘管理 | 手动录入文本关联页码/百分比，引导引号图标 |
| 图书详情页 | 阅读计时器 | 启动/暂停计时，累计时长关联章节，自动更新进度百分比 |
| 年度统计页 | 月度阅读柱状图 | 每月阅读页数柱状图，recharts渲染，交互数据提示框 |
| 年度统计页 | 标签分布环形图 | 阅读标签分布环形图，点击可钻取明细 |
| 年度统计页 | 累计阅读天数折线图 | 累计阅读天数趋势折线图，悬停高亮数据点 |

## 3. 核心流程

用户打开应用后进入图书管理页，可通过添加按钮输入ISBN或书名添加图书。添加后图书以卡片形式展示在列表中，可按标签和状态筛选。点击卡片进入图书详情页，左侧为章节目录，右侧为当前章节的笔记和书摘列表。用户可启动阅读计时器，计时累计关联当前章节并自动更新阅读进度。导航栏年度统计入口可进入统计仪表盘，查看年度阅读数据可视化。

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：暖白 #faf5ef
- 导航栏：半透明毛玻璃效果，背景 rgba(255,248,240,0.85)，模糊12px
- 进度条渐变：从 #6a994e 到 #a7c957
- 强调色/按钮：#6a994e（自然绿）
- 截止时间标签：圆角胶囊样式
- 字体：正文使用系统字体，标题使用衬线体营造书卷气
- 布局风格：卡片网格 + 侧边栏目录
- 图标风格：线性简洁风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 图书管理页 | 导航栏 | 半透明毛玻璃，左侧Logo+名称，右侧添加按钮+统计入口 |
| 图书管理页 | 筛选栏 | 标签胶囊按钮，状态下拉选择 |
| 图书管理页 | 图书卡片 | 封面缩略图、书名、进度条（渐变填充）、更新时间、悬停上浮3px+柔和阴影 |
| 图书详情页 | 章节目录 | 左侧固定280px，浅灰色分隔线，可自定义章节名 |
| 图书详情页 | 笔记卡片 | 左侧彩色竖条（按标签颜色区分），Markdown渲染 |
| 图书详情页 | 书摘卡片 | 引导引号图标（浅灰#d4c9b0），关联页码 |
| 图书详情页 | 计时器 | 启动/暂停按钮，累计时长显示 |
| 年度统计页 | 柱状图 | 月度阅读页数，悬停数据提示框 |
| 年度统计页 | 环形图 | 标签分布，点击钻取明细 |
| 年度统计页 | 折线图 | 累计阅读天数趋势，悬停高亮 |

### 4.3 响应式

- 桌面端：完整布局，卡片三列，详情页左右分栏
- 平板端（≤768px）：导航栏折叠为汉堡菜单，卡片变两列，详情页目录转为顶部滑动选项卡
- 移动端（≤480px）：卡片单列，详情页全屏展示

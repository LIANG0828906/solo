## 1. 产品概述

厨房故事是一款面向家庭用户的智能厨房助手应用，帮助用户管理食材库存、根据现有食材智能推荐菜谱，并提供分步骤的烹饪引导体验。
- 解决家庭食材浪费和"不知道做什么菜"的痛点，目标用户为注重饮食健康的家庭烹饪爱好者
- 通过食材匹配和营养均衡算法，让每一份食材都得到充分利用，提升家庭烹饪的效率和乐趣

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家庭用户 | 无需注册 | 管理食材库存、获取菜谱推荐、使用烹饪引导 |

### 2.2 功能模块
1. **食材库存管理页**：食材增删改查、分类分组展示、过期预警、删除动画
2. **菜谱推荐页**：食材选择器、智能推荐引擎、推荐结果卡片、补购清单
3. **烹饪引导页**：步骤展示、计时器、翻页动画、完成庆祝效果、经验值累计

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 食材库存管理 | 食材列表 | 按分类分组展示所有食材，过期食材置顶并显示黄色警告条 |
| 食材库存管理 | 添加/编辑食材 | 弹窗表单，包含名称、分类、数量、单位、保质期字段 |
| 食材库存管理 | 删除食材 | 卡片向左滑出消失动画，确认删除 |
| 菜谱推荐 | 食材选择器 | 从现有库存中选择2-5种食材作为推荐依据 |
| 菜谱推荐 | 推荐结果 | 展示前5个推荐菜谱卡片，悬停上浮效果，点击展开完整信息 |
| 菜谱推荐 | 补购清单 | 缺少食材高亮显示，可一键加入补购清单 |
| 烹饪引导 | 步骤展示 | 分步骤展示文字说明和计时器 |
| 烹饪引导 | 翻页过渡 | 步骤间平滑页面翻转动画 |
| 烹饪引导 | 完成庆祝 | 小星星粒子飘落动画，自动累计烹饪经验值 |

## 3. 核心流程

用户打开应用后首先进入食材库存管理页，添加并管理自己的食材。当想做饭时，切换到菜谱推荐页，从库存中选择现有食材，系统根据匹配度、营养均衡和难度偏好综合评分，展示前5个推荐结果。用户点击感兴趣的菜谱后进入烹饪引导页，按步骤完成烹饪，每步可启动计时器，完成后获得经验值奖励。

```mermaid

    "H" --> "I: 点击菜谱展开详情"

```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暖木色（#8B6F47）配橄榄绿（#6B8E23），辅以奶油白（#FFF8F0）背景
- 按钮风格：圆角柔和按钮，悬停时微微上浮并加深阴影
- 字体：标题使用 Noto Serif SC（衬线体，温暖感），正文使用 Noto Sans SC（清晰易读）
- 布局风格：卡片式布局，顶部磨砂玻璃导航栏
- 图标风格：线性图标配以自然元素的装饰性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 食材库存管理 | 导航栏 | 磨砂玻璃效果顶部导航，半透明背景，模糊滤镜 |
| 食材库存管理 | 食材卡片 | 圆角浅灰背景，阴影，左侧色条标识分类，过期黄色警告条 |
| 食材库存管理 | 分类分组 | 每个分类用折叠标题分隔，点击展开/收起 |
| 菜谱推荐 | 食材选择器 | 圆角标签式多选，选中变绿高亮 |
| 菜谱推荐 | 推荐卡片 | 圆角卡片，悬停上浮+扩大阴影，匹配度进度条 |
| 菜谱推荐 | 缺少食材标记 | 红色高亮文字+购物袋图标 |
| 烹饪引导 | 步骤卡片 | 大字体步骤编号，文字说明，内嵌计时器 |
| 烹饪引导 | 翻页动画 | CSS 3D翻转过渡，左出右进 |
| 烹饪引导 | 完成动画 | Canvas 小星星粒子飘落，经验值弹出数字 |

### 4.3 响应式设计
- 桌面优先设计，移动端自适应
- 桌面：三栏导航+内容区，卡片网格布局（3列）
- 平板：侧边栏收起为汉堡菜单，卡片双列
- 手机：底部Tab导航，卡片单列，全宽布局，触控优化（更大的点击区域和滑动操作）

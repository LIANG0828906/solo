## 1. 产品概述

InvoiceHub 是一款面向小型企业财务人员的在线发票管理与统计分析应用，提供一站式发票录入、审核、归档服务，并自动生成按客户和月份的营收趋势图表，帮助企业高效管理发票数据、洞察经营状况。

- 核心目标：简化发票管理流程，提供直观的营收数据分析
- 目标用户：小型企业财务人员
- 市场价值：无需复杂ERP系统，轻量高效地完成发票全生命周期管理

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 财务人员 | 本地应用，无需注册 | 发票录入、编辑、审核、归档、统计分析 |

### 2.2 功能模块

1. **发票列表页**：表格展示所有发票，支持状态筛选、搜索、排序
2. **发票详情页**：查看发票详情，编辑信息，执行审核/归档操作
3. **统计分析页**：按日期范围筛选，展示营收趋势折线图和客户营收饼图

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 发票列表页 | 筛选栏 | 全部/待审核/已审核/已归档状态筛选，关键词搜索 |
| 发票列表页 | 数据表格 | 发票信息展示，列排序，点击跳转详情页 |
| 发票详情页 | 状态标签 | 展示当前发票状态（不同颜色区分） |
| 发票详情页 | 编辑表单 | 可编辑所有发票字段，含字段校验 |
| 发票详情页 | 操作按钮 | 审核通过、归档操作，含加载动画 |
| 统计分析页 | 日期选择器 | 选择统计的日期范围 |
| 统计分析页 | 营收趋势图 | 按月统计总营收的折线图 |
| 统计分析页 | 客户营收饼图 | 按客户分组统计营收占比 |
| 全局组件 | Toast提示 | 操作成功/失败的浮层提示 |
| 全局组件 | 警告条 | 数据存储异常时的底部警告条 |
| 全局组件 | 侧边导航 | 页面切换导航栏 |

## 3. 核心流程

财务人员打开应用 → 侧边导航选择功能模块 → 录入或管理发票 → 查看统计分析数据

```mermaid
flowchart TD
    A["打开应用"] --> B["加载IndexedDB数据到Store"]
    B --> C["侧边导航选择模块"]
    C --> D["发票列表页"]
    C --> E["统计分析页"]
    D --> F["筛选/搜索/排序发票"]
    D --> G["点击行跳转详情页"]
    G --> H["编辑发票信息"]
    G --> I["审核通过/归档操作"]
    H --> J["字段校验"]
    J --> K["保存到IndexedDB并更新Store"]
    I --> L["更新状态并同步存储"]
    K --> M["Toast成功提示"]
    L --> M
    M --> N["返回列表页"]
    E --> O["选择日期范围"]
    O --> P["渲染趋势折线图和客户饼图"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#3498DB（蓝色），#27AE60（绿色成功），#E74C3C（红色错误），#F39C12（橙色待审核）
- **背景色**：主背景 #F5F6FA，卡片背景 #FFFFFF，侧边栏 #2C3E50
- **文字色**：正文 #2C3E50，辅助文字 #7F8C8D
- **按钮风格**：圆角4px，悬停变暗，过渡0.2s ease
- **字体**：Google Fonts Inter
- **布局风格**：侧边导航 + 主内容区，卡片式布局
- **阴影效果**：卡片阴影 0 2px 8px rgba(0,0,0,0.08)，悬停加深
- **图标风格**：简洁线条图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局布局 | 侧边导航 | 220px宽，深色背景，选中项左侧蓝色竖条，背景变深 |
| 全局布局 | 主内容区 | 左右边距40px，上下边距30px，页面切换fade-in动画 |
| 发票列表页 | 筛选按钮 | 选中态#3498DB背景白字，未选中态透明背景灰字 |
| 发票列表页 | 数据表格 | 表头#F8F9FA固定，条纹交替行，行高40px，悬停#EBF5FB |
| 发票列表页 | 排序列头 | 点击显示上下箭头，颜色#7F8C8D |
| 发票详情页 | 状态标签 | 待审核#F39C12、已审核#27AE60、已归档#95A5A6 |
| 发票详情页 | 表单字段 | 边框#BDC3C7，聚焦#3498DB，内阴影0.5px |
| 发票详情页 | 操作按钮 | 保存#27AE60，悬停#219A52，含加载Spinner |
| 统计分析页 | 日期选择器 | 原生date picker，边框#BDC3C7，聚焦#3498DB |
| 统计分析页 | 图表容器 | 白色背景，20px留白，圆角8px |
| 全局组件 | Toast | 绿色浮层，0.3s淡入，停留2s，1s淡出 |
| 全局组件 | 警告条 | 底部红色条36px高，可关闭 |

### 4.3 响应式

- 桌面端优先设计，侧边栏固定220px宽度
- 小屏幕可折叠侧边栏，主内容区域自适应
- 表格区域支持横向滚动

### 4.4 动效设计

- 页面切换：0.3s opacity 0→1 fade-in
- 按钮悬停：0.2s ease 颜色过渡
- 卡片悬停：0.2s ease 阴影加深
- 表格行悬停：0.2s ease 背景色变为浅蓝
- Toast消息：0.3s淡入，停留2s，1s淡出

## 1. 产品概述

在线活动门票预订与管理平台，面向小型活动主办方和参与者，提供活动创建、售票发布、座位选择、电子门票生成与订单管理的全流程服务。主办方可以高效创建活动并管理座位排布，用户可以便捷浏览活动、在线选座购票并获取含二维码的电子门票。

- 目标用户：小型活动主办方（会议、演出、展览等）及活动参与者
- 核心价值：一站式活动售票解决方案，零门槛创建活动，直观座位选择体验，即时电子门票交付

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 活动主办方 | 无需注册（演示模式） | 创建活动、设置座位排布与票价、管理售票 |
| 普通用户 | 无需注册（演示模式） | 浏览活动、搜索筛选、选座购票、查看电子门票、查看订单历史 |

### 2.2 功能模块

1. **活动列表页（首页）**：活动卡片网格展示、日期范围筛选、名称关键词搜索、筛选淡入动画
2. **活动创建页**：主办方表单填写、座位排布设置、票价配置
3. **活动详情页**：活动介绍、交互式座位图、购物车面板、购票确认
4. **电子门票页**：门票信息展示、Canvas二维码生成、PNG下载
5. **订单历史页**：已购门票列表、时间排序、门票详情回看

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 活动列表页 | 活动卡片网格 | 展示所有活动的卡片网格，每张卡片含渐变海报缩略图（纯色渐变+活动名首字母）、日期、票价区间，CSS Grid响应式布局（桌面4列/平板2列/手机1列） |
| 活动列表页 | 搜索筛选栏 | 按日期范围筛选、按名称关键词搜索，筛选结果0.3秒淡入动画过渡 |
| 活动创建页 | 活动信息表单 | 填写活动名称、描述、日期时间、场馆地址、座位排布（行列数）、票价，提交后生成活动ID |
| 活动详情页 | 活动信息展示 | 显示活动名称、描述、日期时间、场馆地址 |
| 活动详情页 | 交互式座位图 | 按比例缩放的座位图，空闲绿色/已售灰色，悬停高亮+座位号提示，点击选座加入购物车（蓝色边框闪烁），自适应容器宽度 |
| 活动详情页 | 购物车面板 | 右侧面板显示已选座位数量和总价，确认购票调用API，成功后座位图实时更新 |
| 电子门票页 | 门票信息卡片 | 淡绿色边框+轻微阴影，显示活动名称、日期、场馆、座位号（第X排第X列）、订单编号、购票时间 |
| 电子门票页 | 二维码生成 | 基于Canvas生成二维码（内容为订单编号），点击下载按钮保存门票区域为PNG图片 |
| 订单历史页 | 订单列表 | 展示所有已购门票卡片，含活动信息缩略和座位详情，按订单时间倒序排列，点击可查看电子门票 |

## 3. 核心流程

**购票主流程**：用户浏览活动列表 → 搜索/筛选找到目标活动 → 进入活动详情页查看座位图 → 点击空闲座位选座 → 确认购物车 → 提交购票 → 跳转电子门票页 → 查看含二维码的门票 → 可下载PNG

**活动创建流程**：主办方点击创建活动 → 填写活动信息表单 → 设置座位行列和票价 → 提交保存 → 自动生成活动ID → 活动出现在列表中

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：蓝绿渐变（#2C9EAF → #3B82F6）
- 页面背景：浅灰色 #F5F7FA
- 卡片样式：白色背景，柔和阴影，圆角 12px
- 按钮样式：主色填充，悬停饱和度加深并上移2px，0.2秒transition
- 字体：正文14-16px，标题20-28px，使用系统无衬线字体栈
- 布局风格：顶部Sticky导航栏（毛玻璃效果），内容区域最大宽度1200px居中
- 图标风格：线性图标（lucide-react）
- 交互反馈：所有可交互元素均有hover反馈和0.2秒transition

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 活动列表页 | 导航栏 | 半透明毛玻璃效果，品牌Logo，搜索框，"创建活动"按钮，"我的订单"链接 |
| 活动列表页 | 筛选栏 | 日期范围选择器，关键词搜索输入框 |
| 活动列表页 | 活动卡片 | 白色卡片，渐变海报区（纯色渐变+首字母），日期和票价区间，hover上浮阴影加深 |
| 活动创建页 | 创建表单 | 居中表单卡片，输入字段带标签，行列数输入，票价输入，主色提交按钮 |
| 活动详情页 | 信息区 | 活动名称大标题，描述文本，日期时间图标行，场馆地址图标行 |
| 活动详情页 | 座位图 | 自适应宽度座位网格，绿色空闲/灰色已售方块，hover高亮+tooltip，蓝色选中边框闪烁 |
| 活动详情页 | 购物车面板 | 右侧固定面板，已选座位列表，总价，确认购票按钮 |
| 电子门票页 | 门票卡片 | 淡绿色边框(#D1FAE5)，轻微阴影，活动信息排列，Canvas二维码区域，下载PNG按钮 |
| 订单历史页 | 订单卡片 | 白色卡片，活动信息缩略，座位详情，时间标签，点击hover高亮 |

### 4.3 响应式设计

- 桌面优先设计策略
- 活动卡片网格：桌面4列（≥1024px）、平板2列（≥640px）、手机1列（<640px）
- 座位图区域自适应容器宽度，座位方块尺寸响应式变化
- 导航栏移动端简化
- 购物车面板在移动端变为底部弹出面板

### 4.4 页面过渡动画

- 购票确认：slide-fade动画过渡
- 门票生成：slide-fade动画过渡
- 筛选结果切换：0.3秒淡入动画
- 座位选中：蓝色边框闪烁一次
- 卡片hover：0.2秒上浮+阴影加深

## 1. 产品概述
社区支持农业（CSA）管理系统，帮助农场管理者规划和追踪整个农业生产周期，解决土地规划、种植排期、会员配送等各环节信息分散、无法协同的问题，提供直观的收益预测和风险评估能力。

- 核心目标：整合农场规划、会员管理、收益预测三大模块，实现农业生产全生命周期可视化管理
- 目标用户：社区农场管理者、农业合作社运营人员

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 农场管理者 | 系统内置 | 完整的规划、会员、数据管理权限 |

### 2.2 功能模块
1. **农场规划模块**：种植季管理、地块规划、泳道图可视化、生长阶段追踪
2. **会员订阅与配送模块**：会员信息管理、套餐订阅、配送历史、状态管理
3. **收益预测与风险评估模块**：预期收益柱状图、天气风险仪表盘、虫害风险评估

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 规划大屏 | 农场规划模块 | 泳道图展示地块卡片，拖拽排序，生长阶段标签，悬停显示详情 |
| 规划大屏 | 风险评估模块 | Canvas绘制风险仪表盘，显示干旱/霜冻概率，虫害风险等级 |
| 会员面板 | 会员订阅模块 | 网格展示会员头像卡片，按状态筛选，翻转动画切换状态 |
| 会员详情 | 会员订阅模块 | 底部滑入面板，显示配送历史、下次配送日期、偏好备注 |
| 收益预测 | 收益预测模块 | Canvas绘制周度收益柱状图，渐变填充，浮动数值标签 |

## 3. 核心流程

### 3.1 农场规划流程
管理者创建种植季 → 设置季期起止日期 → 添加地块信息（面积、土壤、作物、预计产量）→ 拖拽调整地块排序 → 系统自动计算生长阶段 → 实时更新预计收获日期

```mermaid
flowchart LR
    A["创建种植季"] --> B["设置季期日期"]
    B --> C["添加地块信息"]
    C --> D["拖拽排序地块"]
    D --> E["系统计算生长阶段"]
    E --> F["实时更新收获预测"]
```

### 3.2 会员管理流程
管理者添加/导入会员 → 设置订阅套餐类型 → 分配配送时段 → 管理会员状态（活跃/暂停/取消）→ 查看配送历史 → 记录偏好备注

```mermaid
flowchart LR
    A["添加会员信息"] --> B["设置订阅套餐"]
    B --> C["分配配送时段"]
    C --> D["管理会员状态"]
    D --> E["查看配送历史"]
    E --> F["记录偏好备注"]
```

### 3.3 收益预测流程
系统读取种植计划 → 结合历史产量数据 → 按周计算预期收益 → 生成柱状图 → 分析天气和虫害风险 → 输出风险评分

```mermaid
flowchart LR
    A["读取种植计划"] --> B["结合历史产量"]
    B --> C["按周计算收益"]
    C --> D["生成收益柱状图"]
    D --> E["分析风险因素"]
    E --> F["输出风险评分"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：草绿色 #8BC34A，大地棕色 #795548
- **辅助色**：叶菜绿 #4CAF50，根菜橙 #FF9800，果实红 #F44336
- **背景**：白色 #FAFAFA，带轻微纸张纹理
- **文字**：深灰 #333333
- **卡片风格**：12px圆角，柔和阴影
- **按钮风格**：圆角8px，悬停微放大效果
- **字体**：Roboto 字体家族，标题粗体，正文常规
- **布局风格**：左侧固定侧边栏（240px）+ 右侧自适应主内容区
- **动效风格**：平滑过渡0.3s，卡片悬停放大1.08倍，状态切换翻转动画0.5s

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|----------|----------|-------------|
| 规划大屏 | 泳道图 | 拖拽排序、彩色卡片、生长阶段标签、悬停放大、淡入过渡 |
| 规划大屏 | 风险仪表盘 | Canvas圆形弧段、动态颜色、中央数字、渐变填充 |
| 会员面板 | 会员网格 | 4列网格、圆形头像、首字母生成、渐变背景、状态筛选 |
| 会员详情 | 滑入面板 | 半透明遮罩、底部滑入、0.3s ease-out过渡 |
| 收益图表 | 柱状图 | Canvas绘制、周度X轴、渐变柱体、浮动数值标签 |
| 全局导航 | 侧边栏 | 深色背景#3E2723、Logo、草绿色指示条、悬停效果 |
| 全局导航 | 响应式 | 768px以下折叠为顶部导航、4列变2列 |

### 4.3 响应式设计
- **桌面端（>768px）**：左侧240px固定侧边栏，主内容区4列卡片布局
- **移动端（≤768px）**：顶部导航栏，主内容区2列卡片布局
- **触摸优化**：卡片点击区域≥48px，滑动手势支持

### 4.4 性能指标
- 会员搜索筛选响应延迟 < 200ms
- 收益图表渲染 ≤ 150ms
- 首次加载总时间 ≤ 1.5秒（懒加载+代码分割）

## 1. 产品概述
手工艺作品记录与展示平台，帮助陶艺、木工、编织等手工艺人系统性地记录和展示从原料到成品的创作过程，解决粉丝无法清晰了解制作难度和耗时的痛点。

- 核心目标：为手工艺人提供专业的作品创作过程记录工具，提升作品展示的专业性和粉丝互动体验
- 目标用户：手工艺创作者（陶艺、木工、编织、皮艺、纸艺）及手工艺爱好者
- 市场价值：填补手工艺领域专业创作过程记录工具的空白，建立创作者与粉丝之间的深度连接

## 2. 核心功能

### 2.1 用户角色
| 角色 | 核心权限 |
|------|----------|
| 创作者 | 创建/编辑作品、管理步骤、查看统计数据、开启评论 |
| 访客 | 浏览作品、查看创作步骤、评分留言 |

### 2.2 功能模块
1. **个人主页**：作品统计概览（作品总数、总耗时、平均评分、总收藏数）、作品卡片网格展示
2. **作品创建**：作品信息录入（名称、类别）、步骤添加（图片、描述、耗时）、步骤拖拽排序
3. **作品详情**：作品信息展示、步骤流程展示（带连接线）、评论板块（评分+留言）
4. **评论系统**：星级评分、文字留言、评论列表（时间倒序、滑入动画）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 个人主页 | 统计面板 | 四个数字卡片展示核心数据，数字从0递增动画 |
| 个人主页 | 作品网格 | 响应式卡片网格，hover发光效果，点击跳转详情 |
| 作品创建页 | 作品表单 | 名称输入、类别选择（带材质色块标签） |
| 作品创建页 | 步骤管理 | 步骤卡片添加、拖拽排序、删除操作 |
| 作品详情页 | 作品头部 | 作品名称、类别标签展示 |
| 作品详情页 | 步骤流程 | 带圆弧虚线连接的步骤卡片序列 |
| 作品详情页 | 评论板块 | 星级评分、留言输入、评论列表动画展示 |

## 3. 核心流程
用户进入个人主页查看作品统计 → 点击创建新作品 → 填写作品信息和制作步骤 → 拖拽调整步骤顺序 → 提交生成作品页面 → 访客浏览作品 → 查看详细步骤 → 评分并留言。

    F --> G["作品详情页"]

    H --> I["查看步骤"]
    I --> J["评分留言"]

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖色调视觉风格，背景极浅米白 #FDF5E6
- **类别色彩**：陶艺暖棕 #D2B48C、木工浅木 #DEB887、编织草绿 #8FBC8F、皮艺深棕 #8B4513、纸艺米白 #F5F5DC
- **文字色彩**：标题深咖啡色 #3E2723，正文深灰 #424242
- **卡片样式**：圆角 16px，软阴影 0px 8px 24px rgba(0,0,0,0.08)，内边距 24px
- **动画缓动**：统一使用 cubic-bezier(0.4, 0, 0.2, 1)
- **按钮交互**：点击产生水波纹动画（0.4s 圆形扩散）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 个人主页 | 统计卡片 | 四种不同背景色卡片，数字递增动画，hover 发光 |
| 个人主页 | 作品网格 | 响应式布局（桌面3列/平板2列/移动1列），卡片阴影和发光 |
| 作品详情 | 步骤流程 | 圆弧虚线连接步骤卡片，拖拽旋转放大效果 |
| 作品详情 | 评论卡片 | 从右侧滑入动画，随机色圆形头像，星级缩放动画 |

### 4.3 响应式设计
- **桌面端**（>=1024px）：三列网格布局，完整导航栏
- **平板端**（768-1023px）：两列网格布局，简化导航
- **移动端**（<768px）：单列布局，汉堡菜单，隐藏顶部导航
- 所有交互元素针对触摸操作优化，按钮最小尺寸 44px

### 4.4 动画与交互
- **步骤拖拽**：拖拽时卡片旋转 2deg、阴影放大 1.05 倍，0.2s 过渡
- **星级评分**：点击星星缩放 1.15 倍并填充金色 #FFD700，0.2s ease-out
- **类别标签**：悬停时轻微隆起阴影效果
- **评论滑入**：每条评论从右侧滑入，0.3s ease-out
- **数字动画**：统计数字从 0 递增到实际值，1s ease-out
- **卡片发光**：hover 时边缘基于类别色发光，模糊半径 6px，0.3s ease

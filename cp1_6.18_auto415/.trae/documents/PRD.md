## 1. 产品概述

词影留声是一个音画同步朗诵创作平台，用户可录制朗读音频，系统自动转写为文字，配合情感驱动的图片滤镜效果，生成可分享的沉浸式视听作品。

- 解决的问题：传统音频分享缺乏视觉维度，无法体现文字与声音的同步美感
- 目标用户：文学爱好者、朗诵爱好者、有声书创作者、教育工作者
- 产品价值：将声音、文字、图像与情感融为一体，打造沉浸式的视听体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 录制音频、创建作品、编辑作品、生成分享链接 |

### 2.2 功能模块

1. **用户认证模块**：注册、登录、会话管理
2. **个人控制台**：作品列表展示、新建作品、删除作品
3. **作品编辑页**：音频波形可视化、文本时间轴编辑、图片管理、情感分析
4. **播放预览**：音频播放、文本同步高亮、图片情感滤镜
5. **分享页面**：只读展示、播放控制、品牌水印

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |

| 注册页 | 注册表单 | 邮箱/密码注册、跳转登录 |
| 控制台 | 作品列表 | 瀑布流展示、新建作品入口 |
| 编辑页 | 音频波形 | 可视化显示、播放进度指示 |
| 编辑页 | 文本时间轴 | 逐句时间戳编辑、误差≤200ms |
| 编辑页 | 图片管理 | 上传JPG/PNG（≤5MB）、裁剪16:9 |

| 分享页 | 播放控制 | 进度条、播放/暂停、音量控制 |

## 3. 核心流程

用户注册登录后，在个人控制台创建新作品，录制或上传音频（最长5分钟），系统自动转写文字。用户可调整每句话的时间戳，上传配图后系统进行情感分析。播放时文字随音频逐词高亮，图片根据情感关键词动态应用滤镜。最后生成分享链接供他人访问。

## 4. 界面设计

### 4.1 设计风格

- **色彩系统**：深色主题
  - 主背景：#0D0D1A
  - 卡片背景：#1A1A2E
  - 主文字：#E0E0E0
  - 强调色1：#4ECDC4（青绿色）
  - 强调色2：#FF6B6B（珊瑚红）
  - 高亮色：#FFD700（金色）
- **圆角规范**：卡片16px，图片封面12px
- **字体**：展示字体采用 Playfair Display，正文字体采用 Noto Sans SC
- **导航栏**：固定顶部60px，毛玻璃效果#FFFFFF10，底部边框#FFFFFF15
- **动效**：
  - 悬停：0.2秒缩放+透明度过渡
  - 点击：0.1秒振动缩放（scale 0.95→1）
  - 滤镜切换：0.5秒渐变过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 控制台 | 作品卡片 | 16:9封面、标题、时长、创建时间、悬停上浮效果 |
| 控制台 | 瀑布流布局 | 桌面3列、平板2列、手机1列 |
| 编辑页 | 波形区域 | 高120px、背景#1A1A2E、Canvas频率条、#4ECDC4→#FF6B6B渐变 |
| 编辑页 | 文本编辑 | 时间轴形式、可拖动调整、误差≤200ms |
| 编辑页 | 左右布局 | 6:4比例、平板以下堆叠 |
| 播放预览 | 文本高亮 | #FFD700、字号1.1倍、逐词高亮 |
| 播放预览 | 情感滤镜 | 喜悦：#FFA50015柔光；悲伤：#4A90D915冷调；激昂：对比度+15% |
| 分享页 | 控制栏 | 深色半透明#00000080、圆形按钮40px、#FF6B6B背景、悬停放大110% |
| 分享页 | 音量滑块 | 轨道4px圆角2px、滑块16px |
| 分享页 | 水印 | "由词影留声生成"、12px、透明度30%、底部固定 |

### 4.3 响应式

- 桌面优先设计，适配平板和移动端
- 作品列表：桌面3列 → 平板2列 → 手机1列
- 编辑页：左右6:4布局 → 平板以下上下堆叠
- 触摸设备：增大可点击区域，优化手势操作

### 4.4 性能指标

- 音频解码与波形渲染：≤0.5秒
- 情感分析响应：≤200ms
- 页面加载：首屏≤2秒

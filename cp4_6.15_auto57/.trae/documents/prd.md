## 1. 产品概述

数字故事板（Storyboard Studio）是一款面向内容创作者的轻量级视觉化灵感整理工具，帮助用户快速创建、管理和分享个性化数字故事板。通过卡片化素材管理、时间轴叙事模式、一键分享与导出，解决创作者在收集灵感素材时缺乏直观、可交互的视觉化整理工具的问题。

- 目标用户：内容创作者、设计师、编剧、策划人员
- 核心价值：将零散灵感素材以视觉化卡片形式组织，支持时间轴叙事与协作分享

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 创作者 | 创建和管理故事板的用户 | 完整的创建、编辑、删除、分享、导出权限 |
| 访客 | 通过分享链接访问的用户 | 仅查看，不可编辑 |

### 2.2 功能模块

1. **故事板列表页**：展示所有故事板封面卡片网格，创建新故事板入口
2. **故事板详情页**：素材管理、时间轴视图切换、统计信息、导出功能
3. **分享查看页**：只读模式的故事板展示页面

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 故事板列表页 | 封面卡片网格 | 以网格形式展示所有故事板封面，每个卡片带渐变色背景、标题和描述，点击进入详情 |
| 故事板列表页 | 创建入口 | 顶部创建按钮，点击弹出输入标题和描述的弹窗，生成故事板封面卡片 |
| 故事板详情页 | 素材管理区 | 上传本地图片（jpg/png/webp，≤5MB）或输入外链URL插入素材卡片，瀑布流排列，拖拽排序，悬停操作（放大/删除/备注） |
| 故事板详情页 | 视图切换 | 自由排列模式与水平时间轴模式切换 |
| 故事板详情页 | 时间轴模式 | 素材沿渐变彩色时间线从左到右排列，可拖动指针定位节点，虚线连线，选中卡片脉冲光晕，点击节点弹性弹出备注 |
| 故事板详情页 | 统计信息 | 右上角显示素材总数、创建天数、最近编辑时间 |
| 故事板详情页 | 分享功能 | 生成随机短码分享链接，复制带toast反馈 |
| 故事板详情页 | 导出功能 | 一键html2canvas截取为PNG长图下载，loading遮罩层 |
| 分享查看页 | 只读展示 | 顶部显示原作者昵称和创建天数，相同颜色主题，不可编辑 |

## 3. 核心流程

用户打开应用 → 在列表页创建新故事板（输入标题和描述） → 进入故事板详情页 → 上传图片或输入URL添加素材卡片 → 拖拽调整素材顺序 → 切换到时间轴模式查看叙事脉络 → 点击时间轴节点查看备注详情 → 生成分享链接或导出长图

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：柔和蓝紫渐变 #667eea → #764ba2
- 辅助色：纯白 #FFFFFF、浅灰 #F5F5F7、深灰 #333333
- 卡片圆角：12px，轻微阴影
- 字体：Poppins（英文/标题）、Noto Sans SC（中文正文）
- 布局：卡片化设计，瀑布流素材排列
- 动效：拖拽弹性反馈、悬停上浮动画、脉冲光晕、弹性弹出、200ms hover过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 故事板列表页 | 顶部导航 | 渐变背景头部，应用标题，创建按钮，hover半透明到实色过渡 |
| 故事板列表页 | 封面卡片网格 | 3列网格布局，每张卡片渐变色背景+标题+描述+素材数，hover上浮+阴影加深 |
| 故事板详情页 | 工具栏 | 返回按钮、视图切换（自由/时间轴）、分享按钮、导出按钮、统计信息 |
| 故事板详情页 | 素材卡片 | 瀑布流排列，图片+操作按钮（悬停浮现），骨架屏加载态，拖拽弹性反馈 |
| 故事板详情页 | 时间轴 | 渐变彩色时间线，圆形节点图标，虚线连线，选中脉冲光晕，节点点击弹性动画 |
| 分享查看页 | 页头 | 原作者昵称+创建天数，相同蓝紫渐变主题 |

### 4.3 响应式适配

- 桌面端：多列瀑布流，水平时间轴
- 移动端：单列瀑布流，时间轴改为垂直布局，触控优化拖拽

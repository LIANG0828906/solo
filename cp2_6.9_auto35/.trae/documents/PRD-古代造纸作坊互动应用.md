## 1. 产品概述

一款基于浏览器的古代造纸作坊互动体验应用，让用户身临其境地体验明代宣纸匠人抄纸、晒纸与成品质检的完整工艺流程。
通过直观的交互和精美的动画效果，传承中华传统造纸文化，寓教于乐。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 体验用户 | 无需注册 | 完整体验造纸工艺流程，查看历史记录 |

### 2.2 功能模块

1. **主场景页面**：纸浆调配、抄纸捞纸、压榨晾晒、成品质检
2. **历史记录模块**：最近10次质检记录展示
3. **重置功能**：一键重新开始新批次

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 主场景 | 纸浆调配区 | 通过楮皮、桑皮、麻纤维三种原料按钮调整纸浆浓度（0-100%） |
| 主场景 | 抄纸区 | 点击抄纸帘触发捞纸动画，纸帘平移沥水形成湿纸坯 |
| 主场景 | 压榨区 | 点击压榨杠杆挤出水分，水渍动画反馈 |
| 主场景 | 晒纸区 | 拖拽纸坯到晒纸墙，自动晾晒变色，干燥进度条显示 |
| 主场景 | 质检区 | 点击纸张10处随机位置进行触感检测，波纹动画反馈，综合评分评级 |
| 主场景 | 历史记录 | 底部仿古卷轴样式展示最近10条质检记录，倒序排列 |
| 主场景 | 重置按钮 | 右上角圆形按钮，一键重置所有状态 |

## 3. 核心流程

用户从纸浆调配开始，依次经过抄纸、压榨、晒纸、质检四个环节，形成完整的造纸工作流。

```mermaid
flowchart LR
    A["纸浆调配<br/>调整原料浓度"] --> B["抄纸捞纸<br/>点击抄纸帘"]
    B --> C["纸坯转移<br/>自动到压榨台"]
    C --> D["压榨脱水<br/>点击压榨杠杆"]
    D --> E["晒纸干燥<br/>拖拽到晒纸墙"]
    E --> F["成品质检<br/>点击10处检测"]
    F --> G["评级记录<br/>分数与徽章"]
    G --> H["查看历史<br/>底部卷轴列表"]
    H -->|重新配料| A
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖米色 #e8dcc8（背景）、土黄色 #c4a882（辅色）、深红色 #8b3a3a（强调色）
- **配色方案**：整体采用古代纸坊暖色调，营造古朴典雅的氛围
- **按钮样式**：圆角矩形，悬停缩放1.05倍，阴影加深；点击按压至0.95倍
- **字体**：思源宋体（Source Han Serif），传承古典书卷气息
- **布局风格**：工作区按顺时针环形排列（纸槽→抄纸→压榨→晒纸→质检），形成流畅的工作流
- **视觉元素**：木质纹理、竹编纹理、羊皮纸质感，仿古卷轴样式的历史记录

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|----------|--------|
| 主场景 | 纸槽 | 深棕色 #6b3a2a 边框，灰白色纸浆悬浮液 |
| 主场景 | 原料按钮 | 楮皮 #c49a6c、桑皮 #d4b08a、麻纤维 #b8a088 三色按钮 |
| 主场景 | 抄纸帘 | 竹编纹理矩形框，半透明，捞纸平移动画1.2秒 |
| 主场景 | 压榨台 | 木质手柄杠杆，下压动画0.5秒，水渍渗出动画2秒 |
| 主场景 | 晒纸墙 | 土黄色 #c4a882 墙面，竖排木格栅，晾晒渐变4秒 |
| 主场景 | 质检台 | 木纹桌面 #8b6b4a，点击波纹扩散动画，数字滚动评分0.6秒 |
| 主场景 | 评级徽章 | 金 #ffd700、银 #c0c0c0、铜 #cd7f32 三色边框 |
| 主场景 | 历史记录 | 羊皮纸色 #f5deb3 卷轴样式，淡入动画0.3秒 |
| 主场景 | 重置按钮 | 深红色 #8b3a3a 圆形按钮，悬停放大1.1倍 |

### 4.3 响应式设计
- **桌面端**（≥768px）：工作区按顺时针环形排列
- **移动端**（<768px）：工作区改为垂直排列，各区域高度自适应
- **触摸优化**：拖拽操作支持触摸事件，按钮最小尺寸48px

### 4.4 动画与交互
- 所有动画使用CSS transform和opacity实现，避免重排
- 悬停效果：缩放+阴影变化，transition 0.2s ease
- 点击反馈：transform scale(0.95)，持续0.1s
- 转场动画：重置时各元素淡出0.5秒后淡入初始状态
- 帧率要求：单次操作帧率保持在55fps以上

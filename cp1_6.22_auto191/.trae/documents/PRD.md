## 1. 产品概述

太空飞船引擎尾焰粒子系统实时编辑与预览工具，解决游戏开发中尾焰效果难以可视化调整的问题。目标用户为游戏开发者和视觉设计师，提供直观的参数调节界面和实时预览画布，支持预设保存与导出。

## 2. 核心功能

### 2.1 功能模块
1. **参数控制面板**：6个可调节滑块控制粒子发射参数
2. **实时预览画布**：Canvas 2D 渲染飞船模型与粒子尾焰
3. **预设管理系统**：保存、加载、删除命名预设，本地存储
4. **视角控制器**：鼠标拖拽旋转、滚轮缩放
5. **配置导出**：导出JSON配置到剪贴板

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主界面 | 控制面板 | 6个参数滑块（发射速率、粒子寿命、粒子速度、粒子大小、扩散角度、重力影响） |
| 主界面 | 预设列表 | 卡片式展示预设，支持加载、删除、保存新预设 |
| 主界面 | 导出按钮 | 一键导出所有参数和预设为JSON |
| 主界面 | 预览画布 | 实时渲染飞船与粒子系统，支持视角控制 |

## 3. 核心流程

用户打开应用 → 查看默认粒子效果 → 拖动滑块调整参数（实时预览）→ 满意后点击保存预设 → 输入名称保存 → 切换预设查看不同效果 → 点击导出配置复制到剪贴板

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：深空科技风，深色背景配合霓虹渐变
- **主色**：背景 #0F172A，面板 #1E293B，滑块轨道 #334155
- **强调色**：滑块填充 #38BDF8 → #818CF8 渐变，按钮 #6366F1 → #8B5CF6 渐变
- **文字**：标签 #CBD5E1，数值 #60A5FA，标题白色
- **粒子颜色**：中心黄 #FFD700 → 橙 #FF6347 → 半透明红 #FF4500 (alpha 0.3)
- **字体**：等宽字体用于参数标签，12px标签，14px按钮文字
- **动效**：所有过渡 200ms ease-out，预设切换 0.5s 淡入淡出

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|-------|
| 主界面 | 左侧控制面板 | 宽320px，圆角12px，内边距16px，深色背景 |
| 主界面 | 参数滑块 | 轨道高6px圆角3px，圆形手柄直径20px白色带阴影 |
| 主界面 | 预设卡片 | 宽100px高80px，圆角8px，深色#2D3748，标题白色12px |
| 主界面 | 预览画布 | 占右侧70%宽度（最小600px），星点闪烁背景 |
| 主界面 | 按钮 | 渐变背景，白色文字，悬浮上移2px加深阴影 |

### 4.3 响应性
- 桌面端优先设计
- 画布最小宽度600px
- 左侧面板固定宽度320px

### 4.4 视觉效果
- 画布背景随机闪烁星点（CSS动画，1-3px大小，0.5-2秒闪烁周期）
- 飞船由简单几何体组合（机翼机身#A0A0A0，尾喷口红色半透明圆环）
- 粒子呈随机圆锥分布，颜色从中心向外渐变
- 视角操作期间粒子发射不中断

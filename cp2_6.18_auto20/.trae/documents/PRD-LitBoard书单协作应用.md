## 1. 产品概述

LitBoard 是一个面向线上读书会社群的书单协作与排行应用，帮助社群成员创建、分享主题书单，通过评论、评分和点赞功能形成互动，并以实时热度排名激发社群活力。

- 核心价值：为读书会社群提供一个沉浸式的书单交流平台，通过热度排行机制鼓励优质内容创作和社群互动
- 目标用户：读书会社群成员、书籍爱好者、阅读推广者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 社群成员 | 自动识别（本地存储模拟） | 创建书单、发表评论、评分、点赞、浏览排行 |

### 2.2 功能模块
1. **书单列表页（首页）**：热度排行榜、书单卡片展示、创建书单入口
2. **书单详情页**：书单详情展示、评论列表、评分表单、点赞功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书单列表页 | 热度排行榜 | 展示热度前10的书单，每30秒自动刷新，位置变化有平滑动画 |
| 书单列表页 | 书单卡片 | 280px宽卡片，展示标题、描述、标签、热度值、评分，点击进入详情 |
| 书单列表页 | 创建书单表单 | 输入标题、描述（≤500字）、选择标签（最多3个，从预设标签选择） |
| 书单详情页 | 书单信息区 | 展示书单完整信息、平均评分、评论数、点赞按钮 |
| 书单详情页 | 评论列表 | 按时间倒序排列，显示用户名、时间戳、评论内容，淡入动画 |
| 书单详情页 | 评分表单 | 1-5星可点击评分，悬停高亮，点击锁定颜色 |
| 书单详情页 | 评论表单 | 输入评论（≤200字），提交后更新列表 |

## 3. 核心流程

用户浏览首页热度排行榜 → 点击书单卡片进入详情页 → 查看书单内容和评论 → 发表评论/评分/点赞 → 返回首页查看更新后的热度排行

```mermaid

    A["浏览首页热度排行"] --> B["点击书单卡片"]
    B --> C["进入书单详情页"]

    D --> E["发表评论/评分/点赞"]
    E --> F["数据更新至Store"]
    F --> G["热度值重新计算"]
    G --> H["返回首页查看新排行"]
    I["创建新书单"] --> F
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：左侧导航深灰 #1F2937，主内容区浅灰 #F9FAFB，卡片纯白 #FFFFFF
- **强调色**：热度渐变 #FEF3C7 → #FDE68A，星星高亮 #F59E0B
- **布局风格**：左右分栏布局，左侧固定导航栏240px，右侧自适应内容区
- **卡片样式**：圆角12px，浅淡阴影，悬停时阴影加深并上移4px（过渡0.2s）
- **字体**：采用 Playfair Display 作为标题字体（优雅衬线体，体现书籍文化感），Inter 作为正文字体，形成经典与现代的平衡
- **图标风格**：线性图标，简洁优雅

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 书单列表页 | 左侧导航 | 深色背景，Logo区域，用户信息，导航菜单，响应式折叠为汉堡菜单 |
| 书单列表页 | 热度排行榜 | 卡片网格布局，热度第一名使用渐变背景，位置变化过渡动画0.5s |
| 书单列表页 | 创建书单弹窗 | 表单验证，标签多选，字数统计 |
| 书单详情页 | 书单头部 | 大标题，标签云，评分展示，点赞按钮 |
| 书单详情页 | 评论区 | 评论输入框，星级评分组件，评论列表淡入动画 |
| 书单详情页 | 热度展示 | 使用 Recharts 展示热度趋势 |

### 4.3 响应式设计
- **桌面端（≥768px）**：左侧固定导航240px，右侧内容区自适应
- **移动端（<768px）**：左侧导航折叠为顶部汉堡菜单，主内容区占满宽度，卡片单列布局
- **触摸优化**：按钮最小点击区域44px，滑动手势支持

### 4.4 动效设计
- 页面加载：卡片错落出现（staggered animation）
- 排行刷新：卡片位置平滑过渡0.5s
- 卡片悬停：上移4px + 阴影加深，过渡0.2s
- 评论提交：新评论淡入，过渡0.3s
- 星星评分：悬停高亮，点击锁定，缩放反馈

## 1. 产品概述

全球3D地形交互可视化平台，基于Three.js和Vue 3构建，将高程数据以1°×1°经纬度切片渲染为3D地形，支持LOD自适应、区域搜索定位、高度测量、动态光照及性能监控。面向地理信息研究人员、教育工作者及地图爱好者，提供沉浸式地形浏览体验。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览地形、搜索区域、测量高度 |
| 高级用户 | 无需注册 | 所有功能 + 性能数据查看 |

### 2.2 功能模块
1. **主视图页**：3D地形渲染、相机交互、动态光照、LOD切换
2. **控制面板**：区域搜索、高度测量、性能监控、缩放控制

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主视图页 | 3D地形渲染 | 加载高程数据渲染1°×1°切片，LOD按相机距离自动切换（近512×512，远64×64） |
| 主视图页 | 区域搜索 | 输入城市名模糊匹配定位，1.5秒平滑相机过渡动画 |
| 主视图页 | 高度测量 | 点击地形两点显示直线距离和垂直高差（±5m精度），带箭头线条可视化 |
| 主视图页 | 动态光照 | 光源模拟东南方向太阳照射，60秒周期旋转 |
| 主视图页 | 性能监控 | 实时FPS和顶点数显示，FPS<30自动降LOD级别 |
| 控制面板 | 搜索框 | 模糊匹配下拉列表，历史记录 |
| 控制面板 | 进度条 | 地形加载进度0%→100%线性增长 |
| 控制面板 | 加载提示 | "数据加载中..."闪烁脉冲动画（透明度0.3↔1.0） |

## 3. 核心流程

用户打开应用 → 初始化Three.js场景 → 加载默认视角地形切片 → 用户可进行以下操作：
- 拖拽旋转/滚轮缩放浏览地形
- 搜索城市名定位区域（1.5秒过渡动画）
- 点击两点测量距离和高差
- 实时查看FPS和顶点数

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝色（#0a1628）和墨绿色（#0d3b2e）
- 地形颜色：低海拔深绿（#1a5c3a）→ 高海拔浅黄（#d4c472）
- 天空盒：渐变蓝（#0a1628 → #1a3a5c → #4a8ab5）
- 按钮：圆角8px，悬浮放大scale(1.1)动画，点击涟漪水波效果
- 面板：半透明毛玻璃效果（backdrop-filter: blur）
- 字体：Orbitron（标题）+ Noto Sans SC（正文）
- 布局：左侧3D视图80%宽度 + 右侧面板20%宽度
- 图标：lucide-vue-next

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主视图页 | 3D地形 | Three.js Canvas，渐变天空盒，LOD地形网格，动态光照 |
| 主视图页 | 测量线 | 带箭头线条连接两点，悬浮信息卡片显示距离和高差 |
| 控制面板 | 搜索框 | 输入框 + 模糊匹配下拉列表 + 历史记录 |
| 控制面板 | 性能监控 | FPS数字 + 顶点数数字，每秒刷新 |
| 控制面板 | 进度条 | 底部线性进度条0%→100%，加载文字脉冲动画 |
| 控制面板 | 工具按钮 | 测量/定位/重置按钮，悬浮放大+涟漪效果 |

### 4.3 响应式设计
- 桌面优先设计（≥768px）：左侧80% 3D视图 + 右侧20%面板
- 移动端适配（<768px）：3D视图全屏，面板变为底部滑出式抽屉
- 触控优化：支持双指缩放、单指旋转

### 4.4 3D场景指引
- 环境：渐变蓝色天空盒，深色大气氛围
- 光照：方向光模拟东南方向太阳，60秒周期旋转；环境光补充暗部
- 相机：透视相机，初始俯视角45°，支持轨道控制旋转/缩放
- 构图：地形居中，相机距离根据LOD级别调整
- 交互：轨道控制器拖拽旋转、滚轮缩放、射线点击检测
- 后期处理：可选雾效增加深度感
- 性能预算：目标45FPS+，射线检测<100ms，切片加载<2秒

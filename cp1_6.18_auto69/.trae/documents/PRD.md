## 1. 产品概述
幻境拼诗是一款古风主题的诗句拼字游戏，用户通过拖拽散落的汉字/词语碎片拼接成完整的五言或七言诗句，在沉浸式的水墨风格交互中体验古典诗词之美。
- 核心玩法：拖拽碎片拼诗，匹配成功解锁下一句，最终完成整首诗
- 目标用户：古诗词爱好者、休闲游戏玩家、国风文化追求者
- 产品价值：将传统文化与互动游戏结合，寓教于乐，提升诗词学习趣味性

## 2. 核心 Features

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通玩家 | 无需注册 | 完整游戏体验、拼诗、使用提示和洗牌功能 |

### 2.2 Feature 模块
1. **主游戏画布**：宣纸背景、散落词语碎片、拖拽交互
2. **拼诗区域**：横向格子容器、自动吸附对齐、匹配反馈
3. **匹配检测**：实时诗句匹配、成功动画反馈
4. **进度系统**：诗句解锁、进度条、计时、连击分数
5. **辅助面板**：已拼诗句展示、未解锁占位、提示和洗牌按钮
6. **动效系统**：水墨粒子动画、碎片悬停效果、匹配成功反馈

### 2.3 页面详情
| 页面名称 | 模块名称 | Feature 描述 |
|---------|----------|--------------|
| 游戏主界面 | 宣纸画布 | 径向渐变米白到浅黄背景，边缘卷曲阴影效果 |
| 游戏主界面 | 词语碎片 | 随机分布，单个汉字或双字词，拖拽交互，悬停放大加深阴影 |
| 游戏主界面 | 拼诗格子区 | 横向排列虚线边框格子，自动吸附对齐 |
| 游戏主界面 | 匹配反馈 | 金色边框高亮、粒子散落动画、进度条前进 |
| 右侧面板 | 诗句列表 | 已拼诗句展示、未解锁灰色占位 |
| 右侧面板 | 操作按钮 | 提示按钮（高亮未使用碎片）、重新洗牌按钮 |
| 底部区域 | 进度条 | 渐变填充显示当前诗句完成进度 |

## 3. 核心流程
用户进入游戏 → 画布上随机散落词语碎片 → 拖拽碎片到拼诗区格子 → 系统检测是否匹配目标诗句 → 匹配成功显示金色边框和粒子动画 → 进度条前进 → 解锁下一句 → 完成整首诗。

```mermaid
flowchart TD
    A["进入游戏"] --> B["初始化画布和碎片"]
    B --> C["用户拖拽碎片"]
    C --> D["碎片吸附到格子"]
    D --> E["检测匹配"]
    E --> F{匹配成功?}
    F -->|是| G["金色边框+粒子动画"]
    G --> H["进度条前进"]
    H --> I{"诗句全部完成?}
    I -->|是| J["解锁下一句"]
    J --> K["重置碎片"]
    K --> C
    I -->|否| C
    F -->|否| C
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：米白 #F5F0E8、浅黄 #EDE4D4、金色 #D4A843、深棕 #2D2D2D
- **按钮风格**：圆角设计，悬停变色过渡动画 0.2s
- **字体**：书法风格标题字体，古典雅致正文字体
- **布局**：左画布 + 右面板的经典布局，留白充足
- **视觉元素**：水墨粒子效果、宣纸纹理、古典边框

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI Elements |
|---------|----------|-------------|
| 游戏主界面 | 宣纸画布 | 径向渐变背景、边缘阴影、固定尺寸 |
| 游戏主界面 | 词语碎片 | 白色卡片、棕色边框、圆角 4px、悬停放大 1.1 倍、过渡 0.2s |
| 游戏主界面 | 拼诗格子 | 横向排列、虚线边框、间距 4px、自动吸附 |
| 游戏主界面 | 匹配动画 | 金色边框高亮、30 个金色粒子扩散、0.8s 动画 |
| 右侧面板 | 诗词列表 | 深色背景 #2D2D2D、圆角 12px、书法标题、已拼/未拼状态 |
| 右侧面板 | 操作按钮 | 提示按钮棕褐色 #6B5B45、洗牌按钮红棕色 #8B4513 |
| 底部区域 | 进度条 | 宽 600px 高 8px、未填充 #E0D4C4、已填充金色渐变 |

### 4.3 响应式
- 桌面端优先设计
- 宽度 < 768px 时：格子宽 30px 高 38px，碎片宽 50px 高 32px
- 移动端触摸优化，拖拽流畅
- 面板在小屏可折叠或移至底部

### 4.4 性能要求
- 碎片拖拽 60FPS 流畅运行
- 动画帧率不低于 55FPS
- 使用 CSS transform 和 GPU 加速
- 避免不必要的重排重绘

## 1. 产品概述
在线活动作品收集与展示墙应用，帮助社区活动组织者高效收集参与者提交的图片/短视频，并提供审核管理与可视化作品墙展示功能，替代传统邮箱收集方式，提升收集整理效率与作品展示体验。

- 目标用户：社区活动组织者（审核管理）与活动参与者（上传作品）
- 核心价值：一键上传→审核→展示的完整闭环，瀑布流作品墙提供沉浸式浏览体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 参与者 | 无需注册 | 上传作品、浏览展示墙、搜索筛选 |
| 组织者 | 登录认证 | 审核作品、管理展示墙、发送拒绝通知 |

### 2.2 功能模块

1. **展示墙页面**：瀑布流作品展示、图片懒加载、悬停交互、全屏预览、搜索筛选
2. **上传页面**：拖拽/点击上传、文件类型与大小校验、进度条、上传成功动画
3. **审核管理页面**：待审核列表、预览模态框、通过/拒绝操作、拒绝邮件模板

## 3. 核心流程

### 3.1 参与者上传流程
参与者进入上传页 → 拖拽或点击选择JPG/PNG/MP4文件（≤50MB） → 系统校验文件格式与大小 → 显示蓝色渐变进度条 → 上传成功 → 作品进入待审核状态 → 卡片淡入动画出现在展示墙末尾（标记为审核中）

### 3.2 组织者审核流程
组织者登录 → 进入审核管理页 → 查看待审核列表 → 点击审核按钮 → 弹出模态框预览作品 → 点击通过/拒绝 → 通过的作品移至"已发布"区域 → 拒绝的作品进入"已拒绝"区域 → 可通过邮件模板通知参与者

### 3.3 访客浏览流程
访客进入展示墙 → 浏览瀑布流作品（懒加载） → 鼠标悬停查看作品信息 → 点击全屏预览 → 使用搜索筛选查找作品

## 4. 界面设计

### 4.1 设计风格
- 主色调：深灰背景（#1e1e2e），渐变靛蓝强调色（#4f46e5）
- 按钮样式：圆角8px，靛蓝渐变填充，悬停时亮度提升
- 字体：标题使用 Outfit（粗体600-700），正文使用 DM Sans（常规400）
- 布局风格：顶部固定导航栏 + 内容区全宽瀑布流
- 图标风格：Lucide线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 导航栏 | 半透明磨砂玻璃效果（backdrop-filter: blur(10px)），固定顶部，Logo+导航链接+登录按钮 |
| 展示墙 | 搜索筛选栏 | 搜索输入框+日期区间选择器，深色卡片容器，靛蓝边框聚焦态 |
| 展示墙 | 瀑布流作品区 | 三列Masonry布局（桌面），卡片圆角12px，3D悬停效果（perspective+rotateX/Y），图片懒加载 |
| 展示墙 | 作品卡片 | 缩略图+悬停浮层（标题+上传者，背景模糊），scale(1.05)放大，过渡动画0.3s |
| 展示墙 | 全屏模态框 | rgba(0,0,0,0.8)背景，内容scale 0.8→1动画，淡出关闭动画 |
| 展示墙 | 空状态 | 灰色纸飞机SVG图标+微动效（上下浮动）+"暂无相关作品"文字 |
| 上传页 | 拖拽上传区 | 虚线圆角矩形，虚线呼吸闪烁动画，云朵上传图标，文字提示 |
| 上传页 | 进度条 | 蓝色渐变（#4f46e5→#7c3aed），圆角4px，宽度百分比动画 |
| 审核管理 | 审核列表 | 横向三区域（待审核/已发布/已拒绝），卡片式作品项 |
| 审核管理 | 审核模态框 | 作品预览+元信息+通过/拒绝按钮+拒绝原因文本框 |

### 4.3 响应式适配
- 桌面端（≥1024px）：三列瀑布流，每列宽度自适应
- 平板端（768px-1023px）：两列瀑布流
- 手机端（<768px）：单列瀑布流
- 导航栏在手机端折叠为汉堡菜单
- 卡片宽度随视口自动调整，保持间距一致

### 4.4 动效规范
- 卡片悬停：transform: perspective(1000px) rotateX/Y + scale(1.05)，过渡0.3s ease
- 模态框打开：背景遮罩淡入0.3s，内容scale(0.8)→scale(1) 0.3s cubic-bezier(0.34,1.56,0.64,1)
- 模态框关闭：背景与内容同时淡出0.2s
- 上传进度条：蓝色渐变位移动画（background-position循环）
- 虚线边框：opacity呼吸动画（0.4↔1，2s循环）
- 新卡片出现：opacity 0→1 + translateY(20px→0)，0.5s ease-out
- 空状态纸飞机：上下浮动4px，2s ease-in-out无限循环

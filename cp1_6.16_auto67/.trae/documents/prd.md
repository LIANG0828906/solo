# 街头美食市集商户入驻与订单聚合系统 PRD

## 1. 产品概述

街头美食市集商户入驻与订单聚合系统是一个面向美食节主办方、摊位商户和顾客的综合平台。主办方可以集中管理多个摊位商户，商户发布菜单和实时库存，顾客通过统一入口浏览所有摊位、下单并取餐，实现订单状态的实时同步与追踪。

- 核心目标：打造沉浸式的美食市集购物体验，提升点餐效率和用户满意度
- 目标用户：美食节主办方、摊位商户、到场顾客
- 市场价值：数字化传统美食市集，提升运营效率和顾客体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 顾客 | 浏览摊位、查看菜单、下单、追踪订单状态、取餐 |
| 商户 | 管理菜单、更新库存、处理订单、更新订单状态 |
| 主办方 | 管理所有摊位、监控整体运营情况 |

### 2.2 功能模块

1. **首页地图**：全屏滚动的美食市集地图，展示所有摊位位置和基本信息
2. **摊位详情**：双列网格展示菜品、库存状态、购物车交互
3. **购物车**：悬浮气泡、右侧滑出面板、商品管理、下单确认
4. **订单追踪**：订单列表、状态进度条、实时更新
5. **商户订单管理**：订单队列、状态切换（制作中/已完成/已取餐）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页地图 | 市集地图 | 暖色调渐变背景、手绘风格摊位图标、浮动动画、点击弹出信息卡片 |
| 首页地图 | 摊位信息卡 | 摊位名称（毛笔字体）、菜系标签、排队等待时间、进度条、查看菜单按钮 |
| 摊位详情 | 菜品网格 | 双列布局、菜品图片、名称价格、描述截断、库存徽章 |
| 摊位详情 | 菜品详情抽屉 | 底部上滑面板、数量加减器、加入购物车按钮 |
| 摊位详情 | 购物车气泡 | 右下角悬浮、商品数量徽章、弹跳动画 |
| 摊位详情 | 购物车面板 | 右侧滑入、商品列表、数量调整、总价计算、下单按钮 |
| 订单追踪 | 订单列表 | 时间倒序、订单卡片、四阶段进度条、平滑填充动画 |
| 商户端 | 订单面板 | 订单队列卡片、状态标签、完成/取餐按钮 |

## 3. 核心流程

顾客浏览市集地图 → 点击摊位查看信息 → 进入摊位详情 → 浏览菜品 → 选择菜品加入购物车 → 确认下单 → 查看订单状态 → 取餐

```mermaid
flowchart LR
    A["浏览市集地图"] --> B["点击摊位图标"]
    B --> C["查看摊位信息卡"]
    C --> D["进入摊位详情页"]
    D --> E["浏览菜品网格"]
    E --> F["点击菜品查看详情"]
    F --> G["调整数量加入购物车"]
    G --> H["打开购物车面板"]
    H --> I["确认下单"]
    I --> J["订单进入制作队列"]
    J --> K["商户完成制作"]
    K --> L["顾客取餐"]
    L --> M["订单完成"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：#E85D2C（橙红色）
- **辅助色**：#F4A261（沙色）
- **背景色**：#FFF8EE（米白色）
- **文字色**：#3D2B1F（深褐色）
- **卡片样式**：圆角12px、轻微盒子阴影（0 4px 12px rgba(0,0,0,0.1)）
- **按钮效果**：悬停放大1.05倍、阴影加深
- **字体**：毛笔字体用于摊位名称、圆润无衬线字体用于正文

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页地图 | 地图背景 | 橙色到焦糖色渐变、傍晚灯光氛围 |
| 首页地图 | 摊位图标 | 手绘风格、阴影、浮动动画、点击交互 |
| 首页地图 | 信息卡片 | 半透明毛玻璃效果、圆角、淡入动画 |
| 摊位详情 | 菜品卡片 | 圆角方形图片、库存徽章（绿/黄/红）、缩放动画 |
| 摊位详情 | 底部抽屉 | 0.3秒上滑动画、缓出函数 |
| 购物车 | 悬浮气泡 | 右下角固定、红色数字徽章、弹跳动画 |
| 购物车 | 侧滑面板 | 右侧滑入、分割线、加粗总价 |
| 订单追踪 | 进度条 | 四阶段圆点、连接线、平滑填充动画 |
| 商户订单 | 状态标签 | 橙色脉冲动画（制作中）、绿色静态（已完成）、灰色淡出（已取餐） |

### 4.3 响应式设计

- 桌面端优先，移动端自适应
- 移动端（宽度小于768px）：双列网格改为单列
- 横向滑入动画改为垂直滚动
- 地图图标按屏幕比例缩放
- 触控友好的按钮尺寸

### 4.4 动画与交互

- 列表项加载：从左到右依次淡入，间隔0.1秒
- 购物车数量变化：弹跳动画（CSS keyframes）
- 库存状态切换：缩放动画
- 页面过渡：水平滑入动画
- 订单状态脉冲：制作中状态橙色呼吸效果

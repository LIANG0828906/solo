## 1. 产品概述

在线食谱分享社区是一个面向烹饪爱好者的轻量级垂直社交平台，用户可以上传和分享拿手菜谱，搜索、收藏和评论他人作品，形成一个充满生活气息的美食交流圈子。

- 目标用户：家庭厨师、烹饪爱好者、美食博主
- 核心价值：降低美食分享门槛，构建高质量食谱内容社区

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名密码注册 | 创建菜谱、点赞、评论、收藏、搜索筛选 |

### 2.2 功能模块

1. **首页**：瀑布流菜谱展示、搜索筛选栏、固定导航栏
2. **菜谱详情页**：完整食谱展示、点赞功能、评论区
3. **创建/编辑菜谱页**：动态表单、食材管理、步骤图片拖拽排序、标签选择
4. **用户认证**：注册登录、用户信息展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索筛选栏 | 关键词实时搜索（防抖300ms）、标签/难度/评分下拉筛选、筛选切换动画 |
| 首页 | 瀑布流卡片列表 | 固定宽度300px卡片、无限滚动加载、卡片淡入动画、hover交互效果 |
| 首页 | 导航栏 | 半透明磨砂玻璃效果、Logo、搜索入口、创建菜谱按钮、用户头像 |
| 菜谱详情页 | 食谱展示 | 主图、菜名、作者信息、食材清单、分步说明（带图）、标签 |
| 菜谱详情页 | 点赞功能 | 点赞按钮、缩放动画、实时计数 |
| 菜谱详情页 | 评论区 | 评论列表、左侧滑入动画、发表评论表单 |
| 创建菜谱页 | 动态表单 | 菜名、简介输入、食材动态增删行、步骤拖拽排序、图片上传、标签多选 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖橙米白系
  - 背景色：`#fdf6ec`（米白色）
  - 主色：`#e67e22`（暖橙色）
  - 文字色：`#3e2723`（深棕色）
- **按钮/卡片圆角**：统一 12px
- **字体**：Noto Sans SC（圆润Web安全中文字体）
- **图标**：Lucide React 图标库
- **视觉质感**：柔和阴影、磨砂玻璃导航、微交互动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 导航栏 | 半透明背景 `backdrop-filter: blur(8px)`、暖橙品牌色Logo、创建按钮 |
| 首页 | 搜索筛选 | 圆角搜索框、下拉选择器、过渡动画 0.4s 交叉淡入淡出 |
| 首页 | 卡片瀑布流 | 300px固定宽、淡入动画 0.5s ease-out、hover上移5px+阴影加深 |
| 详情页 | 食谱主体 | 大图、分步图文、标签徽章 |
| 详情页 | 点赞按钮 | 心形图标、点击缩放0.9倍回弹 |
| 详情页 | 评论列表 | 左侧滑入动画 0.3s、头像气泡布局 |
| 创建页 | 表单 | 分段式布局、动态增删行按钮、拖拽手柄、图片预览区 |

### 4.3 响应式设计

- 桌面端优先设计，瀑布流多列布局
- 平板端自适应2-3列
- 移动端单列瀑布流，触控优化

## 5. 性能指标

| 指标 | 目标值 |
|------|--------|
| 页面初始加载时间 | ≤ 2秒 |
| 瀑布流滚动帧率 | ≥ 55fps |
| 评论提交后显示延迟 | ≤ 300ms |
| 搜索响应（防抖后） | ≤ 100ms |
